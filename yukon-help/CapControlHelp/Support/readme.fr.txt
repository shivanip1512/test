Le modèle XML+XSL crée du langage XML affichable grâce à des
navigateurs prenant en charge le XSL (Extensible Stylesheet
Language). Les règles XSL par défaut incluses dans le modèle
transforment le XML généré en un ensemble de résultats HTML
pouvant être affichés dans IE5 ou une version ultérieure.
Vous pouvez néanmoins personnaliser ces règles si vous
souhaitez les utiliser dans un autre but.

Pour de plus amples informations sur l'utilisation et la
personnalisation des projets créés à partir de ce modèle,
veuillez vous reporter à la section du guide des modèles
dans l'aide en ligne.
