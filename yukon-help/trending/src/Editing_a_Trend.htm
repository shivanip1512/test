Editing a Trend

To edit a trend:

 

  1. Select the trend to edit from the Trend list.

  2. From the Trend menu, select Edit. This displays the Yukon Trending Reports Editor dialog box.

  3. The Name text box displays the name entered during trend creation. Modify as necessary.

  4. The Left Axis and Right Axis areas display the axis scaling defined during trend creation. Modify as necessary.
    To manually specify the minimum or maximum scaling of either the left or right axis

    1. Clear the Auto Scaling check box for either the left or right axis.

    2. Enter a new minimum scaling value in the Min text box.
      This defines the low graph values for display on the axis selected.

    3. Enter a new maximum scaling value in the Max text box.
      This defines the high graph values for display on axis selected.

  5. Select either Graph Points or Usage Points from the Sort By drop-down list.

  6. The lower right of the dialog box displays the points selected during trend creation. Modify as necessary.

    • To add every point of a point type for a device, click on the + to display the device’s point types, select the point type that contains the points to add and click the >> button.

    • To add a single graph point attached to a device, click on the + to display the device’s point types, click on the + of the point type that contains the point to add, select the point, and click the >> button.

    • To remove a graph point, select the point in the table and click the << button.

  7. To change the text label for a point, double-click in the Label column and rename the label.

  8. To change the color of a point, click in the Color column and select a new color.

  9. To select a different axis, click in the Axis column and select an axis. Each point is plotted on the left axis by default.

  10. To change display type, click in the Type column and select a new type. The types are:

    • Graph - displays kW and kVar points as graph.

    • Usage - displays usage (kWh and kVarh) as a summary.

    • Yesterday - displays trend data from the previous day.

    • Peak Graph - displays trend data for the day with the highest peak value.

    • Usage Graph - displays usage (kWh and kVarh) as a graph and as a summary.

    • mm/dd/yy - displays trend data for the day specified in mm/dd/yy format

    • Marker - displays the point value as a threshold marker. The threshold marker can be changed by changing the value of the point.

  11. To change the multiplier, double-click in the Multiplier column and enter a new value for the multiplier. A multiplier alters point values to make them more visible on a graph.

  12. To change the how the data is point displayed, select a display type from the Renderer drop-down list.

  13. To add a value marker to the trend, click the Add Value Marker button.

    1. Enter a value for marker in the text box.
      The marker value must fall within the trend data values or the min and max axis values to be visible in a graph.

    2. Click OK to add the value marker to the trend.

  14. Select a graph point from the Primary Point drop-down list. This is the point from which the six highest value readings are taken for the Current Peaks Table in the Summary tab.

  15. Click OK.
