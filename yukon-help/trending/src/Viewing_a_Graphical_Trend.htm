Viewing a Graphical Trend

Perform the following steps to view a trend as a graph:

 

  1. Double-click on the trend to view. The trend that is displayed by default shows today’s current data.

  2. To change the time period of a trend displaying current data, select a time period from the Time Period drop-down list:

    • Today

    • Prev 2 Days

    • Prev 3 Days

    • Prev 7 Day

    • Event – If the Event option is selected, the number of past point events to display in the trend can be selected from the Number of Events spin box.

  3. To display historical data, select the Historical radio button, select a start date from the Starting Date drop-down list, and select a time period going forward from the Time Period drop-down list. The time periods are:

    • 1 Day

    • 3 Days

    • 1 Week

    • 4 Weeks

    • 5 Weeks

    • 1 Month

    • Event – If the Event option is selected, the number of past point events to display in the trend can be selected from the Number of Events spin box.

  4. To change the graph format, select the type of graph from the View menu. The types of graphs are:

    • Default (By Style) – How the trend data is displayed depends on the Renderer type selected for the point in the Trending Report Editor.

    • Line Graph – Displays the trend data as a line graph.

    • Line/Shapes Graph – Displays the trend data as a line graph and displays the points.

    • Line/Area Graph – Displays the trend data as a line graph and displays the area.

    • Line/Area/Shapes Graph – Displays the trend data as a line graph and displays the area and points.

    • Step Graph – Displays the trend data as a step graph.

    • Step/Shapes Graph – Displays the trend data as a step graph and displays the points.

    • Step/Area Graph – Displays the trend data as a step graph and displays the area.

    • Step/Area/Shapes Graph – Displays the trend data as a step graph and displays the area and points.

    • Step Line Graph – Displays the trend data as a step graph.

    • Line and Shapes Graph – Displays the trend data as a line graph and displays the points.

    • Bar Graph – Displays the trend data as a bar graph.

    • 3D Bar Graph – Displays the trend data as a 3D bar graph.
