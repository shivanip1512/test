Exporting a Trend

Perform the following steps to export a trend:

  1. Double-click on the trend to export in the Trends list.

  2. Select the type data to export by clicking on one of the following tabs:

    • Graph – Click on the Graph tab to export graphical data.

    • Tabular – Click on the Tabular tab to export tabular data.

    • Summary – Click on the Summary tab to export the current peaks and current usage tables.

  3. Select Export from the File menu. The Save dialog box is displayed.

  4. Select the folder to which to export the data.

  5. Enter a name for the exported data in the File Name text box.

  6. Select the type of format to which to export the data from the Files of type drop-down list.

    • Graphical data can be exported to the following formats:

  • Comma Separated (.csv)

  • JPEG Image Format (.jpeg)

  • Portable Network Graphics (.png)

  • Portable Document Format (.pdf)

    • Tabular data can be exported to the following formats:

  • Web Page HTML (.html)

  • Comma Separated (.csv)

    • Summary data can be exported to the following format:

  • Web Page HTML (.html)

  1. Click the Save button to export the data to the folder selected in step 4.
