Exporting a Trend

To export a trend:

  1. From the Trends list, double-click on the trend to export.

  2. Select the type data to export.

    1. To export the graphical data, click the Graph tab.

    2. To export the tabular data, click the Tabular tab.

    3. To export the summary data, click the Summary tab.

  3. From the File menu, select Export.

  4. Select a location to save the data to using the Save In drop-down list.

  5. Enter a name for the data file in the File Name text box.

  6. Select the type of format to export to from the Files of type drop-down list.

    • Graphical data can be exported to the following formats:

- Comma Separated (.csv)

- JPEG Image Format (.jpeg)

- Portable Network Graphics (.png)
- Portable Document Format (.pdf)

    • Tabular data can be exported to the following formats:

- Web Page HTML (.html)
- Comma Separated (.csv)

    • Summary data can be exported to the following format:

- Web Page HTML (.html)

  1. Click the Save button to export the data to the selected directory.
