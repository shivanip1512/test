Graphical Trend Display Options

The following options can be selected for a trend in a graphical format.

  • Graph Multipliers – To display the trend with multipliers, select Graph Multiplier from the Options menu. The multipliers alter point values to make them more visible on the graph.

  • Plot Min/Max Values – To display the minimum and maximum values of the trend for the selected time period, select Plot Min/Max Values from the Options menu.

  • Load Duration – To display a load duration of a trend, select Load Duration from the Options menu. The Load Duration graph orders data from the largest value to smallest value over a percentage of time.

  • Legend Values – To change the values of the legend, select Legend from the Options menu, and select the following options:

    • Show Minimum/Maximum – Displays the minimum and maximum values of the trend in the legend.

    • Show Load Factor – Displays load factor percentages of the trend in the legend.

  • Time Resolution – To change the time resolution of the data, select Resolution from the Options menu, and select one of the following options:

    • None – No change to the time resolution.

    • Seconds – Displays the last value to be recorded if multiple point values were recorded within the same second.

    • Minutes – Displays the last value to be recorded if multiple point values were recorded within the same minute.

  • Axis Labels – To change the axis labels:

    1. Select Advanced from the Options menu.

    2. To edit the domain axis properties, change the values in the Label and Load Duration Label text boxes.

    3. The edit axis properties, change the values in the Left Label and Right Label text boxes.

    4. Click the OK button to save the changes to the axis.

  • Reset Peaks – To reset the start date for peak graph data:

    1. Select Reset Peaks from the Options menu, and then select one of the following options:

      • Selected Trend Peaks – Resets the peak start date for the current trend selected.

      • All Peak Points – Resets the peak start date to the same date for all of the trends defined.

  1. When a reset option is selected, a dialog box is displayed which lists the dates to which the peak data can be reset:

  2. Select the new start date for the peak data:

  • Today – Peak data starts from today’s date.

  • First Date of Month – Peak data starts from the first day of the current month.

  • First Date of Year – Peak data starts from the first day of the current year.

  • Selected Date – Peak data starts from the day selected.

  1. Click the Reset Peaks button to reset the peak data to the selected start date.

  • Zoom Feature – To zoom in on the graph, move the mouse cursor to the upper left corner of the area to magnify, then click and drag the mouse cursor down and to the right. To zoom out, click and drag the mouse cursor up and to the left.
