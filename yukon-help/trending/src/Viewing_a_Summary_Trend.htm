Viewing a Summary Trend

The summary trend displays two tables, Current Peaks Table and Current Usage Table.

 

The Current Peaks Table displays the six highest value readings of the associated peak point.

The Current Usage Table displays the start and end usage values for the selected period along with the usage value difference.

 

To view a summary trend:

 

  1. From the Trends list, double-click on the trend to view.

  2. Click the Summary tab.
    The Current Peaks Table and Current Usage Table are displayed.

  3. To display current summary data, select the Current radio button and select a time period going backwards from the Time Period drop-down list:

    • Today

    • Prev 2 Days

    • Prev 3 Days

    • Prev 7 Day

    • Event - if the Event option is selected, the number of past point events to display in the trend can be selected from the Number of Events spin box.

  4. To display historical summary data, select the Historical radio button, select a start date from the Starting Date drop-down list, and select a the time period going forward from the Time Period drop-down list. The time periods are:

    • 1 Day

    • 3 Days

    • 1 Week

    • 4 Weeks

    • 5 Weeks

    • 1 Month
