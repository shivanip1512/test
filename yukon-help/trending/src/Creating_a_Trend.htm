Creating a Trend

Perform the following steps to create a trend:

  1. Select Create from the Trend menu. The Yukon Trending Reports Editor window is displayed.

  2. Enter a name for the trend in the Name text box.

  3. To specify the minimum and/or maximum value for the left axis:

    1. Clear the Auto Scaling check box for either in the Left Axis area.
      Note:       The left axis automatically scales to fit the data in a trend by default.

    2. Enter a minimum value for the left axis in the Min text box. This defines the smallest graph value to display on the left axis.

    3. Enter a maximum value for the left axis in the Max text box. This defines the largest graph value to display on the left axis.

To specify the minimum and/or maximum value for the right axis:

    1. Clear the Auto Scaling check box for either in the Right Axis area.
      Note:       The right axis automatically scales to fit the data in a trend by default.

    2. Enter a minimum value for the right axis in the Min text box. This is the smallest graph value to display on the right axis.

    3. Enter a maximum value for the right axis in the Max text box. This is the largest graph value to display on the right axis.

  1. Select the type of points to display in the Devices list from the Sort By drop-down list. The points types are:

    • Graph Points

    • Usage Points

  2. Add points to display in the trend.

    • To add all of the points on a device to the trend:

      1. Select the device that contains the points to add from the Devices list.

      2. Click on the + (Plus symbol) located to the left of the device name to display all of the point types on the device.

      3. Click the >> (Add) button to add all of the points on the device to the trend.

    • To add a single point on a device to the trend:

      1. Select the device that contains the point to add from the Devices list.

      2. Click on the + (Plus symbol) located to the left of the device name to display the point types of the device.

      3. Click on the + (Plus symbol) located to the left of the point type that contains the point to add to the trend.

      4. Select the point to add to the trend.

      5. Click the >> (Add) button to add the selected point to the trend.

    • To remove a point from the trend, select the point that is assigned to the trend and click the << (Remove) button.

    • To remove multiple points from the trend:

      1. Hold the Ctrl key on the keyboard while selecting the points to remove from the trend.

      2. Click the << (Remove) button to remove the selected points from the trend.

  3. To change the text label for a point assigned to the trend, double-click in the Label column of the point and enter a new label.

  4. To change the color of a point assigned to the trend, click in the Color column of the point and select a new color from the drop-down list.

  5. To select a different axis on which to plot the data for a point assigned to the trend, click in the Axis column of the point and select an axis from the drop-down list.
    Note:     Each point assigned to the trend is plotted on the left axis by default.

  6. To change the display type for a point assigned to the trend, click in the Type column of the point and select a different display type from the drop-down list. The display types are:

    • Graph – Displays kW and kVar points as graph.

    • Usage – Displays usage (kWh and kVarh) as a summary.

    • Yesterday – Displays trend data from the previous day.

    • Peak Graph – Displays trend data for the day with the highest peak value.

    • Usage Graph – Displays usage (kWh and kVarh) as a graph and as a summary.

    • mm/dd/yy – Displays trend data for the day specified in mm/dd/yy format.

    • Marker – Displays the point value as a threshold marker. The threshold marker can be changed by changing the value of the point.

  7. To change the multiplier of a point assigned to the trend, double-click in the Multiplier column of the point and enter a new multiplier. A multiplier alters the value of a point to make it more visible on a graph.
    Note:     The Graph Multiplier option must be selected from the Options menu of the application to enable the multiplier.

  8. To change the how the point data is displayed, click in the Style column of the point a display and select a display style from the drop-down list. The display styles are:

    • Line – Displays the trend data as a line graph.

    • Line/Shapes – Displays the trend data as a line graph and displays the points.

    • Line/Area – Displays the trend data as a line graph and displays the area.

    • Line/Area/Shapes – Displays the trend data as a line graph and displays the area and points.

    • Step – Displays the trend data as a step graph.

    • Step/Shapes – Displays the trend data as a step graph and displays the points.

    • Step/Area – Displays the trend data as a step graph and displays the area.

    • Step/Area/Shapes – Displays the trend data as a step graph and displays the area and points.

    • Bar – Displays the trend data as a bar graph.

    • 3D Bar – Displays the trend data as a 3D bar graph.

  9. To add a value marker to the trend:

    1. Click the Add Value Marker button.

    2. Enter a value for marker in the text box. The marker value must fall within the trend data values or the Min and Max axis values to make the marker value visible in the trend.

    3. Click the OK button to add the value marker to the trend.

  10. To assign a primary point to the trend, select a point from the Primary Point drop-down list. This is the point from which the six highest value readings are taken for the Current Peaks Table in the Summary tab.

  11. Click the OK button to save the trend.
