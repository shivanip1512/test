Menu Bar

The Menu Bar provides five menus, which perform the following functions:

 

 

  • File - use this menu to export data, print data, and exit the application.

  • Trend - use this menu to create, edit, and delete trends.

  • View - use this menu to view a trend in different graph formats.

  • Options - use this menu to display the graph multiplier, min/max values, load duration, legends, resolution, and advanced options.

  • Help - use this menu to display online help and version information.
