Printing a Trend

To print a trend graph:

 

  1. From the Trends list, double-click on the trend to print.

  2. Select the type data to print.

    1. To print the graphical data, click the Graph tab.

    2. To print the tabular data, click the Tabular tab.

    3. To print the summary data, click the Summary tab.

  3. Select Print from the File menu.

  4. Click OK to print.
