Printing a Trend

Perform the following steps to print a trend graph:

 

  1. Double-click on the trend to print in the Trends list.

  2. Select the type data to print by clicking on one of the following tabs:

    • Graph – Click on the Graph tab to print graphical data.

    • Tabular – Click on the Tabular tab to print tabular data.

    • Summary – Click on the Summary tab to print the current peaks and current usage tables.

  3. Select Print from the File menu.

  4. Click the OK button to print the trend.
