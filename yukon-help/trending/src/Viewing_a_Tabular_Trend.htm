Viewing a Tabular Trend

To view a tabular trend:

 

  1. From the Trends list, double-click on the trend to view.

  2. Click the Tabular tab.

  3. To display current data, select the Current radio button and select a time period going backwards from the Time Period drop-down list:

    • Today

    • Prev 2 Days

    • Prev 3 Days

    • Prev 7 Day

    • Event - Displays a number of past point events

  4. If the Event option is selected from the Time Period drop-down list, the number of point events to display in the trend can be selected from the Number of Events spin box.

  5. To display historical data, select the Historical radio button, select a start date from the Starting Date drop-down list, and select a the time period going forward from the Time Period drop-down list. The time periods are:

    • 1 Day

    • 3 Days

    • 1 Week

    • 4 Weeks

    • 5 Weeks

    • 1 Month

  6. A slide bar appears above the table if the trend contains more than one day of data. To display tabular data for a certain day, move the slider to that day.

  7. To change the time resolution in the tabular data, select the Resolution option from the Options menu. The resolution options are:

    • None - no change to the time resolution

    • Seconds - makes all points that were recorded within the same second appear together

    • Minutes - makes all points that were recorded within the same minute appear together
