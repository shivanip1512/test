Viewing a Tabular Trend

Perform the following steps to view a tabular trend:

 

  1. Double-click on the trend to view.

  2. Click the Tabular tab.

  3. To display current data, select the Current radio button and select a time period going backwards from the Time Period drop-down list:

    • Today

    • Prev 2 Days – Select the day to view by moving the slide bar that appears above the table.

    • Prev 3 Days – Select the day to view by moving the slide bar that appears above the table.

    • Prev 7 Day – Select the day to view by moving the slide bar that appears above the table.

    • Event – If the Event option is selected, the number of past point events to display in the trend can be selected from the Number of Events spin box.

  4. To display historical data, select the Historical radio button, select a start date from the Starting Date drop-down list, and select a time period going forward from the Time Period drop-down list. The time periods are:

    • 1 Day

    • 3 Days – Select the day to view by moving the slide bar that appears above the table.

    • 1 Week – Select the day to view by moving the slide bar that appears above the table.

    • 4 Weeks – Select the day to view by moving the slide bar that appears above the table.

    • 5 Weeks – Select the day to view by moving the slide bar that appears above the table.

    • 1 Month – Select the day to view by moving the slide bar that appears above the table.

    • Event – If the Event option is selected, the number of past point events to display in the trend can be selected from the Number of Events spin box.
