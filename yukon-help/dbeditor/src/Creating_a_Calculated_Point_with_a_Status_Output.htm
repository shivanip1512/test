Creating a Calculated Point with a Status Output

If Calculated - Status Output was selected as the point type:

 

  1. Select a state group from the State Group drop-down list. The state group defines the messages that are displayed for the different states of a point.

  2. Select the initial state of the state group from the Initial State drop-down list. The initial state depends on the state group selected.

  3. Click the Next button.

  4. Select a calculation update type from the Update Type list. The update types are:

    • On First Change – Performs the calculation whenever any point in the calculation changes.

    • On All Change – Performs the calculation only when all of the points in the calculation change.

    • On Timer – Performs the calculation after the timer expires.

    • On Timer + Change – Performs the calculation only after the timer expires and all of the points in the calculation change.

    • Constant – Returns a constant value.

    • Historical – Totalizes when all specified points are available.

  5. If On Timer or On Timer + Change is selected as the update type, select how often to perform a calculation from the Periodic Rate drop-down list.

  6. Click the Finish button. The new point appears in the Point Editor.
