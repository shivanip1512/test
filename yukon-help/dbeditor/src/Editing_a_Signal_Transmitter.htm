Editing a Signal Transmitter

Perform the following steps to edit a signal transmitter:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select one of the following:

    • Device

    • Transmitter

  3. Double-click on the signal transmitter to edit. The General tabbed page is displayed in the Device Editor.

  4. Click the Scan Rate tab to display the Scan Rate tabbed page.

  5. If the signal transmitter is a RTU-LMI, click the Series 5 Settings tab to display the Series 5 Settings tabbed page.

If the signal transmitter is a Paging Tap Terminal, SNPP Terminal, or WCTP Terminal, click the Paging tab to display the Paging tabbed page.

If the signal transmitter is a TNPP Terminal, click the TNPP Settings tab to display the TNPP Settings tabbed page.

  1. Click the Apply button to save the changes to the signal transmitter or click the OK button to save the changes to the signal transmitter and close the Device Editor.
