ExpressCom Load Group

If the load group is an ExpressCom load group:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the load group in the kW Capacity text box. This is the estimated kilowatt capacity of the load control devices in the load group. To disable the kilowatt capacity feature, enter a value of zero.

  3. Select the control priority of ExpressCom messages from the Control Priority drop-down list. A message with a medium priority is ignored if a message with a high priority has already been received by the load control device.

  4. If the load group is an ExpressCom load group, a different control priority for ExpressCom messages can be selected from the Control Priority drop-down list. A message with a medium priority is ignored if a message with a high priority has already been received by the load control device.

  5. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  6. Disable or enable control of the group by selecting or clearing the Select the Disable Control check box. Select this check box to disable control of the load group. Clear this check box to enable control of the load group.

  7. Select the communication route to use for the load group from the Communication Route drop-down list. This is the route or route macro that is used to transmit commands to all of the load control devices in this group.

  8. Click the Next button.

  9. In the Geographical Address Usage area, select the check boxes of the group addresses to use when commands are sent to the load group. The Geographical Address Usage options are:

    • GEO – Group commands are addressed to load control devices that are configured with the Geographical address entered in step 12.

    • Substation – Group commands are addressed to load control devices that are configured with the Substation address entered in step 13.

    • Feeder – Group commands are addressed to load control devices that are configured with the Feeder addresses entered in step 15.

    • Zip – Group commands are addressed to load control devices that are configured with the Zip address entered in step 15.

    • User – Group commands are addressed to load control devices that are configured with the User Defined Address entered in step 16.

To configure the load group to send commands to a single load control device, select the Serial check box and enter the serial number of the load control device in the Serial text box. Valid values are 1 to 4,294,967,295.

Note: When an address check box is selected in the Geographical Address Usage area, the address field in the Geographical Addressing area is activated and it turns blue.

  1. In the Load Address Usage area, select the check boxes of the load addresses to use when commands are sent to a load group. The Load Address Usage options are:

    • Load – Control commands are addressed to the relays of load control devices that are configured with the loads selected in step 19.

    • Program – Control commands are addressed to the relay of load control devices that are configured with the Program address entered in step 19.

    • Splinter – Control commands are addressed to the relay of load control devices that are configured with the Splinter address entered in step 19.

Note: When the Load check box is selected in the Load Address Usage area, the name of the Configured Loads area changes to Targeted Loads and all of the Load address fields are activated and they turn blue.

Note: When a Program or Splinter check box is selected in the Load Address Usage area, the associated address field in the Load Addressing area is activated and it turns blue.

  1. Enter the Service Provider ID address of the load group in the SPID text box. The Service Provider ID address is sent with all group address messages and is used to identify the service provider/owner utility. Valid values are 1 to 65,534.

  2. Enter the Geographical address of the load group in the GEO text box. The Geographical address identifies a desired geographical area, determined by the utility. Valid values are 1 to 65,534.

  3. Enter the Substation address of the load group in the Substation text box. The Substation address is typically used to identify the utility substation on which the target equipment is located. Valid values are 1 to 65,534.

  4. To specify the Feeder addresses of the load group, click the Set Feeder Address button and select the check boxes of the bits to use. The Feeder address is typically used to identify the utility feeder by which the load is fed. This is normally combined with the Substation address, but it can be used alone. Each bit in this address represents a feeder, which allows multiple feeders to be targeted with a single message.

  5. Enter the Zip address of the load group in the ZIP text box. The Zip address is typically used as the postal zip code of the geographic location. This address can also be used for any other addressing options. Valid values are 1 to 16,777,214.

  6. Enter the user defined address of the load group in the User text box. The user defined address is used for additional addressing options. Valid values are 1 to 65,534.

  7. Enter the Program address in the Program text box. The Program address is used to target specific loads. Field devices that control multiple loads contain a separate program address for each load. Valid values are 1 to 254.

  8. Enter the Splinter address in the Splinter text box. The Splinter address is used to target a subset of the loads targeted by the Program address. Field devices that control multiple loads contain a separate splinter address for each load. Valid values are 1 to 254.

  9. If the Program or Splinter address check box was selected in the Load Address Usage area in step 10, select the check box(es) of the load(s) to control in the Configured Loads area.

If the Load address check box was selected in the Load Address Usage area in step 10, select the check box(es) of the load(s) to control in the Targeted Loads area.

  1. Click the Finish button. The new load group appears in the Load Group Editor.
