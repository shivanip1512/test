Creating a Core Database Point

To create a Point:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Point.

  3. Select the type of point to create:

    • Analog - reads the value directly from the transducer or electronic device. Examples include a pseudo point and Voltage or Amperage on a field device.

    • Status - reads the value of the contact or digital input, usually open or closed. Examples include a relay on a load control receiver or a switch status on a capacitor bank. A status point can also have control.

    • Accumulator - measures any quantity that is a simple count accumulation, such as the number of pulses. Examples include kWh energy point or kW demand (energy over time).

    • Calculated - calculates data gathered from other points.

  4. Click Next.

  5. Enter the name for the point in the Name text box.

  6. Select the device on which this point resides from the Device list.

  7. Click Next. Based on the type of point selected, the editor follows one of four paths.

    • Analog

    • Status

    • Accumulator

    • Calculated (Analog Output)

    • Calculated (Status Output)
