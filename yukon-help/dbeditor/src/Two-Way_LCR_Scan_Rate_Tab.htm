Two-Way LCR Scan Rate Tab

The following changes can be made to the scan rate setting on the two-way LCR:

 

  1. To enable scanning to collect demand and status point data, select the Demand & Status Rate check box

    1. A different default scan interval can be selected from the Interval drop-down list.

    2. A different alternate scan interval can be selected from the Alt Interval drop-down list. The alternate scan interval can be activated using the Tabular Data Console or the Yukon Web interface and is used during the “control timeout” period to collect the state of a control point.

    3. A different group can be selected from the Group drop-down list. This determines the priority grouping between different device scans.

  2. To disable scanning, clear the Demand & Status Rate check box.
