Creating a Notification Group

To create a notification group:

  1. From the View menu, select System.

  2. From the Create menu, select Notification Group.

  3. Enter a name for the notification group in the Name text box.

  4. To disable the notification group, select the Disable Group check box.

  5. Click the Finish button. The new group appears in the Notification Group Editor. For instructions on editing a notification group, refer to the related topics
