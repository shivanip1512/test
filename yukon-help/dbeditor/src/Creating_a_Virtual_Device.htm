Creating a Virtual Device

Perform the following steps to create a virtual device:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select Virtual from the list on the left.

  4. Select VIRTUAL SYSTEM from the list on the right.

  5. Click the Next button.

  6. Enter a name for the device in the Device Name text box.

  7. Click the Finish button. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.
