Creating a Season

To add a load control season to the schedule:

 

  1. Click the Create button. The Season Date Creation dialog box appears.

  2. Enter a name for the season in the Season Name text box.

  3. Select the date that is the start of a load control season from the Month and Day drop-down lists in the Start Date area.

  4. Select the date that is the end of a load control season from the Month and Day drop-down lists in the End Date area.

  5. Click the OK button. The season is added to the season schedule.
