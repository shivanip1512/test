Editing a Holiday

Perform the following steps to edit a holiday:

 

  1. Select the holiday to edit from the Assigned Holidays list.

  2. Click the Edit button. The holiday information is displayed in the Edit Holiday Date window.

  3. The name of the holiday can be changed in the Holiday Name text box.

  4. A different holiday month can be selected from the Month drop-down list.

  5. A different holiday date can be selected from the Day drop-down list.

  6. A different holiday year can be selected from the Year drop-down list.

    • To specify a year for the holiday, select a year from the Year drop-down list.

    • To make the holiday repeat each year, select (none) from the Year drop-down list.

  7. Click the OK button to save the changes to the holiday.
