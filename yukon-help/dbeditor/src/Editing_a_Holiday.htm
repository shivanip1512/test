Editing a Holiday

To edit a holiday:

 

  1. Select the holiday to edit from the Assigned Holidays list.

  2. Click Edit.

  3. The name of the holiday can be changed in the Holiday Name text box.

  4. A different holiday month can be selected from the Month drop-down list.

  5. A different holiday date can be selected from the Day drop-down list.

  6. A different holiday year can be selected from the Year drop-down list. To make this holiday repeat each year, select (none).

  7. Click OK to save the changes.
