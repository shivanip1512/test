Creating a Login Group

Perform the following steps to create a login group:

 

  1. From the View menu, select System.

  2. From the Create menu, select Login Group.

  3. Enter a name for the login group in the Group Name text box.

  4. Enter a description of the login group in the Description text box.

  5. Click the Next button.

  6. To display the roles under a role category, click the + button to the left of the role category name.

  7. Select the check boxes of the roles to assign to the login group. When a role is selected, a short description of the role appears in the Description box and the role properties appear in the Properties list.

  8. To modify the properties of an assigned role:

    1. Click in the Value field of the role property to edit.

    2. Enter a new value in the field.

    3. Press the Enter key on the keyboard to save the value entered.

Note: Roles that have been modified are displayed with blue text.

  1. Click the Finish button. The new login group appears in the Login Group Editor. For instructions on editing a login group, refer to the related topics.
