Calculated Point General Tab

The following changes can be made to the general settings of the calculated point:

 

  1. The name of the point can be changed in the Point Name text box.

  2. A different type of timing group, which control how often events are recorded, can be selected from the Timing Group drop-down list. The timing group are:

    • Default – Records events that occur within seconds of each other.

    • SOE – Records events that occur within milliseconds of each other (Sequence of Events).

  3. The point can be disabled or enabled by selecting or clearing the Disable Point check box. Disabling the point may be useful if it is temporarily not functioning.

  4. If the calculated point displays a status output:

    1. A different state group, which defines the messages that are displayed for the different states of a point, can be selected from the State Group drop-down list.

    2. A different initial state of the state group can be selected from Initial State drop-down list. The initial state depends on the state group selected.

    3. To archive the status output, select the Archive Data check box.

If the calculated point displays an analog output:

  1. A different unit of measure can be selected from the Unit of Measure drop-down list.

  2. The number of decimal places to display for the measurement can be changed in the Decimal Digits spin box.

  3. A different state group, which defines the messages that are displayed for the different states of a point, can be selected from the State Group drop-down list.

  4. A different data archive type can be selected from the Data Type drop-down list. The data archive types are:

      • None – Does not archive data.

      • On Change – Archives data when an operand causes the point to change.

      • On Timer – Archives data at the selected archive interval.

      • On Update – Archives data any time the point is updated.

      • On Timer or Update – Archives data at the selected archive interval or any time the point is updated.

    1. If On Timer or On Timer or Update is selected as the data archive type, select how often to archive point data from the Interval drop-down list. The archive intervals are:

      • 30 minute

      • 1 hour

      • 2 hour

      • 6 hour

      • 12 hour

      • Daily

      • Weekly

      • Monthly

  1. A different calculation update type can be selected from the Update Type drop-down list. The calculation update types are:

    • On First Change – Performs the calculation whenever any point in the calculation changes.

    • On All Change – Performs the calculation only when all of the points in the calculation change.

    • On Timer – Performs the calculation after the timer expires.

    • On Timer + Change – Performs the calculation only after the timer expires and all of the points in the calculation change.

    • Constant – Returns a constant value.

    • Historical – Totalizes when all specified points are available.

  2. If On Timer or On Timer + Change is selected as the calculation update type, select how often to perform a calculation from the Rate drop-down list.

  3. If the calculated point displays an analog output, the quality of the calculated point can be forced to always be normal by selecting the Force Quality Normal check box.
