Calculated Point General Tab

The following changes can be made to the general settings of the calculated point:

 

  1. The name of the point can be changed in the Point Name text box.

  2. The Timing Group drop-down list is for DNP devices only. For points on DNP devices, a value of SOE (Sequence of Events) can be selected, which records high speed events that occur within milliseconds of each other.

  3. The point can be disabled or enabled by selecting or clearing the Disable Point check box. Disabling the point may be useful if it is temporarily not functioning.

  4. If the calculated point displays status output:

    1. A different state group can be selected from State Group drop-down list. The state group defines the possible status outputs of the point.

    2. A different initial state can be selected from Initial State drop-down list. The initial state defines the initial status output of the point.

    3. To archive the status output, select the Archive Data check box.

  5. If the calculated point displays analog output:

    1. A different unit of measure can be selected from the Unit of Measure drop-down list. The unit of measure is for the data being read.

    2. The number of decimals can be changed in the Decimal Digits spin box. This is the number of decimal places to display for the measurement.

    3. A different state group can be selected from the State Group drop-down list. The state group defines the messages to display for the different states of a point.

    4. A different data archive type can be selected from the Data Type drop-down list. The data archive types are:

      1. None – does not archive.

      2. On Change – archives when an operand causes the point to change.

      3. On Timer – archives at the selected time interval.

      4. On Update – archives any time the point is updated.

      5. On Timer or On Update – archives at the selected archive interval or any time the point is updated.

    1. If On Timer or On Timer or Update is selected as the data archive type, a different archive interval can be selected from the Interval drop-down list.

  6. A different update type can be selected from the Update Type drop-down list. The update types are:

    • On First Change – performs the calculation whenever any point in the calculation changes.

    • On All Change – performs the calculation only when all of the points in the calculation change.

    • On Timer – performs the calculation after the timer expires.

    • On Timer + Change – performs the calculation only after the timer expires and all of the points in the calculation change.

    • Constant – returns a constant value.

    • Historical – totalizes when all specified points are available.

  7. If On Timer or On Timer + Change is selected as the update type, a different time interval can be selected from the Rate list.

  8. To force the quality of the calculated point to always be normal, select the Force Quality Normal check box.
