Signal Transmitter Series 5 Settings

The following changes can be made to the settings of an RTU-LMI signal transmitter:

 

  1. The tick time, which is the length time that is used when calculating a transmit time, can be changed in the Tick Time text box.

  2. The transmit offset, which is time offset that is used when calculating a transmit time, can be changed in the Transmit Offset text box.

  3. The history of the codes that are sent to the transmitter can be saved by selecting the Save History check box.

  4. The high limit, which is the maximum value of the transmitter analog input value, can be changed in the High Limit text box.

  5. The low limit, which is the minimum value of the transmitter analog input value, can be changed in the Low Limit text box.

  6. The multiplier, which is applied to the transmitter analog input value, can be changed in the Multiplier text box.

  7. The offset, which is the offset of the transmitter analog input value, can be changed in the Offset text box.
