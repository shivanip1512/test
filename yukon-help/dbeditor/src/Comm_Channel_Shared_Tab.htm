Comm Channel Shared Tab

The following changes can be made to the shared port settings of the communication channel:

 

  1. A different type of shared port can be selected from the Shared Port Type drop-down list. This setting is used to tell the Yukon system that the comm port is shared with another system. The shared port type options are:

    • (none)

    • ACS

    • ILEX

  1. If a shared port type is selected, a different socket number can be entered in the Socket Number text box. This defines the socket that the share listener uses to listen. The default socket number is 1025.
