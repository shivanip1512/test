Editing a Macro Load Group

A macro load group contains multiple load groups.

 

Perform the following steps to edit a macro load group:

 

  1. Double-click on the macro load group to edit. The General tabbed page is displayed in the Load Group Editor.

  2. The name of the load group can be changed in the Group Name text box.

  3. Click on the Macro Group tab to display the Macro Group tabbed page, from which the following changes can be made:

    • To assign a load group to the macro load group, select the load group from the Available list and click the Add button. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.

    • To rearrange the order of the assigned load groups, click and drag a load group to move it up or down the list.

    • To unassign a load group from the macro load group, select the load group from the Assigned list and click the Remove button.

  4. Click the Apply button to save the changes to the macro load group or click the OK button to save the changes to the macro load group and close the Load Group Editor.
