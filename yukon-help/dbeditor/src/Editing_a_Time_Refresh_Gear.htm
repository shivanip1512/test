Editing a Time Refresh Gear

To edit a Time Refresh gear:

  1. The name of the gear can be changed in the Gear Name text box.

  2. Select Time Refresh from the Gear Type drop-down list.

  3. A different type of refresh shed can be selected from the Refresh Shed Type drop-down list.

    • Fixed Shed Time - shed time is specified in the gear.

    • Dynamic Shed Time - shed time is dynamically calculated.

  4. A different length of the shed can be selected from the Shed Time drop-down list.

    1. If the refresh shed type is Fixed, select a shed time from the Fixed Shed Time drop-down list.

    2. If the refresh shed type is Dynamic, select a maximum shed time in the Max Shed Time drop-down list.

  5. A different number of groups that are shed for each refresh can be selected from the # of Groups Each Time drop-down list.

  6. A different rate at which the shed command is refreshed or resent can be selected from the Command Resend Rate drop-down list.

  7. A different group selection method can be selected from the Group Selection Method drop-down list. The group selection methods are:

    • Last Controlled - selects the next load group in the list based on the last load group controlled.

    • Always First Group - always selects the first load group in the list.

    • Least Control Time - selects the load group whose seasonal control time is the least.

  8. To ramp in a percentage of the load groups over an interval of time, select the Ramp In check box. Enabling the Ramp In feature, disables the # of Groups Each Time feature.

    1. Enter the percentage of the load groups to ramp in at each interval in the Ramp In Percent text box.

    2. Enter the interval in seconds over which to ramp in a percentage of the load groups in the Ramp In Interval text box.

  9. A different method for stopping control can be selected from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - A restore command is sent to the device.

    • Ramp Out/Time In - The control ramps out when the shed time is over.

    • Ramp Out/Restore - The control ramps out when a restore command is sent to the device.

  10. If Ramp Out/Time In or Ramp Out/Restore is selected from the How to Stop Control drop-down list, select the order to stop the control from the Stop Order drop-down list. The orders for stopping control are:

    • Random - The system selects a random load group to stop.

    • First In First Out - The first group that was controlled is the first group to stop. Then the second group that was controlled is the second group to stop, etc.

  11. If Ramp Out/Time In or Ramp Out/Restore is selected from the How to Stop Control drop-down list:

    1. Enter the percentage of the load groups to ramp out at each interval in the Ramp Out Percent text box.

    2. Enter the interval in seconds over which to ramp out a percentage of the load groups in the Ramp Out Interval text box.

  12. The group capacity reduction percentage can be changed in the Group Capacity Reduction % spin box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop.

  13. A different automatic gear change method can be selected from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear changes only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear changes when the control area priority reaches this priority.

    • Above Trigger - Triggers are defined in Control Areas. Select whether to use trigger one or two in the Trigger Number spin box. (Trigger one is the first entry and trigger two is the second entry of the Control Area’s Trigger drop-down list.) The gear changes when the Control Area threshold value reaches the trigger plus the offset value in the Trigger Offset text box.

  14. Click OK to save the gear.
