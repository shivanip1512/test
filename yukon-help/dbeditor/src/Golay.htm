Golay

If you selected a Golay device:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of the devices in the load group.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Select the Disable Control check box.

  5. Select the route to use for the load group from the Communication Route drop-down list. This is the route or route macro used to reach all of the devices in this group when transmitting commands.

  6. Click the Next button.

  7. Enter the operational address of the load group in the Operation Address text box.

  8. Select the amount of time that a switch will control the selected load from the Nominal Timeout drop-down list. This value is hard coded into the switch.

  9. Click the Finish button. The new load group appears in the Device Editor. For instructions on editing a load group, refer to the related topics.
