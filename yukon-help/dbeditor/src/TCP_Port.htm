TCP Port

If you selected TCP Port:

 

  1. Enter a name for the comm channel in the Description text box. This name is used by the system when it communicates error and informational messages.

  2. Select the baud rate of the port from the Baud Rate drop-down list. This is the communications line speed used by this port.

  3. Click the Finish button. The new communication channel appears in the Comm Channel Editor. For instructions on editing a communication channel, refer to the related topics.
