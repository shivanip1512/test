Thermostat Ramping

Note: All times are in entered in minutes.

 

To create a thermostat ramping gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Thermostat Ramping from the Gear Type drop-down list.

  3. By default, Deltas and Fahrenheit units are used by the gear.

    • To use Absolute values instead of Deltas, click the Deltas button. The button changes to Absolutes.

    • To use Celsius units instead of Fahrenheit units, click the Fahrenheit button. The button changes to Celsius

  4. Select the mode the thermostat must be in to accept commands: Heat Mode or Cool Mode.

  5. To enter a minimum temperature, select the Min check box and enter the minimum temperature in the text box.

  6. To enter a maximum temperature, select the Max check box and enter the maximum temperature in the text box.

  7. To enter Delta setpoints:

    1. Click the push button to select Deltas as the data type.

    2. To enter a Delta B value, select the Delta B check box and enter the delta setpoint in text box.

    3. To enter a Delta D value, select the Delta D check box and enter the delta setpoint in text box.

    4. To enter a Delta F value, select the Delta F check box and enter the delta setpoint in text box.

  8. To enter Absolute setpoints:

    1. Click the push button to select Absolutes as the data type.

    2. To enter an Absolute B value, select the Abs B check box and enter the absolute setpoint in text box.

    3. To enter an Absolute D value, select the Abs D check box and enter the absolute setpoint in text box.

    4. To enter an Absolute F value, select the Abs F check box and enter the absolute setpoint in text box.

  9. To enter a random offset time, select the Rand check box and enter the random offset in the text box. A random offset between 0 minutes and the value Rand is added to any delay time.

  10. To enter the delay time Ta, select the Ta check box and enter the delay time in the text box.

  11. To enter the ramp time Tb, select the Tb check box and enter the ramp time in the text box.

  12. To enter the plateau time Tc, select the Tc check box and enter the plateau time in the text box. During this time the setpoint Abs B is maintained.

  13. To enter the ramp time Td, select the Td check box and enter the ramp time in the text box.

  14. To enter the plateau time Te, select the Te check box and enter the plateau time in the text box. During this time the setpoint Abs D is maintained.

  15. To enter the ramp time Tf, select the Tf check box and enter the ramp time in the text box.

  16. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

  17. Set a group capacity reduction percentage in the Group Capacity Reduction % spin box. The percentage of each groups capacity is used to determine how much to control.

  18. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear changes only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear changes when the control area priority reaches this priority.

    • Above Trigger - Triggers are defined in Control Areas. Select whether to use trigger one or two in the Trigger Number spin box. (Trigger one is the first entry and trigger two is the second entry of the Control Area’s Trigger drop-down list.) The gear changes when the Control Area threshold value reaches the trigger plus the offset value in the Trigger Offset text box.

  19. Click the OK button to save the gear.
