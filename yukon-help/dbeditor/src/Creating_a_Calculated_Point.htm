Creating Calculated Point with Analog Output

If you selected Calculated - Analog Output as the point type:

 

  1. Select an update type from the Update Type list. The update types are:

    • On First Change – performs the calculation whenever any point in the calculation changes.

    • On All Change – performs the calculation only when all of the points in the calculation change.

    • On Timer – performs the calculation after the timer expires.

    • On Timer + Change – performs the calculation only after the timer expires and all of the points in the calculation change.

    • Constant – returns a constant value.

    • Historical – totalizes when all specified points are available.

  2. If On Timer or On Timer + Change is selected as the update type, select a rate from the Periodic Rate drop-down list.

  3. Click Finish. The new point appears in the Point Editor. For instructions on editing a point, refer to the related topics.
