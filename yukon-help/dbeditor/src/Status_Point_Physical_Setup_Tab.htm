Status Point Physical Setup Tab

The following changes can be made to the physical settings of the status point:

 

  1. If the Yukon system obtains physical data from the device on which the point is located or the point is a special Yukon system point, select the Physical Point Offset check box. Clear this check box if this is a pseudo point.

  2. If the Physical Point Offset check box is selected, the point offset can be changed in the Point Offset spin box. This is either the physical input to which the point is wired or a Yukon system defined point.

  3. A different control type can be selected from the Control Type drop-down list. The control types are:

    • None – No control type exists.

    • Latch – The control command is sent and the relay latches in that position. Latch controls require a second command to return to the original state.

    • Normal – The control is a dual relay Select Before Operate (SBO). Normal controls are temporary or held only for a discrete number of milliseconds.

    • Pseudo – The point is controlled by another system. Pseudo controls do not have a physical status point to indicate if control has completed.

    • SBO Latch – Select Before Operate latch command.

    • SBO Pulse – Select Before Operate pulse (normal) command.

  1. If the control type selected is Latch, Normal, SBO Latch, or SBO Pulse:

    1. The control point offset can be changed in the Control Point Offset box. This specifies the physical location used for wiring the control point.

    2. The close times can be changed in the Close Time 1 and Close Time 2 boxes. These times specify how long each relay stays energized.

    3. The control inhibit option can be enabled or disabled by selecting or clearing the Control Inhibit check box. Select this check box to disable control of the point. Clear this check box to enable control of the point.

    4. The command strings can be changed in the text boxes with the Command Strings area. These are the commands that are sent when the Yukon system controls the point.

    5. The command time-out can be changed in the Command TimeOut minutes and seconds boxes. This specifies the length of time to use the alternate scan rate to scan for a state change following control. If a state change is not detected, an alarm occurs.
