Status Point Physical Setup Tab

The following changes can be made to the physical settings of the status point:

 

  1. The physical point offset option can enabled or disabled by selecting or clearing the Physical Point Offset check box. Select this option if the system obtains physical settings from a location other than the hardware. Clear this option if this is a pseudo point.

  2. The point offset can be changed in the Point Offset spin box. This is the physical relay used for wiring.

  3. A different control type can be selected from the Control Type drop-down list. The control types are:

    • None – no control type exists.

    • Latch – control command is sent and the relay latches in that position. Latch controls require a second command to return to the original state.

    • Normal – normal controls are temporary or held only for a discrete number of milliseconds.

    • Pseudo – the point is controlled by another system. Pseudo controls do not have a physical status point to indicate if control has completed.

    • SBO Latch - Select Before Operate latch command.

    • SBO Pulse - Select Before Operate pulse (normal) command.

  1. If a control type other than None or Pseudo is selected:

    1. The control point offset can be changed in the Control Point Offset spin box. This specifies the physical location used for wiring the relay point.

    2. The close times can be changed in the Close Time 1 and Close Time 2 spin boxes. These times specify how long each relay stays energized.

    3. The control inhibit option can be enabled or disabled by selecting or clearing the Control Inhibit check box. Select this check box to disable control of the point. Clear this check box to enable control of the point.

    4. The command strings can be changed in Open and Closed text boxes under Command Strings. These are the commands that are sent when Yukon controls the point.

    5. The command time-out can be changed in the Command TimeOut minutes and seconds spin boxes.
