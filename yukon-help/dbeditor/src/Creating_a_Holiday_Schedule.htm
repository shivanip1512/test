Creating a Holiday Schedule

Perform the following steps to create a holiday schedule:

 

  1. From the View menu, select System.

  2. From the Create menu, select Holiday Schedule. The Holiday Schedule Setup dialog box appears.

  3. Enter a name for the schedule in the Schedule Name text box.

  4. Create, edit, or remove holidays in the Holiday Dates area.

    • To create a holiday, see Creating a Holiday.

    • To edit a holiday, see Editing a Holiday.

    • To remove a holiday, see Removing a Holiday.

  5. Click the Finish button. The new holiday schedule appears in the Holiday Schedule Editor. For instructions on editing a holiday schedule, refer to the related topics.
