Integration Load Group Addressing Tab

The following changes can be made to the addressing settings of the Integration load group:

 

  1. In the Geographical Address Usage area, select the check boxes of the group addresses to use when commands are sent to the load group. The Geographical Address Usage options are:

    • GEO – Group commands are addressed to load control devices that are configured with the Geographical address entered in step 3.

    • Substation – Group commands are addressed to load control devices that are configured with the Substation address entered in step 3.

    • Feeder – Group commands are addressed to load control devices that are configured with the Feeder addresses entered in step 3.

    • Zip – Group commands are addressed to load control devices that are configured with the Zip address entered in step 3.

    • User – Group commands are addressed to load control devices that are configured with the User Defined Address entered in step 3.

To configure the load group to send commands to a single load control device, select the Serial check box and enter the serial number of the load control device in the Serial text box. Valid values are 1 to 4,294,967,295.

Note: When an address check box is selected in the Geographical Address Usage area, the address field in the Geographical Addressing area is activated and it turns blue.

  1. In the Load Address Usage area, select the check boxes of the load addresses to use when commands are sent to a load group. The Load Address Usage options are:

    • Load – Control commands are addressed to the relays of load control devices that are configured with the loads selected in step 4.

    • Program – Control commands are addressed to the relay of load control devices that are configured with the Program address entered in step 4.

    • Splinter – Control commands are addressed to the relay of load control devices that are configured with the  Splinter address entered in step 4.

Note: When the Load check box is selected in the Load Address Usage area, the name of the Configured Loads area changes to Targeted Loads and all of the Load address fields are activated and they turn blue.

Note: When a Program or Splinter check box is selected in the Load Address Usage area, the associated address field in the Load Addressing area is activated and it turns blue.

  1. The Geographical addressing information can be changed in the Geographical Addressing area.

    1. The Service Provider ID address of the load group can be changed in the SPID text box. The Service Provider ID address is sent with all group address messages and is used to identify the service provider/owner utility. Valid values are 1 to 65,534.

    2. The Geographical address of the load group can be changed in the GEO text box. The Geographical address identifies a desired geographical area, determined by the utility. Valid values are 1 to 65,534.

    3. The Substation address of the load group can be changed in the Substation text box. The Substation address is typically used to identify the utility substation on which the target equipment is located. Valid values are 1 to 65,534.

    4. The Feeder addresses of the load group can be changed by clicking the Set Feeder Address button and selecting the check boxes of the bits to use. The Feeder address is typically used to identify the utility feeder by which the load is fed. This is normally combined with the Substation address, but it can be used alone. Each bit in this address represents a feeder, which allows multiple feeders to be targeted with a single message.

    5. The Zip address of the load group can be changed in the ZIP text box. The Zip address is typically used as the postal zip code of the geographic location. This address can also be used for other addressing options. Valid values are 1 to 16,777,214.

    6. The user defined address of the load group can be changed in the User text box. The user defined address is used for additional addressing options. Valid values are 1 to 65,534.

  2. The Load addressing information can be changed in the Load Addressing area.

    1. The program address can be changed in the Program text box. The Program address is used to target specific loads. Field devices that control multiple loads contain a separate program address for each load. Valid values are 1 to 254.

    2. The splinter address can be changed in the Splinter text box. The Splinter address is used to target a subset of the loads targeted by the Program address. Field devices that control multiple loads contain a separate splinter address for each load. Valid values are 1 to 254.

  3. If the Program or Splinter address check box was selected in the Load Address Usage area in step 4, select the check box(es) of the load(s) to control in the Configured Loads area.

  4. If the Load address check box was selected in the Load Address Usage area in step 4, select the check box(es) of the load(s) to control in the Targeted Loads area.
