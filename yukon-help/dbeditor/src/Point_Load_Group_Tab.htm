Point Load Group Tab

The following changes can be made to the control point settings of a Point load group:

 

  1. A different device, on which the control point is attached, can be selected from the Control Device drop-down list.

  2. A different point, on the device that performs the control, can be selected from the Control Point drop-down list.

  3. A different state, which the point needs to be in to start control, can be selected from the Control Start State drop-down list.

  4. Click the Apply button to save the changes to the point group or click the OK button to save the changes to the point group and close the Load Group Editor.
