Creating an RFN Template

Perform the following steps to create an RFN template:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select RFMESH from the list on the left.

  4. Select the type of RFN to create from the list on the right:

    • RFN-410fL – Landis+Gyr FOCUS®

    • RFN-410fX – Landis+Gyr FOCUS AX

    • RFN-410fD – Landis+Gyr FOCUS AX-SD

    • RFN-430A3 – Elster A3 ALPHA®

    • RFN-430kV – GE kV2c

  5. Click the Next button.

  6. Enter the RFN template name in the Device Name text box.

    • For an RFN-410fL, the template name is *RfnTemplate_LGYR_FocuskWh.

    • For an RFN-410fX, the template name is *RfnTemplate_LGYR_FocusAXR

    • For an RFN-410fD, the template name is *RfnTemplate_LGYR_FocusAXD.

    • For an RFN-430A3, the template name is *RfnTemplate_EE_A3R.

    • For an RFN-420kV, the template name is *RfnTemplate_GE_kV2.

  7. Click the Next button.

  8. Enter the meter number to use for the RFN template in the Meter Number text box. The meter number for the RFN template must be well above the normal range of meter numbers used by the utility.

  9. Click the Next button.

  10. Do not enter any values in the Serial Number, Manufacturer, and Model text boxes. These fields must remain blank for the RFN template.

  11. To configure the RFN template so that the default points are added to an RFN created from this template, select the Create Default Points check box. To prevent the default points from being added to an RFN created from this RFN template, clear this check box.

  12. Click the Finish button. The new RFN template is displayed in the Device Editor. For instructions on editing an RFN, refer to the related topics.
