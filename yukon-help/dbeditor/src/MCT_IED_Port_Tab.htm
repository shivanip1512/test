MCT IED Port Tab

The following changes can be made to the IED (Intelligent Electric Device) port settings of an MCT-360 or MCT-370:

 

  1. A different IED can be selected from the Connected IED drop-down list. The IED options are:

    • None

    • Alpha Power Plus

    • Landis + Gyr S4

    • General Electric KV

  2. A different scan rate can be selected from the IED Scan Rate drop-down list. The scan rate options are:

    • 1 minute

    • 2 minute

    • 3 minute

    • 5 minute

  3. The default data class can be changed in the Default Data Class box. This is the data class for the connected IED.

  4. The default data offset can be changed in the Default Data Offset box. The offset is typically used for current/last interval specification.

  5. The password of the IED can be changed in the Password text box.

  6. The option to use IED data for real-time scanning can be enabled or disabled by selecting or clearing the Use IED Data for Real-time Scan check box. Select this check box to enable real-time scanning of IED data. Clear this check box to disable real-time scanning of IED data.
