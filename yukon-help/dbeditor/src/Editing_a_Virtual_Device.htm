Editing a Virtual Device

Perform the following steps to edit a virtual device:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select Device.

  3. Double-click on the virtual device to edit. The General tabbed page is displayed in the Device Editor, from which the following changes can be made:

    1. The name of the virtual device can be changed in the Device Name text box.

    2. The virtual device can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to take the virtual device out of service, which suspends scanning and alarming. Clear this check box to enable the virtual device.

  4. Click the Apply button to save the changes to the virtual device or click the OK button to save the changes to the virtual device and close the Device Editor.
