TCP Terminal Server

If TCP Terminal Server was selected as the port type:

 

  1. Select one of the following communication line options:

    • Dedicated – The communication line is connected to a leased line or data radio connection.

    • Dialup  – The communication line is connected to a modem which dials up a field device.

  2. Click the Next button.

  3. Enter a name for the comm channel in the Description text box. This name is used by the system when it communicates error and informational messages.

  4. Enter the IP address of comm channel in the IP Address text box. This must be a valid decimal-dot IP address.

  5. Enter the port number in the Port Number text box. This is the physical port on the terminal server.

  6. Select the baud rate of the port from the Baud Rate drop-down list. This is the communications line speed used by this port.

  7. The rest of this procedure depends on whether Dedicated or Dialup was selected as the type of communication line.

  • If Dedicated was selected, click the Finish button. The new communication channel appears in the Comm Channel Editor. For instructions on editing a communication channel, refer to the related topics.

  • If Dialup was selected, click the Next button, select the type of modem being used from the drop-down list of modems, and then click the Finish button. The new communication channel appears in the Comm Channel Editor. For instructions on editing a communication channel, refer to the related topics.
