Creating a Control Scenario

Perform the following steps to create a control scenario:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Control Scenario.

  3. Enter a name for the control scenario in the Scenario Name text box.

  4. Select the load program(s) to assign to the control scenario from the Available Programs list. A load program must be assigned to a control area to appear in the Available Programs list.

  5. Click the Assign to Scenario button to assign the selected load program(s) to the control scenario.

  6. To specify a start offset for a load program, click in the program's Start Offset field and enter an offset in mm:ss format. When a control scenario starts, this is the time in minutes and seconds that the load program must wait before it can start.

  7. To specify a stop offset for a load program, click in the program's Stop Offset field and enter an offset in mm:ss format. When a control scenario stops, this is the time in minutes and seconds that the load program must wait before it can start.

  8. To select a different gear with which to start control, click in the program's Start Gear field and select a gear from the drop-down list.

Note: To remove a load program that is assigned to the control scenario, select the load program from the Assigned Programs area and click the Remove from Scenario button.

  1. Click the Finish button. The new control scenario appears in the Control Scenario Editor. For instructions on editing a control scenario, refer to the related topics.
