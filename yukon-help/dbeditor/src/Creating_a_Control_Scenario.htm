Creating a Control Scenario

To create a Control Scenario:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Control Scenario.

  3. Enter a name for the control scenario in the Scenario Name text box.

  4. Select the load program(s) to add to the control scenario from the Available Programs area. Load programs must belong to a control area to appear in this list.

  5. Click Assign to Scenario to add the load program(s) to the control scenario.

  6. To give a load program a start offset, click in the program's Start Offset field and enter an offset in mm:ss format.

  7. To give a load program a stop offset, click in the program's Stop Offset field and enter an offset in mm:ss format.

  8. To select a different gear to start control with, click in the program's Start Gear field and select the gear.

  9. To remove a program assigned to the scenario, select the program from the Assigned Programs are and click Remove from Scenario.

  10. Click Finish. The new control scenario appears in the Control Scenario Editor. For instructions on editing a control scenario, refer to the related topics.
