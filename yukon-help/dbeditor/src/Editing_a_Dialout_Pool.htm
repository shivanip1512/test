Editing a Dialout Pool

Perform the following steps to edit a dialout pool:

 

  1. When the dialout pool comm channel was double-clicked, the General tabbed page was displayed in the Device Editor. The following changes can be made from the General tab of the dial out pool:

    • The name of the comm channel can be changed in the Description text box.

    • The comm channel can be enabled or disable by selecting or clearing the Disable Channel check box. Select this check box to disable the communication channel if it is temporarily not functioning.

  2. Click the Pooled Ports tab to display the Pooled Ports tabbed page, from which the following changes can be made:

    • To assign a comm channel, select the comm channel from the Available list and click the Add button. To select more than one comm channel, hold the Ctrl key on the keyboard while clicking additional comm channels.

    • To rearrange the order of the dialup comm channels, click and drag an comm channel to move it up or down the list.

    • To unassign a dialup comm channel, select the comm channel from the Assigned list and click the Remove button.

  3. Click the Apply button to save the changes to the comm channel or click the OK button to save the changes to the comm channel and close the Comm Channel Editor.
