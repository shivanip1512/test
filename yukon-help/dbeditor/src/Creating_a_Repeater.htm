Creating a Repeater

Before you can create a Repeater, you must create routes.

 

If you selected Repeater as the device type:

 

  1. Enter a name for the repeater in the Device Name text box.

  2. Enter the physical address in the Physical Address text box. For the Repeater 800, Repeater 801, and Repeater 921, the physical address is located on a label inside the unit. For the Repeater 900, the physical address is a thumbwheel setting.

  3. Click Next.

  4. Select the Periodic Health Check check box if you want to enable this scan. The Periodic Health Check scan periodically checks that the device functioning.

    1. Select the scan interval and alternative scan interval from the Interval and Alt Interval drop-down lists.

    2. Select a scan group or enter a new scan group in the Group drop-down list. This is used to group similar scans together.

  5. Under the Optional Scan Window, select the Enabled check box if you want to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the open and close times in the Open and Close text boxes.

  6. Click Next.

  7. Select the route to use with this device from the drop-down list of routes.

  8. Click Finish. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.
