Creating a Repeater

Before you can create a Repeater, you must create CCU routes in the Yukon database.

 

If you selected Repeater as the device type:

 

  1. Enter a name for the repeater in the Device Name text box.

  2. Enter the physical address in the Physical Address text box. For the Repeater 800, Repeater 801, Repeater 902, and Repeater 921, the physical address is located on a label inside the unit. For the Repeater 900, the physical address is a thumbwheel setting.

  3. To create the default points for the repeater, select the Create Default Points check box.

  4. Click Next.

  5. Select the Periodic Health Check check box if you want to enable this scan. The Periodic Health Check scan periodically checks that the device functioning.

    1. Select the default scan interval from the Interval drop-down list.

    2. Select the alternate scan interval from the Alt Interval drop-down list. The alternate scan interval can be activated using the Tabular Data Console or the Yukon Web interface.

    3. Select a scan group or enter a new scan group in the Group drop-down list. This is used to group similar scans together.

  6. Under the Optional Scan Window, select the Enabled check box if you want to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the time at which the scan window starts in the Open text box.

    3. Enter the time at which the scan window stops in the Close text box.

  7. Click Next.

  8. Select the communication route to the repeater from the drop-down list of routes.

  9. Click Finish. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.
