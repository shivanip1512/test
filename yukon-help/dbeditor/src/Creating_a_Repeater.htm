Creating a Repeater

Before a Repeater can be created, a CCU communication route must be created in the Yukon database.

 

Perform the following steps to create a repeater:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select Signal Transmitter from the list on the left.

  4. Select the type of repeater to create from the list on the right:

    • Repeater 900 – Single-Phase and Three-Phase PLC Repeater.

    • Repeater 902 – Single-Phase and Three-Phase PLC Repeater.

    • Repeater 800 – Single-Phase PLC Repeater.

    • Repeater 850 – Single-Phase PLC Repeater.

    • Repeater 801 – Single-Phase PLC Repeater.

    • Repeater 921 – Single-Phase and Three-Phase PLC Repeater.

  5. Click the Next button.

  6. Enter a name for the repeater in the Device Name text box.

  7. Enter the physical address of the repeater in the Physical Address text box. For the Repeater 800, Repeater 801, Repeater 902, and Repeater 921, the physical address is located on a label inside the unit. For the Repeater 900, the physical address is a thumbwheel setting.

  8. To create the default points for the repeater, select the Create Default Points check box. To prevent the default points from being created, clear this check box.

  9. Click the Next button.

  10. To enable periodic health check scans, which periodically check that the repeater is functioning, select the Periodic Health Check check box.

    1. Select the default scan interval from the Interval drop-down list.

    2. Select the alternate scan interval from the Alt Interval drop-down list.

    3. Select a scan group from the Group drop-down list. The scan group is used to assign priorities to the scan category.

  11. Under the Optional Scan Window, select the Enabled check box to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the time at which the scan window starts in the Open text box.

    3. Enter the time at which the scan window stops in the Close text box.

  12. Click the Next button.

  13. Select the communication route to the repeater from the Select the route used with this device drop-down list.

  14. Click the Finish button. The new repeater appears in the Device Editor. For instructions on editing a repeater, refer to the related topics.
