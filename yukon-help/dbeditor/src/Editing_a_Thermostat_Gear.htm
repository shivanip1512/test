Editing a Thermostat Ramping Gear

Note: All times are in entered in minutes.

 

To edit a Thermostat Ramping gear:

  1. Change the name for the gear in the Gear Name text box.

  2. Select Thermostat Ramping from the Gear Type drop-down list.

  3. By default, Deltas and Fahrenheit units are used by the gear.

    • To use Absolute values instead of Deltas, click the Deltas button. The button changes to Absolutes.

    • To use Celsius units instead of Fahrenheit units, click the Fahrenh button. The button changes to Celsius

  4. The mode the thermostat must be in to accept the commands can be changed: Heat Mode or Cool Mode. Select both check boxes to send commands to both modes.

  5. To change the minimum temperature, select the Min check box and enter the new minimum temperature in the text box.

  6. To change a maximum temperature, select the Max check box and enter the new maximum temperature in the text box.

  7. To change Delta setpoints:

    1. Click the push button to select Deltas as the data type.

    2. To change the Delta B value, select the Delta B check box and enter the new delta setpoint in text box.

    3. To change the Delta D value, select the Delta D check box and enter the new delta setpoint in text box.

    4. To change the Delta F value, select the Delta F check box and enter the new delta setpoint in text box.

  8. To change Absolute setpoints:

    1. Click the push button to select Absolutes as the data type.

    2. To change the Absolute B value, select the Abs B check box and enter the new absolute setpoint in text box.

    3. To change the Absolute D value, select the Abs D check box and enter the new absolute setpoint in text box.

    4. To change the Absolute F value, select the Abs F check box and enter the new absolute setpoint in text box.

  9. To change the random offset time, select the Rand check box and enter the new random offset in the text box. A random offset between 0 minutes and the value Rand is added to any delay time

  10. To change the delay time TA, select the TA check box and enter the new delay time in the text box.

  11. To change the ramp time TB, select the TB check box and enter the new amp time in the text box.

  12. To change the plateau time TC, select the TC check box and enter the new plateau time in the text box. During this time the setpoint Abs B is maintained.

  13. To change the ramp time TD, select the TD check box and enter the new ramp time in the text box.

  14. To change the plateau time TE, select the TE check box and enter the new plateau time in the text box. During this time the setpoint Abs D is maintained.

  15. To change the ramp time TF, select the TF check box and enter the new ramp time in the text box.

  16. To change how stop the control, select a new stop method from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

  17. The group capacity reduction percentage can be changed in the Group Capacity Reduction % spin box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop.

  18. A different automatic gear change method can be selected from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear changes only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear changes when the control area priority reaches this priority.

    • Above Trigger - Triggers are defined in Control Areas. Select whether to use trigger one or two in the Trigger Number spin box. (Trigger one is the first entry and trigger two is the second entry of the Control Area’s Trigger drop-down list.) The gear changes when the Control Area threshold value reaches the trigger plus the offset value in the Trigger Offset text box.

  19. Click OK to save the changes.
