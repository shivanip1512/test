Creating a Login

Logins are required to access the Yukon client software. To assign group permissions to a login, edit the login after creating it.

 

To create a login for Yukon access:

 

  1. From the View menu, select System.

  2. From the Create menu, select Login.

  3. To enable the login you are creating, select the Login Enabled check box. Clear this check box if you want to create a login without enabling it.

  4. Enter a username for the login in the User Name text box.

  5. Select the type of password authentication from the Authentication drop-down list. The authentication types are:

    • Normal - The password is stored as plain-text in the Yukon database.

    • Hashed - The password is converted to a sequence of letters and numbers and is then stored in the Yukon database.

    • Radius - A radius login server is used to authenticate passwords.

    • No Login - The password is disabled and the user cannot log in.

  6. If Normal or Hashed is selected as the authentication type, click the Change Password button and enter the password in the New Password and Retype Password text boxes.

  7. (Optional) If the login will be used to access the Yukon STARS application, select the Link to Energy Company check box and select an energy company from the drop-down list.

  8. Click Finish. The new login appears in the Login Editor. For instructions on editing a login, refer to the related topics.
