Creating a Login

A login is required to use the Yukon software and the permissions of a login are controlled by login group(s) to which it is assigned. To assign a login to a login group, refer to Editing a Login.

 

Perform the following steps to create a login for the Yukon system:

 

  1. From the View menu, select System.

  2. From the Create menu, select Login.

  3. To enable the login account that is being creating, select the Login Enabled check box. Clear this check box to create a login account without enabling it.

  4. Enter a username for the login account in the User Name text box.

  5. Select the type of password authentication for the login account from the Authentication drop-down list. The authentication types are:

    • Normal – The password is stored as plain-text in the Yukon database. This is the type of authentication that is required for a Yukon STARS login.

    • Hashed – The password is converted to a sequence of letters and numbers and is then stored in the Yukon database.

    • Radius – A Radius login server is used to authenticate passwords.

    • No Login – The password is disabled and the user cannot log in.

    • Active Directory – An Active Directory server is used to authenticate passwords.

    • LDAP – An LDAP server is used to authenticate passwords.

  6. If either Normal or Hashed is selected as the authentication type, the password for the login account can be created by performing the following steps:

    1. Click the Change Password button.

    2. Enter the password for the login account in the New Password and Retype Password text boxes.

    3. Click the OK button to save the password.

  7. To assign the login to an energy company, select the Link to Energy Company check box and select an energy company from the drop-down list.

  8. Click the Finish button. If a password has not been entered for the login account, enter a password in the dialog box that appears and click the OK button.
    The new login account appears in the Login Editor. For instructions on editing a login account, refer to the related topics.
