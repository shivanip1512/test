Creating a Login

To create a login for Yukon access:

 

  1. From the View menu, select System.

  2. From the Create menu, select Login.

  3. To enable the login you are creating, select the Login Enabled check box. Clear this check box if you want to create a login without activating it.

  4. Enter a username for the login in the User Name text box.

  5. Enter a password for the login in the Password text box.

  6. Re-enter the password for the login in the Retype Password text box.

  7. If the login will be used to access the Yukon STARS application, select the Link to Energy Company check box and select an energy company from the drop-down list.

  8. Click Finish. The new login appears in the Login Editor. For instructions on editing a login, refer to the related topics.
