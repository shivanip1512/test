MCT Load Group Addressing Tab

The following changes can be made to the addressing settings of an MCT load group device:

  1. A different type of address level can be selected from the Address Level drop-down list. The types of address levels are:

    • Bronze – The Bronze address is a group address of the MCT, which is typically used for relay control. Valid values are 1 to 256.

    • Lead – The Lead address is a group address of the MCT, which is typically used for receiving broadcast commands. Valid values are 1 to 4096.

    • MCT Address – The MCT address is the unique address of the MCT. Valid values are 1 to 4,182,015.

  2. If Bronze or Lead is selected as the address level, the group address can be changed in the Address text box. The MCTs that are configured with this group address can be controlled by this load group.

  3. If MCT Address is selected as the addressing type, an MCT can be selected from the MCT Address drop-down list. The MCT that is selected from the drop-down list can be controlled by this load group.

  4. The relays to control in the MCT can be selected using the check boxes in the Relay Usage area. The relays that can be selected are:

    • Relay 1

    • Relay 2

    • Relay 3

    • Relay 4

  5. Click the Apply button to save the changes or OK button to save the changes and close the Load Group Editor.
