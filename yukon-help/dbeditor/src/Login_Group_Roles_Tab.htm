Login Group Roles Tab

The following changes can be made to the roles settings of a login group:

 

  1. The roles assigned to this login group can be changed on this tabbed page.

    1. Click the + next to the Role Categories to expand the roles.

    2. Select the check boxes of role groups to assign to this login group.

    3. Clear check boxes of the role groups to unassign from this login group.

The following login/role groups are pre-defined in the Yukon database:

Role Group

Meaning

Esub Operators Grp

Login group that grants operator access to the Esubstation application.

Esub Users Grp

Login group that grants user access to the Esubstation application.

Operators Grp

Login group that grants operator access to the Yukon system.

Residential Customers Grp

Login group that grants residential customer access to the thermostat Web application.

STARS Operators Grp

Login group that grants operator access to the Yukon STARS applications.

STARS Residential Customers Grp

Login group that grants residential customer access to the Yukon STARS applications.

System Administrators Grp

Login group that grants administrative access to the Yukon system.

Web Client Customers Grp

Login group that grants client access to Yukon Web applications.

Web Client Operators Grp

Login group that grants operator access to Yukon Web applications.

Yukon Grp

Special login group that grants limited user access to the Yukon system. This login group is required for all Yukon login accounts.

   

  1. The properties for the selected role appears on the right side the tabbed page. To edit the properties of a role:

    1. Select the role to edit. The Key and Value properties appear in the list to the right. Click on a key to display a short description of the key in the Description box.

    2. Click in the Value of the property to edit and enter a new value in the field.

    3. Press enter on the keyboard to save the new value.

If the Value fields of a selected role are not editable, select a different role, then select the role to edit and click in the Value field to edit.

Note: Modified values are displayed in blue.
