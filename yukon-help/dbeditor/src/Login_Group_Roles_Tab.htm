Login Group Roles Tab

The following changes can be made to the roles settings of a login group:

 

  1. To change the roles that are assigned to the login group:

    1. Click the + next to a role category name in the Role Categories area to display the roles in that category.

    2. Select the check box(es) of the role(s) to assign to the login group.
      Clear check box(es) of the role(s) to unassign from the login group.

  1. To edit the properties of a role assigned to the login group:

    1. Select the role to edit. The Key and Value properties appear in the Properties list to the right.

    2. Select the key to edit. A short description of the key is displayed in the Description box.

    3. To edit the value of the selected key, click in the Value field of the property and enter a new value.

    4. Press Enter on the keyboard to save the change. The modified value is displayed in blue.

Note: If the Value fields of a selected role are not editable, select a different role first, then select the role to edit and click in the Value field to edit the value.
