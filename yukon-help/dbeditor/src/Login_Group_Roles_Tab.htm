Login Group Roles Tab

The following changes can be made to the roles settings of a login group:

 

  1. The roles assigned to this login group can be changed on this tabbed page.

    1. Click the + next to the Role Categories to expand the roles.

    2. Select the check boxes of role groups to assign to this login group.

    3. Clear check boxes of the role groups to unassign from this login group.

  1. The properties for the selected role appears on the right side the tabbed page. To edit the properties of a role:

    1. Select the role to edit. The Key and Value properties appear in the list to the right. Click on a key to display a short description of the key in the Description box.

    2. To edit the value, click in the Value of the property, enter a new value in the field and press Enter on the keyboard to save the change.

If the Value fields of a selected role are not editable, select a different role, then select the role to edit and click in the Value field to edit.

Note: Modified values are displayed in blue.
