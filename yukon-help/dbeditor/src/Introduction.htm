Introduction

The Yukon® Database Editor is used to add, modify, or remove database items in the Core, Load Management, and System databases.

  • Core database - defines comm channels, signal transmitters, communication routes, MCTs, meters, RTUs, points, and state groups.

  • Load Management database - defines the control scenarios, control areas, load management programs, program constrains, and load groups.

  • System database - defines the alarm categories, baselines, customers, contacts, holiday schedules, logins, login groups, notification groups, season schedules, tags, TOU schedules, and system points.

 

The Database Editor is made up of the following:

 

  • Menu Bar

 

  • Sort By List

 

  • Database Item List

 

  • Database Window

 

  • Message Log

 

Yukon devices, groups, and points are referred to as Database Items. Database Items are created using wizards, which guide a user through the initial setup. After the initial setup is completed, the Database Items can be edited using the Database Editor.
