Introduction

The Yukon™ Database Editor is used to add, modify, or remove database items in the Core, Load Management, and System databases.

  • Core – The Core database defines comm channels, communication routes, MCTs, meters, RTUs, points, signal transmitters, state groups, and two-way LCRs.

  • Load Management – The Load Management database defines the control scenarios, control areas, load groups, load programs, points, and program constrains.

  • System – The System database defines the alarm categories, baselines, customers, contacts, holiday schedules, logins, login groups, notification groups, season schedules, tags, TOU schedules, and system points.

 

The Database Editor is made up of the following items:

  • Menu Bar

  • Sort By List

  • Database Item List

  • Database Window

  • Message Log

Database items, such as devices, groups, and points, are created using database wizards, which guide a user through the initial setup. After the initial setup is completed, database items can be edited using the Database Editor.
