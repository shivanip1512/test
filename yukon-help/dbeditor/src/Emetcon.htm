EMETCON

If you selected an EMETCON device:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of the switches in the load group.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Select the Disable Control check box.

  5. Select the route to use from the Communication Route drop-down list. This is the route or route macro used to reach all of the devices in this group when transmitting commands.

  6. Click Next.

  1. Set the gold address in the Gold Address spin box. A gold address supports a value of 1 to 4.

  2. Set the silver address in the Silver Address spin box. A silver address supports a value of 1 to 60.

  3. Select the relay to use for this group in the Relay to Use area. Select the S option to control all the relays.

  4. Select the type of address to use for control of this group in the Address to Use area.

  5. Click Finish. The new load group appears in the Device Editor. For instructions on editing a load group, refer to the related topics.
