Editing a Simple Thermostat Ramping Gear

Perform the following steps to edit a Simple Thermostat Ramping gear:

  1. The name of the gear can be changed in the Gear Name text box.

  2. Select Thermostat Ramping from the Gear Type drop-down list.

  3. The mode the thermostat must be in to accept the commands can be changed: Heat Mode or Cool Mode.

  4. To change the random start time, between zero minutes and two hours, edit the value in the Random Start Time box. A random offset between zero minutes and the Random Start Time value is added to the start of control.

  5. To change the number of degrees to decrease the temperate during the Pre-Op Cool time or increase the temperature during the Pre-Op Heat Time, edit the value In the Pre-Op (Cool or Heat) Temp box.

  6. To change the length of time that the temperature is decreased during a Pre-Op Cool or the length of time that the temperature is increased during a Pre-Op Heat, edit the value in the Pre-Op (Cool or Heat) Time box.

  7. To change the length of time that the Pre-Op Cool temperature is held or length of time that the Pre-Op Heat temperature is held, edit the value in the Pre-Op (Cool or Heat) Hold box.

  8. To change the number of degrees to increase or decrease the temperate each hour during the load program run time, edit the value in the Ramp °F/Hour box.

  9. To change the maximum number of degrees that the temperature can ramp/change during the load program run time, edit the value in the Max °∆ box.

  10. To change the length of time that the temperature is decreased or increased to the original setpoint before control stops, edit the value in the Ramp Out Time box.

  11. To change the maximum runtime of the load program, edit the value in the Max Runtime box.

  12. To change the rate at which the command is resent, edit the value in the Resend Rate box.

  13. To change how stop the control, select a new stop method from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In – Control stops when the shed time is over.

    • Restore – Control stops when a restore command is sent to the load control device.

  14. A different automatic gear change method can be selected from the When to Change drop-down list. The change methods are:

    • Manually Only – The gear changes only when it is manually switched.

    • After a Duration – The gear changes when the duration ends, which is entered in the Change Duration text box.

    • Priority Change – The gear changes when the priority of the control area, to which this load program is assigned, reaches the priority entered in the Change Priority text box.

    • Above Trigger – The gear changes when the trigger point of the selected trigger rises above the value that is calculated by adding the trigger threshold value and the trigger offset value. The trigger to use for the gear change, trigger one (1) or trigger two (2) of the control area to which this program is assigned, is entered in the Trigger box and the trigger offset value is entered in the Trigger Offset text box.
      Note:       Trigger one (1) is the first entry in the Trigger drop-down list and trigger two (2) is the second entry in the Trigger drop-down list of the control area.

  15. Click the Apply button to save the changes to the gear or click the OK button to save the changes to the gear and close the Load Program Editor.
