Editing a Core Database Point

Core database points allow for the tracking of data. A point may be any of the following types:

  • Analog - reads the value directly from the transducer or electronic device. Examples include a pseudo point and Voltage or Amperage on a field device.

  • Status - reads the value of the contact or digital input, usually open or closed. Examples include a relay on a load control receiver or a switch status on a capacitor bank. A status point can also have control.

  • Accumulator - measures any quantity that is a simple count accumulation, such as the number of pulses. Examples include kWh energy point or kW demand (energy over time).

  • Calculated - calculates data gathered from other points.

 

 To edit a point:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select one of the following:

    • Device

    • IED

    • MCT

    • Meter

    • RTU

    • Transmitter

  3. Click the + next to a device to display the Point type.

  4. Click the + next to the Point type to display the Point.

  5. Double-click the point to modify. This opens the General tabbed page of the Point Editor. The General tabbed page contains different items depending on the type of point.

    • Analog Point General Tab

    • Status Point General Tab

    • Accumulator Point General Tab

    • Calculated Point General Tab

  6. Click the Physical Setup tab to display the Physical Setup tabbed page appears, which contains different options depending on the type of point.

    • Analog Point Physical Setup Tab

    • Status Point Physical Setup Tab

    • Accumulator Point Physical Setup Tab

  7. If the point is an analog, accumulator, or calculated point, click the Limits tab to display the Limits tabbed page.

  8. Click the Foreign Data tab to display the Foreign Data tabbed page, from which Foreign Data Routings can be added, updated, or deleted.

    • Setting Up Foreign Data Routing

    • Updating a Foreign Data Routing Entry

    • Removing Foreign Data Routing Entry

  9. Click the Alarms tab to display the Alarms tabbed page appears.

  10. If the point is a calculated point, click the Calculation tab to display the Calculation tabbed page, from which Calculations can be added, updated, or deleted.

    • Adding a Calculation

    • Updating a Calculation

    • Removing a Calculation

  11. Click Apply to save the changes or OK to save the changes and close the Point Editor.
