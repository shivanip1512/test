Editing a Core Database Point

Core database points allow for the tracking of data. A point may be any of the following types:

  • Analog – Reads the value directly from an electronic device. Examples include a voltage or amperage point on a meter or a pseudo point.

  • Status – Reads the value of the contact or digital input, which is usually open or closed. Examples include a relay on a load control receiver, a switch status on a capacitor bank, or a control point on an RTU.

  • Accumulator – Measures any quantity that is a simple count accumulation, such as the number of pulses. Examples include kWh energy or kW demand (energy over time).

  • Calculated – Calculates a value using data gathered from other points. There are two output types for calculated points:

    • Analog Output – The output is displayed as an analog value.

    • Status Output – The output is displayed as the status of a state group.

 

Perform the following steps to edit a point:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select one of the following:

    • Device

    • IED

    • MCT

    • Meter

    • RTU

    • Transmitter

  3. Click the + next to a device to display the Point type.

  4. Click the + next to the Point type to display the Point.

  5. Double-click the point to edit. The General tabbed page is displayed in the Point Editor.

The General tabbed page contains different items depending on the type of point.

    • Analog Point General Tab

    • Status Point General Tab

    • Accumulator Point General Tab

    • Calculated Point General Tab

  1. Click the Physical Setup tab to display the Physical Setup tabbed page, which contains different options depending on the type of point.

    • Analog Point Physical Setup Tab

    • Status Point Physical Setup Tab

    • Accumulator Point Physical Setup Tab

  2. Click the Limits tab to display the Limits tabbed page.

  3. Click the Foreign Data tab to display the Foreign Data tabbed page, from which Foreign Data Routings can be added, updated, or deleted.

    • Setting Up Foreign Data Routing

    • Updating a Foreign Data Routing Entry

    • Removing Foreign Data Routing Entry

  4. Click the Alarms tab to display the Alarms tabbed page, from which point limits and stale data settings can be configured.

  5. If the point is a calculated point, click the Calculation tab to display the Calculation tabbed page, from which calculations can be added, updated, or deleted.

    • Adding a Calculation

    • Updating a Calculation

    • Removing a Calculation

  6. Click the Apply button to save the changes to the point or click the OK button to save the changes to the point and close the Point Editor.
