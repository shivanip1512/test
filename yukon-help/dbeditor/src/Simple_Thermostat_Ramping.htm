Simple Thermostat Ramping

To create a thermostat ramping gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Simple Thermostat Ramping from the Gear Type drop-down list.

  3. Select the mode the thermostat must be in to accept commands: Heat Mode or Cool Mode.

  4. Enter an offset between zero minutes and two hours in the Random Start Time spin box. A random offset between 0 minutes and the Random Start Time value is added to the start of control.

  5. In the Pre-Op (Cool or Heat) Temp spin box, enter the number of degrees to decrease the temperate during the Pre-Op Cool time or increase the temperature during the Pre-Op Heat Time.

  6. In the Pre-Op (Cool or Heat) Time spin box, enter the length of time that the temperature is decreased during a Pre-Op Cool or the length of time that the temperature is increased during a Pre-Op Heat.

  7. In the Pre-Op (Cool or Heat) Hold spin box, enter the length of time that the Pre-Op Cool temperature is held or length of time that the Pre-Op Heat temperature is held.

  8. Enter the number of degrees to increase or decrease the temperate each hour during the load program run time in the Ramp °F/Hour spin box.

  9. Enter the maximum number of degrees that the temperature can ramp/change during the load program run time in the Max °∆ spin box.

  10. Enter the length of time that the temperature is decreased or increased to the original setpoint before control stops in the Ramp Out Time spin box.

  11. Enter the maximum runtime of the load program in the Max Runtime spin box.

  12. Enter the rate at which the command is resent in the Resend Rate spin box.

  13. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

  14. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear changes only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear changes when the control area priority reaches this priority.

    • Above Trigger - Triggers are defined in Control Areas. Select whether to use trigger one or two in the Trigger Number spin box. (Trigger one is the first entry and trigger two is the second entry of the Control Area’s Trigger drop-down list.) The gear changes when the Control Area threshold value reaches the trigger plus the offset value in the Trigger Offset text box.

  15. Click the OK button to save the gear.
