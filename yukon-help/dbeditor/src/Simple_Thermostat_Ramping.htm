Simple Thermostat Ramping

To create a thermostat ramping gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Simple Thermostat Ramping from the Gear Type drop-down list.

  3. Select the mode the thermostat must be in to accept commands: Heat Mode or Cool Mode.

  4. Enter an offset, between zero minutes and two hours, in the Random Start Time box. A random offset between zero minutes and the Random Start Time value is added to the start of control.

  5. In the Pre-Op (Cool or Heat) Temp box, enter the number of degrees to decrease the temperate during the Pre-Op Cool time or increase the temperature during the Pre-Op Heat Time.

  6. In the Pre-Op (Cool or Heat) Time box, enter the length of time that the temperature is decreased during a Pre-Op Cool or the length of time that the temperature is increased during a Pre-Op Heat.

  7. In the Pre-Op (Cool or Heat) Hold box, enter the length of time that the Pre-Op Cool temperature is held or length of time that the Pre-Op Heat temperature is held.

  8. Enter the number of degrees to increase or decrease the temperate each hour during the load program run time in the Ramp °F/Hour box.

  9. Enter the maximum number of degrees that the temperature can ramp/change during the load program run time in the Max °∆ box.

  10. Enter the length of time that the temperature is decreased or increased to the original setpoint before control stops in the Ramp Out Time box.

  11. Enter the maximum runtime of the load program in the Max Runtime box.

  12. Enter the rate at which the command is resent in the Resend Rate box.

  13. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - Control stops when the shed time is over.

    • Restore - Control stops when a restore command is sent to the load control device.

  14. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only – The gear changes only when it is manually switched.

    • After a Duration – The gear changes when the duration ends, which is entered in the Change Duration text box.

    • Priority Change – The gear changes when the priority of the control area, to which this load program is assigned, reaches the priority entered in the Change Priority text box.

    • Above Trigger – The gear changes when the trigger point of the selected trigger rises above the value that is calculated by adding the trigger threshold value and the trigger offset value. The trigger to use for the gear change, trigger one (1) or trigger two (2) of the control area to which this program is assigned, is entered in the Trigger box and the trigger offset value is entered in the Trigger Offset text box.
      Note:       Trigger one (1) is the first entry in the Trigger drop-down list and trigger two (2) is the second entry in the Trigger drop-down list of the control area.

  15. Click the OK button to save the gear.
