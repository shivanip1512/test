Creating an RTU

If you selected RTU as the device type:

 

  1. Enter a name for the RTU in the Device Name text box.

  2. Enter an address in the Physical Address text box. The physical address is programmed into the device. The Physical Address text box is replaced by a Master Address text box for RTU-DART and RTU-DNP devices. There is no Physical Address for Davis Weather RTU devices.

  3. Click Next. If you selected a Davis Weather RTU, enter a utility meter number into the Utility Meter Number text box and click Next. If you selected RTU-DART, RTU-DNP, RTU-ILEX, or RTU-WELCO, skip to step 4.

  4. Select the check box of the scan categories you want to enable. The scan categories vary depending on the type of RTU selected.

If you selected a Davis Weather RTU, a dialog box appears with the General Data Collection scan category.

If you selected RTU-DART or RTU-DNP, a dialog box appears with the following scan categories

    • Class 0, 1, 2, 3 Scan - collects all of the data on the device

    • Class 1, 2, 3 Scan - collects the data on the device that has changed since the last scan

If you selected RTU-ILEX, a dialog box appears with the following scan categories:

    • General Scan - collects general data

    • Accumulator Rate - collects KYZ pulse points

If you selected RTU-WELCO, a dialog box appears with the following scan categories:

    • Integrity Rate - collects all point values

    • Report by Exception - reports if anything has changed on the device

    • Accumulator Rate - collects KYZ pulse points

    1. Select the default scan interval from the Interval drop-down list.

    2. Select the alternate scan interval from the Alt Interval drop-down list. The alternate scan interval can be activated using the Tabular Data Console or the Yukon Web interface.

    3. Select a scan group or enter a new scan group in the Group drop-down list. This is used to group similar scans together.

  1. Under the Optional Scan Window, select the Enabled check box if you want to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the time at which the scan window starts in the Open text box.

    3. Enter the time at which the scan window stops in the Close text box.

  2. Click Next.

  3. Select a comm channel from the drop-down list of comm channels

  4. The rest of this procedure depends on the type of comm channel selected.

  • If the comm channel is Dedicated, click Finish. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.

  • If the comm channel is Dialup or Dialback, click Next, enter the phone number used to dial this device or dial back from the device, and then click Finish. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.
