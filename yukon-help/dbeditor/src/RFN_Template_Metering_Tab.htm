RFN Template Metering Tab

The following changes can be made to the metering settings of the RFN:

 

  1. The meter number to use for the RFN template can be changed in the Meter Number text box. The meter number for the RFN template must be well above the normal range of meter numbers used by the utility.

  2. A different data collection group can be entered or selected in the Data Collection Group drop-down list.

  3. A different alternate group can be entered or selected in the Alternate Group drop-down list.

  4. A different billing group can be entered or selected in the Billing Group drop-down list.
