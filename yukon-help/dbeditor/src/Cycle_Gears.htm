Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle Gears

To add a Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle from the Gear Type drop-down list.

  3. To disable the ramping of load groups over an interval of time, select the No Ramp (ExpressCom only) check box. The cycle command must be send using the ExpressCom protocol.

  4. Set a control percentage in the Control Percent box. This is the percentage of the cycle period that is control time.

  5. Set a cycle period in the Cycle Period box. This is the cycle period in minutes.

  6. Select a send type for the cycle count from the Cycle Count Send Type drop-down list. The types are:

    • Fixed Count – The same number of cycle counts are sent to the switch each cycle.

    • Count Down – The number of cycles counts sent decreases at each resend.

    • Limited Count Down – Used with LCR 2000s, which do not allow cycle counts to be sent to them. The cycle count is hard coded in switch.

The purpose of Count Down and Limited Count Down is to end up with zero cycle counts at the end of control.

  1. Select a maximum cycle count from the Max Cycle Cnt drop-down list. This limits the number of cycles counts that can be sent.

  2. Select a starting period count from the Starting Period Count drop-down list. This is the count that is used when automatic control starts.

  3. Select a resend rate from the Command Resend Rate drop-down list. This controls how often the cycle command is resent.
    Note:     The resend rate can be in minutes or hours, depending on the selection in the drop-down list to the right of the Command Resent Rate drop-down list.

  4. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Restore - A restore command is sent to the device.

    • Stop Cycle - The period count is set to 0 at the end of control, so the cycle stops the next period.

  5. Set a group capacity reduction percentage in the Group Capacity Reduction % box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop.

  6. For a Target Cycle gear, enter the amount of targeted reduction in the kW Reduction text box.

  7. Select a gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear changes only when it is manually switched.

    • After a Duration - The gear changes when the duration ends, which is entered in the Change Duration text box.

    • Priority Change - The gear changes when the priority of the control area, to which this load program is assigned, reaches the priority entered in the Change Priority text box.

    • Above Trigger - The gear changes when the trigger point of the selected trigger rises above the value that is calculated by adding the trigger threshold value and the trigger offset value. The trigger to use for the gear change, trigger one (1) or trigger two (2) of the control area to which this program is assigned, is entered in the Trigger box and the trigger offset value is entered in the Trigger Offset text box.
      Note:       Trigger one (1) is the first entry in the Trigger drop-down list and trigger two (2) is the second entry in the Trigger drop-down list of the control area.

  8. Click the OK button to save the gear.
