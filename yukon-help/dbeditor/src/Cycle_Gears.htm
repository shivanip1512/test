Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle Gears

To add a Master Cycle, Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select either Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle from the Gear Type drop-down list.

  3. To disable the ramping in of load groups over an interval of time, select the No Ramp (ExpressCom only) check box. The cycle command must be send using the ExpressCom protocol.

  4. Set a control percentage in the Control Percent spin box. This is the percentage of the cycle period that is control time.

  5. Set a cycle period in the Cycle Period spin box. This is the cycle period in minutes.

  6. Select a cycle count send type from the Cycle Count Send Type drop-down list. The types are:

    • Fixed Count - The same number of cycle counts are sent to the switch each cycle.

    • Count Down - The number of cycles counts sent decreases at each resend.

    • Limited Count Down - Used with LCR 2000s, which do not allow cycle counts to be sent to them. The cycle count is hard coded in switch.

The purpose of Count Down and Limited Count Down is to end up with zero cycle counts at the end of control.

  1. Select a maximum cycle count from the Max Cycle Cnt drop-down list. This limits the number of cycles counts that can be sent.

  2. Select a starting period count from the Starting Period Count drop-down list. This is the count that is used when automatic control starts.

  3. Select a resend rate from the Command Resend Rate drop-down list. This controls how often the cycle command is resent.

  4. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Restore - A restore command is sent to the device.

    • Stop Cycle - The period count is set to 0, so the cycle stops the next period.

  5. Set a group capacity reduction percentage in the Group Capacity Reduction % spin box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop

  6. For Target Cycle, enter the amount of targeted reduction in the kW Reduction text box.

  7. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear changes only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear changes when the control area priority reaches this priority.

    • Above Trigger - Triggers are defined in Control Areas. Select whether to use trigger one or two in the Trigger Number spin box. (Trigger one is the first entry and trigger two is the second entry of the Control Area’s Trigger drop-down list.) The gear changes when the Control Area threshold value reaches the trigger plus the offset value in the Trigger Offset text box.

  8. Click OK to save the gear.
