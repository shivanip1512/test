Creating a Control Area

Perform the following steps to create a control area:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Control Area.

  3. Enter a name for the control area into the Name text box.

  4. Select a control internal from the Control Interval drop-down list. This controls how often the Yukon system checks to see if control is needed.

  5. Select the minimum response time from the Min Response Time drop-down list. This is the minimum number of seconds to wait after a control is sent before sending another control.

  6. Select a default state from the Default Daily State drop-down list. This controls what the operational state of the control area is set to each day at midnight. The default states are:

    • No Change – The default state of the control does not change from its current state.

    • Enable/ReEnable – The default state of the control is set to enabled.

    • Disable/Waived – The default state of the control is set to disabled.

  7. Select or clear the Require All Triggers Active check box. This is an optional setting that requires all of the triggers in the control area to be active for control to occur. Select the check box to force all triggers to be active. Clear the check box if all triggers do not have to be active.

  8. Enter the start and stop times of the optional control window the Start Time and Stop Time text boxes. This is the window of time during which the Yukon system is allowed to perform automatic feedback load control. The start and stop times also act as the peak control window.

Note: Changes to the control window times do not take effect until midnight.

  2. Click the New Trigger button to create a trigger. Up to two triggers can be defined per control area.

  3. Click the Next button.

  4. Perform the following steps to assign a load program to the control area:

    1. Select a load program from the Program drop-down list. This is a list of the available load programs in the Yukon system.

    2. Set the start priority in the Assigned Start Priority box. A load program with a start priority of one starts before a load program with a start priority of two.

    3. Set the stop priority in the Assigned Stop Priority box. A load program with a stop priority of one stops before a load program with a stop priority of two.

    4. Click the Add button. The load program is added to Assigned Programs list.

  5. Click the Finish button. The new control area appears in the Control Area Editor. For instructions on editing a control area, refer to the related topics.
