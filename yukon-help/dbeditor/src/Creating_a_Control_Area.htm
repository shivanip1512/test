Creating a Control Area

To create a Control Area:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Control Area.

  3. Enter a name for the control area into the Name text box.

  4. Select a control internal from the Control Interval drop-down list. The control interval is how often Yukon checks to see if control is needed.

  5. Select the minimum response time from the Min Response Time drop-down list. This is the minimum number of seconds to wait after a control is sent before sending another control.

  6. Select a default state from the Default Daily State drop-down list. The control area reverts to this state each day at midnight. The available states are None, Enable/ReEnable, or Disable/Waived.

  7. Select or clear the Require All Triggers Active check box. Select this check box to force all triggers to be active for control to occur. Clear this check box if you do not require all triggers to be active for control to occur.

  8. Under the Optional Control Window, enter a start and stop time in the Start Time and Stop Time text boxes. This is the window of time during which Yukon is allowed to perform automatic feedback load control. The start and stop times also act as the peak control window.

  9. Click the Next button.

  10. Click the New Trigger button to create a trigger. To delete a trigger, see Deleting a Trigger.

  11. Click the Next button.

  12. Select a load program from the Program list. These are the available load programs.

  13. Set the start priority in the Assigned Start Priority box. A load program with a start priority of one starts before a load program with a start priority of two.

  14. Set the stop priority in the Assigned Stop Priority box. A load program with a stop priority of one stops before a load program with a stop priority of two.

  15. Click the Add button. The load program is added to Assigned Programs list.

  16. Click the Finish button. The new control area appears in the Control Area Editor. For instructions on editing a control area, refer to the related topics.
