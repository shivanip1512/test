Ripple Load Group

If the load group is a Ripple load group:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of the load control devices in the load group. To disable the kilowatt capacity feature, enter a value of zero.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Select the Disable Control check box.

  5. Select the communication route to use for the load group from the Communication Route drop-down list. This is the route or route macro that is used to transmit commands to all of the load control devices in this group.

  6. Click the Next button.

  7. Select a shed time from the Shed Time drop-down list that matches the shed time programmed into the load control device. The shed time is the number of minutes a relay controls when a control command is sent. A shed time of Continuous Latch controls the relay until a restore command is sent.

  8. Set a control message using the Control push buttons that matches the control message programmed into the load control device. This is the 50 bit message that causes the relay to control.

  9. Set a restore message using the Restore push buttons that matches the restore message programmed into the load control device. This is the 50 bit message that causes the relay to restore.

  10. Click the Finish button. The new load group appears in the Load Group Editor.
