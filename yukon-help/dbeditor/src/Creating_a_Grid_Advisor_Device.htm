Creating a Grid Advisor Device

Perform the following steps to create a Grid Advisor device:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select Grid Advisor from the list on the left.

  4. Select the type of Grid Advisor device to create from the list on the right:

    • Faulted Circuit Indicator

    • Capacitor Bank Neutral Monitor

  5. Enter a name for the Grid Advisor device in the Name text box.

  6. Select the comm channel for the Grid Advisor device from the Communication Channel drop-down list.

  7. Click the Finish button. The new Grid Advisor device is displayed in the Device Editor. For instructions on editing a Grid Advisor device, refer to the related topics.
