Editing a Customer

A customer is a commercial or industrial user who has signed up for automated meter reading, load management, or capacitor control.

 

Perform the following steps to edit a customer:

 

  1. From the View menu, select System.

  2. From the Sort By list, select CI Customers.

  3. Double-click on the customer to edit. The General tabbed page is displayed in the Customer Editor, from which the following changes can be made:

    • The company name can be changed in the Company Name text box.

    • The home directory can be changed in the Home Directory text box. This is the directory in which customer information is stored.

    • A different owner energy company can be selected from the Owner Energy Company drop-down list.

    • A different primary contact can be selected from the Primary Contact drop-down list. To add a new primary contact, click the New Contact button.

    • A new customer time zone can be selected in the Time Zone drop-down list.

    • The tracking number for the customer can be changed in the Customer Number text box.

  4. Click the Address tab to display the Address tabbed page, from which the following changes can be made:

    • The street address of the customer can be changed in the Address text boxes.

    • The city in which the customer is located can be changed in the City text box.

    • The county in which the customer is located can be changed in the County text box.

    • The state in which the customer is located can be selected from the State drop-down list.

    • The zip code of the customer can be changed in the Zip text box.

  5. Click the Contacts tab to display the Contacts tabbed page, from which contacts can be assigned to or unassigned from the customer.

    • To assign an additional contact, select the contact from the Available Additional Contacts drop-down list and click the Add button.

    • To unassign an additional contact, select the contact from the Assigned Contacts list and click the Remove button.

  6. Click the Trends tab to display the Trends tabbed page, from which trends can be assigned to or unassigned from the customer.

    • To assign a trend, select the trend from the Available Trends list and click the Add button. To select more than one trend, hold the Ctrl key on the keyboard while clicking additional trends.

    • To unassign a trend, select the trend from the Assigned Trends list and click the Remove button. To select more than one trend, hold the Ctrl key on the keyboard while clicking additional trends.

  7. Click the Meters tab to display the Meters tabbed page, from which meters can be assigned to or unassigned from the customer.

    • To assign a meter, select the meter from the Unowned Meters list and click the Add button. To select more than one meter, hold the Ctrl key on the keyboard while clicking additional meters.

    • To unassign a meter, select the meter from the Owned Meters list and click the Remove button. To select more than one meter, hold the Ctrl key on the keyboard while clicking additional meters.

  8. Click the Points tab to display the Points tabbed page, from which points can be added to or removed from the customer. To add a point to a customer:

    1. Click the Add button.

    2. Select the type of point from the Type drop-down list:

      • Demand – Level to which a customer is expected to reduce their load.

      • Settlement – Default amount of curtailment for a customer.

      • Baseline – Baseline of usage.

      • Curtailable – The default load value for this customer that is displayed during event creation. The value is replaced with the actual load when a metering point is assigned for this customer.

      • ContractFrmDemand – The amount of load that customer will be using during an interruption. This value is established by the utility and C&I customer during contract negotiations. If load profile data is available from the meter, the data is shown to the customer to illustrate the level they must achieve during an interruption.

      • AdvBuyThroughkW – Default purchase amount in an economic event if the event price is equal to or below the cost parameter.

      • AdvBuyThrough$ – Strike price for the C&I customer. If the economic event price is less than or equal to this value, the customer automatically elects to purchase through the event instead of interrupting.

      • InterruptHours – Total number of hours the C&I customer can be interrupted in a year.

      • MinimumNotice – Minimum notification time required to include a customer in an event.

      • ContractIntLoad – Amount of load expected from the customer during the contract season. This value is calculated by the utility based on the rules of the tariff. Its primary use is in settlement. Operators can use this value to plan the expected response of an individual customer, but the best indicator is the Current Load parameter if live data is being collected from the meter.

      • CurrentLoad – Used if there is a live meter data point for the customer. This value shows the operators how much load reduction can be expected in an event and allows the operator to verify that the customer has interrupted their load.

      • MinEventDuration – The minimum time that a customer can be included in an event. The parameter is used only when the operator chooses to remove customers from an event prior to the event’s scheduled completion.

      • InterruptHrs24Hr – The maximum number of hours that a customer can be included in all events in a 24 hour period. This parameter is checked at event creation time and the customer will be excluded if their actual or currently scheduled control hours exceed this parameter. An error message is provided.

    3. Enter a name for the point in the Label text box.

    4. Select the device on which the point is located from the Device drop-down list.

    5. Select the point to add to the customer from the Point drop-down list.

To remove a point from the customer, click the Remove button to the right of the Type drop-down list for the point to remove.

  1. Click the Apply button to save the changes to the customer or click the OK button to save the changes to the customer and close the Customer Editor.
