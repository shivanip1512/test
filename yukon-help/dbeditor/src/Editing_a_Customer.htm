Editing a Customer

A customer is a commercial or industrial user who has signed up for automated meter reading, load management, or capacitor control.

 

To edit a customer:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Customers.

  3. Double-click on the customer modify. This opens the General tabbed page of the Customer editor, from which the following changes can be made:

    1. The company name can be changed in the Company Name text box.

    2. The home directory can be changed in the Home Directory text box. This is the directory in which customer information is stored.

    3. A different energy company can be selected from the Owner Energy Company drop-down list.

    4. A different primary contact can be selected from the Primary Contact drop-down list. To add a new primary contact, click the New Contact button.

    5. A new customer time zone can be selected in the Time Zone drop-down list.

    6. The tracking number for the customer can be changed in the Customer Number text box.

  4. Click the Address tab to display the Address tabbed page, from which the following changes can be made:

    1. The address of the customer can be changed in the Address text boxes.

    2. The city in which the customer is located can be changed in the City text box.

    3. The county in which the customer is located can be changed in the County text box.

    4. The state in which the customer is located can be selected from the State drop-down list.

    5. The zip code of the customer can be changed in the Zip text box.

  5. Click the Contacts tab to display the Contacts tabbed page appears, from which contacts can be assigned to or unassigned from the customer.

    • To assign an additional contact, select the contact from the Available Additional Contacts drop-down list and click the Add button.

    • To unassign an additional contact, select the contact from the Assigned Contacts list and click the Remove button.

  6. Click the Trends tab to display the Trends tabbed page appears, from which trends can be assigned to or unassign from the customer.

    • To assign a trend, select the trend from the Available Trends list and click the Add button. To select more than one trend, hold Ctrl key on the keyboard while clicking additional trends.

    • To unassign a trend, select the trend from the Assigned Trends list and click the Remove button. To select more than one trend, hold the Ctrl key on the keyboard while clicking additional trends.

  7. Click the Meters tab to display the Meters tabbed page appears, from which meters can be assigned to or unassign from the customer.

    • To assign a meter, select the meter from the Unowned Meters list and click the Add button. To select more than one meter, hold Ctrl key on the keyboard while clicking additional meters.

    • To unassign a meter, select the meter from the Owned Meters list and click the Remove button. To select more than one meter, hold the Ctrl key on the keyboard while clicking additional meters.

  8. Click the Points tab to display the Points tabbed page appears, from which points can be assigned to the customer. To assign a point to a customer:

    1. Click the Add button.

    2. Select the type of point from the Type drop-down list:

      • Demand Level Point - level to which a customer is expected to reduce their load.

      • Settlement Point - default amount of curtailment for a customer.

      • Baseline - baseline of usage.

      • Curtailable - level to which a customer is expected to curtail their load.

      • ContractFrmDemand - Amount of load that customer will be using during an interruption.

      • AdvBuyThroughkW - Default purchase amount in an economic event if the event price is equal to or below the cost parameter.

      • AdvBuyThrough$ - Strike price for C&I customer.

      • InterruptHours - Total number of hours the C&I customer can be interrupted in a year.

      • MinimumNotice - Minimum notification time required to include a customer in an event.

      • ContractIntLoad - Amount of load expected from the customer during the contract season.

      • CurrentLoad - If this is a live meter data, this value shows the operators how much load reduction can be expected in an event and allows the operator to verify that the customer has interrupted their load.

    3. Enter a name for the point in the Label text box.

    4. Select the device on which the point is located from the Device drop-down list.

    5. Select the point to add to the customer from the Point drop-down list.

  9.  Click the Apply button to save the changes or OK button to save the changes and close the Customer Editor.
