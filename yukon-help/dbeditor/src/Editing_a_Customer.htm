Editing a Customer

To edit a customer:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Customers.

  3. Double-click on the customer modify. This opens the General tabbed page of the Customer editor, from which you may make the following changes:

    1. The company name can be changed in the Company Name text box.

    2. The home directory can be changed in the Home Directory text box. This is the directory in which customer information is stored.

    3. A different energy company can be selected from the Owner Energy Company drop-down list.

    4. A different primary contact can be selected from the Primary Contact drop-down list. To add a new primary contact, click the New Contact button.

    5. A new customer time zone can be selected in the Time Zone drop-down list.

    6. The tracking number for the customer can be changed in the Customer Number text box.

  4. Click the Address tab. The Address tabbed page appears, from which you may make the following changes:

    1. The address can be changed in the Address text boxes.

    2. The city can be changed in the City text box.

    3. The county can be changed in the County text box.

    4. A different state can be selected from the State drop-down list.

    5. The zip code can be changed in the Zip text box.

  5. Click the Contacts tab. The Contacts tabbed page appears, from which you may assign contacts to or unassign contacts from the customer.

    • To assign an additional contact, select the contact from the Available Additional Contacts drop-down list and click Add.

    • To unassign an additional contact, select the contact from the Assigned Contacts list and click Remove.

  6. Click the Trends tab. The Trends tabbed page appears, from which you may assign trends to or unassign trends from the customer.

    • To assign a trend, select the trend from the Available Trends list and click Add. To select more than one trend, hold Ctrl key on the keyboard while clicking additional trends.

    • To unassign a trend, select the trend from the Assigned Trends list and click Remove. To select more than one trend, hold the Ctrl key on the keyboard while clicking additional trends.

  7. Click the Meters tab. The Meters tabbed page appears, from which you may assign meters to or unassign meters from the customer.

    • To assign a meter, select the meter from the Unowned Meters list and click Add. To select more than one meter, hold Ctrl key on the keyboard while clicking additional meters.

    • To unassign a meter, select the meter from the Owned Meters list and click Remove. To select more than one meter, hold the Ctrl key on the keyboard while clicking additional meters.

  8. Click the Points tab. The Points tabbed page appears, from which you can select a demand level point and settlement point for the customer.

    • A demand level point is the level to which a customer is expected to reduce their load.

    • A settlement point is the default amount of curtailment for a customer.

  9. Click Apply to save the changes or OK to save the changes and close the dialog box.
