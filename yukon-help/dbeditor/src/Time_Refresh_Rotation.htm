Time Refresh or Rotation Gear

To add a Time Refresh or Rotation gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select either Time Refresh or Rotation from the Gear Type drop-down list.

  3. For a Time Refresh gear, select the a shed type from the Refresh Shed Type drop-down list.

  4. Select the length of the shed from the Shed Time drop-down list.

    1. If the refresh shed type is Fixed, select a shed time from the Fixed Shed Time drop-down list.

    2. If the refresh shed type is Dynamic, select a max shed time in the Max Shed Time drop-down list.

  5. Select the number of groups from the # of Groups Each Time drop-down list. For a time refresh gear, this is the number of groups that are shed for each refresh. For a rotation gear, the is the number of groups that are shed during each rotation.

  6. For a Time Refresh gear, select a refresh rate from the Refresh/Resend Rate drop-down list. The Refresh/Resend Rate is the rate at which the shed command is refreshed or resent.

For a Rotation gear, select a resend rate from the Command Resend Rate drop-down list. The resend rate is the rate at which the shed command is refreshed or resent.

  1. Select a group selection method from the Group Selection Method drop-down list. The group selection methods are:

    • Last Controlled - starts with the next group based on the last one controlled.

    • Always First Group - always starts with the first group in the list.

    • Least Control Time - starts with the group whose seasonal control time is the least.

  2. To ramp in a percentage of the load groups over an interval of time, select the Ramp In check box. Enabling the Ramp In feature, disables the # of Groups Each Time feature.

    1. Enter the percentage of the load groups to ramp in at each interval in the Ramp In Percent text box.

    2. Enter the interval in seconds to ramp in a percentage of the load groups in the Ramp In Interval text box.

  3. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

    • Ramp Out/Time In - The control ramps out when the shed time is over.

    • Ramp Out/Restore - The control ramps out when a restore command is sent to the device.

  4. If Ramp Out/Time In or Ramp Out/Restore is selected from the How to Stop Control drop-down list, select the order to stop the control from the Stop Order drop-down list. The orders for stopping control are:

    • Random - The system selects a random load group to stop.

    • First In First Out - The first group that was controlled is the first group to stop. Then the second group that was controlled is the second group to stop, etc.

  5. If Ramp Out/Time In or Ramp Out/Restore is selected from the How to Stop Control drop-down list:

    1. Enter the percentage of the load groups to ramp out at each interval in the Ramp Out Percent text box.

    2. Enter the interval in seconds to ramp out a percentage of the load groups in the Ramp Out Interval text box.

  6. Set a group capacity reduction percentage in the Group Capacity Reduction % spin box. The percentage of each groups capacity is used to determine how much load to control.

  7. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear switches only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear switches when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear switches when the control area priority reaches this priority.

    • Above Trigger - The Trigger Number spin box and Trigger Offset text box appear. This change method requires the control area’s threshold triggers to be created before it can be used.  Select whether to use trigger one or two in the Trigger Number spin box. (Trigger one is the first entry and trigger two is the second entry of the Control Area’s Trigger drop-down list.) Enter the offset from the trigger’s threshold in the Trigger Offset text box.

  8. Click OK to save the gear.
