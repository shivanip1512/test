Creating a Season Schedule

A season schedule allows the seasons during which load control can occur to be defined in the Yukon database. A season schedule contains the start and end dates of one or more load control seasons. Once a season schedule is defined, it can be assigned to the program constraint of a load program.

 

Note: Do not define seasons during which load control should not occur.

 

Perform the following steps to create a load control season schedule:

 

  1. From the View menu, select System.

  2. From the Create menu, select Season Schedule. The Seasons Schedule Setup dialog box appears.

  3. Enter a name for the schedule in the Schedule Name text box.

  4. Create, edit, or remove load control seasons in the Season Dates area.

    • To create a season, see Creating a Season.

    • To edit a season, see Editing a Season.

    • To remove a season, see Removing a Season.

  5. Click the Finish button. The new season schedule appears in the Season Schedule Editor. For instructions on editing a season schedule, refer to the related topics.
