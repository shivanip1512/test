Creating a Season Schedule

To create a load control season schedule:

 

  1. From the View menu, select System.

  2. From the Create menu, select Season Schedule. The Seasons Schedule Setup dialog box appears.

  3. Enter a name for the schedule in the Schedule Name text box.

  4. Create, edit, or remove load control seasons in the Season Dates area.

    • To create a season, see Creating a Season.

    • To edit a season, see Editing a Season.

    • To remove a season, see Removing a Season.

  5. Click Finish. The new season schedule appears in the Season Schedule Editor. For instructions on editing a season schedule, refer to the related topics.
