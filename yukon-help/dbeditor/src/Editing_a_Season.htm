Editing a Season

To edit a load control season:

 

  1. Select the season to edit from the Assigned Seasons list.

  2. Click Edit.

  3. The name of the season can be changed in the Season Name text box.

  4. A different month can be selected from the Month drop-down list.

  5. A different day can be selected from the Day drop-down list.

  6. A different year can be selected from the Year drop-down list. Select (none) to make this holiday repeat each year.

  7. Under Start Date, a different month and day can be selected from the Month and Day drop-down lists as the start of the load control season.

  8. Under End Date, a different month and day can be selected from the Month and Day drop-down lists as the end of the load control season.

  9. Click OK to save the changes.
