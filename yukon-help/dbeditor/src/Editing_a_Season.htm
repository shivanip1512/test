Editing a Season

To edit a load control season:

 

  1. Select the season to edit from the Assigned Seasons list.

  2. Click the Edit button. The season information is displayed in the Edit Season Date window.

  3. The name of the load control season can be changed in the Season Name text box.

  4. A different start date for the load control season can be selected from the Month and Day drop-down lists in the Start Date area.

  5. A different end date for the load control season can be selected from the Month and Day drop-down lists in the End Date area.

  6. Click the OK button to save the changes.
