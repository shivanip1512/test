VERSACOM

If you selected a VERSACOM device:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of the switches in the load group. To disable the kilowatt capacity feature, enter a value of zero.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Disable Control check box. Select this check box to disable control. Clear this check box to enable control.

  5. Select the route to use for the load group from the Communication Route drop-down list. This is the route or route macro used to reach all of the devices in this group when transmitting commands.

  6. Click the Next button.

  7. Set the utility address in the Utility Address spin box. A utility address is used to identify the utility and optional region or member system.

  8. Set the section address in the Section Address spin box. A section address is a secondary addressing method based on a device’s substation, bus, or geographic region.

  9. Use the push buttons to select a Class Address. A class address identifies a customer according to a utility classification scheme or type of load.

  10. Use the push buttons to select a Division Address. A division address provides finer control resolution, or identifies particular sub-groupings of load type.

  11. Select the address levels to use when commands are sent in the Address Usage area. The Utility address is required and is selected by default.
    To control a device using its individual address, select the Serial Address check box and enter the physical address of the device in the Serial Address text box.

  12. Select the check boxes of the relays to control for this group in the Relay Usage area.

  13. Click the Finish button. The new load group appears in the Load Group Editor. For instructions on editing a load group, refer to the related topics.
