Creating an RFN

Perform the following steps to create an RFN:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select RFMESH from the list on the left.

  4. Select the type of RFN to create from the list on the right:

    • RFN-410fL – Landis+Gyr FOCUS®

    • RFN-410fX – Landis+Gyr FOCUS AX

    • RFN-410fD – Landis+Gyr FOCUS AX-SD

    • RFN-430A3 – Elster A3 ALPHA®

    • RFN-430kV – GE kV2c

  5. Click the Next button.

  6. Enter a name for the RFN in the Device Name text box.

  7. Click the Next button.

  8. Enter the meter number of the RFN in the Meter Number text box. This meter number is assigned by the utility.

  9. Click the Next button.

  10. Enter the serial number of the RFN in the Serial Number text box.

  11. Enter the name of the manufacturer, for the type of RFN selected in step 4, in the Manufacturer text box The name of the manufacturer must be entered exactly as shown in the following table:

  1. To create the default points for the RFN, select the Create Default Points check box. To prevent the default points from being created for the RFN, clear this check box.

  2. Click the Finish button. The new RFN is displayed in the Device Editor. For instructions on editing an RFN, refer to the related topics.
