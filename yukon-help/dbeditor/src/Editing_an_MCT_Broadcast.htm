Editing an MCT Broadcast

The MCT Broadcast Editor contains a General tab and an MCT Assignment tab.

 

To modify an MCT Broadcast group:

 

  1. Select Core from the View menu.

  2. From the Sort By list, select MCT Broadcast.

  3. Double-click the MCT Broadcast group to display the General tab of the MCT Broadcast editor.

  4. Click the MCT Assignment tab to display the MCT Assignment tabbed page.

  5. Click Apply to save the changes or OK to save the changes and close the Device Editor.
