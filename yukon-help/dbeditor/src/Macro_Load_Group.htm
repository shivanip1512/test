Macro Load Group

Perform the following steps to create a macro load group:

 

  1. Enter a name for the macro load group in the Group Name text box.

  2. Click the Next button.

  3. Assign load groups to or unassign load groups from the Macro Load Group.

    • To assign a load group to the macro load group, select the load group from the Available list and click the Add button. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional load groups.

    • To unassign a load group from the macro load group, select the load group from the Assigned list and click the Remove button. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional load groups.

  4. Click the Finish button. The new macro load group appears in the Load Group Editor.
