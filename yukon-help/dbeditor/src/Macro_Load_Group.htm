Macro Load Group

If you selected Macro Load Group:

 

  1. Enter a name for the macro load group in the Group Name text box.

  2. Click Next.

  3. You may assign load groups to or unassign load groups from the Macro Load group.

    • To assign a load group, select the group from the Available list and click Add. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.

    • To unassign a load group, select the group from the Assigned list and click Remove. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.

  4. Click Finish. The new macro load group appears in the Device Editor. For instructions on editing a macro load group, refer to the related topics.
