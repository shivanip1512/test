Creating a Device

To create a Device:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select a device category from the list on the left.

  4. Select a specific device from the list on the right.

  5. Click the Next button.

  6. The rest of the procedure varies depending on the type of device selected.

    • Signal Transmitter

    • Electronic Meter

    • RTU

    • Virtual

MCTs and Repeaters cannot be created until routes have been defined.

    • MCT Broadcast
