Load Program Notification Tab

The following changes can be made to the notification settings of the load program:

  • To send out a notification message before a load program starts, select the Program Start check box and enter how many minutes before the load program starts to send the message. If this check box is not selected, a notification message is not sent out when the load program starts.

  • To send out a notification message after the load program stops, select the Program Stop check box and enter how many minutes after the load program stops to send the message. If this check box is not selected, a notification message is not sent out when the load program stops.

  • To assign a notification group to the load program, select the notification group from the Notification Groups list and click the Add button. To select more than one notification group, hold Ctrl key on the keyboard while clicking additional notification groups.

  • To unassign a notification group from the load program, select the notification group from the Assigned Groups list and click the Remove button. To select more than one notification group, hold the Ctrl key on the keyboard while clicking additional notification groups.
