UDP Terminal Server

If you selected UDP Terminal Server:

 

  1. Select one of the following communication line options:

    • Dedicated - communication line is connected to a leased line or data radio connection.

    • Dialup - communication line is connected to a dial-up modem.

  2. Click Next.

  3. Enter a name for the comm channel in the Description text box. This name is used by the system when it communicates error and informational messages.

  4. Enter the port number in the Port Number text box. This is the port on which the system listens for data from the two-way CBCs.

  5. Select the baud rate of the port from the Baud Rate drop-down list. This is the communications line speed used by this port.

  6. The rest of this procedure depends on whether you selected Dedicated or Dialup as the type of communication line.

  • If you selected Dedicated, click Finish. The new communication channel appears in the Comm Channel Editor. For instructions on editing a communication channel, refer to the related topics.

  • If you selected Dialup, click Next, select the type of modem being used from the drop-down list of modems, and then click Finish. The new communication channel appears in the Comm Channel Editor. For instructions on editing a communication channel, refer to the related topics.
