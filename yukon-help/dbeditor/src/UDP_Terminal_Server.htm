UDP Terminal Server

If you selected UDP Terminal Server:

 

  1. Enter a name for the comm channel in the Description text box. This name is used by the system when it communicates error and informational messages.

  2. Enter the port number in the Port Number text box. This is the port on which the system listens for data from the two-way CBCs.

  3. Select the baud rate of the port from the Baud Rate drop-down list. This is the communications line speed used by this port.

  4. Click the Finish button. The new communication channel appears in the Comm Channel Editor.
