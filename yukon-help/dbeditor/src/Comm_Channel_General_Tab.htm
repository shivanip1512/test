Comm Channel General Tab

The following changes can be made to the general settings of the communication channel:

 

  1. The name of the comm channel can be changed in the Description text box. This comm channel name that the Yukon system uses when it communicates error and informational messages.

  2. If the comm channel type is Local Serial Port, Local Dialup, or Local Dialback, a different port can be selected from the Physical Port drop-down list. This is the local hardware port used by the Yukon system.

If the comm channel type is Terminal Server, the IP address and port number of the terminal server can be changed in the IP Address and Port Number text boxes.

If the comm channel type is UDP, the port number can be changed in the Port Number text boxes. This is the port on which the Yukon system listens for data from two-way CBCs.

  1. A different baud rate can be selected from Baud Rate drop-down list. This is the communication line speed of the comm channel.

  2. If comm channel is used to communicate with a CCU, a different protocol wrap can be selected from the Protocol Wrap drop-down list.

  • If comm channel is used to communicate with a CCU-711, select IDLC from the drop-down list, unless directed otherwise.

  • If comm channel is used to communicate with a CCU-710, select None from the drop-down list.

  • If comm channel is used to communicate with an LCU, TCU, or RTU, select IDLC from the drop-down list only if the firmware of device supports it.
    If any of these devices share a comm channel with a CCU-711, all of the devices on the comm channel must use IDLC.

  1. The carrier detect wait option can be enabled or disable by selecting or clearing the Carrier Detect Wait check box. The carrier detect wait option is disabled by default.

If the Carrier Detect Wait check box is selected, the number of milliseconds to wait for carrier detect can be changed in the Wait for carrier detect text (DCD) text box.

  1. If the comm channel type is UDP and the comm channel is for a CBC with a Lantronix module that uses AES-CBC 128 encryption, the communication with the CBC can be encrypted. To encrypt the communication with the CBC:

    1. Select the Enable Encryption check box

    2. Enter an encryption key, which is entered as 32 hexadecimal characters, in the Key (Hex) text box.

  2. The comm channel can be enabled or disable by selecting or clearing the Disable Channel check box. Select this check box to disable the channel if the communication channel is temporarily not functioning.
