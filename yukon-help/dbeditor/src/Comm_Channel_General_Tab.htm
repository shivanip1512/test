Comm Channel General Tab

The following changes can be made to the general settings of the communication channel:

 

  1. The name of the comm channel can be changed in the Description text box.

  2. If the comm channel type is Local Port, a different port can be selected from the Physical Port drop-down list. This is the local hardware port used by the software.

    If the comm channel type is TCP Terminal Server, the IP address and port number of the terminal server can be changed in the IP Address and Port Number text boxes.

    If the comm channel type is UDP Terminal Server, the port number can be changed in the Port Number text boxes. This is the port on which the Yukon system listens for data from two-way CBCs.

  3. A different baud rate can be selected from Baud Rate drop-down list. This is the communications line speed used by the port

  4. A different protocol wrap can be selected from the Protocol Wrap drop-down list. If the comm channel has a CCU-711 attached to it, select IDLC, unless directed otherwise. Select None for CCU-710s. RTUs, TCUs, and LCUs may not use IDLC depending upon the firmware in these devices. If any of these devices share a communication line with a CCU-711, both devices must use IDLC.

  5. The carrier detect wait option can be enabled or disable by selecting or clearing the Carrier Detect Wait check box. The carrier detect wait option is disabled by default.

    If the Carrier Detect Wait check box is selected, the number of milliseconds to wait for carrier detect can be changed in the Wait for carrier detect text (DCD) text box.

  6. The comm channel can be enabled or disable by selecting or clearing the Disable Channel check box. Select this check box to disable the channel if the communication channel is temporarily not functioning.
