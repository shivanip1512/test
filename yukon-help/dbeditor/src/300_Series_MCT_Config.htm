300 Series MCT Config

If you selected 300 Series MCT as the config type:

  1. Enter a name for the config in the Configuration Name text box.

  2. Select which mode to assign MCT:

    • On-Peak/Off-Peak Mode - records the maximum demand readings for either the On-Peak or Off-Peak interval.

    • Min/Max Mode - records the minimum and maximum demand readings.

  3. Channel 1 is enabled by default. To enable additional channels, select the Channel 2 or Channel 3 check boxes.

  4. For each channel enabled, complete the following:

    • Select whether the channel is wired for 3-Wire (KYZ) or 2-Wire (KY).

    • If the Ke value is not known, enter the Mp and Kh values into the appropriate text boxes and click the = button to calculate Ke.

    • If the Ke value is known, enter it into the Ke text box.

  5. Click Finish. The new MCT Config appears in the Config Editor.
