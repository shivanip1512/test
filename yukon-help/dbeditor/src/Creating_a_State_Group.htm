Creating a State Group

To create a State Group:

 

  1. From the View menu, select Core.

  2. From the Create menu, select State Group.

  3. Enter a name for the state group in the State Group Name text box.

  4. Set the number of states in the Number of States spin box. There can be up to 10 states.

  5. Click Finish. The new state group appears in the State Editor. For instructions on editing a state group, refer to the related topics.
