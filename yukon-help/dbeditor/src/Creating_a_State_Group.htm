Creating a State Group

State groups contain message text and color-codes for the possible states of a point.

 

Perform the following steps to create a state group:

 

  1. From the View menu, select Core.

  2. From the Create menu, select State Group.

  3. Select the type of state group to create:

    • Status State Group – State group used by status points and calculated points with a status output.

    • Analog State Group – State group used by analog, accumulator, and calculated points with an analog output.

  4. Enter a name for the state group in the State Group Name text box.

If status state group was selected in step 3, enter the number of states in the Number of States box.

  1. Click the Finish button. The new state group appears in the State Editor. For instructions on editing a state group, refer to the related topics.
