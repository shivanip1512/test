Editing a Device

To edit a device:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select one of the following:

    • Device

    • IED

    • MCT

    • Meter

    • RTU

    • Transmitter

  3. Double-click a Device. This opens the General tabbed page of the Device Editor.

  4. If the device is an electronic meter or MCT, click the Metering tab to display the Metering tabbed page.

  5. Click the Scan Rate tab to display the Scan Rate tabbed page.

  6. If the device is an MCT-410, click the MCT Settings tab to display the MCT Settings tabbed page.

  7. If the device is a Paging Tap Terminal, SNPP Terminal, or WCTP Terminal; click the Paging tab to display the Paging tabbed page.

  8. If the device is an MCT-360 or MCT-470, click the IED Port tab to display the IED Port tabbed page.

  9. Click the Apply button to save the changes or OK button to save the changes and close the Device Editor.
