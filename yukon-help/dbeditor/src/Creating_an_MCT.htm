Creating an MCT

Before you can create an MCT, you must create a route.

 

If you selected MCT as the device type:

 

  1. Enter a name for the MCT in the Device Name text box.

  2. Enter the physical address of the MCT in the Physical Address text box. For the MCT-470, MCT-370, MCT-360, and MCT-318 this is the address located on the inside of the door. For the MCT-430 and MCT-410, this is the address located on the label directly above or below the meter’s LCD. For the MCT-310 and MCT-200 series, this is the number on the faceplate, minus the leading 1 and 0s. For instance, if the faceplate number on an MCT-310i is 100020464, enter 20464 in this field.

  3. To create the default points for the MCT, select the Create default points check box.

  4. Click the Next button.

  5. Enter the meter number in the Meter Number text box. This meter number is assigned by the utility.

  6. Click the Next button.

  7. Select the check box of the scan categories you want to enable. The scan categories vary depending on the type of MCT selected. The categories are:

    • Demand & Status Rate - collects demand and status point values

    • Demand & Voltage Rate - collects demand and voltage point values

    • Demand Rate - collects demand point values

    • Status Rate - collects status point values

    • Periodic Health Check - periodically checks that the device is alive

    • Accumulator (Energy) Rate - collects kWh point values

    1. Select the default scan interval from the Interval drop-down list.

    2. Select the alternate scan interval from the Alt Interval drop-down list. The alternate scan interval can be activated using the Tabular Data Console or the Yukon Web interface and is used during the “control timeout” period to collect the state of a control point.

    3. Select a scan group from the Group drop-down list. This is used to group similar scans together.

  8. Under the Optional Scan Window, select the Enabled check box if you want to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the time at which the scan window starts in the Open text box.

    3. Enter the time at which the scan window stops in the Close text box.

  9. Click the Next button.

  10. Select the communication route to the MCT from the drop-down list of routes.

  11. Click the Finish button. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.
