Creating an MCT

Before you can create an MCT, you must create a route for the MCT.

 

If you selected MCT as the device type:

 

  1. Enter a name for the MCT in the Device Name text box.

  2. Enter the physical address of the MCT in the Physical Address text box. For the MCT-470, MCT-370, MCT-360, and MCT-318 this is the address located on the inside of the door. For the MCT-430 series and MCT-410 series, this is the address located on the label directly above or below the meter’s LCD. For the MCT-310 and MCT-200 series, this is the serial number on the faceplate, minus the leading 1 and 0s. For instance, if the serial number on an MCT-310i is 100020464, enter 20464 in this field.

  3. To create the default points for the MCT, select the Create default points check box.

  4. Click Next.

  5. Enter the meter number in the Meter Number text box. This meter number is assigned by the utility.

  6. Click Next.

  7. Select the check box of the scan categories you want to enable. The scan categories vary depending on the type of MCT selected. The categories are:

    • Demand & Status Rate - collects demand and status point values

    • Demand & Voltage Rate - collects demand and voltage point values

    • Demand Rate - collects demand point values

    • Status Rate - collects status point values

    • Periodic Health Check - periodically checks that the device is alive

    • Accumulator (Energy) Rate - collects kWh point values

    1. Select the scan interval and alternative scan interval from the Interval and Alt Interval drop-down lists.

    2. Select a scan group or enter a new scan group in the Group drop-down list. This is used to group similar scans together.

  8. Under the Optional Scan Window, select the Enabled check box if you want to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the open and close times in the Open and Close text boxes.

  9. Click Next.

  10. Select the route used with the device from the drop-down list of routes.

  11. Click Finish. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.
