Creating an MCT

Before an MCT can be created, one of more communication routes must be created.

 

Perform the following steps to create an MCT:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select MCT from the list on the left.

  4. Select the type of MCT to create from the list on the right:

    • MCT-470 – Surface-mount MCT with four sets of KYZ pulse inputs and two Form C control outputs.

    • MCT-430A – MCT-430 meter based on the ABB/Elster ALPHA Plus meter.

    • MCT-430S4 – MCT-430 meter based on the Landis+Gyr S4/S4e meter.

    • MCT-430SL – MCT-430 meter based on the Itron SENTINEL meter.

    • MCT-430A3 – MCT-430 meter based on the ABB/Elster A3 ALPHA meter.

    • MCT-420fL – MCT-420 meter based on the Landis+Gyr FOCUS meter.

    • MCT-420fLD – MCT-420 meter based on the Landis+Gyr FOCUS meter remote service disconnect.

    • MCT-420cL – MCT-420 meter based on the Itron CENTRON meter.

    • MCT-420cLD – MCT-420 meter based on the Itron CENTRON meter with remote service disconnect.

    • MCT-410iL – MCT-410 meter based on the Senus iCon meter.

    • MCT-410cL – MCT-410 meter based on the Itron CENTRON meter.

    • MCT-410fL – MCT-410 meter based on the Landis+Gyr FOCUS meter.

    • MCT-370 – Surface-mount MCT with three sets of KYZ pulse inputs and two Form C control outputs.

    • MCT-360 – Surface-mount MCT with one set of KYZ pulse inputs and one Form C control output.

    • MCT-318L – Surface-mount MCT with three sets of KYZ pulse inputs, two dedicated status inputs, and two control outputs.

    • MCT-310CT – MCT-310 with a current transformer.

    • MCT-310IDL – MCT-310 meter with a remote service disconnect and additional memory for load profile applications.

    • MCT-310IL – MCT-310 meter with additional memory for load profile applications.

  5. Click the Next button.

  6. Enter a name for the MCT in the Device Name text box.

  7. Enter the physical address of the MCT in the Physical Address text box. For the MCT-470, MCT-370, MCT-360, and MCT-318, this is the address located on the inside of the door. For the MCT-430, MCT-420, and MCT-410, this is the address located on the label directly above or below the meter’s LCD. For the MCT-310 meter, this is the number on the faceplate, minus the leading 1 and 0s. For instance, if the faceplate number on an MCT-310i is 100020464, enter 20464 in this field.

  8. To create the default points for the MCT, select the Create Default Points check box.

  9. Click the Next button.

  10. Enter the meter number of the MCT in the Meter Number text box. This meter number is assigned by the utility.

  11. Click the Next button.

  12. Select the check box of the scan categories to enable. The scan categories for an MCT are:

    • Demand & Status Rate – Collects demand and status point values.

    • Accumulator (Energy) Rate – Collects kWh point values.

When the check box of a scan category is selected, the following scan options are enabled:

    • Interval – Default scan interval of the selected scan category.

    • Alt Interval – Alternate scan interval of the selected scan category.

    • Group – Scan group of the selected scan category. The scan group is used to assign priorities to the scan categories.

  1. Under the Optional Scan Window, select the Enabled check box to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the time at which the scan window starts in the Open text box.

    3. Enter the time at which the scan window stops in the Close text box.

  2. Click the Next button.

  3. Select the communication route for the MCT from the Select the route used with this device drop-down list.

  4. Click the Finish button. The new MCT is displayed in the Device Editor. For instructions on editing an MCT, refer to the related topics.
