Control Scenario General Tab

The following changes can be made to the general settings of a control scenario:

 

  1. The name for the control scenario can be changed in the Scenario Name text box.

  2. To add a load program to the control scenario, select a load program from the Available Programs list and click the Assign to Scenario button.
    Note:     A load program must be assigned to a control area to appear in the Available Programs list.
    To remove a load program that is assigned to the control scenario, select a load program from the Assigned Programs list and click the Remove from Scenario button.

  3. To specify a start offset for a load program, click in the Start Offset field of the assigned program and enter a time offset in mm:ss format. When a control scenario starts, this is the time in minutes and seconds that the load program must wait before it can start.

  4. To specify a stop offset for a load program, click in the Stop Offset field of the assigned program and enter a time offset in mm:ss format. When a control scenario stops, this is the time in minutes and seconds that the load program must wait before it can stop.

  5. To select a different gear with which to start control for a load program, click in the Start Gear field of the assigned program and select a gear from the Start Gear drop-down list.
