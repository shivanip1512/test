Control Scenario General Tab

The following changes can be made to the general settings of a control scenario:

  1. The name for the control scenario can be changed in the Scenario Name text box.

  2. To add a load program to the control scenario, select the load program to add from the Available Programs area and click the Assign to Scenario button. Load programs must belong to a control area to appear in this list.

  3. To give a load program a start offset, click in the program's Start Offset field and enter a time offset in mm:ss format. When a control scenario starts, this is the time in minutes and seconds that the load program must wait before it can start.

  4. To give a load program a stop offset, click in the program's Stop Offset field and enter a time offset in mm:ss format. When a control scenario stops, this is the time in minutes and seconds that the load program must wait before it can stop.

  5. To select a different gear to start control with, click in the program's Start Gear field and select the gear.

  6. To remove a program assigned to the scenario, select the program from the Assigned Programs are and click the Remove from Scenario button.
