Device Metering Tab

The following changes can be made to the metering settings of the device:

 

  1. The meter number can be changed in the Meter Number text box. The meter number is assigned by the utility.

  2. A different data collection group can be entered or selected in the Data Collection Group drop-down list.

  3. A different alternate group can be entered or selected in the Alternate Group drop-down list.

  4. A different billing group can be entered or selected in the Billing Group drop-down list.

  5. A different custom collection group can be entered or selected in the Custom Group 1 drop-down list.

  6. A different custom collection group can be entered or selected in the Custom Group 2 drop-down list.

  7. A different custom collection group can be entered or selected in the Custom Group 3 drop-down list.

  8. If the device is an MCT, a different demand interval can be selected from the Last Interval Demand Rate drop-down list.

  9. If the device can collect voltage readings, a different voltage averaging interval can be selected from the Voltage Averaging Interval drop-down list.

  10. If the device can perform Load Profile Collection, the interval at which load profile data is collected can be selected from the Load Profile Demand Rate drop-down list. The channels from which to collect data (Channel 1, Channel 2, Channel 3, etc.) must be selected using the check boxes.

  11. If the device is an MCT-470, the intervals at which to collect load profile data can be selected from the Load Profile Rate #1 and Load Profile Rate #2 drop-down lists. The channels from which to collect data (Channel 1, Channel 2, Channel 3, etc.) must be selected using the check boxes.

  12. If the device can perform Voltage Profile Collection, the interval at which voltage profile data is collected can be selected from the Voltage Profile Demand Rate drop-down list. The Channel #4 (Volts) check box must be selected to record voltage data.

 

Before using an MCT for data collection, the MCT must be initialized with the putconfig EMETCON install command in Yukon Commander.
