Device Metering Tab

The following changes can be made to the metering settings of the device:

 

  1. The meter number can be changed in the Meter Number text box. The meter number is assigned by the utility.

  2. A different data collection group can be entered or selected in the Data Collection Group drop-down list.

  3. A different alternate group can be entered or selected in the Alternate Group drop-down list.

  4. A different billing group can be entered or selected in the Billing Group drop-down list.

  5. A different demand interval can be selected from the Last Interval Demand Rate drop-down list.

  6. If the device can collect voltage readings, a different voltage averaging interval can be selected from the Voltage Averaging Interval drop-down list.

  7. If the device can perform Load Profile Collection, the interval at which load profile data is collected can be selected from the Load Profile Demand Rate drop-down list. The channels to collect data from can be selected using the check boxes.

  8. If the device can perform Voltage Profile Collection, the interval at which voltage profile data is collected can be selected from the Voltage Profile Demand Rate drop-down list. The Channel #4 (Volts) check box must be selected to record voltage data.

 

Before using an MCT for data collection, the MCT must be initialized with the putconfig EMETCON install command in Yukon Commander.
