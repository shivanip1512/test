Editing a TOU Schedule

TOU Schedules define the rates to use and when to use the rates.

A TOU Schedule can be downloaded to an MCT by using the Yukon Commander application.

To edit a tag:

 

  1. From the View menu, select System.

  2. From the Sort By list, select TOU Schedule.

  3. Double-click on the schedule to modify. This opens the General tabbed page of the TOU Editor, from which the following changes can be made:

    1. The name for the TOU schedule can be changed in the Schedule Name text box.

    2. A different default rate of the TOU schedule can be selected from the Default Rate drop-down list. This is the rate that is used if no rate is specified in the schedule.

    3. To edit a TOU Day, select the day to edit from a day of the week drop-down list.

      1. The name of the TOU Day can be changed in the Name text box.

      2. To add additional rates to the TOU Day, enter a time of day in the text box and select a rate from the drop-down list. Note that a TOU Day always starts at midnight (00:00) and the rate for the midnight hour can be changed from the drop-down list.

      3. Click Save Day Settings to save the changes to the TOU Day.

    4. Select the TOU Days to use for each day of the week.

  1. Click Apply to save the changes or OK to save the changes and close the TOU Editor.
