Editing a TOU Schedule

TOU Schedules define the rates to use and when to use the rates. A TOU Schedule can be downloaded to an MCT by using the Yukon Commander application.

Perform the following steps to edit a TOU schedule:

 

  1. From the View menu, select System.

  2. From the Sort By list, select TOU Schedule.

  3. Double-click on the schedule to edit. The General tabbed page is displayed in the TOU Editor, from which the following changes can be made:

    1. The name for the TOU schedule can be changed in the Schedule Name text box.

    2. A different default rate of the TOU schedule can be selected from the Default Rate drop-down list. This is the rate that is used if no rate is specified in the schedule.

    3. To edit the settings of a TOU Day, select the TOU day to edit from one of the day week drop-down lists. For example, select Daily1 from the Sun: drop-down list

      1. The name of the TOU Day can be changed in the Name text box.

      2. To add additional rates to the TOU Day, enter a time of day in the text box and select a rate from the drop-down list.
        Note:         A TOU Day always starts at midnight (00:00) and the rate for the midnight hour can be changed from the drop-down list.

      3. Click the Save Day Settings button to save the changes to the TOU Day.

    4. Select the TOU Day to use for each day of the week (Sun, Mon, Tue, Wed, Thu, Fri, and Sat).

  1. Click the Apply button to save the changes to the TOU schedule or click the OK button to save the changes to the TOU schedule and close the TOU Editor.
