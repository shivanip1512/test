Analog Point Physical Setup Tab

The following changes can be made to the physical settings of the analog point:

  1. The physical point offset option can enabled or disabled by selecting or clearing the Physical Point Offset check box. Select this check box if the system obtains physical points from a location other than the hardware. Clear this check box if this is a pseudo point.

  2. If the Physical Point Offset check box is selected, the point offset can be changed in the Point Offset spin box. This is the physical field wired input.

  3. The multiplier can be changed in the Multiplier text box. The multiplier converts pulses into actual values and is used to calculate the unit value.

  4. The data offset can be changed in the Data Offset text box. This value is added to the point when making a calculation. For example, it is sometimes the Y-intercept for linear calculations.
