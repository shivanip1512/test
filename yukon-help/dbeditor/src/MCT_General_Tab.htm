MCT General Tab

The following changes can be made to the general settings of the MCT:

 

  1. The name of the MCT can be changed in the Device Name text box.

  2. The physical address of the MCT can be changed in the Physical Address text box. For the MCT-470, MCT-370, MCT-360, and MCT-318, this is the address located on the inside of the door. For the MCT-430, MCT-420, and MCT-410, this is the address located on the label directly above or below the meter’s LCD. For the MCT-310, this is the number on the faceplate, minus the leading 1 and 0s. For instance, if the faceplate number on an MCT-310i is 100020464, enter 20464 in this field.

  3. The MCT can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to take the MCT out of service, which suspends scanning and alarming. Clear this check box to enable the MCT.

  4. A different communication route can be selected from the Communication Route drop-down list.

  5. If the MCT supports MCT configs, a different MCT configuration can be selected from the MCT Config drop-down list.
