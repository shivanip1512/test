Comm Channel

Communication channels are defined and used as the point of connection to send or receive data. Exchange of data, such as with a field device, is done through a communication channel. Any number of channels can be used with the system. The types of channels available are:

  • Local Serial

  • Local Serial with Dialup

  • Terminal Server

  • Terminal Server with Dialup

 

The first step in creating a database is to define the communication channels that will be used by the system.
