Golay Load Group

If the load group is a Golay load group:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the load group in the kW Capacity text box. This is the estimated kilowatt capacity of the load control devices in the load group. To disable the kilowatt capacity feature, enter a value of zero.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Select the Disable Control check box. Select this check box to disable control of the load group. Clear this check box to enable control of the load group.

  5. Select the communication route to use for the load group from the Communication Route drop-down list. This is the route or route macro that is used to transmit commands to all of the load control devices in this group.

  6. Click the Next button.

  7. Enter the address of the load group in the Operational Address text box.

  8. Select the length of time that the load control devices are configured to control a load from the Nominal Timeout drop-down list. This value is hard coded into the load control devices.

  9. Click the Finish button. The new load group appears in the Load Group Editor.
