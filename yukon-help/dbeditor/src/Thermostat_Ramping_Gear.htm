Thermostat Ramping Gear

To create a thermostat ramping gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Thermostat Ramping from the Gear Type drop-down list.

  3. By default, delta values and Fahrenheit units are used by the gear.

    • To use absolute values instead of delta values, click the Deltas button to change the setting to Absolutes.

    • To use Celsius units instead of Fahrenheit units, click the Fahrenheit button to change the setting to Celsius.

  4. The mode a thermostat must be in to accept the thermostat ramping commands can be changed:

    • Select the Heat Mode check box to configure the thermostat ramping commands to only be accepted by thermostats in heat mode.

    • Select the Cool Mode check box to configure the thermostat ramping commands to only be accepted by thermostats in cool mode.

    • Select the Heat Mode and Cool Mode check boxes to configure the thermostat ramping commands to only be accepted by thermostats in heat mode or cool mode.

  5. To specify the minimum temperature that is allowed by the gear, select the Min check box and enter a minimum temperature in the text box.

  6. To specify the maximum temperature that is allowed by the gear, select the Max check box and enter a maximum temperature in the text box.

  7. If the ramping temperatures are displayed as delta values, perform the following steps to specify the delta values of the ramping gear:

    1. To specify the number of degrees to precool/preheat (Delta B), select the Delta B check box and enter a delta value in text box.

    2. To specify the number of degrees to ramp up (Delta D), select the Delta D check box and enter a delta value in text box.

    3. To specify the number of degrees to ramp out (Delta F), select the Delta F check box and enter a delta value in text box.

  8. If the ramping temperatures are displayed as absolute values, perform the following steps to specify the absolute values of the ramping gear:

    1. To specify the precool/preheat temperature (Absolute B), select the Abs B check box and enter an absolute value in text box.

    2. To specify the ramp up temperature (Absolute D), select the Abs D check box and enter an absolute value in text box.

    3. To specify the ramp out temperature (Absolute F), select the Abs F check box and enter an absolute value in text box.

Note: All times are in entered in minutes.

  1. To specify the random offset time, select the Rand check box and enter the length of the random offset time in the text box.
    Note:     If the Rand check box is selected, a random offset time between 0 minutes and the value Rand is added to any delay time

  2. To specify the delay time (Ta), select the Ta check box and enter the length of the delay time in the text box.

  3. To specify the precool/preheat time (Tb), select the Tb check box and enter the length of the precool/preheat time in the text box.

  4. To specify the precool/preheat hold time (Tc), select the Tc check box and enter the length of the precool/preheat hold time in the text box.

  5. To specify the ramp up time (Td), select the Td check box and enter the length of the ramp up time in the text box.

  6. To specify the ramp up hold time (Te), select the Te check box and enter the length of the ramp up hold time in the text box.

  7. To specify the ramp out time (Tf), select the Tf check box and enter the length of the ramp out time in the text box.

  8. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - Control stops when the shed time is over.

    • Restore - Control stops when a restore command is sent to the load control device.

  9. Set a group capacity reduction percentage in the Group Capacity Reduction % spin box. The percentage of each load group's capacity is used to determine how much to control.

  10. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only – The gear changes only when it is manually switched.

    • After a Duration – The gear changes when the duration ends, which is entered in the Change Duration text box.

    • Priority Change – The gear changes when the priority of the control area, to which this load program is assigned, reaches the priority entered in the Change Priority text box.

    • Above Trigger – The gear changes when the trigger point of the selected trigger rises above the value that is calculated by adding the trigger threshold value and the trigger offset value. The trigger to use for the gear change, trigger one (1) or trigger two (2) of the control area to which this program is assigned, is entered in the Trigger box and the trigger offset value is entered in the Trigger Offset text box.
      Note:       Trigger one (1) is the first entry in the Trigger drop-down list and trigger two (2) is the second entry in the Trigger drop-down list of the control area.

  11. Click the OK button to save the gear.
