Repeater General Tab

The following changes can be made to the general settings of a repeater:

 

  1. The name of the repeater can be changed in the Device Name text box.

  2. The physical address of the repeater can be changed in the Physical Address text box. For the Repeater 800, Repeater 801, Repeater 902, and Repeater 921, the physical address is located on a label inside the unit. For the Repeater 900, the physical address is a thumbwheel setting.

  1. The repeater can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to disable the repeater, which suspends scanning and alarming. Clear this check box to enable the repeater.

  2. A different communication route can be selected from the Communication Route drop-down list.
