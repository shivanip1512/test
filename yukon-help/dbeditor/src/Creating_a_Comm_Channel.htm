Creating a Comm Channel

To create a Comm Channel:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Comm Channel.

  3. Select Local Serial Port, TCP Terminal Server, or UDP Terminal Server.

  • Examples of Local Serial Ports are:

    • Built-in serial ports

    • Digi® PortServer®

    • Digi ClassicBoard®

  • Examples of TCP Terminal Servers are:

    • Digi® PortServer®

    • Other terminal servers

  • The UDP Terminal Server comm channel is for Cannon Technologies' two-way Capacitor Bank Controllers.

  1. Click Next. Based on the type of port selected, the editor will follow one of three paths.

  • Local Serial Port

  • TCP Terminal Server

  • UDP Terminal Server
