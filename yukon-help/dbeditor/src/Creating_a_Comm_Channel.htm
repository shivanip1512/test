Creating a Comm Channel

Perform the following steps to create a comm channel:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Comm Channel.

  3. Select the type of communication port from the following list:

  • Local Serial Port – Examples of a Local Serial Port are built-in serial ports, a Digi® PortServer®, and a Digi ClassicBoard®.

  • TCP Terminal Server – Examples of a TCP Terminal Server are a Digi PortServer and other terminal servers.

  • UDP Terminal Server – The UDP Terminal Server option is only used for Distribution Automation applications.

  • TCP Port – The TCP Port option is only used for Distribution Automation applications.

  1. Click the Next button.

  2. To continue creating a comm channel, click on the type of communication port that was selected in step 2.

  • Local Serial Port

  • TCP Terminal Server

  • UDP Terminal Server

  • TCP Port
