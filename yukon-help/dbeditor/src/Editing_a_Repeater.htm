Editing a Repeater

Perform the following steps to edit a repeater:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select one of the following:

    • Device

    • Transmitter

  3. Double-click on the repeater to edit. The General tabbed page is displayed in the Device Editor.

  4. Click the Scan Rate tab to display the Scan Rate tabbed page.

  1. Click the Apply button to save the changes to the repeater or click the OK button to save the changes to the repeater and close the Device Editor.
