Electronic Meter General Tab

The following changes can be made to the general settings of an electronic meter:

 

  1. The name of the electronic meter can be changed in the Device Name text box.

  2. The electronic meter can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to take the electronic meter out of service, which suspends scanning and alarming. Clear this check box to enable the electronic meter.

  3. A different communication channel can be selected from the Communication Channel drop-down list.

  4. If the electronic meter is an ION-7330, ION-7700, or ION-8300 meter:

    1. The master address can be entered in the Master Address text box.

    2. The slave address can be changed in the Slave Address text box.

  5. For all electronic meters except ION meters, the password of the electronic meter can be entered in the Password text box.

  6. If the electronic meter is an ALPHA Power Plus, FULCRUM, QUANTUM, TransData MARK-V, VECTRON meter, a master/slave relationship can be selected from the Slave Address drop-down list. For example, one meter may act as a communication gateway for other meters. The slave address options are:

    • Standalone – the meter is a stand-alone device.

    • Master – the meter is a master device and can initiate a read or write command with any slave device.

    • Slave 1 – the meter is slave 1 and can receive a read or write command from the master meter.

    • Slave 2 – the meter is slave 2 and can receive a read or write command from the master meter.

    • Slave 3 – the meter is slave 3 and can receive a read or write command from the master meter.

    • Slave 4 – the meter is slave 4 and can receive a read or write command from the master meter.

  7. If the communication channel is a dialup or dialback comm channel, the phone number that is used to dial the electronic meter or the phone number that the electronic meter uses to dial back can be changed in the Phone Number text box.

To edit the advanced dialup properties of the electronic meter:

    1. Click the Advanced button.

    2. The minimum and maximum connect times can be entered in the Minimum Connect Time and Maximum Connect Time boxes. The minimum time defines the amount of time following a successful communication that the port is not allowed to hang up. After minimum time expire, the port either remains connected until the maximum time is reached, or reverts to normal priority based on queuing and hangs up.

    3. A baud rate other than (Use Port Baud) can be selected from the Baud Rate drop-down list.

    4. Click the OK button to save the advanced dialup properties.
