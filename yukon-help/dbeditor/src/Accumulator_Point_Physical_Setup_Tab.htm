Accumulator Point Physical Setup Tab

The following changes can be made to the physical settings of the accumulator point:

 

  1. If the Yukon system obtains physical data from the device on which the point is located or the point is a special Yukon system point, select the Physical Point Offset check box. Clear this check box if this is a pseudo point.

  2. If the Physical Point Offset check box is selected, the point offset can be changed in the Point Offset spin box. This is either the physical input to which the point is wired or a Yukon system defined point.

  3. The multiplier can be changed in the Multiplier text box. The multiplier converts pulses into actual values and is used to calculate the unit value.

  4. The data offset can be changed in the Data Offset text box. This value is added to the point when making a calculation.
