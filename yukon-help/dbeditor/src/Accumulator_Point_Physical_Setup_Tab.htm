Accumulator Point Physical Setup Tab

The following changes can be made to the physical settings of the accumulator point:

 

  1. The physical point offset option can enabled or disabled by selecting or clearing the Physical Point Offset check box. Select this option if the system obtains physical settings from a location other than the hardware. Clear this option if this is a pseudo point.

  2. If the Physical Point Offset check box is selected, the point offset can be changed in the Point Offset spin box. This is the physical field wired input.

  3. If the deadband Enable check box is selected, the deadband value can be changed in the spin box. This is the amount a reading must deviate before the value is read and updated.

  4. The multiplier can be changed in the Multiplier text box. The multiplier converts pulses into actual values and is used to calculate the unit value.

  5. The data offset can be changed in the Data Offset text box. This value is added to the point when making a calculation. For example, the data offset can be the Y-intercept for linear calculations.
