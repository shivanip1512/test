Point Alarms Tab

The following changes can be made to the alarm settings:

 

  1. A different notification group can be selected from the Group drop-down list. This is the group that is notified of all alarm conditions.

  2. The option to be notified when an alarm is acknowledged can be enabled or disabled by selecting or clearing the Acknowledge check box. If this check box is selected, the notification group receives a message when an alarm is acknowledged.

  3. The option to be notified when an alarm is cleared can be enabled or disabled by selecting or clearing the Clear check box. If this check box is selected, the notification group receives a message when an alarm is cleared.

  4. An alarm category for each alarm condition can be selected from the drop-down lists in the Category column. The notification group within the alarm category is notified when the alarm condition occurs. Alarm categories can be edited in the System view of the Database Editor.

  5. Notification attributes can be selected from the Notify drop-down list for each alarm state row:

    • None - the notification is sent out normally to the alarm's notification group.

    • Exclude Notify - the alarm's notification group does not receive notification of the alarm condition.

    • Auto Ack - the alarm is automatically acknowledged.

    • Exclude Notify & Auto Ack - the alarm's notification group does not receive notification of the alarm condition and the alarm is automatically acknowledged.

  6. The option to disable all alarms can be enabled or disabled by selecting or clearing the Disable All Alarms check box. If this check box is selected, all of the alarms for this point are disabled.
