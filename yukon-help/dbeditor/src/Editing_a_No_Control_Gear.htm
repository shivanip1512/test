Editing No Control

Perform the following steps to edit a No Control gear:

  1. The name of the gear can be changed in the Gear Name text box.

  2. Select No Control from the Gear Type drop-down list.

  3. A different gear change method can be selected from the When to Change drop-down list. The change methods are:

    • Manually Only – The gear changes only when it is manually switched.

    • After a Duration – The gear changes when the duration ends, which is entered in the Change Duration text box.

    • Priority Change – The gear changes when the priority of the control area, to which this load program is assigned, reaches the priority entered in the Change Priority text box.

    • Above Trigger – The gear changes when the trigger point of the selected trigger rises above the value that is calculated by adding the trigger threshold value and the trigger offset value. The trigger to use for the gear change, trigger one (1) or trigger two (2) of the control area to which this program is assigned, is entered in the Trigger box and the trigger offset value is entered in the Trigger Offset text box.
      Note:       Trigger one (1) is the first entry in the Trigger drop-down list and trigger two (2) is the second entry in the Trigger drop-down list of the control area.

  4. Click the Apply button to save the changes to the gear or click the OK button to save the changes to the gear and close the Load Program Editor.
