MCT Group Addressing Tab

The following changes can be made to the addressing settings of an MCT group device:

  1. Click the Addressing tab in the Device Editor.

  2. A different type addressing can be selected from the Address Level drop-down list. The types of addressing are Bronze, Lead, and MCT Address.

  3. If you selected Bronze or Lead as the addressing type, the MCT address can be changed in the Address text box. This allows all of the devices using this address to be controlled.

  4. If you selected MCT Address as the addressing type, a different MCT can be selected from the MCT Address drop-down list. This allows the specified MCT to be controlled.

  5. The relays to control can be selected using the check boxes in the Relay Usage area.

  6. Click Apply to save the changes or OK to save the changes and close the dialog box.
