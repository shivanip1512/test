Editing a Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle Gear

Perform the following steps to edit a Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle gear:

  1. The name of the gear can be changed in the Gear Name text box.

  2. Select either Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle from the Gear Type drop-down list.

  3. To disable the ramping in of load groups over an interval of time, select the No Ramp (ExpressCom only) check box. The cycle command must be send using the ExpressCom protocol.

  4. The control percentage can be changed in the Control Percent spin box. This is the percentage of the cycle period that is control time.

  5. The cycle period can be changed in the Cycle Period spin box. This is the cycle period in minutes.

  6. A different cycle count send type can be selected from the Cycle Count Send Type drop-down list. The types are:

    • Fixed Count - The same number of cycle counts are sent to the switch.

    • Count Down - The number of cycles counts sent decreases at each resend.

    • Limited Count Down - Used with LCR 2000s, which do not allow cycle counts to be sent to them. The cycle count is hard coded in switch.

The purpose of Count Down and Limited Count Down is to end up with zero cycle counts at the end of control.

  1. A different maximum cycle count can be selected from the Max Cycle Cnt drop-down list. This limits the number of cycles counts that can be sent.

  2. A different starting period count can be selected from the Starting Period Count drop-down list. This is the count that is used when automatic control starts.

  3. A different resend rate can be selected from the Command Resend Rate drop-down list. This controls how often the cycle command is resent.

  4. A different method for stopping control can be selected from the How to Stop Control drop-down list. The methods for stopping control are:

    • Restore - Control stops when a restore command is sent to the load control device.

    • Stop Cycle - When the period count is set to 0, the control stops the next period.

  5. The group capacity reduction percentage can be changed in the Group Capacity Reduction % spin box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop.

  6. For Target Cycle, the amount of targeted reduction can be changed in the kW Reduction text box.

  7. A different automatic gear change method can be selected from the When to Change drop-down list. The change methods are:

    • Manually Only – The gear changes only when it is manually switched.

    • After a Duration – The gear changes when the duration ends, which is entered in the Change Duration text box.

    • Priority Change – The gear changes when the priority of the control area, to which this load program is assigned, reaches the priority entered in the Change Priority text box.

    • Above Trigger – The gear changes when the trigger point of the selected trigger rises above the value that is calculated by adding the trigger threshold value and the trigger offset value. The trigger to use for the gear change, trigger one (1) or trigger two (2) of the control area to which this program is assigned, is entered in the Trigger box and the trigger offset value is entered in the Trigger Offset text box.
      Note:       Trigger one (1) is the first entry in the Trigger drop-down list and trigger two (2) is the second entry in the Trigger drop-down list of the control area.

  8. Click the Apply button to save the changes to the gear or click the OK button to save the changes to the gear and close the Load Program Editor.
