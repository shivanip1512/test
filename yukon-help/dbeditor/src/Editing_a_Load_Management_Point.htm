Editing a Load Management Point

The Point Editor includes tabs for General, Physical Setup, Limits, Foreign Data, and Alarms. The General and Physical Setup tabs provide different options depending on the type of point.

 

To edit a point:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Load Group, Load Program, or Control Area.

  3. Click the + next to an item in the database list to display the Point type.

  4. Click the + next to the Point type to display the Point.

  5. Double-click the Point to modify.
    This opens the General tabbed page of the Point Editor. The General tabbed page contains different options depending on the type of point.

    • Analog Point

    • Status Point

  6. Click the Physical Setup tab. The Physical Setup tabbed page contains different options depending on the type of point.

    • Analog Point

    • Status Point

  7. Click the Limits tab. The Limits tabbed page appears. This is available only for analog points.

  8. Click the Foreign Data tab. The Foreign Data tabbed page appears. This allows you to add, update, or delete Foreign Data Routings.

    • Setting up Foreign Data Routing

    • Updating Foreign Data Routing

    • Deleting Foreign Data Routing

  9. Click the Alarms tab. The Alarms tabbed page appears.

  10. Click Apply to save the changes or OK to save the changes and close the dialog box.
