Editing a Load Management Point

Load management points allow for the tracking of load management data. A load management point may be one of two types:

  • Analog - an analog point reads a value directly from the transducer or electronic device.

  • Status - a status point reads the value of a contact, usually open or closed. An example of a status point might be the relay on a verification load control receiver.

 

Perform the following steps to edit a load management point:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Load Group, Load Program, or Control Area.

  3. Click the + next to an item in the database list to display the Point type.

  4. Click the + next to the Point type to display the Point.

  5. Double-click on the point to edit. The General tabbed page is displayed in the Point Editor. The General tabbed page contains different options depending on the type of point.

    • Analog Point

    • Status Point

  6. Click the Physical Setup tab to display the Physical Setup tabbed page, which contains different options depending on the type of point.

    • Analog Point

    • Status Point

  7. Click the Limits tab to display the Limits tabbed page, which is available only for analog points.

  8. Click the Foreign Data tab to display the Foreign Data tabbed page, which allows Foreign Data Routings to added, updated, or deleted.

    • Setting up Foreign Data Routing

    • Updating Foreign Data Routing

    • Deleting Foreign Data Routing

  9. Click the Alarms tab to display the Alarms tabbed page.

  10. Click the Apply button to save the changes to the point or click the OK button to save the changes to the point and close the Point Editor.
