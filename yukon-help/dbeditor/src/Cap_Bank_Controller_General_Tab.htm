Cap Bank Controller General Tab

To modify general settings for a cap bank controller:

 

  1. Change the name of the cap bank controller in the Device Name text box.

  2. Select or clear the Disable Controls check box. Select this check box to globally disallow control operations on this device. Clear this check box if there are control points on this device and you wish to use them.

  3. Change the serial number in the Serial Number text box. This is the serial number of the CBC.

  4. Select a different route from the Communications Route list.

  5. Select or clear the Disable Device check box. Select the check box to take the device out of service (suspends scanning and alarming). Clear the check box to enable the device.
