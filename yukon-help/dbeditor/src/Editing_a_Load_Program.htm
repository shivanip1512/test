Editing a Load Program

To edit a Load Program:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Load Program.

  3. Double-click the load program to modify. This opens the General tab of the Load Program Editor.

  4. Depending on the type of load program, follow one of the following paths.

    • Direct Control

    • Curtailment

    • Energy Exchange

  5. Click Apply to save the changes or OK to save the changes and close the dialog box.
