Editing a Load Program

One or more Load Programs can be defined in the database. A Load Program defines how the load is controlled, the load groups to control, and the customers to notify. When a Load Program is created, a status point is automatically added to the load program. The status point keeps track of when the load program is controlling and when it is stopped.

 

To edit a Load Program:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Load Program.

  3. Double-click the load program to modify. This opens the General tab of the Load Program Editor.

  4. Click the Gears tab to display the Gears tabbed page. Gears define the different rates at which a load program can controls loads. Different gear ratios result in different amounts of reduction.

    • To add a gear, click the Add button. To edit a gear, select the gear to edit from the Gear drop-down list. The control gears that can be added or edited are:

      • Time Refresh - timed commands are sent repeatedly to control.

      • Rotation - the gear rotates through a list of groups to control.

      • Master Cycle - the Master Station sends commands to the LCR to perform cycling.

      • Smart Cycle - the LCR hardware performs cycling.

      • TrueCycle - the LCR uses an algorithm to perform cycling.

      • Magnitude Cycle - uses an algorithm to perform cycling, which targets a percentage of the max current the field device draws.

      • Target Cycle - control is targeted to a specific amount of kW reduction.

      • Latching - one command is sent to open and another command is sent to close.

      • Thermostat Ramping - used to perform thermostat temperature ramping.

      • No Control - no control command is sent.

    • To remove a gear, see Removing a Gear.

  5. Click the Control Window tab to display the Control Window tabbed page.

  6. Click the Groups tab to display the Groups tabbed page.

  7. Click the Notification tab to display the Notification tabbed page.

  8. If the Allow Member Programs role property is enabled within the Database Editor role, click the Member Control tab to display the Member Control tabbed page.

  9. Click Apply to save the changes or OK to save the changes and close the Load Program Editor.
