Updating a Foreign Data Routing Entry

Perform the following steps to update a foreign data routing entry:

 

  1. Select the foreign data routing entry to update from the Foreign Data Routing list.

  2. A different interface type can be selected from the Interface drop-down list. The available interfaces are: ACS, ACSMULTI, BEPC, CYGNET, DNPSLAVE, DSM2FILEIN, DSM2IMPORT, INET, LIVEDATA, LODESTAR_ENH, LOADSTAR_STD, MULTISPEAK LM, OPC, PI, RCCS, RDEX, STEC, SYSTEM, TELGYR, TEXTEXPORT, TEXTIMPORT, TRISTATE, TRISTATESUB, VALMET, WABASH, and XA21LM.

  3. A different data direction can be selected from the Direction drop-down list. This controls whether the data is received, received for control, sent, or sent for control.

  4. Depending on the type of interface selected, different options appear under in the Data Parameters area:

    • ACS – Select Pseudo, Real, or Calculated from the Category drop-down list, enter the remote name in the Remote text box, and enter the point name to import or export in the Point text box.

    • ACSMULTI – Select Pseudo, Real, or Calculated from the Category drop-down list, enter the remote name in the Remote text box, enter the point name to import or export in the Point text box, and enter a destination or source address in the Destination/Source text box.

    • BEPC – Enter the ID of the coop in the Coop ID text box and enter the name of the file to export in the Filename text box.

    • CYGNET – Enter the point ID to import or export in the PointID text box.

    • DNPSLAVE – Enter the DNP master ID in the MasterID text box, enter the DNP slave ID in the SlaveID text box, enter the point offset in the Offset text box, enter the destination or source address in the Destination/Source text box, and enter the point multiplier in the Multiplier text box.

    • DSM2FILEIN – Enter the option number in the Option Number text box and enter the point ID to import in the Point ID text box.

    • DSM2IMPORT or VALMET – Enter the point name to import or export in the Point text box.

    • INET or RCCS – Enter the device name in the Device text box, enter the point name to import or export in the Point text box, and enter the destination or source address in the Destination/Source text box.

    • LIVEDATA – Enter the address in the Address text box and select the type of data to import from the Data Type drop-down list.

    • LODESTAR_ENH or LOADSTAR_STD – Enter the customer in the Customer text box, enter the channel in the Channel text box, enter the drive path in the DrivePath text box, and enter the filename of the file to import in the Filename text box.

    • MUILTISPEAK LM – Enter the load management object ID to import in the ObjectID text box.

    • OPC – Enter the name of the OPC server in the Server Name text box, enter the name of the OPC Group in the OPC Group text box, and enter the name of the OPC item to import or export in the OPC Item text box.

    • PI – Enter the tag name of the point to import in the Tag Name text box. If the polling method is being used, enter how often to poll the PI database in the Period (sec) text box. If the notification method is being used, do not enter a value in the Period (sec) text box.

    • RDEX – Enter the name or the point to import or export in the Translation text box and enter the destination or source address in the Destination/Source text box.

    • STEC, TRISTATE, or TRISTATESUB – Select the point to import from the Point list.

    • SYSTEM – Enter the client name in the Client text box.

    • TELEGYR – Enter the name of the point to import in the Point text box and select a group from the Group drop-down list.

    • TEXTEXPORT – Enter the point ID to export in the PointID text box.

    • TEXTIMPORT – Enter the point ID to import in the PointID text box, enter the location to which to save the file in the DrivePath text box, and enter the name of the file in the FileName text box.

    • WABASH – Enter the name of the schedule in SchedName text box, enter the location to which to save the control file in the Path text box (c:\Yukon\Server\Export\), and enter the name of the control file to export in the File Name text box (control.txt).

    • XA21LM – Enter the point number of the point on the XA/21 system to import or export in the Translation text box.

  5. Click the Update button to update the foreign data routing.
