Editing a CBC DNP 6510

To modify the general settings for cap bank controllers:

  1. Change the name in the Device Name text box. Because this is the name used to refer to the cap bank, choose a descriptive name.

  2. Select or clear the Disable Controls check box. Select the check box to globally disallows control operations on this device. Clear the check box if there are control points on this device and you want to use them.

  3. Change the master address in the Master Address text box.

  4. Select a different comm channel from the Communications Channel drop-down list.

  5. Change the slave address in the Slave Address text box.

  6. Change the post communication wait time in the Post Comm Wait text box.

  7. Select or clear the Disable Device check box. Select the check box to take the device out of service (suspends scanning and alarming).

  8. Click Apply to save the changes or OK to save the changes and close the dialog box.

 

To modify the scan rate settings:

  1. Click the Scan Rate tab in the Device Editor.

  2. Select the category check boxes depending on what you want scanned.
    The two scan rate categories are:

    • General Scan - collects general data

    • Accumulator Rate - collects KYZ pulse points

  3. Select different intervals from the Interval and Alt Interval drop-down lists and for each selected category. This determines how often each scan is completed.

  4. Select a different group from the Group drop-down list for each selected category. This determines the priority grouping between different device scans.

  5. Clear or select the Enabled check box, or select a different the type of scan from the Type drop-down list and enter different open and close times in the Open and Close text boxes.

  6. Click Apply to save the changes or OK to save the changes and close the dialog box.

 

To modify the alarm settings:

  1. Click the Alarms tab in the Comm Channel Editor.

  2. Clear or select the Low Daily Performance check box.

  3. Clear or select the Low Monthly Performance check box.

  4. Clear or select the Communications Failure check box.

  5. Change the threshold in the Low Performance Threshold text box.
    The threshold is set to 00% by default.

  6. Click Apply to save the changes or OK to save the changes and close the dialog box.
