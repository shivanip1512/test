Editing a Season Schedule

Season schedules allow you to define when load control can occur. Season schedules contain the start and end dates of load control seasons. Once a season schedule is defined, it can be assigned to the program constraints of a load program.

 

To edit a season schedule:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Season Schedule.

  3. Double-click a season schedule to display the General tabbed page of the Season Schedule Editor.

  4. Under Assigned Season is a list of load control season created for this season schedule.

    • To create a season, see Creating a Season.

    • To edit a season, see Editing a Season.

    • To remove a season, see Removing a Season.

  5. Click Apply to save the changes or OK to save the changes and close the Season Editor.
