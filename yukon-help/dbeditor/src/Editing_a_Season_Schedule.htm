Editing a Season Schedule

Season schedules define when load control can occur. Season schedules contain the start and end dates of load control seasons. Once a season schedule is defined, it can be assigned to a program constraint.

 

To edit a season schedule:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Season Schedule.

  3. Double-click on the season to edit. The General tabbed page is displayed in the Season Schedule Editor, from which the following changes can be made:

    1. The name of the season schedule can be changed in the Schedule Name text box.

    2. The Assigned Seasons list displays the load control seasons that are defined in the season schedule.

      • To create a season, see Creating a Season.

      • To edit a season, see Editing a Season.

      • To remove a season, see Removing a Season.

  4. Click the Apply button to save the changes to the season schedule or click the OK button to save the changes to the season schedule and close the Season Schedule Editor.
