State Group General Tab

The following changes can be made to the general settings of the state group:

 

  1. The name of the state group can be changed in the State Group Name text box.

  2. The number of states defined in the state group can be changed in the Number of States box. which changes the number of states displayed in the Configuration area.

  3. The text that is displayed for a state can be changed in the State Text text box.

  4. A different color, which is used to represent a state, can be selected from the Color drop-down list.

  5. A different image, which is used to represent a state, can be selected by performing the following steps:

    1. Click the Image button. The Image Selection window is displayed.

    2. To filter the images by category, select an image category from the Category list. The categories are:

      • (All Images) – All of the images are displayed.

      • buttons – Images that are used as buttons in oneline drawings are displayed.

      • demo – Images that are used for demo oneline drawings are displayed.

      • logos – Images that are used as logos in oneline drawings are displayed.

      • oneline – Images that are used to create oneline drawings are displayed.

    3. Select the image to represent the state and click the OK button.

  • To add an image to the Yukon system using the Image Selection window:
    Note:     The recommended size of an image is 4 KB or less.

      1. Click the Add Image(s) button. The Select Image File window is displayed.

      2. Navigate to the image to add to the Yukon system and select the image.
        Note:         To add multiple images, hold down the Ctrl or Shit key which selecting images.

      3. Click the Open button to add the image to the Yukon system.

  • To edit the properties of an image in the Image Selection window:

    1. Right-click on the image and select Edit.

    2. The name of the image can be changed in the Name text box.

    3. A different category, to which the image is assigned, can be selected from the Category drop-down list.

    4. Click the OK button to save the changes to the image.

  • To deselect an image in the Image Selection window, hold down the Ctrl key and left-click on the image.

  • To delete an image from the Yukon system, right-click on the image and select Delete.
