State Group General Tab

The following changes can be made to the general settings of the state group:

 

  1. The name of the state group can be changed in the State Group Name text box.

  2. The number of states can be changed in the Number of States spin box. The number of states you select modifies the number of enabled states under the Configuration area.

  3. The state text for each state can be changed in the State Text text boxes.

  4. A different color for each state can be selected from the Color drop-down lists.

  5. A different image for each state can be selected by clicking the Image buttons.

    • To add an image, click the Add Image(s) button and select the image to add. The recommended size of the image is 4 KB or less.

    • To edit the image properties, right-click on the image and select Edit. This allows you to change the name of the image and assign image categories.

    • To delete an image, right-click on the image and select Delete.

    • To deselect an image, hold down the Ctrl key and left-click the image.
