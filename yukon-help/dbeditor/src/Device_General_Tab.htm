Device General Tab

The following changes can be made to the general settings of the device:

 

  1. The name of the device can be changed in the Device Name text box.

  2. The physical address can be changed in the Physical Address text box. The physical address is the factory-supplied address as labeled on the device. For MCTs, this is the number on the faceplate, minus the leading 1 and 0s. For instance, if the faceplate number on an MCT 310 is 100020464, enter 20464 in this field.

    This text box only appears for certain devices. The     Physical Address text box is replaced by a Master Address or Slave Address text box for some RTUs.

  3. The device can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to take the device out of service, which suspends scanning and alarming. Clear this check box to enable the device.

  4. A different communication channel can be selected from the Communication Channel drop-down list. For MCT or repeater devices, a different route from can be selected the Communication Route drop-down list.

  5. For MCTs, an MCT configuration can be selected from the MCT Config drop-down list.

  6. For certain signal transmitters. the number of milliseconds to wait can be changed in the Post Communication Wait spin box. The post communication wait defines the length of time to wait after communication before this device can be communicated with again.

  7. For RTU-DART, RTU-DNP, RTU-MODBUS, and ION devices, the DNP slave address can be changed in the Slave Address text box.

  8. For CCUs, a different power amplifier card usage can be selected from the CCU Amp Use Type drop-down list. This defines how the amplifier cards are used by the communication routes. The amp card use options are:

    • Amp1 - Use amp card 1.

    • Amp2 - Use amp card 2.

    • Alternating - Alternate between using amp card 1 and amp card 2 after each transmission.

    • Default 1 Fail 2 - Use amp card 1 by default and switch to amp card 2 if amp card 1 fails.

    • Default 2 Fail 1 - Use amp card 2 by default and switch to amp card 1 if amp card 2 fails.

    • Alt w/Failover - Alternate between using amp card 1 and amp card 2 after each transmission, and switch to the other amp card if one of amp cards fail.

  9. For the FULCRUM and VECTRON meters, a master/slave relationship can be selected from the Slave Address drop-down list. For example, one meter may act as a communication gateway for other meters. The slave address options are:

    • Standalone - the meter is a stand-alone device.

    • Master - the meter is a master device and can initiate a read or write command with any slave device.

    • Slave 1 - the meter is slave 1 and can receive a read or write command from the master meter.

    • Slave 2 - the meter is slave 2 and can receive a read or write command from the master meter.

    • Slave 3 - the meter is slave 3 and can receive a read or write command from the master meter.

    • Slave 4 - the meter is slave 4 and can receive a read or write command from the master meter.

  10. For SIXNET meters, the name of the SIXNET log file can be changed in the Log File text box. The name of the log file must include the .6dl extension. In addition, the destination station address can be changed in the Station Address in the text box. This is the receiving station number and is required to communicate with the meter.

  11. For meters assigned to dialup or dialback communication channels, the phone number can be changed in the Phone Number text box

  12. To edit the dialup or dialback properties, click the Advanced button.

    1. The minimum and maximum connect times can be changed in the Minimum Connect Time and Maximum Connect Time boxes. The minimum time defines the amount of time following a successful communication that the port is not allowed to hang up. After this time, the port either remains connected until the maximum time is reached, or reverts to normal priority based on queuing and hangs up.

    2. A different baud rate can be selected from the Baud Rate drop-down list.

    3. Click the OK button to save the advanced dialup properties.
