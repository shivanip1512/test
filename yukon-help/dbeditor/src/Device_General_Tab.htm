Device General Tab

The following changes can be made to the general settings of the device:

 

  1. The name of the device can be changed in the Device Name text box.

  2. The physical address can be changed in the Physical Address text box. This text box only appears for certain devices. The Physical Address text box is replaced by a Master Address text box for RTU-DART, RTU-DNP, and RTU-MODBUS devices.

    The physical address is the factory-supplied address as labeled on the device. For MCTs, this number is the serial number on the faceplate, minus the leading 1 and 0s. For instance, if the faceplate number on an MCT 310 is 100020464, enter 20464 in this field.

  3. The device can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to take the device out of service, which suspends scanning and alarming. Clear this check box to enable the device.

  4. A different communication channel can be selected from the Communication Channel drop-down list. For MCT or repeater devices, a different route from can be selected the Communication Route drop-down list.

  5. For MCTs, an MCT configuration can be selected from the MCT Config drop-down list.

  6. The number of milliseconds to wait can be changed in the Post Communication Wait spin box. The post communication wait defines the length of time to wait before this device can be communicated with again.

  7. For RTU-DART, RTU-DNP, RTU-MODBUS, and ION devices, the slave address can be changed in the Slave Address text box.

  8. For CCUs, a different amp card usage can be selected from the CCU Amp Use Type drop-down list. This defines how the amp cards are used by the route. The amp use options are:

    • Amp1

    • Amp2

    • Alternating

    • Default 1 Fail 2

    • Default 2 Fail 1

    • Alt w/Failover

  9. For certain electronic meters, a different slave address can be selected from the Slave Address drop-down list. The slave address defines the behavior of dial-up devices in terms of master/slave relationships. For example, some devices may act as a communication gateway for other slave devices. The slave address options are:

    • Standalone

    • Master

    • Slave 1

    • Slave 2

    • Slave 3

    • Slave 4

  10. For SIXNET meters, the name of the SIXNET log file can be changed in the Log File text box The name of the log file must include the .6dl extension. In addition, the station address can be changed in the Station Address in the text box.

  11. For meters assigned to dialup or dialback communication channels, the phone number can be changed in the Phone Number text box

  12. To edit the dialup or dialback properties, click the Advanced button.

    1. The minimum and maximum connect times can be changed in the Minimum Connect Time and Maximum Connect Time boxes. The minimum time defines the amount of time following a successful communication that the port is not allowed to hang up. After this time, the port either remains connected until the maximum time is reached, or reverts to normal priority based on queuing and hangs up.

    2. A different baud rate can be selected from the Baud Rate drop-down list.

    3. Click OK to save the advanced dialup properties.
