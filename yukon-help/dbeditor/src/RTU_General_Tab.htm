RTU General Tab

The following changes can be made to the general settings of an RTU:

 

  1. The name of the RTU can be changed in the Device Name text box.

  2. If the RTU is an RTM or RTU-WELCO device, the address of the RTU can be changed in the Physical Address text box.

If the RTU is an RTU-DNP or RTU-MODBUS device:

    1. The master address can be changed in the Master Address text box.

    2. The slave address can be changed in the Slave Address text box.

  1. The RTU can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to take the RTU out of service, which suspends scanning and alarming. Clear this check box to enable the RTU

  2. A different communication channel can be selected from the Communication Channel drop-down list.

  3. If the RTU is an RTU-DNP, RTU-MODBUS, or RTU-WELCO device, the number of milliseconds to wait can be changed in the Post Communication Wait spin box. The post communication wait defines the length of time to wait after communication before this RTU can be communicated with again.

  4. If the communication channel is a dialup or dialback comm channel, the phone number that is used to dial the RTU or the phone number that the RTU uses to dial back can be changed in the Phone Number text box.

To edit the advanced dialup properties of the RTU:

    1. Click the Advanced button.

    2. The minimum and maximum connect times can be entered in the Minimum Connect Time and Maximum Connect Time boxes. The minimum time defines the amount of time following a successful communication that the port is not allowed to hang up. After this time, the port either remains connected until the maximum time is reached, or reverts to normal priority based on queuing and hangs up.

    3. A baud rate other than (Use Port Baud) can be selected from the Baud Rate drop-down list.

    4. Click the OK button to save the advanced dialup properties.
