Editing a Rotation Gear

Perform the following steps to edit a Rotation gear:

  1. The name of the gear can be changed in the Gear Name text box.

  2. Select Rotation from the Gear Type drop-down list.

  3. To change the length of the shed command, select a different shed time from the Shed Time drop-down list.

  4. To change the number groups that are shed each time the shed command is refreshed or resent, select a different number of groups from  the # of Groups Each Time drop-down list.

  5. To change the rate at which the shed command is refreshed or resent, select a different rate from the Command Resend Rate drop-down list.

  1. A different group selection method can be selected from the Group Selection Method drop-down list. The group selection methods are:

    • Last Controlled – The gear selects the next load group in the list, based on the last load group controlled.

    • Always First Group – The gear always selects the first load group in the list.

    • Least Control Time – The gear selects the load group in the list with the least amount of seasonal control time.

  2. A different method for stopping control can be selected from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In – Control stops when the shed time is complete.

    • Restore – Control stops when a restore command is sent to the load control device.

  3. The group capacity reduction percentage can be changed in the Group Capacity Reduction % spin box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop

  4. A different gear change method can be selected from the When to Change drop-down list. The change methods are:

    • Manually Only – The gear changes only when it is manually switched.

    • After a Duration – The gear changes when the duration ends, which is entered in the Change Duration text box.

    • Priority Change – The gear changes when the priority of the control area, to which this load program is assigned, reaches the priority entered in the Change Priority text box.

    • Above Trigger – The gear changes when the trigger point of the selected trigger rises above the value that is calculated by adding the trigger threshold value and the trigger offset value. The trigger to use for the gear change, trigger one (1) or trigger two (2) of the control area to which this program is assigned, is entered in the Trigger box and the trigger offset value is entered in the Trigger Offset text box.
      Note:       Trigger one (1) is the first entry in the Trigger drop-down list and trigger two (2) is the second entry in the Trigger drop-down list of the control area.

  5. Click the Apply button to save the changes to the gear or click the OK button to save the changes to the gear and close the Load Program Editor.
