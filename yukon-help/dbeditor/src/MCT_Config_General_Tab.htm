MCT Config General Tab

The following changes can be made to the general settings of an MCT Config:

 

  1. The name of the MCT Config can be changed in the Configuration Name text box.

  2. The mode of operation can be changed by selecting either the On-Peak/Off-Peak Mode or Min/Max Mode.

    • On-Peak/Off-Peak Mode – Configures the MCT to record the maximum demand readings for either the On-Peak or Off-Peak interval.

    • Min/Max Mode – Configures the MCT to record the minimum and maximum demand readings.

  3. The check box for channel 1 is selected by default and cannot be deselected.

    1. The way channel 1 is wired can be changed by selecting the 3-Wire (KYZ) or 2-Wire (KY) radio button.

    2. If the Ke value (kWh per pulse) for channel 1 is known, it can be changed in the Ke text box.
      If the Ke value for channel 1 is not known:

      1. Click the Recalculate button.

      2. Enter the Mp value (pulse constant) and Kh value (watt-hour or disk constant) into the appropriate text boxes.

      3. Click the = (equals) button to calculate the Ke value.

  4. To enable channel 2, select the Channel 2 check box.

    1. The way channel 2 is wired can be changed by selecting the 3-Wire (KYZ) or 2-Wire (KY) radio button.

    2. If the Ke value (kWh per pulse) for channel 2 is known, it can be changed in the Ke text box.
      If the Ke value for channel 2 is not known:

      1. Click the Recalculate button.

      2. Enter the Mp value (pulse constant) and Kh value (watt-hour or disk constant) into the appropriate text boxes.

      3. Click the = (equals) button to calculate the Ke value.

  5. To enable channel 3, select the Channel 3 check box.

    1. The way channel 3 is wired can be changed by selecting the 3-Wire (KYZ) or 2-Wire (KY) radio button.

    2. If the Ke value (kWh per pulse) for channel 3 is known, it can be changed in the Ke text box.
      If the Ke value for channel 3 is not known:

      1. Click the Recalculate button.

      2. Enter the Mp value (pulse constant) and Kh value (watt-hour or disk constant) into the appropriate text boxes.

      3. Click the = (equals) button to calculate the Ke value.
