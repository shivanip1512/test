MCT Config General Tab

The following changes can be made to the general settings of an MCT Config:

 

  1. The name of the MCT Config can be changed in the Configuration Name text box.

  2. If the config is for an 200 Series MCT, the Ke value can be changed in the Ke text box.
    If the Ke value is not known, click the Recalculate button, enter the Mp and Kh values into the appropriate text boxes, and then click the = button to calculate Ke.

  3. If the config is for an 300 Series MCT:

    1. The mode of operation can be changed by selecting either the On-Peak/Off-Peak Mode or Min/Max Mode.

    2. Channel 1 is enabled by default. Additional channels can be enabled by selecting the Channel 2 or Channel 3 check boxes.

    3. For each channel enabled, select whether the channel is wired for 3-Wire (KYZ) or 2-Wire (KY).

    4. For each channel enabled, the Ke value can be changed in the Ke text box.
      If the Ke value is not known, click the Recalculate button, enter the Mp and Kh values into the appropriate text boxes, and then click the = button to calculate Ke.
