Route Repeater Setup Tab

Note: Do not use the Route tools unless instructed to by Cannon Technologies support.

 

The following changes can be made to the repeater setup settings of the route:

 

  1. Repeaters can be assigned to or unassigned from the route.

    • To assign a repeater, select the repeater from the Available list and click Add. To select more than one repeater, hold Ctrl key on the keyboard while clicking additional repeaters.

    • To unassign a repeater, select the repeater from the Assigned list and click Remove. To select more than one repeater, hold the Ctrl key on the keyboard while clicking additional repeater.

  2. To calculate the fixed and variable bits of the CCU and repeater, select Advanced Setup. The fixed bits and variable bits are automatically calculated by the Yukon software.

    If you assigned a repeater or unassigned a repeater without applying the change, a dialog box appears asking if you want to apply the changes.

    • Click Yes to permanently apply the changes and allow the software to update the fixed and variable bits.

    • Click No to not apply the changes and to not allow the software to update the fixed and variable bits.

  3. To keep the bits from changing during regeneration, select the Do not allow auto bit change (Lock) check box.

  4. To select the route for bit recalculation during regeneration, select the Recalculate fixVar bits (Reset) check box.

  5. Click OK to save the changes.

  6. To regenerate route bits, select Regenerate Route Roles from the Tools menu. Do not use the Route tools unless instructed to by Cannon Technologies support.

    1. To regenerate only the selected route roles, select the Regenerate selected route role(s) check box.

    2. To regenerate all route roles, select the Regenerate all route roles check box.

  7. Click OK.
