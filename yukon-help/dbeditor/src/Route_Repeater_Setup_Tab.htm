Route Repeater Setup Tab

The following changes can be made to the repeater setup settings of the route:

 

  1. Repeaters can be assigned to or unassigned from the route.

    • To assign a repeater to the route, select the repeater from the Available list and click the Add button. To select more than one repeater, hold Ctrl key on the keyboard while clicking additional repeaters.

    • To unassign a repeater from the route, select the repeater from the Assigned list and click the Remove button. To select more than one repeater, hold the Ctrl key on the keyboard while clicking additional repeaters.

    • To rearrange the order of the assigned repeaters, click and drag a repeater to move it up or down in the list. The repeater physically closest to the CCU should be moved to the top of the Assigned list. The repeater physically farthest from the CCU should be moved to the bottom of the Assigned list.

    • To save the changes to the list of assigned repeaters and enable the Advanced Setup button, click the Apply button.

  2. To configure the fixed and variable bits of the CCU and repeaters, click the Advanced Setup button. The fixed bits and variable bits are automatically calculated by the Yukon software.
    Note:     Do not edit the Advanced Setup settings unless instructed to do so by Cooper Power Systems Customer Service.

    1. If another CCU is using the same fixed or variable bits, that CCU appears in the Other CCUs using these bit roles area.
      To exclude the fixed and variable bits of a CCU from future suggestions, select the check box of the CCU to exclude.

    2. To display suggested values for the fixed and variable bits, click the Get Suggestion button.
      If another CCU is using the suggested fixed or variable bits, that CCU appears in the       CCUs using suggested bit roles area.

    3. To use the suggested values for the fixed and variable bits, click the Use Suggestion button.

    4. To prevent a user from modifying the repeaters assigned the route, select the User Locked check box.

    5. Click the OK button to save changes to the Advanced Route Setup window.
