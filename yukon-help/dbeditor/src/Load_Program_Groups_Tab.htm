Load Program Groups Tab

The following changes can be made to the load groups assigned to the load program:

  • To assign a load group to the load program, select the load group from the Available Load Groups list and click the Add button. To select more than one load group, hold Ctrl key on the keyboard while clicking additional load groups.

  • To rearrange the order of the assigned load groups, click and drag a load group to move it up or down in the list.

  • To unassign a load group from the load program, select the group from the Assigned Load Groups list and click the Remove button. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.
