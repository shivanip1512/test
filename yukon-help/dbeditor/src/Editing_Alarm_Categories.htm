Editing Alarm Categories

When an alarm is activated, the associated contact and groups are notified. The alarm categories are predefined in the Yukon database.

 

To edit an alarm category:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Alarm Categories.

  3. Double-click on the alarm category to modify. This opens the General tabbed page of the Alarm Category Editor. You may make the following changes:

    1. The name of the alarm category can be changed in the Name text box

    2. A different notification group can be selected from the Notification Group drop-down list.

  4. Click the Apply button to save the changes or OK button  to save the changes and close the Alarm Category Editor.
