Editing Alarm Categories

When an alarm is activated, the associated contact and groups are notified. The alarm categories Category 1 through Category 100 are predefined in the Yukon database.

 

Perform the following steps to edit an alarm category:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Alarm Categories.

  3. Double-click on the alarm category to edit. The General tabbed page is displayed in the Alarm Category Editor, from which the following changes can be made::

    1. The name of the alarm category can be changed in the Name text box

    2. The notification group assigned to the alarm category can be changed by selecting a different group from the Notification Group drop-down list.

  4. Click the Apply button to save the changes to the alarm category or click the OK button to save the changes to the alarm category and close the Alarm Category Editor.
