Load Program General Tab

The following changes can be made to the general settings of the load program:

 

  1. The name of the load program can be changed in the Name text box.

  2. A different operational state can be selected from the Operational State drop-down list.

    • Automatic – The program controls when the trigger threshold is exceeded or when the program is manually started.

    • Manual Only – The program must be manually started to perform control.

    • Timed – The program controls during the specified control windows.

  3. A different constraint can be selected from the Program Constraint drop-down list. A program constraint controls when the load program can run.

  4. A different trigger offset for this program, can be entered in the Trigger Offset text box. When the load program is assigned to a control area, the trigger offset is added to the trigger threshold. The trigger point defined in the control area must exceed the trigger offset plus the trigger threshold before the load program can start control. The default value of the trigger offset is zero.

  5. A different restore offset for this program, can be entered in the Restore Offset text box. When the load program is assigned to a control area, the restore offset is subtracted from the control area trigger threshold. The trigger point defined in the control area must go below the control area trigger threshold minus the trigger offset before the load program can stop control. A restore offset with a positive value causes the trigger threshold to be lowered and a restore offset with a negative value causes the trigger threshold to be raised. The default value of the restore offset is zero.
