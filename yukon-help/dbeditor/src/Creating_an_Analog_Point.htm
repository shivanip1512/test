Creating an Analog Point

If you selected Analog as the point type:

 

  1. Select a unit of measure from the Unit of Measure list. The unit of measure is for the data being read.

  2. Set the number of decimal places in the Decimal Places box. This is the number of decimal places to display for the measurement.

  3. Set a multiplier in the Multiplier text box. A multiplier can convert pulses into kWh (visual meter reading) and is used to calculate the unit value.

  4. Set the data offset in the Data Offset text box. The data offset is usually associated with metering and is added to the reading during a calculation.

  5. Enter the number of dials/digits displayed in the meter reading in the Meter Digits (Dials) spin box. This allows a rollover to occur when the reading passes the specified number of digits.

  6. Click Next.

  7. If the system is to obtain physical data from the hardware or if it is a special Yukon point, select the Physical Point Offset check box. Clear this check box for a pseudo point.

  8. If the Physical Point Offset check box is selected, set the offset value in the Point Offset spin box. This is the physical relay used for wiring.

  9. Click Finish. The new point appears in the Point Editor. For instructions on editing a point, refer to the related topics.
