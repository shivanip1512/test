ExpressCom Group Addressing Tab

The following changes can be made to the addressing settings of the ExpressCom group:

 

  1. In the Geographical Address Usage area, select the check boxes of the geographical addresses to use when commands are sent. When an address check box is selected, the address field is activated and turns blue.

  2. In the Load Address Usage area, select the check boxes of the load addresses to use when commands are sent. When an address check box is selected, the address field is activated and turns blue.

  3. Addressing information can be changed in the Addressing area.

    1. The service provider ID can be changed in the SPID Address text box. The service provider ID is sent with all group address messages and is used to identify the service provider (owner utility) in the case of any communications cross talk or some shared communications network.

    2. The geographical address can be changed in the GEO Address text box. The geographical address identifies a desired geographical area (determined by the utility).

    3. The substation address can be changed in the SUB Address text box. The substation address is intended to identify the utility substation on which the target equipment is located.

    4. The feeder address can be changed in the FEED Address text box. The feeder address is intended to identify the utility feeder by which the load is fed. This is normally combined with the substation address, but can be used alone.

    5. The zip address can be changed in the ZIP Address text box. The zip address is intended to be the postal zip code. This address can also be used for any other addressing options.

    6. The user defined address can be changed in the USER Address text box. The user defined address is intended to be user definable and can be used by the user for any other addressing options.

    7. If the Serial check box is selected in the Geographical Address Usage area, the serial number can be changed in the Serial text box. This applies to a single device and does not use group addressing.

    8. The program address can be changed in the PROG Address text box. The program address is used to target program loads. Field devices that control multiple loads contain a separate program address for each load.

    9. The splinter address can be changed in the SPLINTER text box. The splinter address is used to target a subset of a program load address. Field devices that control multiple loads contain a separate splinter address for each load.

  4. (Optional) If the program or splinter check box is selected in the Load Address Usage area, the Configured Loads area can be used to select the check boxes of the loads targeted by the program or splinter addresses.

  5. If the load check box is selected in the Load Address Usage area, the Configured Loads area changes to Targeted Loads. Select the check boxes of the loads to control for this group in the Targeted Loads area.
