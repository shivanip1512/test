Accumulator Point General Tab

The following changes can be made to the general settings of the accumulator point:

 

  1. The name of the point can be changed in the Point Name text box.

  2. The Timing Group drop-down list is for DNP devices only. For points on DNP devices, a value of SOE (Sequence of Events) can be selected, which records high speed events that occur within milliseconds of each other.

  3. The point can be disabled or enabled by selecting or clearing the Disable Point check box. Disabling the point may be useful if it is temporarily not functioning.

  4. A different unit of measure can be selected from the Unit of Measure drop-down list. The unit of measure is for the actual data being read.

  5. The number of dials/digits displayed in the meter reading can be changed in the Meter Dials spin box. This allows a rollover to occur when the reading passes the specified number of digits.

  6. The number of decimals to display for the measurement can be changed in the Decimal Positions spin box.

  7. A different state group can be selected from the State Group drop-down list. The state group defines the messages to display for the different states of a point.

  8. A different archive type can be selected from the Data Archive Type drop-down list. The data archive types are:

    • None – does not archive.

    • On Change – archives when an operand causes the point to change.

    • On Timer – archives at the selected archive interval.

    • On Update – archives any time the point is updated.

    • On Timer or Update - archives at the selected archive interval or any time the point is updated.

  9. If On Timer or On Timer or Update is selected as the data archive type, a different archive interval can be selected from the Archive Interval drop-down list.
