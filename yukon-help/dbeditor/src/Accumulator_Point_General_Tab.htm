Accumulator Point General Tab

The following changes can be made to the general settings of the accumulator point:

 

  1. The name of the point can be changed in the Point Name text box.

  2. A different type of timing group, which control how often events are recorded, can be selected from the Timing Group drop-down list. The timing group are:

    • Default – Records events that occur within seconds of each other.

    • SOE – Records events that occur within milliseconds of each other (Sequence of Events).

  3. The point can be disabled or enabled by selecting or clearing the Disable Point check box. Disabling the point may be useful if it is temporarily not functioning.

  4. A different unit of measure can be selected from the Unit of Measure drop-down list.

  5. The number of dials/digits to display for the meter reading can be changed in the Meter Digits (Dials) spin box. When the meter reading surpasses the specified number of digits, the reading rolls over to zero.

  6. The number of decimal places to display for the measurement can be changed in the Decimal Positions spin box.

  7. A different state group, which defines the messages to display for the different states of a point, can be selected from the State Group drop-down list.

  8. A different data archive type can be selected from the Data Archive Type drop-down list. The data archive types are:

    • None – Does not archive data.

    • On Change – Archives data when an operand causes the point to change.

    • On Timer – Archives data at the selected archive interval.

    • On Update – Archives data any time the point is updated.

    • On Timer or Update – Archives data at the selected archive interval or any time the point is updated.

  9. If On Timer or On Timer or Update is selected as the data archive type, select how often to archive point data from the Interval drop-down list. The archive intervals are:

    • 30 minute

    • 1 hour

    • 2 hour

    • 6 hour

    • 12 hour

    • Daily

    • Weekly

    • Monthly
