Setting Up Foreign Data Routing

To set up foreign data routing:

 

  1. Select an interface from the Interface drop-down list. The available interfaces are: ACS, ACSMULTI, BEPC,  CYGNET, DSM2FILEIN, DSM2IMPORT, INET, LIVEDATA, LODESTAR_ENH, LOADSTAR_STD, PI, RCCS, RDEX, STEC, SYSTEM, TELGYR, TEXTEXPORT, TEXTIMPORT, TRISTATE, VALMET, WABASH, and XA21LM. The Data Parameters change depending on the type of interface you select.

  2. Select a data direction from the Direction drop-down list. This controls whether the data is received or sent.

  3. Depending on the Interface selected, different options appear under Device Parameters:

    • If you selected ACS, select Real or Pseudo from the Category drop-down list, and enter the remote name in the Remote text box and the point name in the Point text box.

    • If you selected ACSMULTI, select Real or Pseudo from the Category drop-down list, enter the remote name in the Remote text box, enter the point name in the Point text box, and enter a destination or source in the Destination/Source text box.

    • If you selected CYGMET, TEXTEXPORT, or TEXTIMPORT enter the point id into the PointID text box.

    • If you selected DSM2FILEIN, enter the option number in the Option Number text box and the point ID in the Point ID text box.

    • If you selected BEPC, DSM2IMPORT or VALMET, enter the  point name into the Point text box.

    • If you selected INET or RCCS, enter the  device name in the Device text box, the point name in the Point text box, and the destination/source in the Destination/Source text box.

    • If you selected LIVEDATA, enter the address in the Address text box and select the type of data from the Data Type drop-down list.

    • If you selected LODESTAR_ENH or LOADSTAR_STD, enter the customer in the Customer text box, the channel in the Channel text box, the drive path in the DrivePath text box, and the filename in the Filename text box.

    • If you selected PI, enter the tag name of the point in the Tag Name text box. If the polling method is being used, enter how often to poll the PI database in the Period (sec) text box. If the notification method is being used, do not enter a value in the Period (sec) text box.

    • If you selected RDEX, enter the translation name in the Translation text box and the destination/source in the Destination/Source text box.

    • If you selected STEC or TRISTATE, select the point from the Point drop-down list.

    • If you selected SYSTEM, enter the client name in the Client text box.

    • If you selected TELEGYR, enter the  point name in the Point text box and select the group from the Group drop-down list.

    • If you selected WABASH, enter the schedule name in SchedName text box, enter the file path in the Path text box, and enter the name of the file in the File Name text box.

    • If you selected XA21LM, enter the point number of the point on the XA/21 system in the Translation text box.

  4. Click Add.
