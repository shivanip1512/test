Ripple Load Group Message Tab

Perform the following steps for the message settings of the Ripple load group:

 

  1. Select a value from the Shed Time drop-down list that matches the shed time programmed into the load control device. The shed time is the number of minutes a relay controls when a control command is sent. A shed time of Continuous Latch controls the relay until a restore command is sent.

  2. Set a control message using the Control push buttons that matches the control message programmed into the load control device. This is the 50 bit message that causes the relay to control.

  3. Set a restore message using the Restore push buttons that matches the restore message programmed into the load control device. This is the 50 bit message that causes the relay to restore.
