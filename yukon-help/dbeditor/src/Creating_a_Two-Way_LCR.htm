Creating a Two-Way LCR

To create a two-way LCR:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select Two Way LCR from the list on the left.

  4. Select LCR-3102 from the list on the right.

  5. Click the Next button.

  6. Enter a name for the LCR in the Device Name text box.

  7. Enter the factory-supplied serial number of the LCR in the Serial Number text box. The serial number is located on a label that is affixed to the LCR board inside the LCR enclosure.

  8. To create the default points for a two-way LCR, select the Create default points check box.

  9. Click the Next button.

  10. To enable scanning to collect demand and status point value from the LCR, select the Demand & Status Rate check box.

    1. Select the default scan interval from the Interval drop-down list.

    2. Select the alternate scan interval from the Alt Interval drop-down list. The alternate scan interval can be activated using the Tabular Data Console or the Yukon Web interface and is used during the “control timeout” period to collect the state of a control point.

    3. Select a scan group from the Group drop-down list. Groups allow priorities to be assigned to scan categories.

  11. Click the Next button.

  12. Select the communication route for the LCR from the drop-down list of routes.

  13. Click the Finish button. The new LCR appears in the Device Editor. For instructions on editing a two-way LCR, refer to the related topics.
