Creating a Two-Way LCR

Perform the following steps to create a two-way LCR:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select Two Way LCR from the list on the left.

  4. Select LCR-3102 from the list on the right.

  5. Click the Next button.

  6. Enter a name for the LCR in the Device Name text box.

  7. Enter the factory-supplied serial number of the LCR in the Serial Number text box. The serial number is located on a label that is on the LCR board inside the LCR enclosure.

  8. To create the default points for the LCR, select the Create Default Points check box.

  9. Click the Next button.

  10. To enable Demand & Status Rate scan, which collects demand and status point values from the LCR, select the Demand & Status Rate check box.

  11. If the check box of the Demand & Status Rate scan category is selected, the following scan options are enabled:

    • Interval – Default scan interval of the selected scan category.

    • Alt Interval – Alternate scan interval of the selected scan category.

    • Group – Scan group of the selected scan category. The scan group is used to assign a priority to the scan category.

  1. Click the Next button.

  2. Select the communication route for the LCR from the Select the route used with this device drop-down list.

  3. Click the Finish button. The new LCR appears in the Device Editor. For instructions on editing the LCR, refer to the related topics.
