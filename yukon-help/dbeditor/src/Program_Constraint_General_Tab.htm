Program Constraint General Tab

The following changes can be made to the general settings of a program constraint:

  1. The name of the program constraint can be changed in the Constraint Name text box.

  2. A different load control season schedule can be selected from the Season Schedule drop-down list.

  3. The maximum number of seconds that a load group can be active during a control session can be changed in the Max - Activate text box. This constraint is reset when the load program becomes inactive.

  4. The maximum number of times a load group can be active in one day can be changed in the Max - Daily Ops text box. This constraint is reset at the end of the day.

  5. The minimum number of seconds that a load group must be active before it can be stopped can be changed in the Min - Activate text box. This constraint is reset after the load group stops

  6. The minimum number of seconds to wait after a load group stops before the load group can run again can be changed the Min - Restart text box. This constraint is reset when the load program becomes inactive.

  7. Different check boxes can be selected for which days of the week the load program is available under the Day Selection area

  8. A different holiday schedule can be selected from the Holiday Schedule drop-down list.

  9. If a holiday schedule is selected, select what to do on holidays in the Holiday Usage area:

    • Exclude - control cannot occur on the days defined in the holiday schedule.

    • Force - control can occur on the days defined in the holiday schedule, even if the day is not selected in the Day Selection area.

  10. The maximum number of hours that a load group can be controlled for each time period (Daily, Monthly, Seasonal, Annual) can be changed under the Max Hour Allowance area. A value of zero disables the time constraint.
