Program Constraint General Tab

The following changes can be made to the general settings of a program constraint:

  1. The name of the program constraint can be changed in the Constraint Name text box.

  2. A different season schedule can be selected from the Season Schedule drop-down list.

  3. The maximum number of seconds that a load group is allowed to be controlled can be changed in the Max - Activate text box. This constraint is reset when the load program becomes inactive.

  4. The maximum number of times that a load group is allowed to be controlled in one day can be changed in the Max - Daily Ops text box. This constraint is reset at the end of the day.

  5. The minimum number of seconds that a load group must be controlled, before it is allowed to stop, can be changed in the Min - Activate text box. This constraint is reset after the load group stops.

  6. The minimum number of seconds to wait after a load group stops being controlled, before it is allowed to be controlled again, can be changed the Min - Restart text box. This constraint is reset when the load program becomes inactive.

  7. To specify the days of the week that control can or cannot occur, select or clear the day check boxes in the Day Selection area.
    To prevent control from occurring on a day of the week, clear the check box for that day. To allow control to occur on a day of the week, select the check box for that day.
    Note:     The program constraint allows control to occur every day of the week by default.

  8. A different holiday schedule can be selected from the Holiday Schedule drop-down list.

  9. If a holiday schedule is selected, select what to do on holidays in the Holiday Usage area:

    • Exclude – Control cannot occur on the days defined in the holiday schedule.

    • Force – Control can occur on the days defined in the holiday schedule, even if the day is not selected in the Day Selection area.

  10. The maximum number of hours that a load group is allowed to be controlled for each time period (Daily, Monthly, Seasonal, and Annually) can be changed under the Max Hour Allowance area. A value of zero disables the time constraint.
