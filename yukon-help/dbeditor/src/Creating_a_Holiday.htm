Creating a Holiday

To add a holiday to the schedule:

 

  1. Click the Create button. The Holiday Date Creation dialog box appears.

  2. Enter the name of the holiday in the Holiday Name text box.

  3. Select the holiday month from the Month drop-down list.

  4. Select the day of the holiday from the Day drop-down list.

  5. Select the holiday year from the Year drop-down list. To make the holiday repeat each year, select (none).

  6. Click the OK button.
