Creating a Holiday

Perform the following steps to add a holiday to the schedule:

 

  1. Click the Create button. The Holiday Date Creation dialog box appears.

  2. Enter the name of the holiday in the Holiday Name text box.

  3. Select the month of the holiday from the Month drop-down list.

  4. Select the day of the holiday from the Day drop-down list.

  5. To specify a year for the holiday, select a year from the Year drop-down list.

To make the holiday repeat each year, select (none) from the Year drop-down list.

  1. Click the OK button. The holiday is added to the holiday schedule.
