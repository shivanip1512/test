Editing a Gear

To edit a gear:

 

  1. Select the gear to edit from the drop-down list of gears and click Edit.

  2. Modify the settings under Gear Settings as necessary.

Time Refresh/Rotation

To edit a Time Refresh or Rotation gear:

  1. Change the name of the gear in the Gear Name text box.

  2. Select either Time Refresh or Rotation from the Gear Type drop-down list.

  3. Select a different time from the Shed Time drop-down list. This is the shed time sent to the switches in the field.

  4. Select a different number of groups from the # of Groups Each Time drop-down list. For a time refresh gear, this is the number of groups that are shed for each refresh. For a rotation gear, the is the number of groups that are shed during each rotation.

  5. Select a different rate from the Refresh/Resend Rate drop-down list. This field changes to Send Rate if the gear type is Rotation. The Refresh/Resend Rate is the rate at which the shed command is refreshed or resent. The Send Rate is the rate at which the shed command is sent.

  6. Select a different group selection method from the Group Selection Method drop-down list. The group selection methods are:

    • Last Controlled - starts with the next group based on the last one controlled.

    • Always First Group - always starts with the first group in the list.

    • Least Control Time - starts with the group whose seasonal control time is the least.

  7. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

  8. Set a different group capacity reduction percentage in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much load to control.

  9. Select a different automatic gear change method from the When to Change drop-down list. The change methods are”

    • Manually Only - The gear switches only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear switches when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear switches when the control area priority reaches this priority.

  10. Click OK.

Master Cycle/Smart Cycle

To edit a Master Cycle or Smart Cycle gear:

  1. Change the name of the gear in the Gear Name text box.

  2. Select either Master Cycle or Smart Cycle from the Gear Type drop-down list.

  3. Set a different control percentage in the Control Percent box. This is the percentage of off time.

  4. Set a different cycle period in the Cycle Period box. This is the cycle period in minutes.

  5. Select a different cycle count send type from the Cycle Count Send Type drop-down list. The types are:

    • Fixed Count - The same number of cycle counts are sent to the switch.

    • Count Down - The number of cycles counts sent decreases at each resend.

    • Limited Count Down - Used with LCR 2000s, which do not allow cycle counts to be sent to them. The cycle count is hard coded in switch.

The purpose of Count Down and Limited Count Down is to end up with zero cycle counts at the end of control. This field does not display if the gear type is Master Cycle.

  1. If necessary, select a different maximum cycle count from the Max Cycle Cnt drop-down list. This limits the number of cycles counts that can be sent. This field does not display if the gear type is Master Cycle.

  2. Select a different starting period count from the Starting Period Count drop-down list. This is the count that is used when automatic control starts. This field does not display if the gear type is Master Cycle.

  3. Select a different resend rate from the Command Resend Rate drop-down list. This controls how often the cycle command is resent. This field does not display if the gear type is Master Cycle.

  4. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

    • Stop Cycle - The period count is set to 0, so the cycle stops the next period.

  5. Set a different group capacity reduction percentage in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much load to control.

  6. Select a different automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear switches only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear switches when the control area priority reaches this priority.

  7. Click OK.

Latching

To edit a Latching gear:

  1. Change the name of the gear in the Gear Name text box.

  2. Select Latching from the Gear Type drop-down list.

  3. Select a different starting state from the Control Start State drop-down list. This controls whether an open or close command is sent at the beginning of control.

  4. Set a different group capacity reduction percentage in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much load to control.

  5. Click OK.
