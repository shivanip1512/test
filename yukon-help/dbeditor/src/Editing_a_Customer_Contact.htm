Editing a Customer Contact

To edit a Customer Contact:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Customer.

  3. Click the + box next to a customer name to display the customer contacts.

  4. Double-click a customer contact. This opens the General tabbed page of the Customer Editor.
    You may make the following changes:

    1. Change names and phone numbers in the First Name, Last Name, Phone, and Direct Phone text boxes.

    2. Select a different recipient in the Recipient drop-down list. This recipient list allows you to select an e-mail address (associated with a notification recipient) that may, or may not, be specific to this contact person.

  5. Click the Login tab. The Login tabbed page appears.
    You may make the following changes:

    1. Select the Login Enabled check box to associate login information with a contact.

    2. Enter the user ID in the User ID text box.

    3. Enter the password in the Password text box.
      Enter the password again in the Retype Password text box.

    4. Select what the user has access to under the Type area. The options are:

      • Readmeter

      • Curtailment

      • Load Control

      • Energy Exchange

    5. Select the access options from the drop-down list. The options are:

      • Control

      • View-Only

  6. Click Apply to save the changes or OK to save the changes and close the dialog box.
