Creating a Billing File

The billing file wizard allows you to manually generate billing files in a variety of formats.

 

To create a billing file:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Billing File.

  3. Select the type of billing file format from the File Format drop-down list. The formats are:

    • SEDC - Southeastern Data Cooperative format.

    • CADP - Central Area Data Processing Cooperative format.

    • CADPXL2 - Central Area Data Processing Cooperative format.

    • WLT-40 - Westinghouse format.

    • CTI-CSV - Cannon Technologies format.

    • OPU - Owatonna Public Utilities format.

    • DAFFRON - Daffron & Associates, Inc. format.

    • NCDC - North Central Data Cooperative format.

    • CTI2 - Cannon Technologies Standard 2 format.

    • SEDC 5.4 - Southeastern Data Cooperative format.

    • NISC-Turtle - National Information Solutions Cooperative & Turtle format.

    • NISC-NCDC - NISC's & North Central Data Cooperative format.

    • NCDC-Handheld - North Central Data Cooperative Handheld format

    • NISC TOU (kVarH) - NISC's TOU format.

    • SIMPLE_TOU - Cannon Technologies format for use with electric data only with the option for a simple On-Peak and Off-Peak TOU/TOD rate

    • EXTENDED_TOU - Cannon Technologies format for use with multiple data types (electric, reactive, water, and gas) and optionally up to four TOU/TOD rates

  4. Select an end date for the billing file from the Billing End Date drop-down calendar.

  5. Enter the number of previous days to search for demand readings in the Demand Days Previous text box.

  6. Enter the number of previous days to search for energy readings in the Energy Days Previous text box.

  7. Select a billing group type from the Billing Group Type drop-down list. The billing group types are:

    • Collection Group

    • Alternate Group

    • Billing Group

  8. Depending on the billing group type selected, select a group from the list below Billing Group Type. The groups are defined on the Metering tabbed page of MCTs.

  9. If necessary, change the output directory path in the Output File Directory text box.

  10. To append the billing file to an existing file, select the Append to File check box.

  11. To generate the billing file without the point multiplier figured in, select the Remove Multiplier check box.

  12. Click Generate File to generate the billing file. The billing file is saved in the directory specified in the Output File Directory field.
