Grid Advisor General Tab

The following changes can be made to the general settings of the Grid Advisor device:

 

  1. The name of the Grid Advisor device can be changed in the Device Name text box.

  2. The serial number of the Grid Advisor device can be changed in the Serial Number text box.

  3. The Grid Advisor device can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to take the Grid Advisor device out of service. Clear this check box to enable the Grid Advisor device.

  4. If the communication channel is a dialup channel, the phone number that is used to dial the Grid Advisor device can be changed in the Phone Number text box.

To edit the advanced dialup properties of the Grid Advisor device:

    1. Click the Advanced button.

    2. The minimum and maximum connect times can be entered in the Minimum Connect Time and Maximum Connect Time boxes. The minimum time defines the amount of time following a successful communication that the port is not allowed to hang up. After minimum time expires, the port either remains connected until the maximum time is reached, or reverts to normal priority based on queuing and hangs up.

    3. A baud rate other than (Use Port Baud) can be selected from the Baud Rate drop-down list.

    4. Click the OK button to save the advanced dialup properties.
