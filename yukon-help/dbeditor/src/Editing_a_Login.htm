Editing a Login

A login is required to use the Yukon software and the permissions of a login are controlled by the login group(s) to which it is assigned.

 

Perform the following steps to edit a login on the Yukon system:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Login.

  3. Double-click on the login to edit. The General tabbed page is displayed in the Login Editor.

  4. Click the Role Groups tab to display the Role Groups tabbed page.

  5. Click the Apply button to save the changes to the login or click the OK button to save the changes to the login and close the Login Editor.
