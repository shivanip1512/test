Editing a Login Account

Login accounts are required to access the Yukon client software. To assign group permissions to a login, edit the login after creating it.

 

To edit a Login:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Login.

  3. Double-click a login account. This opens the General tab of the Login Editor.

  4. Click the Roles Group tab to display the Roles Group tabbed page.

  5. Click Apply to save the changes or OK to save the changes and close the Login Editor.
