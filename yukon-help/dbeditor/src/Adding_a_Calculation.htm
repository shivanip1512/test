Adding a Calculation

To add a calculation:

 

  1. Select the type of calculation to add from the Type list. The types of calculations are:

    • Operation – allows you to select a point for an operand and a basic calculation for the operation.

    • Constant – allows you to enter a constant number for the operand and a basic calculation for the operation.

    • Function – allows you to select a point for an operand and a function for the operation.

  2. If you selected Operation:

    1. Select a device from the Operand drop-down list

    2. Select a point associated with the device from the drop-down list below the Operand drop-down list.

    3. Select an operation from the Operation drop-down list. The operations are:

      +

      Adds the point values.

      -

      Subtracts the point values.

      *

      Multiplies the point values.

      /

      Divides the point values.

      Push

      Pushes the point values for use in functions.

  3. If you selected Constant:

    1. Enter a number in the Operand text box.

    2. Select an operation from the Operation drop-down list. The operations are:

      +

      Adds the constant to the point values.

      -

      Subtracts the constant from the point values.

      *

      Multiplies the constant to the point values.

      /

      Divides the constant from the point values.

      Push

      Pushes the constant for use in functions.

  1. If you selected Function:

    1. Select the Use Point check box if you want to use a point as the operand

    2. Select a device from the Operand drop-down list

    3. Select a point associated with the device from the drop-down list below the Operand drop-down list.

    4. Select an operation from the Operation drop-down list. The operations are:

      Absolute Value

      Calculates the absolute value of the operand value.

      Addition

      Adds the point operand values.

      ArcTan

       

      Calculates the arctangent of the operand.

      Baseline

      Allows a baseline to be selected.

      Baseline Percent

      Calculates a percentage of a baseline.

      Binary Encode

      Encodes multiple status point input values into a binary value, which is then converted into a decimal result.

      DemandAvg15

      Calculates the 15 minute average of multiple intervals of a demand point.

      DemandAvg30

      Calculates the 30 minute average of multiple intervals of a demand point.

      DemandAvg60

      Calculates the 60 minute average of multiple intervals of a demand point.

      Division

      Divides the point operand values.

      Float From 16Bit

      Combines two 16 bit words into a 32 bit float. To use this calculation, first push the least significant word (LSW), then push the most significant word (MSW), and then call the function Float From 16Bit without an operand.

      Geq than

      If the value is greater than or equal to the operand, a value of 1 is returned. If the value is not greater than or equal to the operand, a value of 0 is returned.

      Get Interval Minutes

      Returns the interval in minutes of the current calculation point. The Use Point option must be unchecked when this calculation is used.

      Get Point Limit

      Returns the requested limit of the given point. A value 1, 2, 3, or 4 is pushed before the function is called. Depending on the value pushed, the limits returned are High Limit 1, High Limit 2, Low Limit 1, and Low Limit 2 respectively. The Use Point option must be checked and a point must be included for this calculation.

      Greater than

      If the value is greater than the operand, a value of 1 is returned. If the value is not greater than the operand, a value of 0 is returned.

      Intervals to Value

      Performs a regression calculation based on an input point and returns the number of intervals until a value is reached. This calculation has several parameters that must be included:

       - First push a 1 if the limit to be reached is a high limit or 0 for a low limit.

       - Second, push a value for the depth of the regression.

       - Third, push the limit value to be reached.

       - Last, push the point value on which to perform the regression.

      KVa from KW/KQ

      Push KW, push KQ, and call the function KVa from KW/KQ without an operand.

      KVa from KW/KVar

      Push KW, push KVar, and call the function KVAR from KW/Var without an operand.

      KVar from KW/KQ

      Push KW, push KQ, and call the function KVAR from KW/KQ without an operand.

      KW from KVa/KVar

      Push KVa, push KVar, and call the function KW from KVa/KVar without an operand.

      Leq than

      If the value is less than or equal to the operand, a value of 1 is returned. If the value is not less than or equal to the operand, a value of 0 is returned.

      Less than

      If the value is less than the operand, a value of 1 is returned. If the value is not less than the operand, a value of 0 is returned.

      Linear Slope

      Performs a regression calculation based on an input point and returns the slope, either positive or negative. This calculation has several parameters that must be included:

       - First push a 1 to use historical data or 0 to not use historical data.

       - Second, push a value for the maximum depth of the regression

       - Third, push a value for the minimum depth of the regression

       - Last, push the point value on which to perform the regression.

      Logical AND

      If both values are non-zero, a value of 1 is returned.

      If either of the values are zero, a value of 0 is returned.

      Logical NOT

      If both values are zero, a value of 1 is returned.

      If neither of the values are zero, a value of 0 is returned.

      Logical OR

      If either of the values are non-zero, a value of 1 is returned.

      If neither of the values are non-zero, a value of 0 is returned.

      Logical XOR

      If both values are zero, a value of 0 is returned.

      If either of the values are non-zero, a value of 1 is returned.

      If both of the values are non-zero, a value of zero is returned.

      Max

      Calculates the maximum value of the Operands.

      Max Difference

      Calculates the maximum difference between three operands (A, B, and C). Max Difference = (|A-B| + |B-C| + |C-A|) / 2

      Mid Level Latch

      Continually compares status values and returns the last value until both status values are true. When both values are true a value of 1 is returned.

      Point 1 PUSH

      Point 2 PUSH

      FUNCTION Mid Level Latch

       

      Result =

      0 0 -> 0

      1 0 -> Last Value Calculated (no change)

      0 1 -> Last Value Calculated (no change)

      1 1 -> 1

      Min

      Calculates the minimum value of the Operands.

      Modulo Divide

      A modulo is the integer that remains after two integers are divided.

      Multiplication

      Multiplies the point operand values.

      P-Factor KW/KVar

      Push KW, push KVar and call the function Power Factor KW/KVar without an operand.

      P-Factor KW/KQ

      Push KW, push KQ and call the function Power Factor KW/KQ without an operand.

      P-Factor KW/KVa

      Push KW, push KVa and call the function Power Factor KW/KVa without an operand.

      Regression

      Performs a linear regression of a point. Pushed constants specify the number of values on which to base the calculation and number of minutes forward to calculate the value.

      Square Root

      Calculates the square root of the Operands.

      Squared

      Calculates the squared value of the Operands.

      State Timer

      Number of seconds since the state value has changed.

      Subtraction

      Subtracts the point operand values.

       

      True,False,Condition

      Returns different values depending on whether the condition is true or false.
