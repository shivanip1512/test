Creating a Dialout Pool

If you selected Dialout Pool:

  1. Click Next.

  2. Enter a name for the comm channel in the Description text box. This name is used by the system when it communicates error and information messages. The name is also used for comm channel status alarms.

  3. Click Next.

  4. To assign a comm channel to the dialout pool, select the comm channel from the Available list and click Add. To select more than one comm channel, hold the Ctrl key on the keyboard while clicking additional comm channels.

  5. To rearrange the order of the assigned comm channels, click and drag a comm channel to move it up or down in the list.

  6. To remove a comm channel from the dialout pool, select the comm channel from the Assigned list and click Remove.

  7. Click Finish to save the dialout pool. The new dialout pool appears in the Comm Channel Editor. For instructions on using this editor, refer to the related topics.
