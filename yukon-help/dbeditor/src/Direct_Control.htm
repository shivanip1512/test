Direct Control

Direct Control programs send controls to field hardware to reduce load.

 

If you selected Direct Control:

 

  1. Click the Add button to add a new gear.

 

Gears define the different rates at which a load program can controls loads. Different gear ratios result in different amounts of reduction.

The control gears are:

    • Time Refresh - timed commands are sent repeatedly to control.

    • Rotation - the gear rotates through a list of groups to control.

    • Master Cycle - the Master Station sends commands to the LCR to perform cycling.

    • Smart Cycle - the LCR hardware performs cycling.

    • TrueCycle - the LCR uses an algorithm to perform cycling.

    • Latching - one command is sent to open and another command is sent to close.

    • Thermostat Ramping - used to perform thermostat temperature ramping.

    • No Control - no control command is sent.

  1. Click Next.

  2. Select the Use Window 1 or Use Window 2 check box to activate one or both control windows.

  3. If you selected a control window check box, select a start time and a stop time for the control window from the Hr and Min drop-down lists.

  4. Click Next.

  5. To assign a load group to the load program, select the group from the Available Groups list and click Add. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.

  6. To unassign a load group to the load program, select the group from the Assigned Groups list and click Remove. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.

  7. Click Next.

  8. To send out a notification message before control starts, select the Program Start check box and enter how many minutes before control starts to send the message. If this check box is not selected, a notification message is not sent out when the control starts.

  9. To send out a notification message after control stops, select the Program Stop check box and enter how many minutes after control stops to send the message. If this check box is not selected, a notification message is not sent out when the control stops.

  10. To assign a notification group, select the group from the Notification Groups list and click Add. To select more than one notification group, hold the Ctrl key on the keyboard while clicking additional groups.

  11. To unassign a notification group, select the group from the Assigned Groups list and click Remove. To select more than one notification group, hold the Ctrl key on the keyboard while clicking additional groups.

  12. Click Finish. The new load program appears in the Program Editor. For instructions on editing a load program, refer to the related topics.
