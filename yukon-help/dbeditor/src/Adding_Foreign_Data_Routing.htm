Adding a Foreign Data Routing

To add foreign data routing:

 

  1. Select an interface from the Interface drop-down list. The available interfaces are: ACS, CYGNET, DSM2IMPORT, INET, RCCS, RDEX, STEC, SYSTEM, TRISTATE, TELGYR, and VALMET . The Data Parameters change depending on the type of interface you select.

  2. Select a direction from the Direction drop-down list.

  3. Depending on the Interface selected, different options appear under Device Parameters:

    • If you selected ACS, select Real or Pseudo from the Category drop-down list, and enter a remote name in the Remote text box and a point name in the Point text box.

    • If you selected CYGMET, enter a point id into the PointID text box.

    • If you selected DSM2IMPORT or VALMET, enter a point name into the Point text box.

    • If you selected INET or RCCS, enter a device name in the Device text box, a point name in the Point text box, and a destination/source in the Destination/Source text box.

    • If you selected RDEX, enter a translation name in the Translation text box and a destination/source in the Destination/Source text box.

    • If you selected STEC or TRISTATE, select a point from the Point drop-down list.

    • If you selected SYSTEM, enter a client name in the Client text box.

    • If you selected TELEGYR, enter a point name in the Point text box and select a group from the Group drop-down list.

  4. Click Add.
