Route General Tab

The following changes can be made to the general settings of the route:

 

  1. The name of the route can be changed in the Route Name text box.

  2. A different transmitter can be selected from the Signal Transmitter drop-down list.

  3. The route can be assigned as the default communication route by selecting the Default Route check box. Default route means that, if not otherwise specified, commands are sent on this route.

  4. If the signal transmitter is a CCU, the bus number can be changed in the Bus Number spin box. This defines the CCU port number used by this communication route.
