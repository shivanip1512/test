Route General Tab

The following changes can be made to the general settings of the route:

 

  1. The name of the route can be changed in the Route Name text box.

  2. A different transmitter can be selected from the Signal Transmitter drop-down list.

  3. The route can be assigned as the default by selecting the Default Route check box. Default route means that, if not otherwise specified, load control commands are sent on this route.

  4. If this is a CCU or Repeater route, the bus number can be changed in the Bus Number spin box. The bus number defines the substation bus number used by this route.
