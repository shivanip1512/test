Route General Tab

The following changes can be made to the general settings of a communication route:

 

  1. The name of the route can be changed in the Route Name text box.

  2. A different signal transmitter can be selected from the Signal Transmitter drop-down list.
    Note: The Signal Transmitter drop-down list only displays signal transmitters that match the type of transmitter that was selected when the route was created,

  3. The route can be assigned as the default communication route by selecting the Default Route check box. Unless specified otherwise, default routes are the routes that the Yukon system uses by default when it communicates with field devices.

  4. If the signal transmitter is a CCU, the bus number can be changed in the Bus Number box. This is the bus of the CCU that is used by the communication route.
