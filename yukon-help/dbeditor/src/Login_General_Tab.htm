Login General Tab

The following changes can be made to the general settings of a login:

  1. The login can be enabled or disable by selecting or clearing the Login Enabled check box. Select this check box to enable the login. Clear this check box to disable the login.

  2. The name of the login can be changed in the User Name text box.

  3. The authentication type of the login can be changed by selecting a different type from the drop-down list:

    • Normal – The password is stored as plain-text in the Yukon database. This is the type of authentication that is required for a Yukon STARS login.

    • Hashed – The password is converted to a sequence of letters and numbers and it is then stored in the Yukon database.

    • Radius – A Radius login server is used to authenticate passwords.

    • No Login – The password is disabled and the user cannot log in.

    • Active Directory – An Active Directory server is used to authenticate passwords.

    • LDAP – An LDAP server is used to authenticate passwords.

  4. If either Normal or Hashed is selected as the authentication type, the Change Password button is displayed, which allows the password to be changed.
    To change the password of the login:

    1. Click the Change Password button.

    2. Enter a password in the New Password and Retype Password text boxes.

    3. Click the OK button to save the password.

  5. To assign the login to an energy company, select the Link to Energy Company check box and select an energy company from the drop-down list.

To unassign the login from the energy company, clear the Link to Energy Company check box.
