Local Serial Port

If you selected Local Serial Port:

 

  1. Select one of the following communication line options:

    • Dedicated - communication line is connected to a leased line or data radio connection.

    • Dialup - communication line is connected to a modem which dials up a field device.

    • Dialback - communication line is connected to a modem on a field device which dials back to the master station.

    • Dialout Pool - a pool of dialup comm channels. To create a dialout pool, see Creating a Dialout Pool.

  2. Click Next.

  3. Enter a name for the comm channel in the Description text box. This name is used by the system when it communicates error and informational messages.

  4. Select the comm channel port from the Physical Port drop-down list. This defines the local hardware port used by the Yukon software.

  5. Select the baud rate of the port from the Baud Rate drop-down list. This is the communications line speed used by the port.

  6. The rest of this procedure depends on whether you selected Dedicated, Dialup, or Dialback as the type of communication line.

  • If you selected Dedicated, click Finish.
    The new communication channel appears in the Comm Channel Editor. For instructions on editing a communication channel, refer to the related topics.

  • If you selected Dialup or Dialback, click Next, select the type of modem being used from the drop-down list of modems, and then click Finish. The new communication channel appears in the Comm Channel Editor. For instructions on editing a communication channel, refer to the related topics.
