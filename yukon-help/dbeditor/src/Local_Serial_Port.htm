Local Serial Port

If Local Serial Port was selected as the port type:

 

  1. Select one of the following communication line options:

    • Dedicated – The communication line is connected to a leased line or data radio connection.

    • Dialup – The communication line is connected to a modem which dials up a field device.

    • Dialback – The communication line is connected to a modem on a field device which dials back to the master station.

    • Dialout Pool – The communication line is a pool of dialup comm channels. To create a dialout pool, see Creating a Dialout Pool.

  2. Click the Next button.

  3. Enter a name for the comm channel in the Description text box. This comm channel name that the Yukon system uses when it communicates error and informational messages.

  4. Select the comm channel port from the Physical Port drop-down list. This is the local hardware port used by the Yukon system.

  5. Select the baud rate of the port from the Baud Rate drop-down list. This is the communication line speed of the comm channel.

  6. The rest of this procedure depends on the type of communication line selected in step 1.

  • If the communication line is Dedicated, click the Finish button. The new communication channel appears in the Comm Channel Editor.

  • If the communication line is Dialup or Dialback:

    1. Click the Next button.

    2. Select the type of modem that is used by the communication line from the Select the type of Modem drop-down list.

    3. Click the Finish button. The new communication channel appears in the Comm Channel Editor.
