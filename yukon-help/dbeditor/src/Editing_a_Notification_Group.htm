Editing a Notification Group

A notification group controls who receives notifications and the information sent in the notification.

 

To edit a notification group:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Notification Group.

  3. Double-click on the notification group to modify. This opens the General tabbed page of the Notification Group editor. You may make the following changes:

    1. The name of the group can be changed in the Name text box.

    2. The group can be disabled or enabled by selecting or clearing the Disable Group check box. Select this check box to disable the group. Clear this check box to enable the group.

  4. Click the Notifications tab to display the Notifications tabbed page appears, from which customers and contacts can be assigned to or removed from the notification group.

    1. To assign a customer to a notification group, select the check box next to the customer name.

    2. To assign a contact to a notification group, click the + next to the customer name and select the check box next to the contact name.

    3. To send email notifications to the customer contact(s), select the Send Emails check box.

    4. If the Yukon system includes an Interactive Voice Response (IVR) server, select the Make Phone Calls check box to call the customer contact and transmit the notification over the phone.

  5. Click the Apply button to save the changes or OK button to save the changes and close the Notification Group Editor.
