Editing a Notification Group

A notification group defines the recipients of system notifications.

 

Perform the following steps to edit a notification group:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Notification Group.

  3. Double-click on the notification group to edit. The General tabbed page is displayed in the Notification Group Editor, from which the following changes can be made:

    1. The name of the notification group can be changed in the Name text box.

    2. The notification group can be disabled or enabled by selecting or clearing the Disable Group check box. Select this check box to disable the notification group. Clear this check box to enable the notification group.

  4. Click the Notifications tab to display the Notifications tabbed page, from which customers and contacts can be assigned to or removed from the notification group.

    1. To assign a customer to a notification group, select the check box next to the customer name.
      To assign a contact to a notification group, click the + next to the customer name and select the check box next to the contact name.

    2. To send email notifications to the customer contact(s), select the Send Emails check box.

    3. If the Yukon system includes an Interactive Voice Response (IVR) server, select the Make Phone Calls check box to instruct the IVR server to call the customer contact and transmit the notification over the phone.

  5. Click the Apply button to save the changes to the notification group or click the OK button to save the changes to the notification group and close the Notification Group Editor.
