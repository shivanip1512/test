Adding a Load Program

Perform the following steps to add a load program to the control area:

 

  1. Select the load program to assign to the control area from the Program drop-down list. This is a list of the available load programs in the Yukon system.

  2. Set a start priority for the load program in the Assigned Start Priority box. A program with a start priority of one starts before a program with a start priority of two.

  3. Set a stop priority for the load program in the Assigned Stop Priority box. A program with a stop priority of one stops before a program with a stop priority of two.

  4. Click the Add button. The load program is added to the Assigned Programs list of the control area.
