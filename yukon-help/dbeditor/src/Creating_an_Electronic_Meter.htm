Creating an Electronic Meter

Perform the following steps to create an electronic meter:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Device.

  3. Select Electronic Meters from the list on the left.

  4. Select the type of electronic meter to create from the list on the right:

    • ALPHA POWER PLUS – Elster ALPHA Plus® (A2) meter.

    • ALPHA A1 – Elster ALPHA® (A1) meter.

    • FULCRUM – Itron FULCRUM® meter.

    • ION-7330 – Power Measurement/PowerLogic ION 7330™ meter.

    • ION-7700 – Power Measurement/PowerLogic ION 7700™ meter.

    • ION-8300 – Power Measurement/PowerLogic ION 8300™ meter.

    • Landis+Gyr S4 – Landis+Gyr S4/S4e meter.

    • QUANTUM – Itron QUANTUM® meter.

    • SENTINEL – Itron SENTINEL® meter.

    • FOCUS – Landis+Gyr FOCUS® meter.

    • TRANSDATA MARK-V – TransData MARK-V Energy Meter.

    • VECTRON – Itron VECTRON® meter.

  5. Click the Next button.

  6. Enter a name for the meter in the Device Name text box.

If the meter is an ION meter, enter the slave address of the meter in the Slave Address text box.

  1. Click the Next button.

  2. Enter the meter number of the electronic meter in the Meter Number text box. This is the meter number assigned by the utility.

  3. Click the Next button.

  4. Select the check boxes of the scan categories to enable. The scan categories vary depending on the type of electronic meter selected in step 4.

  • If the meter is an ION meter, the following scan categories are displayed:

      • Integrity Rate – Collects all point values from the electronic meter.

      • General Data Collection – Collects general data from the electronic meter.

      • Eventlog Collection – Collects event logs from the electronic meter.

  • If the meter is not an ION meter, the following scan category is displayed:

      • General Data Collection – Collects general data.

When the check box of a scan category is selected, the following scan options are enabled:

    • Interval – Default scan interval of the selected scan category.

    • Alt Interval – Alternate scan interval of the selected scan category.

    • Group – Scan group of the selected scan category. The scan group is used to assign priorities to the scan categories.

  1. Under the Optional Scan Window, select the Enabled check box to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the time at which the scan window starts in the Open text box.

    3. Enter the time at which the scan window stops in the Close text box.

  2. Click the Next button.

  3. Select the comm channel for the electronic meter from the Select a Communication Channel drop-down list.

The rest of this procedure depends on the type of comm channel selected.

  • If the selected comm channel is Dedicated, click the Finish button.

The new electronic meter appears in the Device Editor. For instructions on editing an electronic meter, refer to the related topics.

  • If the selected comm channel is Dialup or Dialback:

    1. Click the Next button.

    2. Enter the phone number used to dial this electronic meter or dialback from this electronic meter in the Phone Number text box.

    3. Click the Finish button.

The new electronic meter appears in the Device Editor. For instructions on using this editor, refer to the related topics.
