Creating an Electronic Meter

If you selected Electronic Meter as the device type:

 

  1. Enter a name for the meter in the Device Name text box.

    For ION meters, enter the slave address of the meter in the Slave Address text box, which appears below the Device Name text box.

  2. Click Next.

  3. Enter the meter number in the Meter Number text box. This meter number is assigned by the utility.

  4. Click Next.

  5. Select the check boxes of the scan categories you want to enable. The scan categories vary depending on the type of signal transmitter selected.

If you selected an ION device, a dialog box appears with the following scan categories:

    • Integrity Rate - collects all point values

    • General Data Collection - collects general data

    • Eventlog Collection - collects the devices event log

If you selected one of the other devices, a dialog box appears with one scan category:

    • General Data Collection - collects general data

    1. Select the scan interval and alternative scan interval from the Interval and Alt Interval drop-down lists.

    2. Select a scan group or enter a new scan group in the Group drop-down list. This is used to group similar scans together.

  1. Under the Optional Scan Window, select the Enabled check box if you want to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the open and close times in the Open and Close text boxes.

  2. If you selected SIXNET as the meter device, click Next and perform steps a and b. Otherwise, skip to step 8.

    1. Enter the name of the SIXNET log file in the Log File text box. The name of the log file must include the .6dl extension.

    2. Enter the SIXNET station address in the Station Address text box.

  3. Click Next.

  4. Select a comm channel from the drop-down list of comm channels

  5. The rest of this procedure depends on the type of comm channel selected.

  • If the comm channel is dedicated, click Finish. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.

  • If the comm channel is dialup or dialback, click Next, enter the phone number used to dial this device or dialback from the device, and then click Finish. The new device appears in the Device Editor. For instructions on editing a device, refer to the related topics.
