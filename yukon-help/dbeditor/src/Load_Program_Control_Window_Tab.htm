Load Program Control Window Tab

The following changes can be made to the control windows settings of the load program:

 

  • If a control window 1 is enabled, a different start time and a stop time can be selected from the Hr and Min drop-down lists. To disable control window 1, clear the Use Window 1 check box.

  • If a control window 2 is enabled, a different start time and a stop time can be selected from the Hr and Min drop-down lists. To disable control window 2, clear the Use Window 2 check box.
