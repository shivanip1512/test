Load Program Control Window Tab

The following changes can be made to the control windows settings of the load program:

  • If the Use Window 1 check box is selected:

    • A different start time for the control window can be selected from the Hr and Min drop-down lists in the Start area

    • A different stop time for the control window can be selected from the Hr and Min drop-down lists in the Stop area.

    • To disable control window 1, clear the Use Window 1 check box.

  • If the Use Window 2 check box is selected:

    • A different start time for the control window can be selected from the Hr and Min drop-down lists in the Start area

    • A different stop time for the control window can be selected from the Hr and Min drop-down lists in the Stop area.

    • To disable control window 2, clear the Use Window 2 check box.
