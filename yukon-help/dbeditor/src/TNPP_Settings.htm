Signal Transmitter TNPP Settings

The following changes can be made to the TNPP settings of a TNPP terminal:

 

  1. The source address, which indicates the node from which the packet originated, can be changed in the Origin Address text box.

  2. The destination address, which indicates the node that processes the transactions in the packet, can be changed in the Destination Address text box.

  3. The inertia value, which is the maximum number of nodes that the packet is allowed to pass through before it reaches the destination, can be changed in the Inertia text box.

  4. A different protocol, which is used for sending pages, can be selected from the Protocol drop-down list.

  5. A different data format, which is the format of the data used for sending pages, can be selected from the Data Format drop-down list.

  6. A different identifier format, which is the type of format used for sending pages, can be selected from the Identifier Format drop-down list.

    • The CAP page format contains all the raw information for sending a message to a pager.

    • The ID page format only contains the pager's PIN and it must be translated at a paging terminal.

  7. The pager ID, which is the ID of the outgoing pages, can be changed in the Pager ID text box.

  8. The RF channel designator can be changed in the Channel text box.

  9. The function code can be changed in the Function Code text box.

  10. The RF zone designator can be changed in the Zone text box.
