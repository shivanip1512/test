Comm Channel Modem Tab

The following changes can be made to the modem settings of the communication channel:

 

  1. A different modem can be selected from the Modem Type drop-down list. This setting defines the type of modem being used by this comm channel. The supported modems are:

    • U.S. Robotics Sportster

    • U.S. Robotics Courier

    • Motorola

    • Telenetics

  1. The initialization string, which is used to initialize and configure the modem settings, can be changed in the Initialization String text box.

  2. If the comm channel type is Local Dialup:

    1. The phone number prefix can be changed in the Prefix Number text box. Common values are 9 or 1.

    2. The phone number suffix can be changed in the Suffix Number text box. Common values are commas and numbers.
