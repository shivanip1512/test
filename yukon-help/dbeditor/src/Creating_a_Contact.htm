Creating a Contact

To create a contact:

 

  1. From the View menu, select System.

  2. From the Create menu, select Contact.

  3. Enter the first name of the contact in the First Name text box.

  4. Enter the last name of the contact in the Last Name text box.

  5. To assign a login to the contact, select a login from the Yukon Login drop-down list.

  6. Select the type of notification from the Notification Type drop-down list. The notification types are:

    • Email

    • Phone Number

    • Email to Pager

    • Fax Number

    • Home Phone

    • Work Phone

    • Voice PIN

    • Cell Phone

    • Email to Cell

    • Call Back Phone

    • IVR Login

    1. Depending on the notification type, enter either the contact’s e-mail address or phone number into the Address text box.

    2. To disable the notification address entered, select the Disable Usage check box.

    3. Click Add. The contact information appears in the table at the bottom of the dialog box. To remove a type of notification for the contact, select the notification and click Remove.

  7. Click Next.

  8. Enter the address in the Address text boxes.

  9. Enter the city in the City text box.

  10. Enter the county in the County text box.

  11. Select the state from the State drop-down list.

  12. Enter the zip code in the Zip text box.

  13. Click Finish. The new contact appears in the Contact Editor. For instructions on editing a contact, refer to the related topics.
