Editing a Contact

A contact is a person who can be notified of events occurring on the Yukon system.

 

Perform the following steps to edit a contact:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Contacts.

  3. Double-click on the contact to edit. The General tabbed page is displayed in the Contact Editor, from which the following changes can be made

    1. The first name of the contact can be changed in the First Name text box.

    2. The last name of the contact can be changed in the Last Name text box.

    3. A different Yukon login account can be selected from the Yukon Login drop-down list.

    4. To add a notification e-mail or phone number for the contact:

      1. Select the type of notification from the Notification Type drop-down list. The notification types are:

        • Email

        • Phone Number

        • Email to Pager

        • Fax Number

        • Home Phone

        • Work Phone

        • Voice PIN

        • Cell Phone

        • Email to Cell

        • Call Back Phone

        • IVR Login

      2. Depending on the notification type selected in the previous step, enter either the e-mail address, phone number, or pin/login of the contact into the Address text box.

      3. To prevent the e-mail address or phone number from being used for notification purposes, select the Disable Usage check box.

      4. Click the Add button. The notification information is added to the list at the bottom of the Contact Editor.
        To remove notification information from the contact, select the notification information to remove from the list and click the         Remove button.

  4. Click the Address tab to display the Address tabbed page, from which the following changes can be made:

    1. The address of the contact can be changed in the Address text boxes.

    2. The city in which the contact is located can be changed in City text box.

    3. The county in which the contact is located can be changed in the County text box.

    4. The state in which the contact is located can be changed by selecting a different state from the State drop-down list.

    5. The zip code of the contact can be changed in the Zip text box.

  5. Click the Apply button to save the changes to the contact or click the OK button to save the changes to the contact and close the Contact Editor.
