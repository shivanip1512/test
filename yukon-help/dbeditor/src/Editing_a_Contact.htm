Editing a Contact

A contacts is a person who can be notified of events occurring on the Yukon system.

 

To edit a contact:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Contacts.

  3. Double-click on the contact to modify. This opens the General tabbed page of the Contact editor, from which the following changes can be made

    1. The contact's first name can be changed in the First Name text box.

    2. The contact's last name can be changed in the Last Name text box.

    3. A different login can be selected from the Yukon Login drop-down list.

    4. To add a notification e-mail or phone number for the contact:

      1. Select the type of notification from the Notification Type drop-down list. The notification types are:

        • Email

        • Phone Number

        • Email to Pager

        • Fax Number

        • Home Phone

        • Work Phone

        • Voice PIN

        • Cell Phone

        • Email to Cell

        • Call Back Phone

        • IVR Login

      2. Depending on the notification type, enter either the contact’s e-mail address, phone number, or pin/login into the Address text box.

      3. To disable the notification address entered, select the Disable Usage check box.

      4. Click Add. The contact's notification information appears in the list.

    5. To remove a notification for the contact, select the notification from the list and click Remove.

  4. Click the Address tab to display the Address tabbed page, from which the following changes can be made:

    1. The address of the contact can be changed in the Address text boxes.

    2. The city can be changed in City text box.

    3. The county can be changed in the County text box.

    4. A different state can be selected from the State drop-down list.

    5. The zip code can be changed in the Zip text box.

  5. Click Apply to save the changes or OK to save the changes and close the Contact Editor.
