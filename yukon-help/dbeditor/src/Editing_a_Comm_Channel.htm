Editing a Comm Channel

A comm channel is needed to create a signal transmitter, electronic meter, or RTU.

 

Perform the following steps to edit a comm channel:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select Comm Channel.

  3. Double-click on the comm channel to edit. The General tabbed page is displayed in the Comm Channel Editor.

If the comm channel is Dialout Pool, see Editing a Dialout Pool to edit the comm channel.

  1. Click the Timing tab to display the Timing tabbed page.

  2. Click the Modem tab to display the Modem tabbed page.

  3. Click the Shared tab to display the Shared tabbed page.

  4. Click the Apply button to save the changes to the comm channel or click the OK button to save the changes to the comm channel and close the Comm Channel Editor.
