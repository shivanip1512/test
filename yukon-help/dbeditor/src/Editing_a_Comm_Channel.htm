Editing a Comm Channel

A comm channels is needed to create devices. The exchange of data between Yukon and field devices is performed through a comm channel.

 

To edit a Comm Channel:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select Comm Channel.

  3. Double-click the comm channel to modify. This opens the General tabbed page of the Device Editor. To edit a Dialout Pool comm channel, see Editing a Dialout Pool.

  4. Click the Timing tab to display the Timing tabbed page.

  5. Click the Modem tab to display the Modem tabbed page.

  6. Click the Shared tab to display the Shared tabbed page.

  7. Click the Apply button to save the changes or OK button to save the changes and close the Comm Channel Editor.
