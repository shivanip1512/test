Point Group Tab

The following changes can be made to the control point settings of a Point Group:

 

  1. A different device on which the control point is attached can be selected from the Control Device drop-down list.

  2. A different point on the device that performs the control can be selected from the Control Point drop-down list.

  3. A different state the point needs to be in to start control can be selected from the Control Start State drop-down list.

  4. Click Apply to save the changes or OK to save the changes and close the Load Group Editor.
