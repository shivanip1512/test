Point Group Tab

The following changes can be made to the control point settings of a Point Group:

 

  1. A different control device can be selected from the Control Device drop-down list. This is the device on which the control point is attached.

  2. A different control point can be selected from the Control Point drop-down list. This is the point that performs the control.

  3. A different start state can be selected from the Control Start State drop-down list. This is the state the point needs to be in to start control.

  4. Click Apply to save the changes or OK to save the changes and close the dialog box.
