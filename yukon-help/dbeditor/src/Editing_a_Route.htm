Editing a Route

A route is defined as a communication path to a field device. These paths are used by the system to control different types of communication on the distributed on the system. All routes originate from a signal transmitter defined in the system.

 

Perform the following steps to edit a route:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select Route.

  3. Double-click on the route to edit. The General tabbed page is displayed in the Route Editor.

If the route is a macro route, see Editing a Macro Route to edit the route.

  1. If the route is for a repeater, click the Repeater Setup tab to display the Repeater Setup tabbed page.

  2. Click the Apply button to save the changes to the route or click the OK button to save the changes to the route and close the Route Editor.
