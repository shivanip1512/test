Editing a Login Group

The Login Group Editor contains a General tab and a Roles tab.

To modify a login group:

  1. From the View menu, select System.

  2. From the Sort By list, select Login Group.

  3. Double-click the login group to modify. This opens the General tab of the Login Group Editor.

  4. Click the Roles tab. The Roles tabbed page appears.

  5. Click Apply to save the changes or OK to save the changes and close the dialog box.
