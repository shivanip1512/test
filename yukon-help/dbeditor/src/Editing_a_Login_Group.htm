Editing a Login Group

Login groups control the permissions of a Yukon login.

Perform the following steps to edit a login group:

  1. From the View menu, select System.

  2. From the Sort By list, select Login Group.

  3. Double-click on the login group to edit. The General tabbed page is displayed in the Login Group Editor.

  4. Click the Roles tab to display the Roles tabbed page.

  5. Click the Apply button to save the changes to the login group or click the OK button to save the changes to the login group and close the Login Group Editor.
