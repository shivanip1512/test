Creating an Accumulator Point

If you selected Accumulator as the point type:

 

  1. Select a measurement unit from the Unit of Measure drop-down list. The unit of measure is for the data being read.

  2. Enter a multiplier in the Multiplier text box. The multiplier converts pulses into engineering units and is used to calculate the unit value.

  3. Enter a data offset in the Data Offset text box. This is associated with metering and is added to the reading during calculation.

  4. Enter the number of dials/digits displayed in the meter reading in the Meter Digits (Dials) spin box. This allows a rollover to occur when the reading passes the specified number of digits.

  5. Select either Dial or Demand as the reading type. A Dial (Pulse) reading measures the total amount of electrical energy consumed. A Demand reading measures the maximum amount of power drawn through the meter during a specific billing period.

  6. Click Next.

  7. If the system is to obtain physical data from the hardware or if it is a special Yukon point, select the Physical Point Offset check box. Clear this check box. Clear this check box for a pseudo point.

  8. If Physical Point Offset check box is selected, set the offset value in the Point Offset spin box. This is the physical input used for wiring.

  9. Click Finish. The new point appears in the Point Editor. For instructions on editing a point, refer to the related topics.
