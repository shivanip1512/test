Creating an Accumulator Point

If Accumulator was selected as the point type:

 

  1. Select a measurement unit from the Unit of Measure drop-down list.

  2. Enter a multiplier in the Multiplier text box. The multiplier converts pulses into engineering units and it is used to calculate the unit value.

  3. Enter a data offset in the Data Offset text box. This is associated with metering and is added to the reading during calculation.

  4. Enter the number of dials/digits to display for the meter reading in the Meter Digits (Dials) box. When the meter reading surpasses the specified number of digits, the reading rolls over to zero.

  5. Select type of reading in the Reading Type area. The reading types are:

    • Dial (Pulse) – Measures the total amount of electrical energy consumed.

    • Demand – Measures the maximum amount of power drawn through the meter during a specific billing period.

  6. Click the Next button.

  7. If the Yukon system obtains physical data from the device on which the point is located or the point is a special Yukon system point, select the Physical Point Offset check box. Clear this check box if this is a pseudo point.

  8. If the Physical Point Offset check box is selected, the point offset can be changed in the Point Offset spin box. This is either the physical input to which the point is wired or a Yukon system defined point.

  9. Click the Finish button. The new point appears in the Point Editor.
