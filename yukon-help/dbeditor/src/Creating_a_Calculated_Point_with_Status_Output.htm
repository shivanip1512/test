Creating a Calculated Point with Status Output

If you selected Calculated - Status Output as the point type:

 

  1. Select the group that defines the possible states for the point from the State Group drop-down list. The state group defines the messages to display for the different states of a point.

  2. Select the initial state of the state group from the Initial State drop-down list. The initial state depends on the state group selected.

  3. Click Next.

  4. Select an calculation update type from the Update Type list. The update types are:

    • On First Change – performs the calculation whenever any point in the calculation changes.

    • On All Change – performs the calculation only when all of the points in the calculation change.

    • On Timer – performs the calculation after the timer expires.

    • On Timer + Change – performs the calculation only after the timer expires and all of the points in the calculation change.

    • Constant – returns a constant value.

    • Historical – totalizes when all specified points are available.

  5. If On Timer or On Timer + Change is selected as the update type, select a rate from the Periodic Rate drop-down list. This is the interval that controls how often a calculation is performed.

  6. Click Finish. The new point appears in the Point Editor. For instructions on editing a point, refer to the related topics.
