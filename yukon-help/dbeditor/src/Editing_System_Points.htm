Editing System Points

System points allow you to monitor the status of the Yukon application servers. The following system points are pre-defined:

 

  • Calc Monitor

  • Cap Control Monitor

  • Dispatch Monitor

  • FDR Monitor

  • MACS Monitor

  • Porter Monitor

  • Porter Work Count

  • Scanner Monitor

 

To view the status of the system points, create a custom display in the Yukon TDC application and add the system points from the System Device object type.

 

Alarms can be assigned to the system points by performing the steps below. If a system point goes into an alarm condition, an alarm appears in the Yukon TDC application. The following changes can be made to the alarm settings:

 

  1. A different notification group can be selected from the Group drop-down list. This is the group that is notified of all alarm conditions.

  2. A different customer contact can be selected from the Contact drop-down list. This is the contact who is notified of all alarm conditions.

  3. To create a new contact, click the Create new button.

    1. Enter the name and e-mail address of the new contact in the First Name, Last Name, and E-mail Address text boxes.

    2. Click OK to save the new contact and return to the Alarms tabbed page.

  4. The option to be notified when an alarm is acknowledged can be enabled or disabled by selecting or clearing the Acknowledge check box. If this check box is selected, the notification group and contact receive a message when an alarm is acknowledged.

  5. The option to be notified when an alarm is cleared can be enabled or disabled by selecting or clearing the Clear check box. If this check box is selected, the notification group and contact receive a message when an alarm is cleared.

  6. Different alarm categories for each alarm condition can be selected from the drop-down lists in the Category column. Alarm categories can be edited in the System view of the Database Editor.

  7. Notification attributes can be selected from the Notify drop-down list for each alarm state row:

    • None - the notification is sent out normally.

    • Exclude Notify - the notification group/contact do not receive notification of the alarm condition.

    • Auto Ack - the alarm is automatically acknowledged.

    • Exclude Notify & Auto Ack - the notification group/contact do not receive notification of the alarm condition and the alarm is automatically acknowledged.

  8. The option to disable all alarms can be enabled or disabled by selecting or clearing the Disable All Alarms check box. If this check box is selected, all of the alarms for this point are disabled.
