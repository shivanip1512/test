Editing System Points

System points allow the Yukon system to monitor the status of Yukon application servers. The following system points are pre-defined:

  • Calc Monitor

  • Cap Control Monitor

  • Dispatch Monitor

  • FDR Monitor

  • MACS Monitor

  • Porter Monitor

  • Porter Work Count

  • Scanner Monitor

 

To view the status of the system points, create a custom display in the Tabular Data Console application and add the system points from the System Device object type.

In addition, alarms can be assigned to the system points. If a system point goes into an alarm condition, an alarm appears in the Tabular Data Console application.

 

To assign an alarm to a system point, complete the following steps on the Alarms tab of a system point:

 

  1. From the View menu, select System.

  2. From the Sort By list, select System Points.

  3. Double-click on the system point edit. The General tabbed page is displayed in the Point Editor.

  4. Click on the Alarms tab to display the Alarms tabbed page.

  5. A notification group can be selected from the Group drop-down list. This is the notification group that is notified of all alarm conditions.

  6. The option to be notified when an alarm is acknowledged can be enabled or disabled by selecting or clearing the Acknowledge check box. If this check box is selected, the notification group receives a message when an alarm is acknowledged.

  7. The option to be notified when an alarm is cleared can be enabled or disabled by selecting or clearing the Clear check box. If this check box is selected, the notification group receives a message when an alarm is cleared.

  8. An alarm category for each alarm condition can be selected from the drop-down lists in the Category column. The notification group within the alarm category is notified when the alarm condition occurs. Alarm categories can be edited in the System view of the Database Editor.

  9. Notification attributes can be selected from the Notify drop-down list for each alarm state row:

    • None – The notification is sent out normally to the notification group of the alarm.

    • Exclude Notify – The notification group of the alarm does not receive notification of the alarm condition.

    • Auto Ack – The alarm is automatically acknowledged.

    • Exclude Notify & Auto Ack – The notification group of the alarm does not receive notification of the alarm condition and the alarm is automatically acknowledged.

  10. The option to disable all alarms can be enabled or disabled by selecting or clearing the Disable All Alarms check box. If this check box is selected, all of the alarms for this point are disabled.

  11. Click the Apply button to save the changes to the system point or click the OK button to save the changes to the system point and close the Point Editor.
