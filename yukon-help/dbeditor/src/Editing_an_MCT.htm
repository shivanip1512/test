Editing an MCT

Perform the following steps to edit an MCT:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select one of the following:

    • Device

    • MCT

    • Meter Number

  3. Double-click on the MCT to edit. The General tabbed page is displayed in the Device Editor.

  4. Click the Metering tab to display the Metering tabbed page.

  5. Click the Scan Rate tab to display the Scan Rate tabbed page.

  6. If the device is an MCT-410, click the MCT Settings tab to display the MCT Settings tabbed page.

  7. If the device is an MCT-360 or MCT-470, click the IED Port tab to display the IED Port tabbed page.

  8. Click the Apply button to save the changes to the MCT or click the OK button to save the changes to the MCT and close the Device Editor.
