Creating a TOU Schedule

TOU Schedules define the rates to use and when to use the rates.

A TOU Schedule can be downloaded to an MCT by using the Yukon Commander application.

To create a TOU schedule:

  1. From the View menu, select System.

  2. From the Create menu, select TOU Schedule.

  3. Enter a name for the TOU schedule in the Schedule Name text box.

  4. Select the default rate of the TOU schedule from the Default Rate drop-down list. This is the rate that is used if no rate is specified in the schedule.

  5. Select a TOU Day to edit from any day of the week drop-down list.

    1. The name of the TOU Day can be changed in the Name text box.

    2. To add additional rates to the TOU Day, enter a time of day in the text box and select a rate from the drop-down list. Note that a TOU Day always starts at midnight (00:00) and the rate for the midnight hour can be changed from the drop-down list.

    3. Click Save Day Settings to save the changes to the TOU Day.

  6. Select the TOU Days to use for each day of the week.

  7. Click Finish. The new TOU schedule appears in the TOU Editor. For instructions on editing a TOU schedule, refer to the related topics.
