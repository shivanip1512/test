Creating a TOU Schedule

Perform the following steps to create a TOU schedule:

  1. From the View menu, select System.

  2. From the Create menu, select TOU Schedule.

  3. Enter a name for the TOU schedule in the Schedule Name text box.

  4. Select the default rate of the TOU schedule from the Default Rate drop-down list. This is the rate that is used if no rate is specified in the schedule.

  5. To edit the settings of a TOU day in the TOU Day Editor, select the TOU day to edit from one of the day week drop-down lists. For example, select Daily1 from the Sun: drop-down list.

    1. The name of the TOU Day can be changed in the Name text box.

    2. To add additional rates to the TOU Day, enter a time of day in the text box and select a rate from the drop-down list.
      Note: A TOU Day always starts at midnight (00:00) and the rate for the midnight hour can be changed from the drop-down list.

    3. Click the Save Day Settings button to save the changes to the TOU Day.

  6. Select the TOU Day to use for each day of the week (Sun, Mon, Tue, Wed, Thu, Fri, and Sat).

  7. Click the Finish button. The new TOU schedule appears in the TOU Editor. For instructions on editing a TOU schedule, refer to the related topics.
