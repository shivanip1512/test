Load Program Member Control Tab

The following changes can be made to the member control settings of the load program:

Note: The Member Control tab is visible only if the allow_member_programs role property within the Database Editor role is set to true.

The Member Control tab allows the load program to be configured as a master load program. Subordinate load programs are then assigned to the master load program. When a master load program is running, subordinate load programs cannot be started. When a subordinate load program is running and the master load program is started, the master load program takes over control for the subordinate load programs.

To enable member control, select the Allow Member Control check box and assign the subordinate load programs to the master load program.

  • To assign a subordinate load program to a master program, select a load program from the Available Subordinates list and click the Add button. To select more than one load program, hold Ctrl key on the keyboard while clicking additional load programs.

  • To rearrange the order of the assigned load programs, click and drag a load program to move it up or down in the list.

  • To unassign a subordinate load from group from a master program, select a load program from the Assigned Subordinates list and click the Remove button. To select more than one load program, hold the Ctrl key on the keyboard while clicking additional load programs.

To disable member control, clear the Allow Member Control check box.
