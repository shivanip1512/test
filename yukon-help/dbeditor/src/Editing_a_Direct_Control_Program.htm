Editing a Direct Control Program

The following changes can be made to a direct control program:

 

  1. Click the Gears tab.
    Gears define the different rates at which a load program can controls loads. Different gear ratios result in different amounts of reduction.

    • To add a gear, click the Add button. To edit a gear, select the gear to edit from the Gear drop-down list. The control gears that can be added or edited are:

      • Time Refresh - timed commands are sent repeatedly to control.

      • Rotation - the gear rotates through a list of groups to control.

      • Master Cycle - the Master Station sends commands to the LCR to perform cycling.

      • Smart Cycle - the LCR hardware performs cycling.

      • TrueCycle - the LCR uses an algorithm to perform cycling.

      • Latching - one command is sent to open and another command is sent to close.

      • Thermostat Ramping - used to perform thermostat temperature ramping.

      • No Control - no control command is sent

    • To remove a gear, see Removing a Gear.

  2. Click the Control Window tab.

    1. The control windows can be enabled or disabled by selecting or clearing the Use Window 1 or Use Window 2 check boxes.

    2. If a control window is enabled, a different start time and stop time can be selected from the Hr and Min drop-down lists.

  3. Click the Groups tab. You may assign load groups to or unassign load groups from the load program.

    • To assign a load group, select the group from the Available Load Groups list and click Add. To select more than one group, hold Ctrl key on the keyboard while clicking additional groups.

    • To rearrange the order of the assigned load groups, click and drag a load group to move it up or down in the list.

    • To unassign a group, select the group from the Assigned Load Groups list and click Remove. To select more than one group, hold the Ctrl key on the keyboard while clicking additional groups.

  4. Click the Notification tab. You may assign notification groups to or unassign notification groups from the load program.

    • To send out a notification message before control starts, select the Program Start check box and enter how many minutes before control starts to send the message. If this check box is not selected, a notification message is not sent out when the control starts.

    • To send out a notification message after control stops, select the Program Stop check box and enter how many minutes after control stops to send the message. If this check box is not selected, a notification message is not sent out when the control stops.

    • To assign a notification group, select the group from the Notification Groups list and click Add. To select more than one group, hold Ctrl key on the keyboard while clicking additional groups.

    • To unassign a notification group, select the group from the Assigned Groups list and click Remove. To select more than one group, hold the Ctrl key on the keyboard while clicking additional groups.
