Editing a Holiday Schedule

A holiday schedule contains the dates of holidays for the year. Assigning a holiday schedule to a program prevents that program from running on holidays.

 

To edit a holiday schedule:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Holiday Schedule.

  3. Double-click on the holiday schedule to modify. This opens the General tabbed page of the Holiday Schedule Editor.

  4. The name of the holiday schedule can be changed in the Schedule Name text box.

  5. Under Assigned Holidays is a list of holidays created for this holiday schedule.

    • To create a holiday, see Creating a Holiday.

    • To edit a holiday, see Editing a Holiday.

    • To remove a holiday, see Removing a Holiday.

  6. Click the Apply button to save the changes or OK button to save the changes and close the Holiday Schedule Editor.
