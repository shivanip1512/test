Editing a Holiday Schedule

A holiday schedule contains the dates of holidays for the year. Assigning a holiday schedule to a program prevents that program from running on holidays.

 

Perform the following steps to edit a holiday schedule:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Holiday Schedule.

  3. Double-click on the holiday schedule to edit. The General tabbed page is displayed in the Holiday Schedule Editor, from which the following changes can be made:

    1. The name of the holiday schedule can be changed in the Schedule Name text box.

    2. The Assigned Holidays list displays the holidays that are defined in the holiday schedule.

      • To create a holiday, see Creating a Holiday.

      • To edit a holiday, see Editing a Holiday.

      • To remove a holiday, see Removing a Holiday.

  4. Click the Apply button to save the changes to the holiday schedule or click the OK button to save the changes to the holiday schedule and close the Holiday Schedule Editor.
