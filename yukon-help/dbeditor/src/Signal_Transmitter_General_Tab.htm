Signal Transmitter General Tab

The following changes can be made to the general settings of a signal transmitter:

 

  1. The name of the signal transmitter can be changed in the Device Name text box.

  2. If the signal transmitter is a CCU-710A, CCU-711, LCU, RTC, or TCU the physical address can be changed in the Physical Address text box. The physical address is the factory-supplied address of the signal transmitter.

    • For a CCU-710A or CCU-711, the physical address is located on the power supply dial inside the CCU enclosure. If the CCU power supply does not have a dial setting, the physical address is zero.

    • For Local Control Units (LCU) and Transmitter Control Units (TCU), the physical address is programmed into the device.

If the signal transmitter is a CCU-721, the physical address can be changed in the Slave Address text box. The physical address of the CCU-721 is accessible from the front panel display.

If the signal transmitter is an RTU-LMI, the physical address can be changed in the Address text box.

  1. The signal transmitter can be disabled or enabled by selecting or clearing the Disable Device check box. Select this check box to disable the signal transmitter, which suspends scanning and alarming. Clear this check box to enable the signal transmitter.

  2. If the signal transmitter is an RTU-LMI, verification of transmitted commands can be disabled or enabled by selected or clearing the Disable Verification check box.

  3. A different communication channel can be selected from the Communication Channel drop-down list.

  4. If the signal transmitter is a CCU-710A, CCU-711, Integration, LCU, RTU-LMI, or TCU signal transmitter, the number of milliseconds to wait can be changed in the Post Communication Wait spin box. The post communication wait defines the length of time to wait after communication before this signal transmitter can be communicated with again.

  5. If the signal transmitter is an RTC, a different listen before talking setting can be selected from the Listen Before Talk drop-down list.

  6. If the signal transmitter is a CCU-710A or CCU-711, a different power amplifier card usage can be selected from the CCU Amp Use Type drop-down list. This defines how the amplifier cards are used by the communication routes. The CCU amp card use options are:

    • Amp1 – Use amp card 1.

    • Amp2 – Use amp card 2.

    • Alternating – Alternate between using amp card 1 and amp card 2 after each transmission (CCU-711 only).

    • Default 1 Fail 2 – Use amp card 1 by default and switch to amp card 2 if amp card 1 fails (CCU-711 only).

    • Default 2 Fail 1 – Use amp card 2 by default and switch to amp card 1 if amp card 2 fails (CCU-711 only).

    • Alt w/Failover – Alternate between using amp card 1 and amp card 2 after each transmission, and switch to the other amp card if one of amp cards fail (CCU-711 only).

  7. If the signal transmitter is a Paging Tap Terminal or TNPP Terminal the password of the terminal account can be changed in the Password text box.

 

If the signal transmitter is an SNPP Terminal:

  • The login of the SNPP account can be changed in the Login text box.

  • The password of the SNPP account can be changed in the Password text box.

 

If the signal transmitter is a WCTP Terminal:

    • The password of the WCTP account can be changed in the Password text box.

    • The sender ID/address of the WCTP account can be changed in the Sender text box.

    • The security code of the WCTP account can be changed in the Security Code text box.

  1. If the communication channel is a dialup comm channel, the phone number that is used to dial the signal transmitter can be changed in the Phone Number text box. To edit the advanced dialup properties of the signal transmitter:

    1. Click the Advanced button.

    2. The minimum and maximum connect times can be entered in the Minimum Connect Time and Maximum Connect Time boxes. The minimum time defines the amount of time following a successful communication that the port is not allowed to hang up. After this time, the port either remains connected until the maximum time is reached, or reverts to normal priority based on queuing and hangs up.

    3. A baud rate other than (Use Port Baud) can be selected from the Baud Rate drop-down list.

    4. Click the OK button to save the advanced dialup properties.
