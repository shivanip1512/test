Menu Bar

 

The Menu Bar provides six menus, which perform the following functions:

 

  • File - use this menu to exit the application or end the Yukon login session.

  • Edit - use this menu to edit, copy, delete, and find Database Items.

  • View - use this menu to select the type of database to view, show or hide the message log, or to refresh the window display.

  • Create - use this menu to create Database Items.

  • Tools - use this menu to regenerate route roles and set default routes

  • Help - use this menu to display the online help, point offset legend, and copyright information.
