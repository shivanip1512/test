MCT Load Group

If the load group is an MCT load group:

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of the load control devices that match the addressing of the load group. To disable the this feature, enter a value of zero.

  3. To take the load group out of service, select the Disable Group check box.

  4. To disable control of the load group, select the Disable Control check box.

  5. Select the route to use from the Communication Route drop-down list. This is the route or route macro used to reach all of the field devices in this group when transmitting commands.

  6. Click the Next button.

  7. Select the type of address level to use from the Address Level drop-down list. The types of address levels are:

    • Bronze –

    • Lead –

    • MCT Address –

  8. If Bronze or Lead is selected as the addressing type, enter the MCT address in the Address text box. This allows all of the devices using this address to be controlled.

  9. If MCT Address is selected as the addressing type, select an MCT to use for addressing from the MCT Address drop-down list. This allows the specified MCT to be controlled.

  10. Select the check boxes of the relays to control for this group in the Relay Usage area.

  11. Click the Finish button. The new load group appears in the Load Group Editor.
