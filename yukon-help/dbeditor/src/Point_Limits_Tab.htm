Point Limits Tab

The following changes can be made to the point limits settings:

 

  1. To enable the high and low limits, select the Limit 1 and/or Limit 2 check boxes.

  2. If Limit 1 is enabled, enter a high and low value for the limit in the High and Low text boxes. If the value goes outside of this range, the limit is violated and an alarm is generated.

  3. If Limit 2 is enabled, enter a high and low value for the limit in the High and Low text boxes. If the value goes outside of this range, the limit is violated and an alarm is generated.

  4. The limit duration of Limit 1 can be changed in the Limit Duration 1 spin box. This specifies the number of seconds that Limit 1 must be exceeded before an alarm is generated.

  5. The limit duration of Limit 2 can be changed in the Limit Duration 2 spin box. This specifies the number of seconds that Limit 2 must be exceeded before an alarm is generated.

  6. To enable the reasonability limits, select the High Limit and/or Low Limit check boxes, then enter reasonability limit values in the text boxes. If a reading exceeds a reasonability limit, the reading is ignored and the last reasonable value is used.
