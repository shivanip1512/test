Menu Bar

The menu bar provides five menus, which perform the following functions:

 

 

  • File – Use this menu to exit the Database Editor application.

  • Edit – Use this menu to edit, copy, change type, delete, and find database items.

  • View – Use this menu to select the type of database to view, show or hide the message log, and refresh the window display.

  • Create – Use this menu to create database items.

  • Help – Use this menu to display the online help and copyright information.
