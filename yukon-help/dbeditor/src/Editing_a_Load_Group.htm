Editing a Load Group

A load group defines the communication protocol and the device addressing needed to control field devices.

 

To edit a Load Group:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Load Group.

  3. Double-click the load group to modify. If the load group is a macro load group, see Editing a Macro Load Group.

    • The General tabbed page appears for all groups.

    • The Addressing tabbed page appears for ExpressCom, VERSACOM, EMETCON, and MCT groups.

    • The Message tabbed page appears for Ripple groups.

    • The Settings tabbed page appears for Golay groups.

    • The Point Group tabbed page appears for Point groups.

  4. Click Apply to save the changes or OK to save the changes and close the Load Group Editor.
