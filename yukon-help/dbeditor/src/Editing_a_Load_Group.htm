Editing a Load Group

A load group defines the communication protocol and the device addressing needed to control field devices.

 

Perform the following steps to edit a load group:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Load Group.

  3. Double-click on the load group to edit. If the load group is a macro load group, see Editing a Macro Load Group.

    • The General tabbed page appears in the Load Group Editor for all load groups.

    • The Addressing tabbed page appears for EMETCON, ExpressCom, Integration, MCT, and VERSACOM load groups.

    • The Settings tabbed page appears for Golay load groups.

    • The Integration Settings tabbed page appears for Integration load groups.

    • The Point Group tabbed page appears for Point load groups.

    • The Message tabbed page appears for Ripple load groups.

  4. Click the Apply button to save the changes to the load group or click the OK button to save the changes to the load group and close the Load Group Editor.
