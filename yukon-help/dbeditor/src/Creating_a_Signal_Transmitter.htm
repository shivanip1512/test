Creating a Signal Transmitter

If you selected Signal Transmitter as the device type:

 

  1. Enter a name for the signal transmitter in the Device Name text box.

  2. Enter an address in the Physical Address text box.

    • For CCU-711s and older CCU-710s, the physical address is located on the power supply dial inside the CCU enclosure. If the CCU’s power supply does not have a dial setting, the physical address is zero.

    • For Transmitter Control Units (TCU) and Local Control Units (LCU), the physical address is programmed into the device.

If you selected Paging Tap Terminal or SNPP Terminal as the device:

    1. Enter a pager number in the Pager Number text box.

    2. If a password is required to connect to the Paging Tap Terminal, select the Password Required check box.

    3. Enter the password connect to the signal transmitter in the Password text box.

If you selected WCTP Terminal as the device:

  1. Enter the pager number in the Pager Number text box.

  2. Enter the sender ID/address of the WCTP account in the Sender text box..

  3. Enter the password of the WCTP account in the Password text box.

  4. Enter the security code of the WCTP account in the Security Code text box.

  1. Click Next. Skip step 4 and 5 for Paging Tap Terminal, SNPP Terminal, and WCTP Terminal signal transmitters.

  2. Select the check boxes of the scan categories you want to enable. The categories vary depending on the type of signal transmitter selected. The categories are:

    • Periodic Health Check - periodically checks that the device is alive

    • Status & Analog - collects both status and analog points.

    • Integrity Rate - collects all point values

    • Report by Exception - reports if anything has changed on the device

    • Accumulator Rate - collects KYZ pulse points

    1. Select the scan interval and alternative scan interval from the Interval and Alt Interval drop-down lists.

    2. Select a scan group or enter a new scan group in the Group drop-down list. This is used to group similar scans together.

  3. Under the Optional Scan Window, select the Enabled check box if you want to define a time when the scans can occur.

    1. Select the type of scan from the Type drop-down list.

    2. Enter the open and close times in the Open and Close text boxes.

  4. Click Next.

  5. Select a comm channel from the drop-down list of comm channels.

  6. The rest of this procedure depends on the type of comm channel selected.

    • If the comm channel is Dedicated, click Finish. The new signal transmitter appears in the Device Editor. For instructions on using this editor, refer to the related topics.

    • If the comm channel is Dialup or Dialback, click Next, enter the phone number used to dial this device or dial back from the device, and then click Finish. The new signal transmitter appears in the Device Editor. For instructions on using this editor, refer to the related topics.
