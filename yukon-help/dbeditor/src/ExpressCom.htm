ExpressCom

If you selected an ExpressCom device:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of the switches in the load group. To disable the kilowatt capacity feature, enter a value of zero.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Select the Disable Control check box.

  5. Select the route to use from the Communication Route drop-down list. This is the route or route macro used to reach all of the devices in this group when transmitting commands.

  6. Click Next.

  7. In the Geographical Address Usage area, select the check boxes of the geographical addresses to use when commands are sent. When an address check box is selected, the address field is activated and turns blue.

  8. In the Load Address Usage area, select the check boxes of the load addresses to use when commands are sent. When an address check box is selected, the address field is activated and turns blue.

  9. Enter the service provider ID in the SPID Address text box. The service provider ID is sent with all group address messages and is used to identify the service provider (owner utility) in the case of any communications cross talk or some shared communications network. Value values are 1 to 65,534.

  10. Enter the geographical address in the GEO Address text box. The geographical address identifies a desired geographical area (determined by the utility). Value values are 1 to 65,534.

  11. Enter the substation address in the SUB Address text box. The substation address is intended to identify the utility substation on which the target equipment is located. Value values are 1 to 65,534.

  12. Enter the feeder address in the FEED Address text box. The feeder address is intended to identify the utility feeder by which the load is fed. This is normally combined with the substation address, but can be used alone. Each bit in this address represents a feeder (supports 16 feeders). This allows a set number of feeders to be targeted with a single message.

  13. Enter the zip address in the ZIP Address text box. The zip address is intended to be the postal zip code. This address can also be used for any other addressing options. Valid values are 1 to 16,777,214.

  14. Enter the user defined address in the USER Address text box. The user defined address is intended to be user definable and can be used by the user for any other addressing options. Valid values are 1 to 65,534.

  15. To control a device using its individual address, select the Serial check box in the Geographical Address Usage area and enter the physical address of the device in the Serial text box. This applies to a single device and does not use group addressing. Up to 4,294,96,295 unique devices are supported.

  16. Enter the program address in the PROG Address text box. The program address is used to target program loads. Field devices that control multiple loads contain a separate program address for each load. Valid values are 1 to 254.

  17. Enter the splinter address in the SPLINTER text box. The splinter address is used to target a subset of a program load address. Field devices that control multiple loads contain a separate splinter address for each load. Valid values are 1 to 254.

  18. (Optional) If the program or splinter check box is selected in the Load Address Usage area, the Configured Loads area can be used to select the check boxes of the loads targeted by the program or splinter addresses.

  19. If the load check box is selected in the Load Address Usage area, the Configured Loads area changes to Targeted Loads. Select the check boxes of the loads to control for this group in the Targeted Loads area.

  20. Click Finish. The new load group appears in the Device Editor. For instructions on editing a load group, refer to the related topics.
