Ripple Group Message Tab

Perform the following steps for the message settings of the ripple group:

 

  1. Select a value from the Shed Time drop-down list that matches the shed time programmed into the device. The shed time is the number of minutes a relay controls when a control command is sent. A shed time of Continuous Latch controls until a restore command is sent.

  2. Set a control message using the Control push buttons that matches the control message programmed into the device. This is the 50 bit message that causes the relay to control.

  3. Set a restore message using the Restore push buttons that matches the restore message programmed into the device. This is the 50 bit message that causes the relay to restore.
