Editing a Macro Route

If the route is a macro route, the Macro Route tabbed page opens when the route is double-clicked.

 

The following changes can be made to the settings of the macro route:

 

  1. The name of the macro route can be changed in the Route Macro Name text box.

  2. Routes can be assigned to or unassigned from the route macro.

    • To assign a route, select the route from the Available list and click Add. To select more than one route, hold the Ctrl key on the keyboard while clicking additional routes.

    • To rearrange the order of the assigned routes, click and drag a route to move it up or down the list.

    • To unassign a route, select the route from the Assigned list and click Remove.

  3. Click OK to save the changes and close the Route Editor.
