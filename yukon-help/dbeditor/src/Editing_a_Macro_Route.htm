Editing a Macro Route

The following changes can be made to the settings of the macro route:

 

  1. The name of the macro route can be changed in the Route Macro Name text box.

  2. Routes can be assigned to or unassigned from the route macro.

    • To assign a route to the macro route, select the route from the Available list and click the Add button. To select more than one route, hold the Ctrl key on the keyboard while clicking additional routes.

    • To rearrange the order of the assigned routes, click and drag a route to move it up or down the list.

    • To unassign a route from the macro route, select the route from the Assigned list and click the Remove button.

  3. Click the Apply button to save the changes to the macro route or click the OK button to save the changes to the macro route and close the Route Editor.
