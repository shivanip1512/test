VERSACOM Group Addressing Tab

The following changes can be made to the addressing settings of the VERSACOM group:

 

  1. The utility address can be changed in the Utility Address spin box. A utility address is used to identify the utility and optional region or member system.

  2. The section address can be changed in the Section Address spin box. A section address is a secondary addressing method based on a device’s substation, bus, or geographic region.

  3. A different class address can be set using the Class Address push buttons. A class address identifies a customer according to a utility classification scheme or type of load.

  4. A different division address can be set using the Division Address push buttons. A division address provides finer control resolution, or identifies particular sub-groupings of load type.

  5. Different address levels can be selected using the check boxes in the Address Usage area.
    If you select Serial Address in the Address Usage area, enter the device serial number in the Serial Address text box.

  6. Different relays to control for this group can be selected using the check boxes in the Relay Usage area.
