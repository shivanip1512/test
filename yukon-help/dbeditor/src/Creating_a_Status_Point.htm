Creating a Status Point

If Status was selected as the point type:

 

  1. Select a state group from the State Group list. The state group defines the messages that are display for the different states of the point.

  2. Select an initial state for the state group from the Initial State list. The initial state depends on the state group selected.

  3. Click the Next button.

  4. If the Yukon system obtains physical data from the device on which the point is located or the point is a special Yukon system point, select the Physical Point Offset check box. Clear this check box if this is a pseudo point.

If the Physical Point Offset check box is selected, set the offset value in the Point Offset spin box. This is either the physical input to which the point is wired or a Yukon system defined point.

  1. If the point is capable of control, select the type of control from the Control Type drop-down list. The four control types are:

    • None – No control type exists.

    • Latch – When a control command is sent, the relay latches into position.  Latch controls require a second command to return to the original state.

    • Normal – Control is performed with a dual relay Select Before Operate (SBO). Normal controls are temporary or held only for a discrete number of milliseconds.

    • Pseudo – The point is controlled by another system. Pseudo controls do not have a physical status point to indicate if control has completed.

    • SBO Latch - Select Before Operate latch command.

    • SBO Pulse - Select Before Operate pulse (normal) command.

If the control type is Latch, Normal, SBO Latch, or SBO Pulse, select a point offset in the Control Offset drop-down list. This is the physical location of the wired the relay point.

  1. Click the Finish button. The new point appears in the Point Editor.
