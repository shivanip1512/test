Creating a Status Point

If you selected Status as the point type:

 

  1. Select a state group from the State Group list. The state group defines the messages to display for the different states of the point.

  2. Select an initial state for the state group from the Initial State list. The initial sate depends on the state group selected.

  3. Click Next.

  4. If the system is to obtain physical data from the hardware, select the Physical Point Offset check box. Clear this check box for a pseudo point.

  5. If the Physical Point Offset check box is selected, set the offset value in the Point Offset spin box. This is the physical input to which the status point is wired or it is a Yukon defined point.

  6. If the point is capable of control, select the type of control from the Control Type drop-down list. The four control types are:

    • None – no control type exists.

    • Latch – when a control command is sent, the relay latches into position.  Latch controls require a second command to return to the original state.

    • Normal – control is performed with a dual relay Select Before Operate (SBO). Normal controls are temporary or held only for a discrete number of milliseconds.

    • Pseudo – the point is controlled by another system. Pseudo controls do not have a physical status point to indicate if control has completed.

    • SBO Latch - Select Before Operate latch command.

    • SBO Pulse - Select Before Operate pulse (normal) command.

  7. For a control type other than None or Pseudo, select a point offset in the Control Offset drop-down list. This is the physical location of the wired the relay point.

  8. Click Finish. The new point appears in the Point Editor. For instructions on editing a point, refer to the related topics.
