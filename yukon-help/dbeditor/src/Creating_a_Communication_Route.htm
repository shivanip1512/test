Creating a Communication Route

If you selected Communication Route as the route type:

  1. Enter a name for the route in the Route Name text box.

  2. Select the signal transmitter for this route from the Signal Transmitter drop-down list.

  3. If a CCU was selected as the signal transmitter click Next. The following question appears: “Does this route use a repeater to rebroadcast communication?”

  • If the route does not use a repeater, select No.

  • If the route does use a repeater, select Yes, click Next, and add the repeater(s) on this route to the Assigned list.

  1. Click Finish. The new route appears in the Route Editor. For instructions on editing a route, refer to the related topics.
