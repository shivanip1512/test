Creating a Communication Route

If Communication Route was selected as the route type:

  1. Enter a name for the communication route in the Route Name text box.

  2. Select the signal transmitter for this communication route from the Signal Transmitter drop-down list.

  3. If a CCU was selected as the signal transmitter, click Next and the following question appears: “Does this route use a repeater to rebroadcast communication?”

  • If the route does use a repeater, select Yes, click Next, and add the repeater(s) on this route to the Assigned list.

  • If the route does not use a repeater, select No.

  1. Click the Finish button. The new communication route appears in the Route Editor.
