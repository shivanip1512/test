Signal Transmitter Scan Rate Tab

The following changes can be made to the scan rate settings of a signal transmitter:

 

  1. The categories to scan can be changed by selecting or clearing the category check boxes. The categories displayed depend on the signal transmitter selected. The categories are:

    • Periodic Health Check – Periodically checks that the signal transmitter is functioning.

    • Integrity Rate – Collects all point values.

    • Report by Exception – Reports if anything has changed on the signal transmitter.

    • Accumulator Rate – Collects KYZ pulse points.

When the check box of a scan category is selected, the following changes can be made to the scan settings:

    1. A different default scan interval can be selected from the Interval drop-down list.

    2. A different alternate scan interval can be selected from the Alt Interval drop-down list.

    3. A different group can be selected from the Group drop-down list. The scan group is used to assign priorities to the scan category.

  1. The optional scan window can be enabled or disable by selecting or clearing the Enabled check box. The Optional Scan Window defines a time when the scans can occur. This is useful if the signal transmitter is sharing a dial-up connection.

    1. A different scan type can be selected from the Type drop-down list.

    2. The time at which the scan window starts can be changed in the Open text box.

    3. The time at which the scan window stops can be changed in the Close text box.
