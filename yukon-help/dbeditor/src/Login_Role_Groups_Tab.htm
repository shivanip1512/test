Login Role Groups Tab

The following login groups are pre-defined in the Yukon database:

Role Group

Meaning

Esub Operators Grp

Login group that grants operator access to the eSubstation application.

Esub Users Grp

Login group that grants user access to the eSubstation application.

Operators Grp

Login group that grants operator access to the Yukon system.

Residential Customers Grp

Login group that grants residential customer access to the thermostat Web application.

STARS Operators Grp

Login group that grants operator access to the Yukon STARS applications.

STARS Residential Customers Grp

Login group that grants residential customer access to the Yukon STARS applications.

System Administrators Grp

Login group that grants administrative access to the Yukon system.

Web Client Customers Grp

Login group that grants client access to Yukon Web applications.

Web Client Operators Grp

Login group that grants operator access to Yukon Web applications.

Yukon Grp

Special login group that grants limited user access to the Yukon system. This login group is required for all Yukon login accounts.

 

The following changes can be made to the roles groups settings of a login:

  1. Select the check boxes of role groups to assign to this login. When a role is selected, a short description of the role appears in the Description box.

  2. To display the permissions assigned to a role group, right-click on a role group and select Roles.

  3. If a role conflict occurs, right-click on one of the conflicting role groups and select Conflicts. The dialog box that appears displays the roles in the selected group that conflict with roles in the other assigned groups.

  4. Clear check boxes of the role groups to unassign from this login.
