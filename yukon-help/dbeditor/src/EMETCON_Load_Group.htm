EMETCON Load Group

If the load group is an EMETCON load group:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the load group in the kW Capacity text box. This is the estimated kilowatt capacity of the load control devices in the load group. To disable the kilowatt capacity feature, enter a value of zero.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Select the Disable Control check box. Select this check box to disable control of the load group. Clear this check box to enable control of the load group.

  5. Select the communication route to use for the load group from the Communication Route drop-down list. This is the route or route macro that is used to transmit commands to all of the load control devices in this group.

  6. Click the Next button.

  7. Set the gold address of the load group in the Gold Address box. The gold address range is 1 to 4 and it is used for maximum load control coverage and it is normally reserved for emergency sheds.

  8. Set the silver address of the load group in the Silver Address box. The silver address range is 1 to 60 and it is used for medium load control coverage and non-emergency sheds.

  9. Select a relay option to use for the load group in the Relay to Use area. The Relay to Use options are:

    • Relay A – Load on physical relay A, which is either a 5 A or 30 A relay.

    • Relay B – Load on physical relay B, which is either a 5 A or 30 A relay.

    • Relay C – Load on physical relay C, which is a 5 A relay.

    • S (all) – Controls all relays (SCRAM).

  10. In the Address to Use area, select the type of addressing to use when commands are sent to the load group. The Address to Use options are:

    • Gold Address – Commands are addressed to load control devices that are configured with the gold address value that was entered in step 7.

    • Silver Address – Commands are addressed to load control devices that are configured with the silver address value that was entered in step 8.

  11. Click the Finish button. The new load group appears in the Load Group Editor.
