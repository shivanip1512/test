Status Point General Tab

The following changes can be made to the general settings of the status point:

 

  1. The name of the point can be changed in the Point Name text box.

  2. A different type of timing group, which control how often events are recorded, can be selected from the Timing Group drop-down list. The timing group are:

    • Default – Records events that occur within seconds of each other.

    • SOE – Records events that occur within milliseconds of each other (Sequence of Events).

  3. The point can be disabled or enabled by selecting or clearing the Disable Point check box. Disabling the point may be useful if it is temporarily not functioning.

  4. A different state group, which defines the messages that are displayed for the different states of a point, can be selected from the State Group drop-down list.

  5. A different initial state can be selected from the Initial State drop-down list. The initial state depends on the state group selected.

  6. The archive data option can be enabled or disabled by selecting or clearing the Archive Data check box. Select this check box to archive point data and send it to the raw point table.
    Note: Status points are typically archived only if they contain data important to the utility.
