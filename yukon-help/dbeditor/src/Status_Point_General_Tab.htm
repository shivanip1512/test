Status Point General Tab

The following changes can be made to the general settings of the status point:

 

  1. The name of the point can be changed in the Point Name text box.

  2. The Timing Group drop-down list controls how often events are recorded. A value of Default records events that occur within seconds of each other. For points on DNP devices, a value of SOE (Sequence of Events) records high speed events that occur within milliseconds of each other.

  3. The point can be disabled or enabled by selecting or clearing the Disable Point check box. Disabling the point may be useful if it is temporarily not functional.

  4. A different state group can be selected from the State Group drop-down list. The state group defines the messages to display for the different states of a point.

  5. A different initial state can be selected from the Initial State drop-down list. The initial state depends on the state group selected.

  6. The archive data option can be enabled or disabled by selecting or clearing the Archive Data check box. Select this check box to send information to the raw point table. Generally, only points of particular significance are sent to this table.
