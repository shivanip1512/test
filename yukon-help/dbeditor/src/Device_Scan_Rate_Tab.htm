Device Scan Rate Tab

The following changes can be made to the scan rate settings of the device:

 

  1. The categories to scan can be changed by selecting or clearing the category check boxes. The categories displayed depend on the device selected. The different categories are:

    • Demand & Status Rate - collects demand and status point values

    • Demand & Voltage Rate - collects demand and voltage point values

    • Demand Rate - collects demand point values

    • Status Rate - collects status point values

    • Periodic Health Check - periodically checks that the device is alive

    • Accumulator Rate - collects KYZ pulse points

    • Accumulator (Energy) Rate - collects kWh point values

    • Integrity Rate - collects all point values

    • Report by Exception - reports if anything has changed on the device

    • General Data Collection - collects general data

    • Eventlog Collection - collects the devices event log

    • Class 0, 1, 2, 3 Scan - collects all of the data on the device

    • Class 1, 2, 3 Scan - collects the data on the device that has changed since the last scan

    1. A different default scan interval can be selected from the Interval drop-down list.

    2. A different alternate scan interval can be selected from the Alt Interval drop-down list. The alternate scan interval can be activated using the Tabular Data Console or the Yukon Web interface and is used during the “control timeout” period to collect the state of a control point.

    3. A different group can be selected from the Group drop-down list. This determines the priority grouping between different device scans.

  2. The optional scan window can be enabled or disable by selecting or clearing the Enabled check box. The Optional Scan Window defines a time when the scans can occur. This is useful if the device is read with a shared dial-up connection.

    1. A different scan type can be selected from the Type drop-down list.

    2. The time at which the scan window starts can be changed in the Open text box.

    3. The time at which the scan window stops can be changed in the Close text box.
