Editing a Baseline

A baseline is a known value that is used for comparison with current values. For example, the average usage can be made a baseline and can be compared to the current usage.

 

To edit a baseline:

  1. From the View menu, select System.

  2. From the Sort By list, select Baseline.

  3. Double-click on the baseline to modify. This opens the General tabbed page of the Baseline editor, from which the following changes can be made:

    1. The name of the baseline can be changed in the Name text box.

    2. The number of previous days used can be changed in the Previous Days Used text box. This is the number of days back to look for valid dates. Yukon looks at the specified number days from the current date to check if a day contains valid baseline data. A valid date is a non-curtailed date; a non-holiday (if a holiday schedule is selected), and a non-excluded day (if any excluded days are selected).

    3. The load percentage can be changed in the Load Percent text box. This is the percentage of validity for an interval value to be considered valid data. If a load percentage greater than zero is entered, all interval readings must exceed this percentage calculated against another point’s readings. This other point is often another baseline point that is calculated with a load percent of zero. The load percent of zero is used to create average data readings.

    4. The number of days in the calculation can be changed in the Days in Calculation text box. This is the minimum number of valid days needed to calculate the baseline..

    5. A different holiday schedule can be selected from the Holiday drop-down list. This prevents holidays from being used in the base line calculation.

    6. The days of the week to exclude from the base line calculation can be changed under Excluded Days. Select the check boxes of the days to excluded from the calculation. Clear all of the check boxes to include all of the days in the calculation.

  4. Click Apply to save the changes or OK to save the changes and close the Baseline Editor.
