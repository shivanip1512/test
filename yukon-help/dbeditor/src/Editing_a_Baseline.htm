Editing a Baseline

A baseline is a known value that is used for comparison with current values. For example, the average usage can be a baseline and it can be compared to the current usage.

 

Perform the following steps to edit a baseline:

  1. From the View menu, select System.

  2. From the Sort By list, select Baseline.

  3. Double-click on the baseline to edit. The General tabbed page is displayed in the Baseline Editor, from which the following changes can be made:

    1. The name of the baseline can be changed in the Name text box.

    2. The number of previous days, from the current date, in which the Yukon system can search for days that contain valid baseline data, can be changed in the Previous Days Used text box. A day must meet the following requirements for the data to be considered valid:

      • Non-curtailed day

      • Non-holiday day (if a holiday schedule is selected)

      • Non-excluded day (if any excluded days are selected)

    3. The load percentage of the baseline can be changed in the Load Percent text box. This is the percentage of validity for an interval value to be considered valid data. If a load percentage greater than zero is entered, all interval readings must exceed this percentage calculated against another point’s readings. This other point is often another baseline point that is calculated with a load percent of zero. The load percent of zero is used to create average data readings.

    4. The number of days needed to calculate the baseline can be changed in the Days in Calculation text box.

    5. A different holiday schedule, which prevents the holidays in the schedule from being used in the baseline calculation, can be selected from the Holiday drop-down list.

    6. The days of the week to exclude from the baseline calculation can be changed in the Excluded Days area. To exclude a day of the week from the calculation, select the check box of that day. To include all of the days of the week in the calculation, clear all of the check boxes.

  4. Click the Apply button to save the changes to the baseline or click the OK button to save the changes to the baseline and close the Baseline Editor.
