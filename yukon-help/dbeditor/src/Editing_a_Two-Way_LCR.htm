Editing a Two-Way LCR

To edit a two-way LCR:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select Device.

  3. Double-click an LCR. This opens the General tabbed page of the Device Editor.

  4. Click on the Metering tab to display the Metering tabbed page and select the demand interval for the LCR from the Last Interval Demand Rate drop-down list.

  5. Click the Scan Rate tab to display Scan Rate tabbed page.

  6. Click the Apply button to save the changes or OK button to save the changes and close the Config Editor.
