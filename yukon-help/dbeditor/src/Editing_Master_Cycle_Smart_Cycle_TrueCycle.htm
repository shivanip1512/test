Editing a Master Cycle, Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle Gear

To edit a Master Cycle, Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle gear:

  1. The name of the gear can be changed in the Gear Name text box.

  2. Select either Master Cycle, Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle from the Gear Type drop-down list.

  3. For Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle; to disable the ramping in of load groups over an interval of time, select the No Ramp (ExpressCom only) check box. The cycle command must be send using the ExpressCom protocol.

  4. The control percentage can be changed in the Control Percent spin box. This is the percentage of off time.

  5. The cycle period can be changed in the Cycle Period spin box. This is the cycle period in minutes.

  6. For Master Cycle, to change how the software selects load groups, select a different method from the Group Selection Method drop-down list.

  7. For Master Cycle, a percentage of the load groups can be ramped in over an interval of time by selecting the Ramp In check box.

    1. Enter the percentage of the load groups to ramp in at each interval in the Ramp In Percent text box.

    2. Enter the interval in seconds to ramp in a percentage of the load groups in the Ramp In Interval text box.

  8. For Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle; a different cycle count send type can be selected from the Cycle Count Send Type drop-down list. The types are:

    • Fixed Count - The same number of cycle counts are sent to the switch.

    • Count Down - The number of cycles counts sent decreases at each resend.

    • Limited Count Down - Used with LCR 2000s, which do not allow cycle counts to be sent to them. The cycle count is hard coded in switch.

The purpose of Count Down and Limited Count Down is to end up with zero cycle counts at the end of control.

  1. For Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle; a different maximum cycle count can be selected from the Max Cycle Cnt drop-down list. This limits the number of cycles counts that can be sent.

  2. For Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle; a different starting period count can be selected from the Starting Period Count drop-down list. This is the count that is used when automatic control starts.

  3. For Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle; a different resend rate can be selected from the Command Resend Rate drop-down list. This controls how often the cycle command is resent.

  4. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over. (Master Cycle only)

    • Restore - A restore command is sent to the device.

    • Stop Cycle - The period count is set to 0, so the cycle stops the next period. (Smart Cycle, TrueCycle, Magnitude Cycle, or Target Cycle only)

    • Ramp Out/Time In - The control ramps out when the shed time is over. (Master Cycle only)

    • Ramp Out/Restore - The control ramps out when a restore command is sent to the device. (Master Cycle only)

  5. If Ramp Out/Time In or Ramp Out/Restore is selected from the How to Stop Control drop-down list, select the order to stop the control from the Stop Order drop-down list. The orders for stopping control are:

    • Random - The system selects a random load group to stop.

    • First In First Out - The first group that was controlled is the first group to stop. Then the second group that was controlled is the second group to stop, etc.

  6. If Ramp Out/Time In or Ramp Out/Restore is selected from the How to Stop Control drop-down list:

    1. Enter the percentage of the load groups to ramp out at each interval in the Ramp Out Percent text box.

    2. Enter the interval in seconds to ramp out a percentage of the load groups in the Ramp Out Interval text box.

  7. The group capacity reduction percentage can be changed in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much load to control.

  8. For Target Cycle, the amount of targeted reduction can be changed in the kW Reduction text box.

  9. A different automatic gear change method can be selected from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear switches only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear switches when the control area priority reaches this priority.

    • Above Trigger - The Trigger Number spin box and Trigger Offset text box appear. This change method requires the control area’s threshold triggers to be created before it can be used.  Select whether to use trigger one or two in the Trigger Number spin box. (Trigger one is the first entry of the Control Area’s Trigger drop-down list.) Enter the offset from the trigger’s threshold in the Trigger Offset text box.

  10. Click OK to save the changes.
