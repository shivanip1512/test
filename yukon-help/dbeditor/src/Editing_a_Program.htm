Editing a Program

The following changes can be made to a load program assigned to a control area:

 

  1. The start priority for an assigned program can be changed by double-clicking in the Start Priority box and entering a new value. A load program with a start priority of one starts before a load program with a start priority of two.

  2. The start priority for an assigned program can be changed by double-clicking in the Stop Priority box and entering a new value. A load program with a stop priority of one stops before a load program with a stop priority of two.

  3. Click Apply to save the changes or OK to save the changes and close the Control Area Editor.
