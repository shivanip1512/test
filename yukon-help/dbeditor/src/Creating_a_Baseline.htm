Creating a Baseline

A baseline is a known value that is used for comparison with current values. For example, the average usage can be a baseline and it can be compared to the current usage.

Perform the following steps to create a baseline:

  1. From the View menu, select System.

  2. From the Create menu, select Baseline.

  3. Enter the name of the baseline in the Name text box.

  4. Enter the number of previous days, from the current date, in the Previous Days Used text box in which the Yukon system can search for days that contain valid baseline data. A day must meet the following requirements for the data to be considered valid:

    • Non-curtailed day

    • Non-holiday day (if a holiday schedule is selected)

    • Non-excluded day (if any excluded days are selected)

  5. Enter the load percent of the baseline in the Load Percent text box. This is the percentage of validity for an interval value to be considered valid data. If a load percentage greater than zero is entered, all interval readings must exceed this percentage calculated against another point’s readings. This other point is often another baseline point that is calculated with a load percent of zero. The load percent of zero is used to create average data readings.

  6. Enter the maximum number days needed to calculate the baseline in the Days in Calculation text box.

  7. To prevent holidays from being used in the baseline calculation, select a holiday schedule that contains the holidays to not use in the calculation, from the Holiday drop-down list.

  8. Select which days of the week to exclude from the baseline in the Excluded Days area. To include all of the days of the week in the calculation, clear all of the check boxes.

  9. Click the Finish button. The new baseline appears in the Baseline Editor. For instructions on editing a baseline, refer to the related topics.
