Creating a Baseline

A baseline is a known value that is used for comparison with current values. For example, the average usage can be made a baseline and can be compared to the current usage.

To create a baseline:

  1. From the View menu, select System.

  2. From the Create menu, select Baseline.

  3. Enter the name of the baseline in the Name text box.

  4. Enter the number of days back to look for valid dates in the Previous Days Used text box. Yukon looks at the specified number days from the current date to check if a day contains valid baseline data. A valid date is a non-curtailed date; a non-holiday (if a holiday schedule is selected), and a non-excluded day (if any excluded days are selected).

  5. Enter the load percent in the Load Percent text box. This is the percentage of validity for an interval value to be considered valid data. If a load percentage greater than zero is entered, all interval readings must exceed this percentage calculated against another point’s readings. This other point is often another baseline point that is calculated with a load percent of zero. The load percent of zero is used to create average data readings.

  6. Enter the maximum number days need to calculate the baseline in the Days in Calculation text box.

  7. Select a holiday schedule to use from the Holiday drop-down list. This prevents holidays from being used in the baseline calculation.

  8. Select the which days of the week to exclude from the baseline in the Excluded Days area. Clear all of the check boxes to include all of the days in the calculation.

  9. Click Finish. The new baseline appears in the Baseline Editor. For instructions on editing a baseline, refer to the related topics.
