Latching Gear

Perform the following step to add a Latching gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Latching from the Gear Type drop-down list.

  3. Select a starting state from the Control Start State drop-down list. This controls whether an open or close command is sent at the beginning of control.

  4. Set a group capacity reduction percentage in the Group Capacity Reduction % spin box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop

  5. Click the OK button to save the gear.
