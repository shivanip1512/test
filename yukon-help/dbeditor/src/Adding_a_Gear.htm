Adding a Gear

Control gears define the different rates at which a load program can controls loads. Different gear ratios net different amounts of reduction.

 

The control methods are:

  • Time Refresh - timed commands are sent repeatedly to control.

  • Rotation - rotates through a list of groups to control.

  • Master Cycle - the Master Station sends command to the LCR to cycle.

  • Smart Cycle - the LCR hardware performs the cycling.

  • Latching - one command is sent to open and an opposite command is sent to close.

  • Thermostat Pre-operate - used to pre-heat or pre-cool.

  • Thermostat Setback - used to modify the temperature setpoint of a thermostat.

 

Time Refresh/Rotation

To add a Time Refresh or Rotation gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select either Time Refresh or Rotation from the Gear Type drop-down list.

  3. Select a time from the Shed Time drop-down list. This is the shed time sent to the switches in the field.

  4. Select the number of groups from the # of Groups Each Time drop-down list. For a time refresh gear, this is the number of groups that are shed for each refresh. For a rotation gear, the is the number of groups that are shed during each rotation.

  5. Select a rate in the Refresh/Resend Rate drop-down list. This field changes to Send Rate if the gear type is Rotation. The Refresh/Resend Rate is the rate at which the shed command is refreshed or resent. The Send Rate is the rate at which the shed command is sent.

  6. Select a group selection method from the Group Selection Method drop-down list. The group selection methods are:

    • Last Controlled - starts with the next group based on the last one controlled.

    • Always First Group - always starts with the first group in the list.

    • Least Control Time - starts with the group whose seasonal control time is the least.

  7. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

  8. Set a group capacity reduction percentage in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much load to control.

  9. Select an automatic gear change method from the When to Change drop-down list. The change methods are”

    • Manually Only - The gear switches only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear switches when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear switches when the control area priority reaches this priority.

  10. Click OK.

Master Cycle/Smart Cycle

To add a Master Cycle or Smart Cycle gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select either Master Cycle or Smart Cycle from the Gear Type drop-down list.

  3. Set a control percentage in the Control Percent box. This is the percentage of off time.

  4. Set a cycle period in the Cycle Period box. This is the cycle period in minutes.

  5. Select a cycle count send type from the Cycle Count Send Type drop-down list. The types are:

    • Fixed Count - The same number of cycle counts are sent to the switch.

    • Count Down - The number of cycles counts sent decreases at each resend.

    • Limited Count Down - Used with LCR 2000s, which do not allow cycle counts to be sent to them. The cycle count is hard coded in switch.

The purpose of Count Down and Limited Count Down is to end up with zero cycle counts at the end of control. This field does not display if the gear type is Master Cycle.

  1. If necessary, select a maximum cycle count from the Max Cycle Cnt drop-down list. This limits the number of cycles counts that can be sent. This field does not display if the gear type is Master Cycle.

  2. Select a starting period count from the Starting Period Count drop-down list. This is the count that is used when automatic control starts. This field does not display if the gear type is Master Cycle.

  3. Select a resend rate from the Command Resend Rate drop-down list. This controls how often the cycle command is resent. This field does not display if the gear type is Master Cycle.

  4. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over.

    • Restore - The control stops when a restore command is sent to the device.

    • Stop Cycle - The period count is set to 0, so the cycle stops the next period.

  5. Set a group capacity reduction percentage in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much load to control.

  6. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear switches only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear switches when the control area priority reaches this priority.

  7. Click OK.

Latching

To add a Latching gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Latching from the Gear Type drop-down list.

  3. Select a starting state from the Control Start State drop-down list. This controls whether an open or close command is sent at the beginning of control.

  4. Set a group capacity reduction percentage in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much load to control.

  5. Click OK.

Thermostat Pre-operate/Thermostat Setback

To create a thermostat pre-operate or thermostat setback gear:

  1. Enter a name for the gear in the Gear Name text box.

  2. Select Thermostat Pre-operate or Thermostat Setback from the Gear Type drop-down list.

  3. By default, Deltas and Fahrenheit units are used by the pre-operate gear.

    • To use Absolute values instead of Deltas, click the Deltas button.

    • To use Celsius units instead of Fahrenheit units, click the Fahren button.

  4. Select the mode the thermostat must be in to accept the commands: Heat Mode or Cool Mode

  5. Enter the minimum temperature in degrees in the Min text box.

  6. Enter the maximum temperature in degrees in the Max text box.

  7. Enter the setpoint delta in degrees in the Delta B text box.
    Enter the absolute setpoint in degrees in the Abs B text box.

  8. Enter the setpoint delta in degrees in the Delta D text box.
    Enter the absolute setpoint in degrees in the Abs D text box.

  9. Enter the setpoint delta in degrees in the Delta F text box.
    Enter the absolute setpoint in degrees in the Abs F text box.

  10. Enter the random offset time in minutes in the Rand text box.
    A random offset between 0 and the value Rand will be added to any delay time

  11. Enter the delay time in minutes in the TA text box.

  12. Enter the ramp time in minutes in the TB text box.

  13. Enter the plateau time in minutes in the TC text box. During this time the setpoint Abs B is maintained.

  14. Enter the ramp time in minutes in the TD text box.

  15. Enter the plateau time in minutes in the TE text box. During this time the setpoint Abs D is maintained.

  16. Enter the ramp time in minutes in the TF text box.

  17. Select how to stop the control from the How to Stop Control drop-down list. The methods for stopping control are:

    • Time In - The control stops when the shed time is over. This option does not display if the gear type is Smart Cycle.

    • Restore - The control stops when a restore command is sent to the device.

    • Stop Cycle - The period count is set to 0, so the cycle stops the next period. This option does not display if the gear type is Master Cycle.

  18. Set a group capacity reduction percentage in the Group Capacity Reduction % box. The percentage of each groups capacity is used to determine how much to control.

  19. Select an automatic gear change method from the When to Change drop-down list. The change methods are:

    • Manually Only - The gear switches only when it is manually switched.

    • After a Duration - Enter a duration in the Change Duration text box. The gear changes when the duration ends.

    • Priority Change - Enter a priority in the Change Priority text box. The gear switches when the priority of the control area reaches the change priority.

    • Above Trigger - Enter a trigger number and offset in the Trigger Number and Trigger Offset text boxes. This feature is not implemented yet.

  20. Click OK.
