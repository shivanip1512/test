Control Area

A Control Area defines a geographical area for controlling loads to reduce the peak for the area. A Control Area can consist of the whole system, portion of the system, substation, or any controllable area.

 

Two types of trigger points can be defined per control area:

  • Status - power supplier when closed-control

  • Threshold - a value goes over a defined level

 

When two control triggers are defined, you must select whether both are required active before control begins for the control area.
