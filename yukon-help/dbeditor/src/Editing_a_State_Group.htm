Editing a State Group

State groups contain message text and color-codes for the possible states of a point.

 

Perform the following steps to edit a state group:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select State Group.

  3. Double-click on the state group to edit. The General tabbed page is displayed in the State Editor.

  4. Click the Apply button to save the changes to the state group or click the OK button to save the changes to the state group and close the State Group Editor.
