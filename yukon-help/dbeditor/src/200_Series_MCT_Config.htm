200 Series MCT Config

If you selected 200 Series MCT as the config type:

  1. Enter a name for the config in the Configuration Name text box.

  2. If the Ke value is not known, enter the Mp and Kh values into the appropriate text boxes and click the = button to calculate Ke.

  3. If the Ke value is known, enter it into the Ke text box.

  4. Click Finish. The new MCT Config appears in the Config Editor.
