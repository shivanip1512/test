MCT Metering Tab

The following changes can be made to the metering settings of the MCT:

 

  1. The meter number of the MCT, which is assigned by the utility, can be changed in the Meter Number text box.

  2. A different data collection group can be entered or selected in the Data Collection Group drop-down list.

  3. A different alternate group can be entered or selected in the Alternate Group drop-down list.

  4. A different billing group can be entered or selected in the Billing Group drop-down list.

  5. A different demand interval can be selected from the Last Interval Demand Rate drop-down list.

  6. If the MCT can collect voltage readings, a different voltage interval can be selected from the Voltage Averaging Interval drop-down list.

  7. If the MCT can perform Load Profile Collection, the interval at which load profile data is collected can be selected from the Load Profile Demand Rate drop-down list. In addition, the check boxes of the channels from which to collect data, Channel #1, Channel #2, and/or Channel #3, must be selected.

  8. If the MCT can perform Voltage Profile Collection, the interval at which voltage profile data is collected can be selected from the Voltage Profile Demand Rate drop-down list. In addition, the check box of the channel from which to collect data, Channel #4 (Volts), must be selected.

 

Note: An MCT must be configured by the Yukon system before it can be used for data collection. To manually configure an MCT, perform the following steps:

    1. Start the Yukon Commander application.

    2. Select the MCT to configure.

    3. Execute the putconfig EMETCON install command.
