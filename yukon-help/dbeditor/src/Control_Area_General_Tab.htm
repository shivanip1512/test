Control Area General Tab

The following changes can be made to a control area:

  1. The name of the control area can be changed in the Name text box.

  2. A different control interval can be selected from the Control Interval drop-down list. The control interval is used to check if more load is required for control.

  3. A different minimum response time can be selected from the Min Response Time drop-down list. This is the minimum number of seconds to wait after control is sent before sending another control.

  4. A different default state can be selected from the Daily Default State drop-down list. The control area reverts to this state each day at midnight. The available default states are No Change, Enable/ReEnable, or Disable/Waived.

  5. The option to require all triggers to be active can be enabled or disabled by selecting or clearing the Require All Triggers Active check box. Select the check box to force all triggers to be active. Clear the check box if you do not need all triggers to be active.

  6. The start and stop times can be changed in the Start Time and Stop Time text boxes. These times serve as the daily time window during which Yukon checks to see if automatic control is needed.
