Control Area General Tab

The following changes can be made to a control area:

 

  1. The name of the control area can be changed in the Name text box.

  2. A different control interval can be selected from the Control Interval drop-down list. This is how often the Yukon system checks to see if control is needed,

  3. A different minimum response time can be selected from the Min Response Time drop-down list. This is the minimum number of seconds to wait after a control command is sent before sending another control command.

  4. A different default state can be selected from the Daily Default State drop-down list. This controls what the operational state of the control area is set to each day at midnight. The default states are:

    • No Change – The default state of the control does not change from its current state at midnight.

    • Enable/ReEnable – The default state of the control is set to enabled at midnight.

    • Disable/Waived – The default state of the control is set to disabled at midnight.

  5. The Require All Triggers Active check box is an optional setting that requires all of the triggers in the control area to be active for control to occur. Select the check box to force all triggers to be active. Clear the check box if all of the triggers do not have to be active.

  6. The start and stop times of the optional control window can be changed in the Start Time and Stop Time text boxes. This is the window of time during which the Yukon system is allowed to perform automatic feedback load control.

Note: Changes to the control window times do not take effect until midnight.
