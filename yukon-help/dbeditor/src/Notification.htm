Curtailment

Curtailment programs notify customers that they need to reduce or curtail their load.

 

If you selected Curtailment:

 

  1. Enter a subject of the message in the Subject text box. This is used as the subject in e-mails.

  2. Enter text in the Header text box. This is the text placed at the beginning each notification message.

  3. Enter text in Footer text box. This is the text placed at the end of each notification message.

  4. Enter text in the Canceled text box. This is the message that is sent to the customer if the curtailment is cancelled.

  5. Enter text in the Stopped Early text box. This is the message that is sent to the customer if the curtailment offer is stopped early.

  6. Set the minimum notification time in the Min Notify Time box. This is the minimum number of minutes before the start of the curtailment period to send out the notification message.

  7. Set the acknowledgment time limit in the Ack Time Limit box. This is the amount of time a customer has to acknowledge the curtailment notification message.

  8. Click Next.

  9. Assign customers to or unassign customers from the curtailment program.

    • To assign a customer, select the customer from the Available list and click Add. To select more than one customer, hold Ctrl key on the keyboard while clicking additional customers.

    • To unassign a customer, select the customer from the Assigned list and click Remove. To select more than one customer, hold Ctrl key on the keyboard while clicking additional customers.

  1. To require acknowledgement from the customer of the notification, select the Ack Required check box for customers in the Available list.

  2. Click Finish. The new load program appears in the Program Editor. For instructions on editing a load program, refer to the related topics.
