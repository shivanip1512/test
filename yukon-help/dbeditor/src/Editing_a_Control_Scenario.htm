Editing a Control Scenario

A control scenario is used to define start and stop offsets for a large number of manually controlled load programs and the starting gears for each load program.

 

Perform the following steps to edit a control scenario:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Scenario.

  3. Double-click on the control scenario to edit. This General tabbed page is displayed in the Control Scenario Editor.

  4. Click the Apply button to save the changes to the control scenario or click the OK button to save the changes to the control scenario and close the Control Scenario Editor.
