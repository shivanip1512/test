Editing a Control Scenario

A control scenario defines when to start load programs and the gears to use for the programs.

 

To edit a control scenario:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Scenario.

  3. Double-click the control scenario to modify. This opens the General tab of the Control Scenario Editor.

  4. Click the Apply button to save the changes or OK button to save the changes and close the Control Scenario Editor.
