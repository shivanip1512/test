RTM Verification Tab

The following changes can be made to the communication verification settings of an RTM:

  • The verification of RTM communication can be disabled or enabled by selecting or clearing the Disable Verification check box. Select this check box to disable verification of RTM communication. Clear this check box to enable verification of RTM communication.

  • To assign an RTC signal transmitter to the RTM for communication verification, select the RTC from the Available Devices list and click the Assign to RTM button. To select more than one RTC, hold the Ctrl key on the keyboard while clicking additional RTCs.

  • To make an RTC resend a command if the RTM detects a communication failure, select the Resend on Fail check box of the RTC in the Assigned Devices area.

  • To unassign an RTC signal transmitter from the RTM, select the RTC from the Assigned Devices list and click the Remove from RTM button. To select more than one RTC, hold the Ctrl key on the keyboard while clicking additional RTCs.
