Editing a Tag

Tags can be used to inhibit control of field devices.

To edit a tag:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Tag.

  3. Double-click on the tag to modify. This opens the General tabbed page of the Tag Editor, from which the following changes can be made:

    1. The name of the tag can be changed in the Tag Name text box.

    2. The level of the tag can be changed in the Tag Level text box. The value must be in decimal format.

    3. To prevent control of a device that has been tagged, select the Inhibit Control check box. To allow control of a device that have been tagged, clear the Inhibit Control check box.

    4. A different color to represent the tag can be selected from the Color drop-down list.

    5. A different image to represent the tag can be selected by clicking the Image button.

  4. Click Apply to save the changes or OK to save the changes and close the Tag Editor.
