Editing a Program Constraint

A program constraint controls when a load program can start.

 

To edit a program constraint:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Constraint.

  3. Double-click the program constraint to modify. This opens the General tab of the Program Constraint Editor.

  4. Click Apply to save the changes or OK to save the changes and close the Program Constraint Editor.
