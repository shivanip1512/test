Creating a Load Management Point

Perform the following steps to create a point:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Point.

  3. Select the type of point:

    • Analog – Reads a value directly from the transducer or electronic device.

    • Status – Reads the value of a contact or digital input, usually open or closed. An example of a status point might be the relay on a verification load control receiver.

  4. Click the Next button.

  5. Enter a name for the point in the Name text box.

  6. Select the device on which to create this point from the Device drop-down list.

  7. Click the Next button.

  8. To continue creating a point, click on the type of point that was selected in step 3 from the list of point types below:

    • Analog

    • Status
