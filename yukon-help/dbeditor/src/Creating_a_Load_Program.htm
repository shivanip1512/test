Creating a Load Program

To create a Load Program:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Load Program.

  3. Select the type of program: Direct Control, Curtailment, or Energy Exchange.

  4. Click Next.

  5. Enter a name for the program in the Name text box.

  6. Select an operational state from the Operational State list.

    • Automatic - the program controls when the trigger threshold is exceeded or when the program is manually started.

    • Manual Only - the program must be manually started to perform control.

    • Timed - the program controls during the specified control windows.

  7. Select the constraint to use from the Program Constraint drop-down list. A program constraint controls when the load program can run.

  8. To specify a trigger offset for this program, enter an offset in the Trigger Offset text box. When the load program is assigned to a control area, the trigger offset is added to the trigger threshold. The trigger point defined in the control area must exceed the trigger offset plus the trigger threshold before the load program can start control. The default value is zero.

  9. To specify a restore offset for this program, enter an offset in the Restore Offset text box. When the load program is assigned to a control area, the restore offset is subtracted from the control area trigger threshold. The trigger point defined in the control area must go below the control area trigger threshold minus the trigger offset before the load program can stop control. A restore offset with a positive value causes the trigger threshold to be lowered, a restore offset with a negative value causes the trigger threshold to be raised. The default value is zero.

  10. Click Next. Based on the type of program selected, the editor follows one of three paths:

    • Direct Control

    • Curtailment

    • Energy Exchange
