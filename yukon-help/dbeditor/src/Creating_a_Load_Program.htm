Creating a Load Program

To create a Load Program:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Load Program.

  3. Enter a name for the program in the Name text box.

  4. Select an operational state from the Operational State list.

    • Automatic - the program controls when the trigger threshold is exceeded or when the program is manually started.

    • Manual Only - the program must be manually started to perform control.

    • Timed - the program controls during the specified control windows.

  5. Select the constraint to use from the Program Constraint drop-down list. A program constraint controls when the load program can run.

  6. To specify a trigger offset for this program, enter an offset in the Trigger Offset text box. When the load program is assigned to a control area, the trigger offset is added to the trigger threshold. The trigger point defined in the control area must exceed the trigger offset plus the trigger threshold before the load program can start control. The default value is zero.

  7. To specify a restore offset for this program, enter an offset in the Restore Offset text box. When the load program is assigned to a control area, the restore offset is subtracted from the control area trigger threshold. The trigger point defined in the control area must go below the control area trigger threshold minus the trigger offset before the load program can stop control. A restore offset with a positive value causes the trigger threshold to be lowered, a restore offset with a negative value causes the trigger threshold to be raised. The default value is zero.

  8. Click Next.

  9. Click the Add button to add a new gear.

    Gears define the different rates at which a load program can controls loads. Different gear ratios result in different amounts of reduction.

    The control gears are:

      • Time Refresh - timed commands are sent repeatedly to control.

      • Rotation - the gear rotates through a list of groups to control.

      • Master Cycle - the Master Station sends commands to the LCR to perform cycling.

      • Smart Cycle - the LCR hardware performs cycling.

      • TrueCycle - the LCR uses an algorithm to perform cycling.

      • Magnitude Cycle - uses an algorithm to perform cycling, which targets a percentage of the max current the field device draws.

      • Target Cycle - control is targeted to a specific amount of kW reduction.

      • Latching - one command is sent to open and another command is sent to close.

      • Thermostat Ramping - used to perform thermostat temperature ramping.

      • No Control - no control command is sent.

  10. Click Next.

  11. Select the Use Window 1 or Use Window 2 check box to activate one or both control windows.

  12. If you selected a control window check box, select a start time and a stop time for the control window from the Hr and Min drop-down lists.

  13. Click Next.

  14. To assign a load group to the load program, select the group from the Available Groups list and click Add. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.

  15. To unassign a load group to the load program, select the group from the Assigned Groups list and click Remove. To select more than one load group, hold the Ctrl key on the keyboard while clicking additional groups.

  16. Click Next.

  17. To send out a notification message before control starts, select the Program Start check box and enter how many minutes before control starts to send the message. If this check box is not selected, a notification message is not sent out when the control starts.

    To send out a notification message after control stops, select the     Program Stop check box and enter how many minutes after control stops to send the message. If this check box is not selected, a notification message is not sent out when the control stops.

  18. To assign a notification group, select the group from the Notification Groups list and click Add. To select more than one notification group, hold the Ctrl key on the keyboard while clicking additional groups.

    To unassign a notification group, select the group from the     Assigned Groups list and click Remove. To select more than one notification group, hold the Ctrl key on the keyboard while clicking additional groups.

  19. Click Finish. The new load program appears in the Program Editor. For instructions on editing a load program, refer to the related topics.
