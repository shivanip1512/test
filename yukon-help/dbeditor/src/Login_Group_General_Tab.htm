Login Group General Tab

The following changes can be made to the general settings of a login group:

 

  1. The name of the login group can be changed in the Group Name text box.

  2. The description of the login group can be changed in the Description text box.

  3. The Members of this Group list displays the logins assigned to this login group.

Note: The members of the login group cannot be modified from this tabbed page.
