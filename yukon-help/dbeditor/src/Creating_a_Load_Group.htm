Creating a Load Group

To create a Load Group:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Load Group.

  3. Select either Load Group or Macro Load Group for the type of group.

  4. Click Next. If you selected Macro Load Group, see Macro Load Group for instructions. If you selected Load Group, continue with step 5 of this procedure.

  5. Select the type of switch to use for this load group from the list of switches.

  6. Click Next. Based on the type of switch selected, the editor follows one of the these paths:

  • LCR 5000 (ExpressCom)

  • LCR 5000 (VERSACOM)

  • LCR 4700 (ExpressCom)

  • LCR 4700 (Golay)

  • LCR 4500 (ExpressCom)

  • LCR 4500 (Golay)

  • LCR 4000 (VERSACOM)

  • LCR 3100 (ExpressCom)

  • LCR 3100 (VERSACOM)

  • LCR 3000 (VERSACOM)

  • LCR 3000 (EMETCON)

  • LCR 2000 (VERSACOM)

  • LCR 1000 (Ripple)

  • LMT 100 Series (EMETCON)

  • DCU-S1180 (Golay)

  • T-STAT (ExpressCom)

  • Point Group

  • MCT Group
