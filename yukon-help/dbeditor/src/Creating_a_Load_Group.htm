Creating a Load Group

Perform the following steps to create a load group:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Load Group.

  3. Select either Load Group or Macro Load Group for the type of group.

  4. Click the Next button. If Macro Load Group was selected, see Macro Load Group for instructions. If Load Group was selected, continue with step 5 of this procedure.

  5. Select the type of load group to create from the list of groups.

    • EMETCON Group – Load group for devices that can be communicated with using the EMETCON protocol.

    • ExpressCom Group – Load group for devices that can be communicated with using the ExpressCom protocol.

    • Golay Group – Load group for devices that use the Golay protocol.

    • Integration Group – Load group for devices that can be communicated with using RF integration.

    • MCT Group – Load group for devices that can be communicated with using Bronze, Lead, and MCT address levels.

    • Point Group – Load group for devices that can be communicated with using a point group.

    • Ripple Group – Load group for devices that can be communicated with using the Ripple protocol.

    • VERSACOM Group – Load group for devices that can be communicated with using the VERSACOM protocol.

  6. Click the Next button.

  7. To continue creating a load group, click on the load group, which was selected in step 5, from the list of load groups below:

  • EMETCON Group

  • ExpressCom Group

  • Golay Group

  • Integration Group

  • MCT Group

  • Point Group

  • Ripple Group

  • VERSACOM Group
