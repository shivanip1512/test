Creating a Route

To create a Route:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Route.

  3. Select either Communication route or Macro route. A communication route is used for a single communication route involving a transmitter. A macro route is used for multiple communication routes.

  4. Click Next. Based on the type of route selected, the editor follows one of two paths.

    • Communication Route

    • Macro Route
