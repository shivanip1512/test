Creating a Route

Perform the following steps to create a route:

 

  1. From the View menu, select Core.

  2. From the Create menu, select Route.

  3. Select the type of route to create:

    • Communication route – A route that is used for a single communication route involving a signal transmitter.

    • Macro route (multiple communications routes) – A route that is used to be combine multiple communication routes.

  4. Click the Next button.

  5. To continue creating a route, click on the type of route that was selected in step 3 from the list of routes below:

    • Communication Route

    • Macro Route
