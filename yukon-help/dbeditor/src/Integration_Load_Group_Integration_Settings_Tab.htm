Integration Load Group Integration Settings Tab

The following changes can be made to the integration settings of the Integration load group:

 

  1. The Utility Enrollment Group ID parameter, which allows control events to be directed to specific group of devices, can be modified by performing the following steps:

    1. Click on the UtilID parameter to select it.

    2. Enter a value for the UtilID parameter in the Value text box.

    3. Click the Update button to save the changes to the parameter.

  2. The Device Class parameter, which identifies the type of device that is being controlled, can be modified by performing the following steps:

    1. Click on the DeviceClass parameter to select it.

    2. Enter a value for the DeviceClass parameter in the Value text box.

    3. Click the Update button to save  the changes to the parameter.

  3. The Device Group parameter, which identifies the type of device that is being controlled, can be modified by performing the following steps:

    1. Click on the DeviceGroup parameter to select it.

    2. Enter a value for the DeviceGroup parameter in the Value text box.

    3. Click the Update button to save  the changes to the parameter.

  4. To add an integration settings parameter, performing the following steps:

    1. Enter the name of the parameter in the Name text box.

    2. Enter a value for the DeviceGroup parameter in the Value text box.

    3. Click the Add button to add the parameter.

To remove an integration settings parameter, performing the following steps:

    1. Click on the parameter to remove.

    2. Click the Remove button to remove the parameter.
