Load Group General Tab

The following changes can be made to the general settings of the load group:

 

  1. The name of the load group can be changed in the Group Name text box.

  2. The kilowatt capacity of the load group can be changed in the kW Capacity text box. This is the estimated kilowatt capacity of the devices in the load group. To disable the estimated kilowatt capacity feature, enter a value of zero.

  3. If the load group is an ExpressCom load group, a different control priority for ExpressCom messages can be selected from the Control Priority drop-down list. A message with a medium priority is ignored if a message with a high priority has already been received by the load control device.

  4. The load group can be disabled or enabled by selecting or clearing the Disable Group check box. Select this check box to disable the load group. Clear this check box to enable the load group.

  5. Control operations can be disabled or enabled by selecting or clearing the Disable Controls check box. Select this check box to globally disallow control operations on this device. Clear this check box if there are control points on this device.

  6. A different communication route for the load group can be selected from the Communication Route drop-down list.

Note: The Communication Route drop-down list does not appear for Point Groups.
