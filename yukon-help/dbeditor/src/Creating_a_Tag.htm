Creating a Tag

Tags can be used to inhibit control of field devices.

Perform the following steps to create a tag:

  1. From the View menu, select System.

  2. From the Create menu, select Tag.

  3. Enter a name for the tag in the Tag Name text box.

  4. Enter a level for the tag in the Tag Level text box. The value must be in decimal format.

  5. To prevent control of a device that has been tagged, select the Inhibit Control check box. To allow control of a device that have been tagged, clear this check box.

  6. Select a color to represent the tag from the Color drop-down list.

  7. Select an image to represent the tag by clicking the Image Select button.

  8. Click the Finish button. The new tag appears in the Tags Editor. For instructions on editing a tag, refer to the related topics.
