Editing a Notification Recipient

To edit a Notification Recipient:

 

  1. From the View menu, select System.

  2. From the Sort By list, select Notification Recipients.

  3. Double-click a notification recipient. This opens the General tabbed page of the Recipient Editor.
    You may make the following changes:

    1. Change the name of the recipient in the Name text box.

    2. Select a different type of notification from the Notification Type list.

    3. Change the e-mail address in the Address text box. This e-mail address can be a paging e-mail address so that the recipient will receive notifications via paging.

    4. Select or clear the Disable check box. Select the check box to disable this recipient. Clear this check box to enable the recipient.

  4. Click Apply to save the changes or OK to save the changes and close the dialog box.
