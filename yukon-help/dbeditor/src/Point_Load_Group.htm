Point Load Group

If the load group is a Point load group:

 

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of the load control devices that match the addressing of the load group. To disable the this feature, enter a value of zero.

  3. Disable or enable the group by selecting or clearing the Disable Group check box. Select this check box to take the load group out of service. Clear this check box to enable the load group.

  4. Disable or enable control of the group by selecting or clearing the Disable Control check box. Select this check box to disable control. Clear this check box to enable control.

  5. Click the Next button.

  6. Select the device on which the control point is attached from the Control Device drop-down list.

  7. Select the point on the device that performs the control from the Control Point drop-down list.

  8. Select the state the point needs to be in to start control from the Control Start State drop-down list.

  9. Click the Finish button. The new load group appears in the Load Group Editor.
