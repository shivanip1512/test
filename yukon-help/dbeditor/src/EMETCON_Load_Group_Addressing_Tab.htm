EMETCON Load Group Addressing Tab

The following changes can be made to the addressing settings of the EMETCON load group:

 

  1. The gold address of the load group can be changed in the Gold Address box. The gold address range is 1 to 4 and it is used for maximum load control coverage and it is normally reserved for emergency sheds

  2. The silver address of the load group can be changed in the Silver Address spin box. The silver address range is 1 to 60 and it is used for medium load control coverage and non-emergency sheds.

  3. A different relay option to use for the load group can be selected in the Relay to Use area. The relay options are:

    • Relay A – Load on physical relay A, which is either a 5 A or 30 A relay.

    • Relay B – Load on physical relay B, which is either a 5 A or 30 A relay.

    • Relay C – Load on physical relay C, which is a 5 A relay.

    • S (all) – Controls all relays (SCRAM).

  4. A different address to use for control of this load group can be selected under the Address to Use area. The address to use options are:

    • Gold Address – Commands are addressed to load control devices that are configured with the gold address value that was entered in step 1.

    • Silver Address – Commands are addressed to load control devices that are configured with the silver address value that was entered in step 2.
