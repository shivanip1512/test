MCT Group

If you selected MCT Group:

  1. Enter a name for the load group in the Group Name text box.

  2. Enter the kilowatt capacity for the group in the kW Capacity text box. This is the estimated kilowatt capacity of field devices that match the addressing of the load group. To disable the this feature, enter a value of zero.

  3. Select the Disable Group check box if you want to take the load group out of service.

  4. Select the Disable Control check box if you want to disable control of the load group.

  5. Select the route to use from the Communication Route drop-down list. This is the route or route macro used to reach all of the field devices in this group when transmitting commands.

  6. Click Next.

  7. Select the type addressing to use from the Address Level drop-down list. The types of addressing are Bronze, Lead, and MCT Address.

  8. If you selected Bronze or Lead as the addressing type, enter the MCT address in the Address text box. This allows all of the devices using this address to be controlled.

  9. If you selected MCT Address as the addressing type, select an MCT to use for addressing from the MCT Address drop-down list. This allows the specified MCT to be controlled.

  10. Select the check boxes of the relays to control for this group in the Relay Usage area.

  11. Click Finish. The new load group appears in the Device Editor. For instructions on using this editor, refer to related topics.
