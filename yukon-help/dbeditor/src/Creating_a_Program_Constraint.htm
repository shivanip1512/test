Creating a Program Constraint

Perform the following steps to create a program constraint for a load program:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Program Constraint.

  3. Enter a name for the program constraint in the Constraint Name text box.

  4. To assign a season schedule to the program constraint, select a schedule from the Season Schedule drop-down list.

  5. The maximum number of seconds that a load group is allowed to be controlled can be changed in the Max - Activate text box. This constraint is reset when the load program becomes inactive.

  6. The maximum number of times that a load group is allowed to be controlled in one day can be changed in the Max - Daily Ops text box. This constraint is reset at the end of the day.

  7. The minimum number of seconds that a load group must be controlled, before it is allowed to stop, can be changed in the Min - Activate text box. This constraint is reset after the load group stops.

  8. The minimum number of seconds to wait after a load group stops being controlled, before it is allowed to be controlled again, can be changed the Min - Restart text box. This constraint is reset when the load program becomes inactive.

  9. To specify the days of the week that control can or cannot occur, select or clear the day check boxes in the Day Selection area.
    To prevent control from occurring on a day of the week, clear the check box for that day. To allow control to occur on a day of the week, select the check box for that day.
    Note:     The program constraint allows control to occur every day of the week by default.

  1. To assign a holiday schedule to the program constraint, select a schedule from the Holiday Schedule drop-down list.

  2. If a holiday schedule is selected, select what to do on a holiday from the options in the Holiday Usage area:

    • Exclude – Control cannot occur on the days defined in the holiday schedule.

    • Force – Control can occur on the days defined in the holiday schedule, even if the day is not selected in the Day Selection area.

  3. The maximum number of hours that a load group is allowed to be controlled for each time period (Daily, Monthly, Seasonal, and Annually) can be entered in the Max Hour Allowance area. A value of zero disables the time constraint.

  4. Click the Finish button. The new program constraint appears in the Program Constraint Editor. For instructions on editing a program constraint, refer to the related topics.
