Creating a Program Constraint

To create a load program constraint:

 

  1. From the View menu, select Load Management.

  2. From the Create menu, select Program Constraint.

  3. Enter a name for the program constraint in the Constraint Name text box.

  4. Select a load control season schedule from the Season Schedule drop-down list.

  5. Enter the maximum number of seconds that a load group can run in the Max - Activate text box. This constraint is reset when the load program becomes inactive.

  6. Enter the maximum number of times a load group can run in one day in the Max - Daily Ops text box. This constraint is reset at the end of the day.

  7. Enter the minimum number of seconds that a load group must run before it can be stopped in the Min - Activate text box. This constraint is reset after the load group stops.

  8. Enter the minimum number of seconds to wait after a load group stops before the load group can run again in the Min - Restart text box. This constraint is reset when the load program becomes inactive.

  9. Select the check boxes for which days of the week the load program is available under the Day Selection area

  10. Select a holiday schedule from the Holiday Schedule drop-down list.

  11. If a holiday schedule is selected, select what to do on holidays in the Holiday Usage area:

    • Exclude - control cannot occur on the days defined in the holiday schedule.

    • Force - control can occur on the days defined in the holiday schedule, even if the day is not selected in the Day Selection area.

  12. The maximum number of hours that a load group can be controlled for each time period (Daily, Monthly, Seasonal, Annual) can be entered under the Max Hour Allowance area. A value of zero disables the time constraint.

  13. Click Finish. The new program constraint appears in the Program Constraint Editor. For instructions on editing a program constraint, refer to the related topics.
