Editing a Control Area

A control area defines geographical areas for controlling loads to reduce the peak for an area. A control area can consist of the whole system, portion of the system, substation, or any controllable area.

 

To edit a control area:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Control Area.

  3. Double-click the control area to modify. This opens the General tab of the Control Area Editor.

  4. Click the Triggers tab to display the Triggers tabbed page. If triggers exists, they appear in the Current Trigger area.

    • To create a new trigger, see Creating a Trigger.

    • To edit a trigger, see Editing a Trigger.

    • To delete a trigger, see Deleting a Trigger.

  5. Click the Programs tab to display the Programs tabbed page. The load programs are assigned to the control area, they appear in the Assigned Programs area.

    • To add a program to a control area, see Adding a Program.

    • To edit a program assigned to a control area, see Editing a Program.

    • To remove a program from a control area, see Removing a Program.

  6. Click Apply to save the changes or OK to save the changes and close the Control Area Editor.
