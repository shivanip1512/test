Editing a Control Area

A control area defines geographical areas for controlling loads to reduce the peak for an area. A control area can consist of the whole system, portion of the system, substation, or any controllable area.

 

Perform the following steps to edit a control area:

 

  1. From the View menu, select Load Management.

  2. From the Sort By list, select Control Area.

  3. Double-click on the control area to edit. The General tabbed page is displayed in the Control Area Editor.

  4. Click the Triggers tab to display the Triggers tabbed page. If triggers exist, they are displayed in the Current Trigger area.

    • To create a new trigger, see Creating a Trigger.

    • To edit a trigger, see Editing a Trigger.

    • To delete a trigger, see Deleting a Trigger.

  5. Click the Programs tab to display the Programs tabbed page. The load programs that are assigned to the control area appear in the Assigned Programs list.

    • To add a load program to a control area, see Adding a Load Program.

    • To edit the settings of a load program assigned to a control area, see Editing an Assigned Load Program.

    • To remove a load program from a control area, see Removing a Load Program.

  6. Click the Apply button to save the changes to the control area or click the OK button to save the changes to the control area and close the Control Area Editor.
