RFN Template General Tab

The following changes can be made to the general settings of the RFN template:

 

  1. The name of the RFN template can be changed in the Device Name text box.

    • For an RFN-410fL, the template name is *RfnTemplate_LGYR_FocuskWh.

    • For an RFN-410fX, the template name is *RfnTemplate_LGYR_FocusAXR

    • For an RFN-410fD, the template name is *RfnTemplate_LGYR_FocusAXD.

    • For an RFN-430A3, the template name is *RfnTemplate_EE_A3R.

    • For an RFN-420kV, the template name is *RfnTemplate_GE_kV2.

  2. Do not enter any values in the Serial Number, Manufacturer, and Model text boxes. These fields must remain blank for the RFN template.

  3. To configure the RFN template so that the default points are added to an RFN created from this template, select the Create Default Points check box. To prevent the default points from being added to an RFN created from this RFN template, clear this check box.
