Editing a Latching Gear

To edit a Latching gear:

  1. The name of the gear can be changed in the Gear Name text box.

  2. Select Latching from the Gear Type drop-down list.

  3. A different starting state can be selected from the Control Start State drop-down list. This controls whether an open or close command is sent at the beginning of control.

  4. The group capacity reduction percentage can be changed in the Group Capacity Reduction % spin box. This is the percentage of each load group’s kW capacity (if specified), which determines the amount of kW reduction that must occur before control can stop

  5. Click OK to save the changes.
