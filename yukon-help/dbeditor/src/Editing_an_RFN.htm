Editing an RFN

Perform the following steps to edit a signal transmitter:

 

  1. From the View menu, select Core.

  2. From the Sort By list, select one of the following:

    • Device

    • IED

    • Meter

    • Meter Number

  3. Double-click on the RFN to edit. The General tabbed page is displayed in the Device Editor.

  4. Click the Metering tab to display the Metering tabbed page.

  5. Click the Apply button to save the changes to the RFN or click the OK button to save the changes to the RFN and close the Device Editor.
