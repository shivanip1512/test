Creating an MCT Config

The information in a config is uploaded to an MCT when a putconfig EMETCON install command is sent to the MCT from Yukon Commander.

To create an MCT config:

  1. From the View menu, select Core.

  2. From the Create menu, select MCT Config.

  3. Select either 200 Series MCT or 300 Series MCT.

  4. Click Next. Based on the type of MCT config selected, the wizard will follow one of two paths.

    • 200 Series MCT

    • 300 Series MCT
