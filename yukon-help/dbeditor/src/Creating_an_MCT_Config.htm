Creating an MCT Config

To manually upload a config to an MCT, start the Yukon Commander application, select the MCT to which to send the config, and execute the putconfig EMETCON install command.

Perform the following steps to create an MCT config:

 

  1. From the View menu, select Core.

  2. From the Create menu, select MCT Config.

  3. Select 300 Series MCT.

  4. Click the Next button.

  5. Enter a name for the config in the Configuration Name text box.

  6. Select the mode of operation for the MCT config:

    • On-Peak/Off-Peak Mode – Configures the MCT to record the maximum demand readings for either the On-Peak or Off-Peak interval.

    • Min/Max Mode – Configures the MCT to record the minimum and maximum demand readings.

  7. The check box for channel 1 is selected by default and cannot be deselected.

    1. Select how the MCT is wired for channel 1 by selecting the 3-Wire (KYZ) or 2-Wire (KY) radio button.

    2. If the Ke value (kWh per pulse) for channel 1 is known, enter it in the Ke text box.
      If the Ke value for channel 1 is not known:

      1. Click the Recalculate button.

      2. Enter the Mp value (pulse constant) and Kh value (watt-hour or disk constant) into the appropriate text boxes.

      3. Click the = (equals) button to calculate the Ke value.

  8. To enable channel 2, select the Channel 2 check box.

    1. Select how the MCT is wired for channel 2 by selecting the 3-Wire (KYZ) or 2-Wire (KY) radio button.

    2. If the Ke value (kWh per pulse) for channel 2 is known, enter it in the Ke text box.
      If the Ke value for channel 2 is not known:

      1. Click the Recalculate button.

      2. Enter the Mp value (pulse constant) and Kh value (watt-hour or disk constant) into the appropriate text boxes.

      3. Click the = (equals) button to calculate the Ke value.

  9. To enable channel 3, select the Channel 3 check box.

    1. Select how the MCT is wired for channel 3 by selecting the 3-Wire (KYZ) or 2-Wire (KY) radio button.

    2. If the Ke value (kWh per pulse) for channel 3 is known, enter it in the Ke text box.
      If the Ke value for channel 3 is not known:

      1. Click the Recalculate button.

      2. Enter the Mp value (pulse constant) and Kh value (watt-hour or disk constant) into the appropriate text boxes.

      3. Click the = (equals) button to calculate the Ke value.

  10. Click the Finish button. The new MCT Config is displayed in the Config Editor. For instructions on editing an MCT config, refer to the related topics.
