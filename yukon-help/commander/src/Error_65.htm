Error 65: No DCD on Return Message

Description

Communication failure between the Master Station and the CCU.

 

Probable Reason

Several possible causes (see solutions below).

 

Solution

Check the following possibilities:

  • Check if CCU is turned off or failed.

  • Check if modem failed.

  • Check if communication (radio link or phone line) to substation is down.

  • Check that the CCU is switched to external.

  • Check if burst of noise interfered with radio transmission. Resend the command.

  • Check if CCU failed to handshake by raising carrier on receive line.

  • Check for improper radio delays.
