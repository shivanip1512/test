Finding a Device

Perform the following steps to find a device in the database item list:

 

  1. Select the type of item to search for from the Sort By drop-down list.

  2. Select Find from the View menu.

  3. Enter the name of the item in the Find text box.

  4. Click the OK button to find the device. If the device is found, the first device that matches the search is highlighted in the database item list.
