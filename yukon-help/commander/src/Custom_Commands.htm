Custom Commands

To add custom commands to the Yukon Commander:

 

  1. Select the database type from the Sort By list.

  2. Select the device to which you want to add custom commands from the database item list.

  3. From the Command menu, select Edit Custom Commands. A dialog box appears that shows the existing commands for the device selected in step 2.

  4. To add a command, click the Add button and perform the following steps:

    1. Enter a user-friendly alias for the command in the Command Label text box.

    2. Enter the actual command to send in the Execute Command text box.

    3. Click OK to add the command to the table of custom commands.

  5. To remove a custom command from the device, select the command and click Remove and click Yes to delete the command.

  6. Click OK to save your changes.

 

The custom command can then be selected from the Command Commands drop-down list.

Command Syntax

Yukon Commander interprets question marks as prompts for data. Quotation marks are used to separate the prompt text from the rest of the command. For example, the following command prompts you to enter the number of minutes to shed.

 

Shed x Minutes = control shed ?‘Minutes to Shed’m

 

When this command is run, a text box appears asking you to enter the Minutes to Shed.

 

Entering a value into the text box and clicking OK causes the command to run, with the value entered replacing ?‘Minutes to Shed’. For example, entering 15 causes the following command to run:

 

control shed 15m

Values That Contain Spaces

To allow for values that contain spaces, move the quotation mark outside of the question mark. For example, the following command prompts for a Load Group, which have names that often contains spaces.

 

Install Addressing = putconfig template ‘?LoadGroupName’

 

When this command is run, a text box appears asking you to enter the LoadGroupName.

 

Entering Afton Summer A/C and clicking OK causes the following command to run:

 

putconfig template ‘Afton Summer A/C’

Values and Prompts That Contain Spaces

Use a second set of quotation marks to allow for values that contain spaces and prompt text that contains spaces. For example:

 

Install Addressing = putconfig template ‘?‘Load Group Name’’

 

When this command is run, a text box appears asking you to enter the Load Group Name.

 

Entering Afton Summer A/C and clicking OK causes the following command to run:

 

putconfig template ‘Afton Summer A/C’
