Custom Commands

Perform the following steps to add, edit, or remove custom commands in the Yukon Commander:

 

  1. Select the devices to display from the Sort By list.

  2. Select the type of device to which to add custom commands from the database item list.

  3. Select Edit Custom Commands from the Command menu. A dialog box appears that shows the existing commands for the device selected in step 2.

  4. To add a custom command to the device:

    1. Click the Add button.

    2. Enter a user-friendly name for the command in the Command Label text box.

    3. Enter the actual command to send to the device in the Execute Command text box.
      Refer to the following help pages for information about command syntax and values that contain spaces:

      • Command Syntax

      • Values that Contain Spaces

      • Values and Prompts that Contain Spaces

    4. Click the OK button to add the custom command to the device.

To edit an existing custom command:

  1. Select the custom command to edit from the list of commands.

  2. Click the Edit button.

  3. The user-friendly name for the command can be changed in the Command Label text box.

  4. The actual command to send to the device can be changed in the Execute Command text box.
    Refer to the following help pages for information about command syntax and values that contain spaces:

      • Command Syntax

      • Values that Contain Spaces

      • Values and Prompts that Contain Spaces

    1. Click the OK button to save changes to the custom command.

To remove a custom command from the device, select the command from the list and click the Remove button and click the Yes button to delete the command.

  1. Click the OK button to save the changes to the custom commands.

After the custom command is added to a device type, the command can be selected from the Command Commands drop-down list for that device type.
