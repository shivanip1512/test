Download TOU Schedule

Perform the following steps to send a TOU Schedule to an MCT-400 series meter:

 

  1. Select Device from the Sort By drop-down list.

  2. Select an MCT-400 series meter from the Devices list.

  3. Select Download TOU Schedule from the Command menu.

  4. Select the TOU schedule to send to the selected MCT from the Select TOU Schedule to Download drop-down list.

  5. Click the Download button to send the TOU schedule to the selected MCT.
