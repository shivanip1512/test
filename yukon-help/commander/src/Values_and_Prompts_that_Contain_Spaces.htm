Values and Prompts that Contain Spaces

Use a second set of quotation marks to allow for values that contain spaces and prompt text that contains spaces. For example:

 

Install Addressing = putconfig template ‘?‘Load Group Name’’

 

When this command is run, a text box appears asking you to enter the Load Group Name.

 

Entering Afton Summer A/C and clicking OK causes the following command to run:

 

putconfig template ‘Afton Summer A/C’
