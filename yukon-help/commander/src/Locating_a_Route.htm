Locating a Route

To find which route a distribution line carrier device is associated with, use the locate route command.

 

To use the locate route command:

 

  1. Select Device from the Sort By drop-down list.

  2. Select a power line communications (PLC) field device from the database item list.

  3. Select Locate Route from the Command menu.

  4. From the drop-down list, select the route or route macro on which to send the command.

  5. Click Locate.

 

If the command is successful, the route the device is associated with is displayed with the text Loopback successful. If the command is not successful, an error message is displayed.
