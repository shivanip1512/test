Locating a Route

Use the locate route command to find the route on which a Power Line Carrier (PLC) field device is located.

 

Perform the following steps to use the locate route command:

 

  1. Select Device from the Sort By drop-down list.

  2. Select a PLC field device from the database item list.

  3. Select Locate Route from the Command menu.

  4. Select the route or route macro on which to send the locate route command from the Select Route to Locate drop-down list.

  5. Click the Locate button to find the route. If the command is successful, the route the device is associated with is displayed with the text Loopback successful. If the command is not successful, an error message is displayed.
