Installing LCR Addressing

To install addressing into an LCR:

 

  1. Select ExpressCom Serial or Versacom Serial from the Sort By drop-down list.

  2. Select the serial number of the device to install addressing into from the database item list or enter the serial number in the Serial # text box and press Enter.

  3. Select Install Addressing from the Command menu.

  4. Select an addressing template from the Load Group Template drop-down list.

  5. Click OK to install the addressing.
