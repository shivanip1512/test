Installing LCR Addressing

Perform the following steps to install addressing into load control device:

 

  1. Select ExpressCom Serial or Versacom Serial from the Sort By drop-down list.

  2. Select the serial number of the load control device to install addressing into from the database item list.
    If the serial number of the load control device is not listed in the database item list, enter the serial number of the load control device in the Serial # text box and press the Enter key on the keyboard to add it to the list.

  3. Select Install Addressing from the Command menu.

  4. Select a load group addressing template from the Load Group Template drop-down list.

  5. Click the OK button to send the load group addressing template to the load control device.
