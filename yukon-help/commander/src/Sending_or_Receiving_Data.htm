Sending or Receiving Data

To enter or select commands for collecting or sending data:

 

  1. Select a database type from the Sort By list. The database types are:

    • Device - list of the field devices in the Yukon system.

    • Meter Number - list of the meter numbers in the Yukon system

    • MCT Broadcast - list of the MCT Broadcast groups.

    • Transmitter - list of signal transmitters in the Yukon system.

    • Load Group - list of the load groups in the Yukon system.

    • Cap Bank Controller - list of the cap bank controllers in the Yukon system.

    • Collection Group - list of the collection groups in the Yukon system.

    • Test Collection Group - list of the test collection groups in the Yukon system.

    • ExpressCom Serial - select the route to send a command on from the Route # drop-down list and enter the serial number of the ExpressCom device to send a command to in the Serial # text box.

    • Versacom Serial - select the route to send a command on from the Route # drop-down list and enter the serial number of the VERSACOM device to send a command to in the Serial # text box.

  2. To refresh the list of database items, select Reload Devices from the View menu.

  3. Select the database item to which you want to send a command from the database item list.

  4. Select the command options you want to enable.

  5. Select a command from the Common Commands drop-down list. In addition, you can enter or select a command in the Execute Command drop-down list.

  6. To run the command, click the Execute button or select Execute from the Command menu. The output result appears in the area below and command information appears in the Command Log window.

  7. To interrupt the command and stop the display of the output, either click the Stop button or select Stop from the Command menu.

  8. To print the output, click the Print button or select Print from the File menu.

  9. To clear the display area, click the Clear All button.

Note: The last five database items that have had commands sent to them appear under the File menu.
