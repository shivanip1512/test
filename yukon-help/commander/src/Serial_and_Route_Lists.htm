Serial # and Route # List

The Serial # text box and Route # drop-down list become active when ExpressCom Serial or Versacom Serial is selected from the Sort By drop-down list.

  • To add a field device to the ExpressCom serial number list:

    1. Select ExpressCom Serial from the Sort By drop-down list.

    2. Enter the serial number of the ExpressCom device into the Serial # text box

    3. Select the communication route to the device box from the Route # drop-down list.

    4. Press the Enter key on the keyboard to add the serial number to the serial number list.

  • To add a field device to the VERSACOM serial number List:

    1. Select VERSACOM Serial from the Sort By drop-down list.

    2. Enter the serial number of the VERSACOM device into the Serial # text box

    3. Select the communication route to the device box from the Route # drop-down list.

    4. Press the Enter key on the keyboard to add the serial number to the serial number list.
