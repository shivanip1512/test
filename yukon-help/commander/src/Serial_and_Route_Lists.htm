Serial # and Route # List

The Serial # text box and Route # drop-down list become active when ExpressCom Serial or Versacom Serial is selected from the Sort By drop-down list.

 

To add an ExpressCom or VERSACOM device to the Database Item List, enter the serial number of the device into the Serial # text box and select the route to the device box from the Route # drop-down list. By default, all default routes are selected for a new device.
