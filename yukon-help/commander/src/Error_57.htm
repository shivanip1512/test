Error 57: E-Word Received in Returned Message

Description

This message only occurs in systems containing repeaters. The repeater did not receive an expected return message from a two-way device. The repeater then placed E-Word in the response field before transmitting message back to the Master Station.

 

Probable Reason

Failed two-way device.

 

Solution

Check the following possibilities:

  • Check that the two-way device is functional.

  • Check for line switching.

  • Check repeater SCU-810.

  • Check the repeater module or RIM (Repeater Module Interface).

  • Check roles in repeater vs. route in the database.
