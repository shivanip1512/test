Error 17: Word 1 NACK

Description

CCU receives a partial return message from any 2-way device.

 

Probable Reason

Inadequate communication throughput.

 

Solution

Any of the following items may correct the problem:

  • Check gas tube arrestor in SCU.

  • Add an SCU-830 enhancer (H-field couplers).

  • Retune the PCA coupled to the CCU.

  • Check gas tube arrestor in CCU.

  • Replace two-way device.

  • Check receiver card in CCU.

  • Add an RPT/PCA to the distribution line.
