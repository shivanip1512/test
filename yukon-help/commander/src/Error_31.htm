Error 31: Timeout Reading from Port

Description

Communication failure between the Master Station and the CCU.

 

Probable Reason

Several possible reasons (see solutions, below).

Check first for something simple including loss of power to the modem, a disconnected cable between computer and modem, etc.

Next check the database setup and verify the baud rate, protocol wrap, comm port, etc.

 

Solution

Any of the following may be the cause of the problem:

  • Check if CCU is turned off or has failed.

  • Check if modem has failed at either Master Station or CCU.

  • Check if communication (radio link or phone line) to substation is down.

  • Check that CCU is switched to external modem.

  • Check if burst of noise interfered radio transmission. Resend the command.

  • Check if connection between Master Station and modem is bad.
