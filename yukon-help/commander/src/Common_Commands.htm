Common Commands

Device Commands

The following commands are available for devices.

CCU, LCU, and RTU

The commands for CCUs, LCUs, and RTUs are:

Command

Description

1 Comm Loopback

Sends a single loopback command. Used to verify proper operation of the communications line.

5 Comm Loopback

Sends five sequential loopback commands. Used to verify proper operation of the communications line.

TCU

The commands for TCUs are:

Command

Description

1 Comm Loopback

Sends a single loopback command. Used to verify proper operation of the communications line.

5 Comm Loopback

Sends five sequential loopback commands. Used to verify proper operation of the communications line.

Get Status Info

Returns status info.

Load Group Commands

The following commands are available for load groups.

 

Macro Load Group

The commands for Macro load groups are:

Command

Description

Shed 5-min

Controls the selected device for 5 minutes

Cycle 50%/30min

Cycles at 50% for 30 minutes

Terminate Cycle

Terminates the cycle

Restore Load

Restores load to all devices

Reset All Counters

Resets all history counters

Set All Lights ON

Turns on all the LEDs of the selected device

Ripple Group

The commands for VERSACOM groups are:

Command

Description

Shed Loads

Sheds load to all devices

Restore Loads

Restores load to all devices

VERSACOM Group

The commands for VERSACOM groups are:

Command

Description

Shed 5-min

Controls the selected device for 5 minutes

Cycle 50%/30min

Cycles the selected device at 50% for 30 minutes

Terminate Cycle

Cancels the cycling of all devices

Restore Load

Restores load to all devices

Set to Out-of-Service

Disables the selected device

Set to In-of-Service

Enables the select device

Reset All Counters

Reset all history counters

Set All Lights ON

Turns on all the LEDs of the selected device

Cap Bank Controller Commands

The commands for CBCs are:

Command

Description

OPEN Cap Bank

Opens the capacitor bank switch.

CLOSE Cap Bank

Closes the capacitor bank switch.

Disable OVUV

Disables Over Voltage/Under Voltage reporting.

Enable OVUV

Enables Over Voltage/Under Voltage reporting.
