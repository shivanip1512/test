Command Syntax

Yukon Commander interprets question marks as prompts for data. Quotation marks are used to separate the prompt text from the rest of the command. For example, the following command prompts you to enter the number of minutes to shed.

 

Shed x Minutes = control shed ?‘Minutes to Shed’m

 

When this command is run, a text box appears asking you to enter the Minutes to Shed.

 

Entering a value into the text box and clicking OK causes the command to run, with the value entered replacing ?‘Minutes to Shed’. For example, entering 15 causes the following command to run:

 

control shed 15m
