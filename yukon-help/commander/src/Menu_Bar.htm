Menu Bar

The Menu Bar provides four menus, which perform the following functions:

 

 

  • File – Use this menu to save output, print output, and exit the application. In addition, the last five database items that have had commands sent to them are displayed in the File menu.

  • View  – Use this menu to find a database item, reload devices, clear the screen, and delete device serial numbers.

  • Command – Use this menu to execute commands, stop commands, locate routes, install addressing, edit custom commands, and set commander advanced options.

  • Help – Use this menu to display online help and version information.
