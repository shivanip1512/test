Command Options

To change the Yukon Commander options, select Options from the Command menu. The Advanced Properties window appears.

 

The Advanced Properties window contains the following options:

  • Command Priority – Specifies the priority of the command. The lowest priority is 1 and the highest priority is 14.

  • Queue Commands – Select this option to queue MCT commands. This option is not selected by default.

  • Show Message Log – Select this option to display the message log. The message log displays information about the commands sent from Yukon Commander. The log includes the date, time, device, and command sent. This option is selected by default.

  • Confirm Command Execution – Select this option to confirm the execution of a command before it is sent. This option is not selected by default.
