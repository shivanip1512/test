Script Schedule without a Script Template

If the schedule type is script, but a script template was not selected:

  1. Enter a name for the script in the Script Name text box.

  2. Edit the script in the text area. The Yukon Command Language supported by MACS is the Tool Command Language (Tcl). MACS adds its own extensions to support retrieving and sending data from and to field devices. This allows for looping and flow control. See the Yukon Command Language Summary for more information.

  3. Click the Finish button to save the script. The schedule appears in the Metering and Control Scheduler list.
    If you are editing the schedule, click the OK button or Apply button to save the changes.
