Load Management

To enable the Load Management features of Yukon, you must purchase the Load Management module. For more information about this module, contact a Cooper Power Systems sales representative.

To use the Load Management display:

  1. From the Display menu, select Load Management.

  2. Select the type load control areas to view from the View drop-down list. The types are:

    • All Control Areas - all control areas

    • Active Areas - control areas that are being controlled

    • Inactive Areas - control areas that are not being controlled

    • Control Point History - threshold control point history

  3. Click a control area to display the programs and groups assigned to that control area.

  4. Click on any of the toolbar buttons for the following functionality:

    • Silence - turns off the audible alarm for all current alarms. The alarms continue to flash until they are acknowledged.

    • Mute/Unmute - mute turns off the audible alarm for all current and future alarms. Unmute turns the audible alarm back on for all current and future alarms. The alarms continue to flash until they are acknowledged.

    • Refresh - refreshes the data on the current display.

    • Enable All – enables all control areas.

    • Disable All – disables all control areas.

    • Start Scenario - allows you to start a control scenario. See Starting a Control Scenario.

    • Stop Scenario - allows you to stop a control scenario. See Stopping a Control Scenario.

  5. Right-click on a control area to display the following menu:

    • Start Program(s) - allows you to start load programs assigned to the control area.

    • Stop Program(s) - allows you to stop load programs assigned to the control area.

    • Enable Program(s) - allows you to enable multiple load programs assigned to the control area.

    • Disable Program(s) - allows you to disable multiple load programs assigned to the control area.

    • Triggers Change - allows you to change the trigger values.

    • Daily Time Change - allows you to change the daily start and stop times.

    • Disable/Waive or Enable/ReEnable - disables or enables the control area.

    • Reset Peak - allows you to reset the peak values for all triggers to zero on the selected control area.

  6. Right-click on a load program to display the following menu:

    • Start/Stop - starts or stops the selected load program. See Starting a Direct Control Program or Starting a Curtailment Program for instructions on starting a load program.

    • Change Gear -

    • Disable/Enable - disables or enables the selected load program.

  7. Right-click on the load group of a direct control program to display the following menu:

    • Shed - allows you to perform a shed of the selected load group.

    • Restore - restores the selected load group.

    • True Cycle - allows you to send a true cycle command to the selected load group.

    • Smart Cycle - allows you to send a smart cycle command to the selected load group.

    • Disable Group/Enable Group - enables or disables the selected load group.

    • Confirm - resends the last command that Yukon recorded was sent to the load group.
