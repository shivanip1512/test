Adding a Column

Upon initial display of the dialog box, the next number in the column number sequence appears in the Column Number list.

 

To add a column:

 

  1. Enter a name for the column in the Title text box.

  2. Select the column type from the Type list.

  3. Click Add. The next column number appears in the Column Number list.
    Note: If you have edited or deleted one or more columns, you will need to click Add at the beginning of this procedure to display the next column number.

  4. Click OK.
