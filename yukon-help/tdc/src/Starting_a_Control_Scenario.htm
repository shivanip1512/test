Starting a Control Scenario

Complete the following steps to start a control scenario.

 

  1. To start a scenario, click on the Start Scenario toolbar button.

  2. Select the control scenario to start from the Scenario drop-down list.

  3. To start the scenario immediately, select the Start Now check box.
    To start the scenario later, clear the Start Now check box, enter a start time in the Start Time text box, and select a date from the drop-down list.

  4. To specify a stop time, enter a time in the Stop Time text box and select a date from the drop-down list.
    If you do not want to specify a stop time, select the Never Stop check box. The program can then only be stopped manually.

  5. Select the gear to use for all of the programs in the control scenario from the Gear drop-down list. Gears define the rates at which loads in a program are controlled.

    To specify gears for individual programs, select a gear from the Gear drop-down list in the table on the right-hand side of the window.

  6. Select whether to Observe or Check the program constraints from the Constraints drop-down list.

    • Observe - if the constraint is violated, the control does not occur.

    • Check - the control is checked before it is sent to the Load Management server and the result is displayed in the Program Constraint Violation window.

      • If the control does not violate a constraint, press the Resubmit button to send the control to the Load Management server.

      • If the control does violate a constraint, either click the Cancel button to cancel control or select the Override check box and press the Resubmit button to send the control to the Load Management server.

  7. By default, every program in a control scenario is selected to start. To specify which programs to start, clear the Select All check box and select the check boxes of the programs to start.

  8. The Start and Stop columns display the start and stop offsets of the programs in the control scenario. The offsets are displayed in the format of “hours:minutes”.

  9. Click OK to start the control scenario.

    • If you selected the Start Now check box, the program starts immediately and the status in the Load Control client display changes to Manual Active.

    • If you specified a Start Time in the future, the program status in the Load Control client display changes to Scheduled. When the start time is reached, the program starts and the status changes to Manual Active.
