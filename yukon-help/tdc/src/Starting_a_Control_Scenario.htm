Starting a Control Scenario

Perform the following steps to start a control scenario.

 

  1. To start a control scenario, click on the Start Scenario toolbar button.

  2. Select the control scenario to start from the Scenario drop-down list.

  3. To start the control scenario immediately, select the Start Now check box.
    To specify when to start the control scenario:

    1. Clear the Start Now check box.

    2. Enter a start time in the Start Time text box.

    3. Select a start date from the drop-down list.

  4. To specify when to stop the control scenario, enter a stop time in the Stop Time text box and select a stop date from the drop-down list.
    To not specify when to stop the control scenario, select the Never Stop check box. The control scenario must be manually stopped if the stop time is set to Never Stop.

  5. To specify the gear for all of the load programs in the control scenario, select a gear from the Gear drop-down list. Gears define the rates at which loads in a load program are controlled.

    To specify gears for individual load programs, select a gear from the Gear drop-down list to the right of the program name in the table on the right-hand side of the window.

  6. Select whether to Observe or Check the program constraints from the Constraints drop-down list.

    • Observe – If the constraint is violated, the control does not occur.

    • Check – The control is checked before it is sent to the Load Management server and the result is displayed in the Program Constraint Violation window.

      • If the control does not violate a constraint, press the Resubmit button to send the control to the Load Management server.

      • If the control does violate a constraint, either click the Cancel button to cancel control or select the Override check box and press the Resubmit button to send the control to the Load Management server.

  7. By default, every load program in a control scenario is selected to start. To specify which load programs to start, clear the Select All check box and select the check boxes of the load programs to start.

  8. The Start and Stop columns display the start and stop offsets of the load programs in the control scenario. The start and stop offsets of the control scenario are defined in the Database Editor and the offsets are displayed in the format HH:MM.

  9. To specify a start offset for a load program in the control scenario, enter the start offset in the Start column, located to the right if the load program name.
    To specify a stop offset for a load program in the control scenario, enter the stop offset in the Stop column, located to the right if the load program name.
    Note: The start and stop offsets are entered in the format of “hours:minutes”.

  10. Click the OK button to start the control scenario.

    • If the Start Now check box was selected in step 3, the load programs that are selected in the control scenario start immediately.
      If a start time/date was selected in step 3, the status of       the load programs that are selected in the control scenario change to Scheduled. When the start time/date is reached, the selected load programs start.
      Note:       The status of the load programs that are selected in the control scenario changes to Manual Active when the load programs start.

 

    • If a stop time/date was selected in step 4, the load programs that are selected in the control scenario stop when the stop time/date is reached.

    • If the Never Stop check box was selected in step 4, right-click on the load program to stop and select the Stop option or refer to Stopping a Control Scenario.
      Note:       The status of the load programs that are selected in the control scenario changes to Inactive when the load programs stop.
