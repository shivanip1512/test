Print Date (Time) Range

To print an alarms and events display containing data for a selected day and time period:

 

  1. From the Display menu, select Alarms and Events.

  2. Select the information to display from the View drop-down.

  3. Use the View Date drop-down to select the date that contains the data you want to print.

  4. From the File menu, select Print Date Range.

  5. Enter a time for the beginning of the time period in the Min. Time text box.

  6. Enter a time for the end of the time period in the Max. Time text box.

  7. Make sure the drop-down boxes have the same date selected.

  8. Click Ok to print the data in the selected time range.
