Print Date (Time) Range

Perform the following steps to print an Alarms and Events display containing data for a selected day and time period:

 

  1. Select Alarms and Events from the Display menu.

  2. Select the display to view from the View drop-down list.

  3. Select the date that contains the data to print from the View Date drop-down list.

  4. Select Print Date Range from the File menu.

  5. Enter a starting time of the time period to print in the Min. Time text box.

  6. Enter a ending time of the time period  to print in the Max. Time text box.

  7. Select the same date from the Min Time and Max Time drop-down lists.

  8. Click the OK button to print the data from the selected time period.
