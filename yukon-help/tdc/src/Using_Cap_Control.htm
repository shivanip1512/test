Using Cap Control

To use the Cap Control display:

 

  1. From the Display menu, select Cap Control.
    Substation buses that are enabled appear green and substation buses that are disabled appear red.

  2. Select an substation area to display from the Sub Area drop-down list. All areas are displayed by default.

  3. Right-click on a substation bus to display the following menu:

    • Confirm Sub - confirms the status of the substation bus.

    • Disable/Enable - disables or enables the substation bus.

    • SubBus Data - displays substation bus information.

  4. Click on any of the Cap Control Client toolbar buttons for the following functionality:

    • Silence/Unsilence –  disables or enables the audible alarm.

    • Refresh – refreshes the data in the table.

    • Enable Area - enables the selected Sub Area.

    • Disable Area - disables the selected Sub Area.

    • Confirm Area - confirms the status of the selected Sub Area.

    • Split Pane View/Multiple Pane View - switches between displaying information in a split pane view or multiple pane view. The multiple pane view is the default.

  5. To display the feeders and capacitor banks on the selected substation bus, click the Split Pane View toolbar button. The split pane view displays the feeders and capacitor banks in separate windows.

 

Feeders Window

The feeders window displays the feeders associated with the selected substation bus.

 

Right-click on a feeder to display a menu with the following options:

  • Disable/Enable - disables or enables the selected feeder.

  • Feeder Data - displays information about the selected feeder.

 

Capacitor Banks Window

The capacitor banks window displays the cap banks associated with the selected feeder or substation bus.

 

Right-click on a cap bank to display a menu with the following options:

  • Ack Alarm - acknowledges an alarm status.

  • Clear Alarm - clears an alarm status.

  • Manual Entry - allows the state to be changed manually.

  • Disable/Enable - disables or enables a cap bank.

  • Confirm - confirms the status of the cap bank.

  • Open/Close - opens or closes a cap bank.

  • Temporary Move - allows a cap bank to be temporarily moved to another feeder.

  • Move Back - moves a cap bank back to its originally assigned feeder.
