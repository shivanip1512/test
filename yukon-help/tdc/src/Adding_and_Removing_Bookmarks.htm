Adding and Removing Bookmarks

Bookmarks allow quick access to commonly used displays.

 

Perform the following steps to add a bookmark:

  1. Select the type of display to bookmark from the Display menu.

  2. Select the display to bookmark from the View menu.
    If Metering and Control Scheduler is selected from the Display menu, select the category to bookmark from the Category menu.

  3. Select Add Bookmark from the Bookmark menu. The bookmark is added to the Bookmark menu.

Perform the following steps to remove a bookmark:

  1. Select Remove Bookmark from the Bookmark menu. The Remove Bookmark dialog box appears.

  2. Select the bookmark to remove from the list of bookmarks.

  3. Click the Delete button to remove the bookmark. The bookmark is removed from the Bookmark menu.
