Using Metering and Control Scheduler

To use the Metering and Control Scheduler display:

 

  1. From the Display menu, select Metering and Control Scheduler.

  2. If no schedules exist, see Creating a MACS Schedule  for instructions on creating a schedule.

  3. Select the category you would like to display from the Category list. All categories are displayed by default.

  4. Select a schedule to manage.

  5. Right-click on schedule to display a menu with the following options:

    • Start/Stop - allows you to start or stop a schedule.

    • Disable/Enable - disables or enables a schedule.

    • Edit - allows you to edit a schedule.

    • Update - refreshes the displayed information.

    • Delete - allows you to delete a schedule.

  6. Click on any of the toolbar buttons for the following functionality.

    • Silence/Unsilence – disables or enables the audible alarm.

    • Refresh -  refreshes the data in the table.

    • Edit –  allows you to edit a schedule.

    • Disable/Enable – disables or enables a schedule.

    • Start/Stop – starts or stops the a schedule.

    • Create – allows you to add a new schedule.

    • Delete – allows you to delete a schedule.

 

NOTES:

    • The Start button starts the schedule even if a start policy is specified. The schedule continues to run until either the stop policy becomes effective or the Stop button is selected.

    • The Stop button stops the schedule even if a stop policy is specified.
