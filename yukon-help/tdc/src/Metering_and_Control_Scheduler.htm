Metering and Control Scheduler

To use the Metering and Control Scheduler display:

 

  1. From the Display menu, select Metering and Control Scheduler.

  2. Click on a schedule to manage. If no schedules exist, see Creating a Schedule for instructions on creating a schedule.

  3. Click on any of the toolbar buttons for the following functionality.

    • Silence - turns off the audible alarm for all current alarms. The alarms continue to flash until they are acknowledged.

    • Mute/Unmute - mute turns off the audible alarm for all current and future alarms. Unmute turns the audible alarm back on for all current and future alarms. The alarms continue to flash until they are acknowledged.

    • Refresh - refreshes the data on the current display.

    • Edit –  allows you to edit a schedule.

    • Disable/Enable – disables or enables a schedule.

    • Start/Stop – starts or stops the a schedule.

    • Create – allows you to add a new schedule.

    • Delete – allows you to delete a schedule.

 

NOTES:

  • The Start button can start a schedule even if a start policy is specified. The schedule continues to run until either the stop policy becomes effective or the Stop button is selected.

  • The Stop button can stop a schedule even if a stop policy is specified.

  1. Right-click on schedule to display a menu with the following options:

    • Start/Stop - allows you to start or stop a schedule.

    • Disable/Enable - disables or enables a schedule.

    • Edit - allows you to edit a schedule.

    • Update - refreshes the displayed information.

    • Delete - allows you to delete a schedule.
