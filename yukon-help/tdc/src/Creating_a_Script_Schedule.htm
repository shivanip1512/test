Script Schedule with a Script Template

If schedule type is script and a script template is selected:

 

  1. Enter a name for the script in the Script Name text box.

  2. Enter a description of the script on the Description text box.

In the Meter Read tabbed page:

  1. Select the meter collection group to read from the Group Name drop-down list.

  2. Enter the number of seconds for the meter read to complete in the Porter Timeout text box.

  3. Enter the location to save the result files to in the File Path text box.

  4. Enter a name for the unsuccessful reads file in the Missed File Name text box.

  5. Enter a name for the successful reads file in the Success File Name text box.

  6. If you selected the Meter Read or IED Meter Read script template:

    1. Enter the number of times to retry missed reads in the Retry Count text box.

    2. Enter the number of times to retry the missed reads before turning off queuing in the Queue Off Count text box. This value must be less than the retry count.

    3. Enter the maximum time in hours to retry missed reads in the Max Retry Hours text box. A value of -1 disables the maximum retry hours limit.

  7. If you selected the IED Meter Read or IED Meter Read Retry script template:

    1. Select the time of use rate to use from the TOU Rate drop-down list.

    2. Select the number of times to resend the demand reset command in the Reset Demand Count text box.

    3. Select the check box of the electronic meter from which to read the frozen demand register.

In the Options tabbed page:

  1. To send an email notification when the meter read script completes, select the Send Email Notification check box.

    1. Select the notification group to send the message to from the Notify Group drop-down list.

    2. Enter the subject of the notification message in the Message Subject text box.

  2. To generate a billing file for the meter read, select the Generate Billing File check box.

    1. Select the billing file format to use from the File Format drop-down list.

    2. Enter the number of previous days to search for demand readings in the Demand Days text box.

    3. Enter the number of previous days to search for energy readings in the Energy Days text box.

    4. Enter a name for the billing file in the File Name text box.

    5. Enter a location to save the billing file in the File Path text box.

In the Text Editor tabbed page:

  1. The script is auto-generated from the settings in the Meter Read and Options tab.

  2. Click Finish. The schedule appears in the Metering and Control Scheduler list.
    If you are editing the schedule, click OK or Apply to save the changes.
