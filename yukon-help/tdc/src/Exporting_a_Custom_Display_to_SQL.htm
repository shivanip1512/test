Exporting a Display to SQL

To export a display SQL:

 

  1. From the Display menu, select Custom Displays.

  2. Select the custom display to export from the View drop-down list.

  3. From the Tools menu, select Export SQL.

  4. Select a directory to which the SQL file will be exported.

  5. The name of the exported SQL file can be changed in the File Name text box..

  6. Click the Save button to export the display.
