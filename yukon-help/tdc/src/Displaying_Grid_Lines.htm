Displaying Grid Lines

To display horizontal grid lines select Grid Lines from the Options menu and then select Horizontal Grid Lines from the Grid Lines menu option.

To display vertical grid lines select Grid Lines from the Options menu and then select Vertical Grid Lines from the Grid Lines menu option.

Note: When a grid line option is selected, a check mark is placed in front of the option in the menu.

 

To hide horizontal grid lines, deselect the Horizontal Grid Lines option from the Options menu.

To hide vertical grid lines, deselect the Vertical Grid Lines option from the Options menu.
