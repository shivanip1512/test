All Alarms and Alarm Categories

The All Alarms display shows all of the current alarms on the Yukon system.

 

Alarm Category 1 through Category 31 show the alarms on the Yukon system for the selected category.

 

For both of the above displays, right-click on an alarm to display a menu with the following functionality:

  • Ack Alarm - acknowledges the selected alarm. The alarm stops flashing and the audible alarms stop, however it still is displayed in the table. This is helpful while handling an alarm event.
