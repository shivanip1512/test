Editing a Custom Display

To edit a custom display:

 

  1. From the Edit menu, select Edit. This opens the Edit Display dialog box.

  2. Select the display to edit from the Display drop-down list.

  3. The name of the display can be changed in the Name text box. This is the name that appears in title bar and the Display drop-down list.

  4. The title of the display can be changed in the Title text box. This is the title that appears in the square box above the custom display data.

  5. The description of the display can be changed in the Description text box. This field is optional and describes the custom display being created.

  6. A new template can be selected from the Template drop-down list.

  7. Points assigned to the custom display appear in the Selected list.

    • To add a point, first click the + next to the device type to display the available devices. Then, select the device or point on the device and click Add. Selecting a device adds all the points on that device to the display. To select more than one device or point, hold the Ctrl key on the keyboard while clicking devices or points.

    • To insert a blank row between two points, double-click the row below where you want the space.

    • To remove a point, select the point from the Selected list and click Remove.

  8. Click OK.
