Starting a Direct Control Program

To start a direct control program:

 

  1. Right-click on a program and select Start.

    To start multiple programs, right-click on the control area that contains the programs and select Start Program(s).

  2. To start the program immediately, select the Start Now check box.
    To start the program later, clear the Start Now check box, enter a start time in the Start Time text box, and select a date from the drop-down list.

    • To start the program immediately, select the Start Now check box.

    • To start the program later, clear the Start Now check box. enter a start time in the Start Time text box, and select a date from the drop-down list.

  1. To specify a stop time, enter a time in the Stop Time text box and select a date from the drop-down list.
    To not specify a stop time, select the Never Stop check box. The program can then only be stopped manually.

    • To specify no stop time, select the Never Stop check box. The program can then only be stopped manually.

    • To specify a stop time, enter a time in the Stop Time text box and select a date from the drop-down list.

  2. Select the gear to use for all of the programs in the control area from the Gear drop-down list. Gears define the rates at which loads in a program are controlled.

    To specify gears for individual programs, select a gear from the Gear drop-down list in the table on the right-hand side of the window.

  3. Select whether to Observe or Check the program constraints from the Constraints drop-down list.

    • Observe - if the constraint is violated, the control does not occur.

    • Check - the control is checked before it is sent to the Load Management server and the result is displayed in the Program Constraint Violation window.

      • If the control does not violate a constraint, press the Resubmit button to send the control to the Load Management server.

      • If the control does violate a constraint, either click the Cancel button to cancel control or select the Override check box and press the Resubmit button to send the control to the Load Management server.

  4. By default, every program in a control area is selected to start. To specify which programs to start, clear the Select All check box and select the check boxes of the programs to start.

  5. Click OK.

    • If you selected the Start Now check box, the program starts immediately and the program status changes to Manual Active.

    • If you specified a start time in the future, the program status changes to Scheduled. When the start time is reached, the program starts and the program status changes to Manual Active.

    • If you selected the Never Stop check box, right click on the program and select Stop to stop the program.

    • If you specified a stop time, the program stops when the stop time is reached.
