Using Load Management

To use the Load Management display:

 

  1. From the Display menu, select Load Management.

  2. Select either All Control Areas or Energy Exchange from the drop-down list.
    If you selected Energy Exchange, see Using Energy Exchange for instructions.

  3. Double-click a control area to display the corresponding load groups/load programs.

  4. Right-click on a control area to display the following menu:

    • Triggers Change - allows you to change the trigger values.

    • Daily Time Change - allows you to change the daily start and stop times.

    • Disable/Waive or Enable/ReEnable - disables or enables the control area.

  5. Click on any of the toolbar buttons for the following functionality:

    • Silence/Unsilence – disables or enables the audible alarm.

    • Refresh –  refreshes the data in the table.

    • Enable Area/Disable Area – disables or enables the selected control area.

    • Enable All – enables all control areas.

    • Disable All – disables all control areas.

    • Show Groups/Show Programs - displays either load groups or load programs of the selected control area.

  6. Right-click on a load program to display the following menu:

    • Start/Stop  – starts or stops the selected load program. See Starting a Direct Control Program or Starting a Curtailment Program for instructions on starting a load program.

    • Disable/Enable  – disables or enables the selected load program.

  7. Right-click on a load group to display the following menu:

    • Group Info  – displays demand limit and acknowledgement information

    • Acknowledge  – used when customers verbally acknowledge curtailment requests
