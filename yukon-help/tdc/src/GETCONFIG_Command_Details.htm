GETCONFIG Command Details

This command is used to retrieve configuration type data from the various supported field devices. The command defaults to the most basic or common configuration data for each device type. Most field device types support multiple pieces of configuration data. To allow for the retrieval of the different configuration data the GETCONFIG command supports additional arguments to specify the other pieces of data.

 

getconfig [model] [options] [address (group|uniq)] [role rolenum] [time [sync]] [intervals|interval (lp|li)] [multiplier kyz offset]  [raw [func] start=0xhexnum [length]]  [ied scan]

 

The arguments are as follows:

 

Argument

Description

model

Returns the model information of the field device.

options

Returns the set options of the field device.

address (group|uniq)

Returns either the group or unique addressing information of the field device.

role rolenum

Returns the role information of the field device.

time [sync]

Returns the current date and time from the field device. When sync is used, the command returns information regarding when the field device received its last time synchronization.

intervals

Returns the last interval demand/voltage and load/voltage profile settings in a field device.

interval (lp|li)

Returns information regarding supported demand interval programming. (lp = Load Profile, li = Last Demand Interval)

multiplier kyz offset

Returns any internal multiplier information from the field device.

raw [func] start=0xhexnum [length]

Downloads a raw block of data to a field device. The func option makes the command download raw function information. The start parameter specifies the starting location in hex. This command downloads a maximum length of 13 bytes if a length is not specified.

ied scan

Returns IED Port configuration data programmed into the field device (i.e. Alpha scan and class information from an MCT-360).

Examples

getconfig model

getconfig options

getconfig role 52

getconfig multiplier kyz 15

getconfig time

getconfig time sync

getconfig intervals

getconfig interval li

getconfig interval lp

getconfig raw start=0x20

getconfig rawfunc start=0x90

getconfig ied scan
