GETCONFIG Command Details

This command is used to retrieve configuration type data from the various supported field devices. To allow for the retrieval of the different configuration data, the GETCONFIG command supports the following arguments.

 

getconfig [model] [options] [disconnect] [address (group|unique)] [role rolenum] [time[sync]] [internal] [intervals|interval (lp|li)] [channels] [lp channel N] [tou [schedule N] [holiday] [multiplier kyz offset] [ied (scan|dnp)] [phasedetect (read|clear)] [thresholds] [centron (ratio|parameters)] [raw [function] start=0xhexnum [length]]

 

The arguments are as follows:

 

Argument

Description

model

Returns the model information of the field device.

options

Returns the set options of the field device.

disconnect

Returns the disconnect configuration of an field device.

address (group|uniq)

Returns either the group or unique addressing information of the field device.

role rolenum

Returns the role information of the field device.

time [sync]

Returns the current date and time from the field device. When sync is used, the command returns information regarding when the field device received its last time synchronization.

internal

Returns all internal statuses from the field device.

intervals

Returns the last interval demand/voltage and load/voltage profile settings in a field device.

interval (lp|li)

Returns information regarding supported demand interval programming. (lp = Load Profile, li = Last Demand Interval)

channels

Returns the current load profile configuration for each channel. Load profile channels can be configured to return IED data or pulse input data.

lp channel N

Returns the current status of load profile on the specified channel.

tou [schedule N]

Returns the current Time of Use state in the field device. The parameter schedule N returns the specified TOU schedule.

holiday

Returns the three TOU holidays in the field device.

multiplier kyz offset

Returns any internal multiplier information from the field device.

ied scan

Returns IED Port configuration data programmed into the field device (i.e. Alpha scan and class information from an MCT).

ied dnp [0-154]

The dnp option returns the DNP status bytes and reports the results. When a point is specified, the command returns the DNP configuration from the MCT.

phasedetect (read|clear)

Returns the phase reading or clears the phase reading.

thresholds

Reads the outage threshold setting.

centron (ratio|parameters)

The ratio option returns the Itron CENTRON multiplier ratio from an MCT-410cL and the parameters option returns the Itron CENTRON display parameters from an MCT-410cL.

raw [func] start=0xhexnum [length]

Downloads a raw block of data to a field device. The func option makes the command download raw function information. The start parameter specifies the starting location in hex. This command downloads a maximum length of 13 bytes if a length is not specified.
