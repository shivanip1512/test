PUTSTATUS Command Details

This command is used to send data to the various supported field devices that clears or resets a status or error condition (or flag). To support the different available functions, the PUTSTATUS command supports the following arguments.

ExpressCom

putstatus xcom [prop (display|terminate|increment)] [rssi] [serial #]

 

The meanings of the arguments are as follows:

 

Argument

Description

xcom

Target type of putstatus message.

prop

Sends a propagation test command.

display

Causes the propagation LED to light and increments the propagation counter.

terminate

Causes the propagation LED to extinguish.

increment

Increments the propagation counter.

rssi

Displays RSSI signal strength using the LCR LEDs.

serial #

Unique identifying address of the field device.

 

Examples

putstatus xcom prop display serial 5345214143

putstatus xcom prop terminate serial 5345214143

putstatus xcom prop increment serial 5345214143

putstatus xcom rssi serial 5345214143

 

EMETCON

 

putstatus EMETCON [reset] [freeze [voltage] [one|two]] [peak (on|off)]

 

The meanings of the arguments are as follows:

 

Argument

Description

EMETCON

Target type of putstatus message.

reset

Resets a CCU.

freeze

Freezes and resets a field device's counters. If one or two is not specified, the command freezes and resets both counters.

voltage

When used with a freeze command, clears the voltage min/max values.

one

Freezes and resets counter one.

two

Freezes and resets counter two.

peak on

Starts the peak period for an MCT-300 series.

peak off

Ends the peak period for an MCT-300 series.

Examples

putstatus EMETCON reset

putstatus EMETCON freeze

putstatus EMETCON freeze one

putstatus EMETCON freeze voltage two

putstatus peak on

putstatus peak off

 

VERSACOM

putstatus [VERSACOM]  [prop (display|terminate|increment)] [pfcount] [ovuv (enable|disable)] [serial #]

 

The meanings of the arguments are as follows:

 

Argument

Description

VERSACOM

Target type of putstatus message. The target type defaults to VERSACOM.

prop

Sends a propagation test command.

display

Causes the propagation LED to light and increments the propagation counter.

terminate

Causes the propagation LED to extinguish.

increment

Increments the propagation counter.

pfcount

Sends a command to clear the power fail count to zero.

ovuv enable|disable

Enables or disables OverVoltage/UnderVoltage detection and protection.

serial #

Unique identifying address of the field device.

 

Examples

putstatus prop display serial 41334090

putstatus prop terminate serial 41334090

putstatus prop increment serial 41334090

putstatus pfcount serial 70123456

putstatus ovuv enable serial 70123456

 

FisherPierce

putstatus fp [serial #] ovuv (enable|disable)

 

The meanings of the arguments are as follows:

 

Argument

Description

fp

Target type of putstatus message.

serial #

Unique identifying address of the field device.

ovuv enable|disable

Enables or disables OverVoltage/UnderVoltage detection and protection.

 

Examples

putstatus fp serial 322126541 ovuv enable

putstatus fp serial 322126541 ovuv disable
