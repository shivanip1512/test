Starting a Curtailment Program

To start a curtailment program:

 

  1. Right-click on a program and select Start.

  2. Select a notification date from the Date drop-down list and enter a notification time in the Time text box. Program customers are notified of curtailment at selected data and time.

  3. Select a curtailment date from the Date drop-down list and enter a curtailment time in the Time text box. The actual curtailment takes place at this date and time.

  4. Select or enter the curtailment duration in the Curtail Duration spin box.

  5. Enter a comment into the User Comment text box if necessary.

  6. Click Start.

    • If the notify date is in the future, the program status changes to Scheduled. When the notification time is reached, the program status changes to Notified.

    • If a response is required from the customer, Unacknowledged appears as the state in the groups table. Otherwise, Not Required appears as the state in the groups table.

  • If the customer acknowledges via the Web, Acknowledged appears as the state in the groups table.

  1. When the curtail date and time is reached, the program status is changed to Manual Active and the control area status is changed to Active. The control area status displays Fully Active if all of its assigned programs are active.
