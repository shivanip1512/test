Editing a Custom Display

To edit a custom display:

 

  1. From the Display menu, select Custom Displays.

  2. Select the display to edit from the View drop-down list.

  3. From the Edit menu, select Edit. This opens the Edit Display dialog box.

  4. The name of the display can be changed in the Name text box. This is the name that appears in title bar and the Display drop-down list.

  5. The title of the display can be changed in the Title text box. This is the title that appears in the square box above the custom display data.

  6. The description of the display can be changed in the Description text box. This field is optional and describes the custom display being created.

  7. To use a new display template, select the Use Template check box and select a template for the display from the Template drop-down list.

  8. Points assigned to the custom display appear in the Selected list.

    • To add a point:

      1. Click the + next to the device type to display the available devices, click the + next to a device to display the point type, and then click the + next to the point type to displays the points.

      2. Select the device or a point on the device to add to the display and click the Add button. Selecting a device adds all the points on that device to the display. To select more than one device or point, hold the Ctrl key on the keyboard while clicking devices or points.

    • To insert a blank row between two points, double-click the row below where you want the space.

    • To remove a point, select the point from the Selected list and click the Remove button.

  9. Click the OK button to save changes to the custom display.
