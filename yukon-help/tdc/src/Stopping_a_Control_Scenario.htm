Stopping a Control Scenario

Complete the following steps to stop a control scenario.

 

  1. To stop a scenario, click on the Stop Scenario toolbar button.

  2. Select the control scenario to stop from the Scenario drop-down list.

  3. To stop the scenario immediately, select the Stop Now check box.
    To stop the scenario at a specific time, clear the Stop Now check box, enter a stop time in the Stop Time text box, and select a date from the drop-down list.

  4. By default, every program in a control scenario is selected to stop. To specify which programs to stop, clear the Select All check box and select the check boxes of the programs to stop.

  5. The Stop column displays the stop offsets of the programs in the control scenario. The stop offsets are displayed in the format of “hours:minutes”.

  6. Click OK to stop the control scenario.
