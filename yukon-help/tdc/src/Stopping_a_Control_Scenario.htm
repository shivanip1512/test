Stopping a Control Scenario

Perform the following steps to stop a control scenario.

 

  1. To stop a scenario, click on the Stop Scenario toolbar button.

  2. Select the control scenario to stop from the Scenario drop-down list.

  3. To stop the scenario immediately, select the Stop Now check box.
    To specify when to stop the scenario:

    1. Clear the Stop Now check box.

    2. Enter a stop time in the Stop Time text box.

    3. Select a stop date from the drop-down list.

  4. By default, every load program in a control scenario is selected to stop. To specify which load programs to stop, clear the Select All check box and select the check boxes of the load programs to stop.

  5. The Stop column displays the stop offsets of the load programs in the control scenario. The stop offsets of the control scenario are defined in the Database Editor and the offsets are displayed in the format HH:MM.

  6. Click the OK button to stop the control scenario.
