Menu Bar

The Menu Bar provides seven menus, which perform the following functions:

 

 

  • File - use this menu to create displays, export data, print data, start new TDCs, end the Yukon login session, or exit the application.

  • Edit - use this menu to edit, copy, and delete displays, in addition to finding text in TDC.

  • Display - use this menu to select the display to view. TDC can contain the following displays:

    • Alarms and Events - this display allows you to view alarms and events that have occurred on the Yukon system.

    • Custom Displays - this display allows you to create and view custom displays.

    • Metering and Control Scheduler - this display allows you to create and manage schedules for collecting meter data and sending controls. You may view the status of schedules as well as modify schedule information or change the state of a schedule.

    • Load Management - this display allows you to view information about control areas, load programs, and load groups. You may interact with the server for operations such as manually starting or stopping control, or changing a threshold.

    • Yukon Servers - this display allows you to start and stop Yukon services.

  • Bookmark - use this menu to add, view, and remove bookmarks.

  • Options - use this menu to set fonts; and display/hide the grid lines, message log, and toolbar.

  • Tools - use this menu create, edit, and delete templates, export data to SQL, and reset season control hours.

  • Help - use this menu to display online help and version information
