Menu Bar

The Menu Bar provides seven menus, which perform the following functions:

 

 

  • File – Use this menu to export data, print data, start new TDCs, or exit the application.

  • Edit – Use this menu to create, edit, copy, and delete displays, in addition to finding text in TDC.

  • Display – Use this menu to select the display to view. TDC can contain the following displays:

    • Alarms and Events – This display is used to view alarms and events that have occurred on the Yukon system.

    • Custom Displays – This display is used to view custom displays.

    • Metering and Control Scheduler – This display is used to create and manage schedules for collecting meter data and sending controls.

    • Load Management – This display is used to view information about control areas, load programs, and load groups.

  • Bookmark – Use this menu to add, view, and remove bookmarks.

  • Options – Use this menu to set fonts; and display/hide the grid lines, the message log, and the toolbar.

  • Tools – Use this menu create, edit, and delete templates, export data to SQL, and reset season control hours.

  • Help  – Use this menu to display online help and Yukon software version information
