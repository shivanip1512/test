Editing a Schedule

To edit a schedule:

 

  1. Select the schedule to be modified.

  2. If the schedule is running, click Stop.

  3. Click Edit.

  4. The schedule name can be changed in the Name text box.

  5. The schedule category can be changed in the Category drop-down list. The category can be changed by either typing the name of the new category or selecting a different category from the drop-down list.

  6. Change the settings under Start Policy. If necessary, select a different start policy from the Start Type Policy drop-down list. For the start type policy, select which holiday schedule to use from the Holiday drop-down list.

    • Manual - schedule starts manually. To start the schedule, select the Start toolbar button or right-click on the schedule and select Start.

    • Date Time - schedule starts on a specified date. Enter the time in the Start Time text box and select a date from the drop-down list. Select the Every Year check box to make the schedule occur every year.

    • Date of Month Time - schedule starts on a specified day each month. Enter the time in the Start Time text box and the day of the month in the text box.

    • Week Day Time - schedule starts on a specified day each week. Enter the time in the Start Time text box and select which days of the week the schedule runs. Select the Use Holiday check box to force this schedule to run on holidays.

  1. Change the settings under Stop Policy. If necessary, select a different stop policy from the Stop Type Policy drop-down list.

    • Manual - schedule stops manually. To stop the schedule, select the Stop toolbar button or right-click on the schedule and select Stop.

    • Absolute Time - schedule runs until a specified stop time. Enter the time in the Stop Time text box.

    • Duration - schedule runs for a specified duration interval. Enter the duration in minutes in the spin box.

    • Until Complete - schedule runs until the script is complete. (Visible only for Script schedules)
