GETVALUE Command Details

This command is used to collect metering type data from the various supported field devices. The command defaults to the most basic metering data for each device type. Most field device types support multiple metering quantities (kWh, current kW, peak kW, kvarh, current kvar, etc.). To allow the collection of the different metering type quantities the GETVALUE command supports additional arguments to specify the other quantities.

 

getvalue [[ied] kwh|kvarh|kvah [rate (a|b|c|d|t)] [update]]

getvalue [[ied] kw|kva|kvar|demand|peak [update]]

getvalue [lp channel (1|4) date time]

getvalue [minmax|peak [frozen] [update]]

getvalue [powerfail]

getvalue [outage (1|2|3|4|5|6)]

 

The arguments are as follows:

 

Argument

Description

ied

Collects IED Port data from field device.

kwh

Collects the energy reading from a metering type device.

kvarh

Collects the reactive reading from a metering type device.

kvah

Collects the voltage-amp reading from a metering type device.

rate (a|b|c|d|t)

Optionally specifies a time-of-use rate.

update

Saves the data into the current/last reading table.

ied

Collects IED Port data from field device.

kw

Collects the current kW from a metering type device.

kva

Collects the current kva from a metering type device.

kvar

Collects the current kvar from a metering type device.

lp channel (1|4) date time

Collects specific blocks or data ranges of Load Profile data from an MCT-410.

demand

Collects the demand reading from a metering type device.

minmax

Collects the minimum and maximum values from a field device.

peak

Collects the peak reading from a metering type device.

frozen

Optionally specifies frozen data instead of current data.

powerfail

Collects the powerfail count from a metering type device.

outage (1|2|3|4|5|6)

Collects the specified outage logs. Specifying 1 or 2 collects outage logs 1 and 2. Specifying 3 or 4 collects outage logs 3 and 4. Specifying 5 or 6 collects outage logs 5 and 6.

Examples

getvalue kwh rate a update

getvalue kvarh update

getvalue ied kwh

getvalue ied demand

getvalue kvah rate b update

getvalue kvar demand

getvalue lp channel 1 12/15/2004 12:00

getvalue peak frozen update

getvalue minmax

getvalue offset 60

getvalue powerfail

getvalue outage 1
