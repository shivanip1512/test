Editing a MACS Schedule

To edit a MACS schedule:

 

  1. Select the schedule to be modified.

  2. Click Edit.

  3. Modify the Name of the schedule if necessary.

  4. Modify the Category if necessary, by either typing the name of the category or selecting a category from the list.

  5. Change the settings under Start Policy. If necessary, select a different start policy from the Start Type Policy drop-down list.

    • Manual - schedule starts manually by selecting Start within the Metering and Control Scheduler window.

    • Date Time - schedule starts on a specified date. Enter the time in the Start Time text box and select a date from the drop-down list. Select the Every Year check box to make the schedule yearly.

    • Date of Month Time - schedule starts on a specified day each month. Enter the time in the Start Time text box and the day of the month in the text box.

    • Week Day Time - schedule starts on a specified day each week. Enter the time in the Start Time text box and select which days of the week the schedule occurs.

If necessary, select a different holiday list from the Holiday drop-down list.

  1. Change the settings under Stop Policy. If necessary, select a different stop policy from the Stop Type Policy drop-down list.

    • Manual - schedule stops manually by selecting Stop within the Metering and Control Scheduler window.

    • Absolute - schedule runs until a specified stop time.

    • Duration - schedule runs for a specified duration interval. Use the spin box to select the number of minutes.
