Yukon Command Language Summary

Yukon adds its own extensions to the Tcl script language that support a core of base commands used to collect or send data from or to field devices. This allows the system to support powerful schedules by using Tcl for schedule flow control, such as loops or delays, along with the extensions for data collection and for data downloading. Each command extension may have its own set of arguments and parameters. To stay with the standard Tcl syntax, a space is used as the delimiter between arguments. The following commands make up the core command extensions.

 

  • SELECT - Command used to select a target field device or group of field devices. It includes a set of arguments for selecting an individual device or a grouping of devices.

  • GETVALUE - Command used to collect metering or analog type data from field devices. Includes a set of arguments used to pick between the many metering quantities often available.

  • PUTVALUE - Companion command to the getvalue command, this command is used to send any metering or analog data or also to perform data freeze and resets.

  • GETSTATUS - Command used to collect status or state information from field devices. Includes a set of arguments used to pick between the many available statuses often available.

  • PUTSTATUS - Command used to change or reset a status in a given device (this does not include controls).

  • GETCONFIG - Command used to collect configuration type data from a field device. Includes a set of arguments used to pick between the many configuration parameters often available in different field devices.

  • PUTCONFIG - Command used to send configuration data to field devices. Includes a set of arguments used to pick between the many configuration parameters often available in different field devices.

  • LOOP/PING - Support command used to verify communications with a two-way device.

  • CONTROL - Command used to send a control command to field devices. Includes a set of arguments used to select between different control points and control types.
