Print Preview

Use the print preview feature to preview the print before sending it to the printer.

 

To perform a print preview:

 

  1. View the display you would like to print.

  2. From the File menu, select Print Preview. The Print Preview dialog box appears.

  3. Use the following toolbar buttons to view the pages:

    • First – displays the first page.

    • Previous – displays the previous page in the sequence.

    • Next – displays the next page in the sequence.

    • Last – displays the last page.

  4. Click Print when you are ready to print.

  5. Click Close after printing or to close the print preview dialog box.
