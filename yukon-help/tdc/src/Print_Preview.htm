Print Preview

Use the print preview feature to preview the print before sending it to the printer.

 

Perform the following steps to perform a print preview:

 

  1. Select the type of display to view from the Display menu.

  2. Select the display to view from the View drop-down list.

  3. Select Print Preview from the File menu. The Print Preview window is displayed.

  4. To view the printable pages in the Print Preview window, use the following toolbar button:

    • First – Clicking this button displays the first printed page.

    • Previous – Clicking this button displays the previous printed page.

    • Next – Clicking this button displays the next printed page.

    • Last – Clicking this button displays the last printed page.

  5. Click the Print button to print the selected View.

  6. Click the Close button close the print preview window.
