Yukon Servers

The following section describe how use the Yukon Servers display to start and stop Yukon services.

 

Note: The Yukon Servers display is accessible only if the user’s login group has a number 1 as the fourth digit of the tdc_rights property under the Application -> Tabular Display Console category.

 

To view or change the status of Yukon services:

 

  1. From the Display menu, select Yukon Servers.

  2. Select which services to display from the Sort By drop-down list. The two choices are Yukon Servers Only or All Servers.
    Yukon Servers Only displays only the Yukon server processes. All Servers displays all of the Windows system processes.

  3. Click on any of the toolbar buttons for the following functionality:

    • Silence - turns off the audible alarm for all current alarms. The alarms continue to flash until they are acknowledged.

    • Mute/Unmute - mute turns off the audible alarm for all current and future alarms. Unmute turns the audible alarm back on for all current and future alarms. The alarms continue to flash until they are acknowledged.

    • Refresh - refreshes the data on the current display.

    • Start All - starts all stopped services.

    • Stop All - stops all running services.

  4. Right click on a service to display a menu with the following options:

    • Start - starts the selected service.

    • Stop - stops the selected service.

    • Refresh - refreshes the status of the selected service.
