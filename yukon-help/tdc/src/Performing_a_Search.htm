Performing a Search

To perform a search:

 

  1. View the display you would like to search.

  2. From the Edit menu, select Find. The Search Table dialog box appears.

  3. Enter the text to search for in the Search Text text box.

  4. Select which options to use under Search Options.

    • Search By Column - select this option to search a single column.

    • Column Name - when Search By Column is selected, select the column to search in from this list.

    • Case Sensitive - select this option to search for case sensitive text.

    • Search from Selected Row Down - select this option to search after the selected row.

  5. Click Search.
