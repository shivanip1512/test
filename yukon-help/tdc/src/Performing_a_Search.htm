Performing a Search

Perform the following steps to search for text in a TDC display:

 

  1. Select the type of display to view from the Display menu.

  2. Select the display to view from the View menu.
    If Metering and Control Scheduler is selected from the Display menu, select the category to view from the Category menu.

  3. Select Find from the Edit menu. The Search Table dialog box appears.

  4. Enter the text to search for in the Search Text text box.

  5. Select which options to use in Search Options area.

    • Search By Column – Select this option to search a single column.

      • Column Name – Select the column to search in from this Column Name drop-down list.

    • Case Sensitive – Select this option to search for case sensitive text.

    • Search from Selected Row Down – Select this option to search after the selected row.

  6. Click the Search button to search for the text.
