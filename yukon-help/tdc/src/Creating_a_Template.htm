Creating a Template

To create a template:

 

  1. From the Tools menu, select Templates, and then select Create. The Create Template dialog box appears.

  2. Enter the template name in the Name text box. This is the name that appears in the Template drop-down list of the Create Display and Edit Display dialog boxes.

  3. When the dialog box appears, the number 1 is selected in the Column Number drop-down list.

  4. Enter a name for the column in the Title text box. The same name cannot be used twice.

  5. Select a column type from the Type drop-down list.

    • Point ID - ID assigned to the point in the Yukon system.

    • Point Name - name of the point defined in the Yukon system.

    • Point Type - type of point defined in the Yukon system.

    • Point State - current state of the point.

    • Device Name - name of the device defined in the Yukon system.

    • Device Type - type of device defined in the Yukon system.

    • Device Current State - current state of the device in the Yukon system.

    • Device ID - ID assigned to the device in the Yukon system.

    • Point Value - value of the point.

    • Point Quality - quality of the point. See Point Qualities for a list of possible qualities.

    • Point Time Stamp - recorded time stamp for the last time the point was updated from the server.

    • UofM - unit of measure for the point. Status points do not have a unit of measure.

    • State - displays the status of the point.

    • Point Image - status image of the point.

    • Quality Count - number of times in row that the point quality has stayed the same when the point has updated.

  6. Click Add. The column appears under Current Layout. Notice that the number 2 is selected in the Column Number drop-down list.

  7. Repeat the above steps to add more columns.

  8. To remove one of the columns, select the column number to remove from the Column Number drop-down list and click Remove.

  9. To remove all of the columns, click Clear.

  10. To resize a column, place the mouse pointer over the column edge and click/drag the edge of the column. If the template is wider than the dialog box, a scroll bar appears. Use the arrows on either side of the scroll bar or click and drag the scroll bar to view the template.

  11. To rearrange the columns, click on the column to move and drag it to the appropriate location.

  12. Click OK.
