Editing a Template

To edit a template:

 

  1. From the Tools menu, select Templates, and then select Edit. A Warning dialog box appears informing you that changes to templates affect their associated displays.

  2. Click OK on the Warning dialog box. The Edit Template dialog box appears.

  3. Select the template to edit from the Template drop-down list.

  4. When the dialog box appears, the next number in the column number sequence appears in the Column Number drop-down list.

  5. To add a column:

    1. Select the column number to add from the Column Number drop-down list.

    2. Enter a unique name for the column in the Title text box.

    3. Select a column type from the Type drop-down list.

      • Point ID - ID assigned to the point in the Yukon system.

      • Point Name - name of the point defined in the Yukon system.

      • Point Type - type of point defined in the Yukon system.

      • Point State - current state of the point.

      • Device Name - name of the device defined in the Yukon system.

      • Device Type - type of device defined in the Yukon system.

      • Device ID - ID assigned to the device in the Yukon system.

      • Point Value - value of the point.

      • Point Quality - quality of the point. See Point Qualities for a list of possible qualities.

      • Point Time Stamp - recorded time stamp for the last time the point was updated from the server.

      • UofM - unit of measure for the point.

      • State - displays the status of the point.

      • Point Image - status image of the point.

      • Quality Count - number of times in row that the point quality has stayed the same when the point has updated.

    4. Click Add. The column appears under Current Layout and the next column number appears in the Column Number drop-down list.

    5. Repeat the above steps to add additional columns.

  6. To edit a column:

    1. Select the column number to modify from the Column Number drop-down list. The column title and type are displayed in their corresponding text boxes.

    2. The name of the column can be changed in the Title text box.

    3. A new column type can be selected from the Type drop-down list.

  7. To remove one of the columns, select the column number to remove from the Column Number drop-down list and click Remove.

  8. To remove all of the columns, click Clear.

  9. To resize a column, place the mouse pointer over the column edge and click/drag the edge of the column. If the template is wider than the dialog box, a scroll bar appears. Use the arrows on either side of the scroll bar or click and drag the scroll bar to view the template.

  10. To rearrange the columns, click on the column to move and drag it to the appropriate location.

  11. Click OK.
