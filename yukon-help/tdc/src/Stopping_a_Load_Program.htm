Stopping a Load Program

Perform the following steps to stop a load program:

 

  1. Right-click on the load program to stop and select Stop.

  2. To stop the load program immediately, select the Stop Now check box.
    To specify when to stop the load program:

    1. Clear the Start Now check box.

    2. Enter a stop time in the Start Time text box.

    3. Select a stop date from the drop-down list.

  3. Click the Submit button to stop the load program.

    • If you selected the Stop Now check box, the load program stops immediately.

    • If you specified a stop time/date in step 2, the load program stops when the stop time/date is reached.
      Note:       The status of the load program changes to Inactive when the load program stops.
