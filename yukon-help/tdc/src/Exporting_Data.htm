Exporting a Data Set

Data can be exported in either comma delimited or HTML format.

 

To export a data set:

 

  1. Display the data to export in the Tabular Data Console.

  2. From the File menu, select Export. The Export Data dialog box appears.

  3. Select the data format from the Format list. The format options are:

    • Comma Delimited (.CSV)

    • HTML 4.0 Page (.HTML)

  4. Click the OK button.

  5. Select a directory to which to save the data.

  6. Modify the file name if necessary.

  7. Click the Save button to export the data.
