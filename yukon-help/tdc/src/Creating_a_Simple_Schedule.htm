Simple Schedule

If the schedule type is simple:

 

  1. Select Device or Load Group from the Sort By list to display either devices or load groups in the area below.

  2. Select the device or load group for which you would like to execute the start and stop commands.

  3. Enter a command in the Start text box or select one from the drop-down list. The command selected provides instructions for the schedule upon starting. See the Yukon Command Language Summary for more information.

  4. Enter a command in the Stop text box or select a stop command from the drop-down list. The selected command provides instructions for the schedule upon stopping.

  5. Select or enter the interval to repeat the commands in Repeat Interval spin box.

  6. Click Finish. The schedule appears in the Metering and Control Scheduler list.
    If you are editing the schedule, click OK or Apply to save the changes.
