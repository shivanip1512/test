Viewing Custom Displays

To view custom displays:

 

  1. From the Display menu, select Custom Displays.

  2. Select the display you would like to view from the Display list.

  3. If the data displayed is more than one page, right click on the title bar box to select which page to view.

  4. Click on any of the Custom Displays toolbar buttons for the following functionality.
    These buttons apply to all alarms within the table:

    • Silence/Unsilence – disables or enables the audible alarm.

    • Refresh – refreshes the data in the table.

    • Clear Display – clears the entire table.

    • Ack Viewable – acknowledges viewable alarms. Alarms stop flashing and the audible alarms stop; however the alarm is still displayed in the table. This is helpful while handling an alarm event.

    • Clear Viewable – acknowledges and removes alarms. Use this feature when an alarm event has been terminated.

  5. Right-click on an custom display to display a menu with the following functionality.

    • Ack Alarm – acknowledges viewable alarms.

    • Clear Alarm – clears an individual alarm.

    • Manual Entry – changes the current state or value of the point.

    • Manual Control - manually controls the displayed device.

    • Tags - allows you to disable/enable the entire device or disable/enable the point.

    • Control - allows you to inhibit/enable control on the entire device or inhibit/enable control on the point.
