Custom Displays

To view custom displays:

 

  1. From the Display menu, select Custom Displays.

  2. Select the display to view from the View drop-down list.

  3. If the data displayed is more than one page long, right click on the title bar box to select which page to view.

  4. Click on any of the Custom Displays toolbar buttons for the following functionality. These buttons apply to all alarms within the table:

    • Silence - turns off the audible alarm for all current alarms. The alarms continue to flash until they are acknowledged.

    • Mute/Unmute - mute turns off the audible alarm for all current and future alarms. Unmute turns the audible alarm back on for all current and future alarms. The alarms continue to flash until they are acknowledged.

    • Refresh - refreshes the data on the current display.

    • Ack Viewable - acknowledges all viewable alarms. Alarms stop flashing and the audible alarms stop. However, alarms still display in the table until the points that caused the alarms change to a non-alarming state.

  5. Right-click on an custom display to display a menu with the following functionality.

    • Ack Alarm – displays a list of active alarms for the point, from which you can acknowledge them. If more than one alarm state is active for the point, the option (ACK ALL) appears in the list. When an alarm is acknowledged, the alarm stops flashing and the audible alarms stop. However, the alarm still displays in the table until the point that caused the alarm changes to a non-alarming state.

    • Manual Entry – allows you to change the status or value of the selected point. If the selected point is alarming, the dialog box displays an Alarm Panel which allows you to clear and acknowledge the alarm.

    • Manual Control - allows you to manually control the selected point.

    • Tags - allows you tag a device to disable/enable the entire device or disable/enable the point.

    • Enable/Disable- allows you to enable/disable an entire device or enable/disable the point on the device.

    • 30 Day Trend - displays a trend containing the past 30 days of data for the point.

    • Force Alt Scan - allows you to force a scan of the device point using the alternate scan rate defined in the device. The duration to use the alternate scan rate can be selected from the Duration drop-down list. If Only Once is selected from the Duration drop-down list, an immediate one-time scan is performed on the device point.
