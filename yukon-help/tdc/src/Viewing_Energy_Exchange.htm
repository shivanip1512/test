Using Energy Exchange

To use Energy Exchange:

 

  1. Double-click a control area to display the corresponding offers/customers.

  2. Right-click on a control area to display the following menu:

    • Triggers Change - allows you to change the trigger values.

    • Daily Time Change - allows you to change the daily start and stop times.

    • Disable/Enable - disables or enables the control area.

  3. Click on any of the toolbar buttons for the following functionality:

    • Silence/Unsilence - disables or enables the audible alarm.

    • Refresh - refreshes the data in the table.

    • Enable Area/Disable Area – disables or enables the selected control area.

    • Create Offer - allows you create an Energy Exchange offer. For instructions, see Creating Energy an Exchange Offer.

    • View Revisions - allows you to view Energy Exchange offer revisions. For instructions, see Viewing an Energy Exchange Offer Revision.

    • Show Offers/Show Customers - switches between showing offers or customers for the selected control area.

  4. Right-click on an offer to display the following menu:

    • Close - closes the offer.

    • Cancel - cancels the offer.

    • Create Revision - allows you to create an offer revision. For instructions, see Revising an Energy Exchange Offer.

    • View Offer - allows you to view the existing offer.

    • View Revisions - allows you to view offer revisions. For instructions, see Viewing an Energy Exchange Offer Revision.

  5. Right-click on a customer to display the following menu:

    • Start - allows you to start the offer for the customer.

    • Disable/Enable - disables or enables the customer.
