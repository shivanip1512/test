Creating a Schedule

Perform the following steps to create a schedule:

 

  1. Select Metering and Control Scheduler from the Display menu.

  2. Click the Create button on the Metering and Control Scheduler toolbar. The Create Schedule window appears.

  3. Enter a name for the schedule in the Name text box. This is the name that appears in the Schedule Name column of the Metering and Control Scheduler area.

  4. Enter a name for the schedule category in the Category field or select a previously entered category from the drop-down list. This is the name that appears in the Category Name column of the Metering and Control Scheduler area.

  5. Select either Simple or Script from the Type drop-down list.

    • Select Script for a complex schedule that requires modification of the MACS script.

    • Select Simple for a basic schedule that does not require modification of the MACS script.

  6. If the schedule type is Script, select a script template to use from the Template drop-down list.

    • No Template – No template.

    • Meter Read – Template for reading usage information from meters.

    • Meter Read Retry – Template for reading usage information from meters that were missed by the Meter Read script.

    • 360/370 IED Meter Read – Template for reading usage information from IEDs attached to MCT-360/MCT-370 devices.

    • 360/370 IED Meter Read Retry – Template for reading usage information from IEDs attached to MCT-360/MCT-370 devices that were missed by the 360/370 IED Meter Read script.

    • Outages Meter Read – Template for reading outage information from meters.

    • Voltage Meter Read – Template for reading voltage information from meters.

    • 400 Series IED Meter Read – Template for reading usage information from IEDs attached to MCT-400 Series devices.

    • 400 Series IED Meter Read Retry – Template for reading usage information from IEDs attached to MCT-400 Series devices that were missed by the 400 Series IED Meter Read script.

  7. In the Start Policy area, select the type of start policy from the Start Type Policy drop-down list.
    Note:     To prevent the schedule from running on holidays, select the holiday schedule to use from the Holiday drop-down list.

    • Manual – The schedule is started manually.

    • Date Time – The schedule starts on a specified date. Enter the start time in the Start Time text box and select a start date from the drop-down list. To make the schedule start every year on the selected date, select the Every Year check box.

    • Date of Month Time – The schedule starts on a specified day each month. Enter the start time in the Start Time text box and the day of the month start the schedule.

    • Week Day Time – The schedule starts on a specified day each week. Enter the start time in the Start Time text box and select the days of the week to start the schedule.

  8. In the Stop Policy area, select a stop policy from the Stop Type Policy drop-down list.

    • Manual – The schedule is stopped manually.

    • Absolute Time – The schedule runs until a specified stop time, which is entered in the Stop Time text box.

    • Duration – The schedule runs for a specified duration, which is entered in minutes.

    • Until Complete – The schedule runs until the script is complete. This stop policy is only available for Script schedules.

  9. Click the Next button.

  10. To continue creating a schedule, click on the type of schedule that was selected in steps 5 and 6.

    • Simple Schedule

    • Script Schedule with a Script Template

    • Script Schedule without a Script Template
