Creating a Custom Display

To create a custom display:

 

  1. From the Display menu, select Custom Displays.

  2. From the Edit menu, select New. This opens the Create Display dialog box.

  3. Enter a name for the display in the Name text box. This is the name that appears in title bar and the Display drop-down list.

  4. Enter a title for the display in the Title text box. This is the title that appears in the title box above the custom display data.

  5. Enter a description of the display in the Description text box. This field is optional and describes the custom display being created.

  6. To use a display template, select the Use Template check box and select a template for the display from the Template drop-down list.

  7. To add a point to the display:

    1. Within the Available area, click the + next to the device type to display the available devices, click the + next to a device to display the point type, and then click the + next to the point type to displays the points.

    2. Select the device or a point on the device to add to the display and click the Add button. Selecting a device adds all the points on that device to the display. To select more than one device or point, hold the Ctrl key on the keyboard while clicking devices or points.

  8. To insert a blank row between two points in the Selected area, double-click below a row to insert the a blank row.

  9. To remove a point or blank row, select the point or row from the Selected area and click the Remove button.

  10. Click the OK button to save the display. The custom display appears in the Display drop-down list when viewing the Custom Displays.
