PUTVALUE Command Details

This command is used to send metering type data to the various supported field devices. The command defaults to the most basic or common metering function for each device type. This command is commonly used for metering maintenance type functions like demand reset commands. To support the different available functions the PUTVALUE command supports additional arguments to specify other functions.

 

putvalue kyz offset [(reset|reading value)] [analog]

 

putvalue [powerfail|ied] reset

 

The arguments are as follows:

 

Argument

Description

kyz offset

Specifies the KYZ offset to read or reset.

reading

Writes the demand reading into a field device. If reading is not specified, the command defaults to reset.

reset

Resets the demand reading in a demand type meter.

analog

Writes the value to the specified analog output point on a DNP device.

powerfail

Resets the powerfail count.

ied

Resets the demand reading for an IED attached to an MCT.

 

Examples

putvalue kyz 1 reset
putvalue kyz 1 reading 297

putvalue analog 5 32

putvalue powerfail reset

putvalue ied reset
