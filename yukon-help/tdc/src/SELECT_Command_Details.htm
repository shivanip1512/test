SELECT Command Details

The select command is used to select a target field device. It can also select a group of devices or devices of a specific type, for example Load Management. Double quotes are required if the name of the device, load group, or collection group contains spaces. When data retrieval commands, program commands, or control commands are used, a previous select command be used to select a target.

 

select [name name] [id number] [group name] [altgroup name] [group name] [route name name] [route id number] [point name name] [point id number]

 

The meanings of the arguments are as follows:

 

Argument

Description

name name

Specifies a device by name.

id number

Specifies a device by Yukon id number.

group name

Specifies a group of devices by a group name. This is the primary data collection group.

altgroup name

Specifies a group of devices by an alternate group name. This is the alternate data collection group.

route name name

Specifies a route by name.

route id number

Specifies a route by Yukon id number.

point name name

Specifies a point by name.

point id number

Specifies a point by Yukon id number.

 

Examples

select name "MCT-360 #1"

select name "MCT-360 #1" point name "Communication Status (CVD)"

select name "MCT-360 #1" point id 362

select name "Summer AC Control"

select id 335

select group "District 1 Group"

select altgroup "District 1 Alternate Group"

select route name "District 1 CCU"

select route id 245
