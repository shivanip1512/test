Viewing Alarms and Events

To view alarms and events:

 

  1. From the Display menu, select Alarms and Events. The number of active alarms appears in the upper right-hand corner of the window.

  2. Select a display to view from the Display drop-down list. The displays include:

    • Event Viewer - displays all Yukon events for the day selected from View Date.

    • All Alarms - displays all active alarms. See All Alarms and Alarm Categories for more information.

    • Category 1 to Category 9 - displays active alarms for the selected category. See All Alarms and Alarm Categories for more information.

  3. Click on any of the toolbar buttons for the following functionality:

    • Silence/Unsilence - disables or enables the audible alarm.

    • Refresh - refreshes the display table.

    • Clear Display - clears the entire table.

    • Ack Viewable - acknowledges all viewable alarms. Alarms stop flashing and the audible alarms stop; however the alarm is still displayed in the table. This is helpful while handling an alarm event.

    • Clear Viewable - acknowledges and removes all viewable alarms. Use this feature when an alarm event has been terminated.
