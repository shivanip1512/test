Graphical Trend Display Options

This section describes the display options of a trend in a graphical format.

Graph Multipliers

To display the graph with multipliers, select Graph Multiplier from the Options menu. The multiplier alters selected point values to make them more visible on the graph.

Min/Max Values

To display the minimum and maximum values of the selected time period, select Plot Min/Max Values from the Options menu.

Load Duration

To display a load duration of a trend, select Load Duration from the Options menu.

Legend Values

To change the values of the legend, select the Legend option from the Options menu. The legend options are:

  • Show Minimum/Maximum - displays the minimum and maximum values in the legend.

  • Show Load Factor - displays load factor percentages in the legend.

 

Time Resolution

 

To change the time resolution of the tabular data, select the Resolution option from the Options menu. The resolution options are:

  • None - no change to the time resolution.

  • Seconds - makes all points that were recorded within the same second appear together.

  • Minutes - makes all points that were recorded within the same minute appear together.

Axis Labels

To change the axis labels, select Advanced from the Options menu.

  1. To edit the domain axis properties, change the values in the Label and Load Duration Label text boxes.

  2. The edit axis properties, change the values in the Left Label and Right Label text boxes.

  3. Click OK to save the changes to the axis.

Reset Peaks

To reset the start date for peak graph data, select Reset Peaks from the Options menu, and then select one of the following:

  • Selected Trend Peaks - resets the peak start date for the current trend selected.

  • All Peak Points - reset the peak start date to the same date for all of the trends defined.

 

The dialog box that appears displays the options for resetting the peak start date.

  1. Select the new start date for the peak data:

- Today - peak data starts from today’s date.

- First Date of Month - peak data starts from the first day of the current month.

- First Date of Year - peak data starts from the first day of the current year.

- Selected Date - peak data starts from the day selected.

  1. Click Reset Peaks to reset the peak data start date.

Zoom Feature

To zoom in on the graph, move the mouse cursor to the upper left corner of the area to magnify, then click and drag the mouse cursor down and to the right.

 

To zoom out, click and drag the mouse cursor up and to the left.
